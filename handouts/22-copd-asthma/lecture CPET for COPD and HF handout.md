## 自學方案

本文件旨在教導臨床醫師如何運用心肺運動測試(Cardiopulmonary Exercise Testing, CPET)的生理學參數，系統性地鑑別呼吸困難(dyspnea)的病因，特別聚焦於心衰竭(Heart Failure)與慢性阻塞性肺病(COPD)的區分。建議先複習基礎心肺生理學後再閱讀本文件。


## CPET基礎概念


### 什麼是CPET?


CPET不僅僅是壓力測試(stress test)，而是一項完整的生理學研究，同時評估心臟、肺臟與骨骼肌在運動壓力下的整合功能。


### Fick原理：CPET的理論基礎


CPET的核心建立在Fick原理之上：

equation
VO_2 = CO \times (CaO_2 - CvO_2)
equation

其中：

- $VO_2$：攝氧量 (oxygen consumption)
- CO：心輸出量 (cardiac output)
- $CaO_2 - CvO_2$：動靜脈氧差 (arteriovenous oxygen difference)


VO$_2$的增加依賴兩個機制：(1) 心輸出量增加，(2) 周邊組織攝氧增加。心衰竭患者因心輸出量受限而無法有效增加VO$_2$；COPD患者則因通氣限制而無法維持足夠的氧合。


### 核心測量參數


| p{4cm} p{7cm}}  **參數** | **正常值** | **臨床意義** |
| --- | --- | --- |
| Peak VO$_2$ | $>84  $>20$ mL/kg/min | 反映整體心肺適能   最重要的預後指標 |
| Anaerobic Threshold (AT) | 40--60 | 無氧代謝開始的時間點   心衰竭患者提早出現 |
| VE/VCO$_2$ slope | $<30$ | 通氣效率指標   數值越高代表效率越差 |
| O$_2$ Pulse (VO$_2$/HR) | 持續上升至plateau | 反映stroke volume $\times$ AV O$_2$ difference   心衰竭時早期plateau |
| Breathing Reserve (BR) | $>20$--$30 | BR = 1 $-$ (VE$_{max}$/MVV)    COPD時明顯下降 |
| Respiratory Exchange Ratio (RER) | $>1.10$達到最大運動 | VCO$_2$/VO$_2$   判斷是否達到最大運動努力 |


## Heart Failure的CPET特徵


### 病理生理學機轉


心衰竭患者運動受限的核心機轉包括：


1. **心輸出量限制**：無法適當增加stroke volume與heart rate
2. **周邊灌流不足**：骨骼肌血流不足導致提早無氧代謝
3. **通氣效率下降**：V/Q mismatch與dead space增加
4. **骨骼肌病變**：肌肉纖維type shift與粒線體功能異常


### 典型CPET表現


| p{11cm}}  **參數** | **Heart Failure典型表現** |  |
| --- | --- | --- |
| Peak VO$_2$ | 明顯下降，通常$<$14--18 mL/kg/min   是心臟移植評估的重要指標 |  |
| Anaerobic Threshold | 提早出現（$<$40 反映心輸出量不足導致組織缺氧 |  |
| VE/VCO$_2$ slope | 明顯上升（$>$34--36）   反映dead space ventilation增加與V/Q mismatch |  |
| O$_2$ Pulse | 早期plateau或下降   反映stroke volume無法隨運動增加 |  |
| Breathing Reserve | 通常正常（$>$30 運動受限非因通氣限制 |  |
| $\Delta$VO$_2$/$\Delta$Work Rate | 下降（正常$\approx$10 mL/min/W）   反映O$_2$ delivery不足 |  |
| Oscillatory Ventilation | 週期性呼吸模式   預後不良的重要指標 |  |


**Oscillatory Ventilation Pattern**是心衰竭特有的呼吸模式，表現為VE的週期性波動（週期約60--90秒）。其機轉與延長的circulatory time有關，是預後極差的獨立預測因子。若在CPET中觀察到此模式，應高度警覺患者可能需要進階心衰竭治療。

## COPD的CPET特徵


### 病理生理學機轉


COPD患者運動受限的核心機轉：


1. **通氣限制**：氣流阻塞導致無法增加minute ventilation
2. **動態過度充氣**：expiratory flow limitation導致air trapping
3. **呼吸肌疲勞**：呼吸功增加導致呼吸肌能量消耗過大
4. **氣體交換異常**：V/Q mismatch導致低血氧


### 典型CPET表現


| p{11cm}}  **參數** | **COPD典型表現** |  |
| --- | --- | --- |
| Peak VO$_2$ | 下降，但通常較心衰竭輕微   受通氣限制而非循環限制 |  |
| Anaerobic Threshold | 通常正常或輕度提早   循環系統功能相對保留 |  |
| VE/VCO$_2$ slope | 可能上升，但機轉不同   反映dead space增加與V/Q mismatch |  |
| O$_2$ Pulse | 通常正常上升至plateau   反映心臟功能相對保留 |  |
| Breathing Reserve | 明顯下降（$<$15 達到通氣極限(ventilatory ceiling) |  |
| SpO$_2$ | 運動中下降   反映氣體交換異常 |  |
| IC (Inspiratory Capacity) | 運動中下降   反映dynamic hyperinflation |  |
| PETCO$_2$ | 可能上升（CO$_2$ retention）   或維持不變 |  |


**Dynamic Hyperinflation**是COPD運動受限的關鍵機轉。運動時因expiratory flow limitation，患者無法完全呼氣，導致end-expiratory lung volume逐漸增加。這會使inspiratory capacity下降、呼吸功增加、呼吸肌處於機械劣勢，最終導致運動終止。可透過連續測量IC來評估。


### Flow-Volume Loop分析


在CPET中，運動時的flow-volume loop與靜態最大flow-volume envelope的關係提供重要資訊：


- **正常人**：運動時tidal flow-volume loop遠離最大envelope
- **COPD**：運動時tidal loop觸及或超過最大envelope（expiratory flow limitation）
- **Heart Failure**：通常不會觸及通氣極限


## Heart Failure vs COPD：系統性鑑別診斷


### 核心鑑別參數對照表


| p{5.5cm} p{5.5cm}}  **參數** | **Heart Failure** | **COPD** |
| --- | --- | --- |
| **運動受限機轉** | Circulatory limitation | Ventilatory limitation |
| **Peak VO$_2$** | $\downarrow\downarrow$ (明顯下降) | $\downarrow$ (輕中度下降) |
| **Anaerobic Threshold** | 明顯提早 ($<$40 | 正常或輕度提早 |
| **VE/VCO$_2$ slope** | $\uparrow\uparrow$ ($>$34--36) | $\uparrow$ (通常較輕微) |
| **O$_2$ Pulse** | Plateau或下降 | 正常上升 |
| **Breathing Reserve** | 正常 ($>$30 | 明顯下降 ($<$15 |
| **SpO$_2$變化** | 通常維持正常 | 常見下降 |
| **PETCO$_2$變化** | 運動中下降 | 可能上升或維持 |
| **運動終止原因** | 腿部疲勞、呼吸困難 | 呼吸困難、無法增加通氣 |
| **Oscillatory breathing** | 可見（預後不良指標） | 不常見 |
| **Dynamic hyperinflation** | 不常見 | 特徵性發現 |


### 鑑別診斷決策流程


importantbox
**CPET鑑別診斷五步驟流程**

**Step 1: Peak VO$_2$是否下降?**

- 若正常 ($>$84
- 若下降：進入Step 2


**Step 2: 運動終止的原因是什麼?**

- Breathing Reserve $<$15
- Breathing Reserve $>$30


**Step 3: O$_2$ Pulse的變化模式?**

- Plateau或下降：支持Heart Failure
- 正常上升：支持COPD或其他非心因性原因


**Step 4: Anaerobic Threshold出現時機?**

- $<$40
- 正常（40--60


**Step 5: 輔助指標確認**

- Oscillatory breathing：Heart Failure（預後差）
- SpO$_2$下降 + Dynamic hyperinflation：COPD
- VE/VCO$_2$ slope $>$34：兩者皆可能，需結合其他指標


importantbox


### 混合型病例的判讀


臨床上，Heart Failure與COPD常共存。判讀要點：


1. **雙重限制模式**：同時出現通氣限制（低BR）與循環限制（早期AT、O$_2$ pulse plateau）
2. **主要限制因素**：判斷何者為主要運動終止原因
3. **治療優先順序**：根據主要限制因素決定治療重點


當患者同時具有Heart Failure與COPD特徵時，建議：(1) 優先處理可逆性因素，(2) 追蹤治療後CPET變化以評估療效，(3) 多專科團隊討論制定最佳治療策略。

## 實用記憶口訣


importantbox

**「HF心不動，COPD氣不通」**


**Heart Failure**：

- O$_2$ pulse**不動**（早期plateau）
- AT**提早動**（提早出現）
- BR正常（心臟問題，不是肺的問題）


**COPD**：

- **氣不通**（BR用光、達通氣極限）
- SpO$_2$下降（氣體交換差）
- O$_2$ pulse正常（心臟沒問題）

### 考題詳解


1. **正確答案: b) Heart failure**      **詳解:** 本病例呈現典型的循環限制模式：(1) Peak VO$_2$明顯下降(55
2. **正確答案: c) O$_2$ pulse反映stroke volume乘以動靜脈氧差**      **詳解:** O$_2$ pulse = VO$_2$/HR。根據Fick原理，VO$_2$ = CO $\times$ (CaO$_2$ - CvO$_2$) = HR $\times$ SV $\times$ (CaO$_2$ - CvO$_2$)。因此O$_2$ pulse = SV $\times$ (CaO$_2$ - CvO$_2$)。O$_2$ pulse plateau是心衰竭（非COPD）的特徵，反映stroke volume無法隨運動增加。
3. **正確答案: b) 主要為通氣限制**      **詳解:** Breathing Reserve僅8
4. **正確答案: c) 數值升高代表通氣效率良好**      **詳解:** VE/VCO$_2$ slope升高代表通氣效率**下降**（需要更多通氣量來排出相同量的CO$_2$），而非良好。這反映dead space增加或V/Q mismatch。正常值確實小於30，心衰竭患者常升高至$>$34--36。
5. **正確答案: d) $<$10 mL/kg/min**      **詳解:** Weber Classification: Class A $>$20, Class B 16--20, Class C 10--16, Class D $<$10 mL/kg/min。Class D代表重度心衰竭，應考慮心臟移植或機械循環支持。
6. **正確答案: b) Oscillatory ventilation pattern**      **詳解:** Oscillatory ventilation pattern（週期性呼吸模式）是心衰竭特有的發現，與延長的circulatory time相關，是預後不良的獨立預測因子。Breathing reserve下降與dynamic hyperinflation是COPD特徵，SpO$_2$下降較非特異性。
7. **正確答案: c) 與expiratory flow limitation相關**      **詳解:** Dynamic hyperinflation是COPD的特徵性發現。由於expiratory flow limitation，患者無法完全呼氣，導致air trapping，使end-expiratory lung volume增加、inspiratory capacity**下降**（非增加）。這會使breathing reserve**下降**（非增加）。
8. **正確答案: b) O$_2$ pulse pattern與Breathing Reserve**      **詳解:** O$_2$ pulse pattern與Breathing Reserve的組合最能區分循環限制（心衰竭：O$_2$ pulse plateau + BR正常）與通氣限制（COPD：O$_2$ pulse正常上升 + BR下降）。Peak VO$_2$兩者皆會下降，VE/VCO$_2$ slope兩者皆可能升高，鑑別力較低。
9. **正確答案: c) 心衰竭合併COPD**      **詳解:** 本病例同時具有循環限制特徵（AT於28
10. **正確答案: c) AT提早出現反映組織氧氣供應不足**      **詳解:** AT正常出現於40--60

## 結語


CPET是評估呼吸困難病因與運動耐受度的金標準檢查。透過系統性分析各項生理參數，可以客觀區分心衰竭（循環限制）與COPD（通氣限制）作為運動受限的主要原因。

**關鍵學習要點:**

1. CPET是整合評估心、肺、肌肉系統的生理學研究
2. 心衰竭特徵：早期AT、O$_2$ pulse plateau、VE/VCO$_2$升高、BR正常
3. COPD特徵：BR明顯下降、SpO$_2$下降、dynamic hyperinflation、O$_2$ pulse正常
4. O$_2$ pulse pattern與Breathing Reserve的組合最具鑑別力
5. 臨床上常見混合型，需判斷主要限制因素以指導治療
