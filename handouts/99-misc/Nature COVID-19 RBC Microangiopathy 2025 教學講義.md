# COVID-19 紅血球微血管病變：缺血性內皮細胞壞死性凋亡誘導溶血與血管病變

**整理：謝慕揚 MD, PhD, FESC**
**日期：2026-01-28**
**原文連結：[Nature — Ischaemic Endothelial Necroptosis Induces Haemolysis and COVID-19 Angiopathy](https://doi.org/10.1038/s41586-025-09076-x)**

---

## 1. 研究背景

### 1.1 COVID-19 微血管病變的重要性

COVID-19 的 microangiopathy（微血管病變）是 SARS-CoV-2 感染的重要併發症：

- 造成急性及慢性併發症（long COVID syndrome）
- 毛細血管功能下降 60-90%
- 傳統認為是由 fibrin（纖維蛋白）和 platelet（血小板）微血栓造成
- 但 anticoagulant therapy（抗凝血治療）效果有限

### 1.2 傳統觀點的問題

過去認為 COVID-19 微血管阻塞主要由於 endotheliopathy（內皮病變）、血管收縮、microthrombus formation（微血栓形成）及 COVID-19-associated coagulopathy。

> **Pearl**: 本研究挑戰了傳統觀點，發現 COVID-19 微血管阻塞的主要原因並非 fibrin 和 platelet 沉積，而是一種全新的 RBC haemolysis（紅血球溶血）機制。

---

## 2. 核心發現摘要

1. **廣泛的內皮細胞死亡：** COVID-19 器官微血管中存在大量 EC death
2. **非傳統機制：** EC death 與 fibrin 或 platelet 沉積無關，而是與 RBC haemolysis 相關
3. **缺血再灌注損傷：** 此 RBC microangiopathy 與 ischaemia-reperfusion injury 相關
4. **跨疾病相關性：** 同樣機制見於 MI、腸道缺血、stroke、septic shock、cardiogenic shock
5. **分子機制：** Ischaemia 誘導 MLKL-dependent EC necroptosis 和 complement-dependent RBC haemolysis
6. **新型止血機制：** 溶血的 RBC membranes 形成 "endo-seal"，獨立於 platelet 和 fibrin
7. **治療標靶：** EC *Mlkl* 基因剔除可減少 RBC haemolysis 和缺血性器官損傷

---

## 3. 研究方法

### 3.1 多模態影像分析

- 分析超過 1,000 條血管（動脈、靜脈、毛細血管）
- 來自 COVID-19 死亡患者的肺、心、腎、肝
- 使用 immunofluorescence (IF)、H&E、scanning electron microscopy (SEM)
- Correlative light and electron microscopy (CLEM) 分析

### 3.2 動物模型

- K18-hACE2 小鼠 + SARS-CoV-2 Delta 株感染
- LAD artery ligation（心臟缺血模型）
- Superior mesenteric artery ligation（腸道 IR 模型）
- Ischaemic stroke 模型；heart transplant 模型

### 3.3 基因剔除小鼠

| 小鼠品系 | 用途 |
|---------|------|
| Mlkl−/− | 全身性 MLKL 缺失，確認 necroptosis 在 RBC haemolysis 的角色 |
| MlklEC−/− | 內皮細胞特異性 MLKL 缺失，確認 EC necroptosis 的角色 |
| MlklRBC−/− | 紅血球特異性 MLKL 缺失，排除 RBC necroptosis 的角色 |
| C9−/− | Complement C9 缺失，確認 membrane attack complex 在 RBC haemolysis 的角色 |

## 4. 詳細結果

### 4.1 COVID-19 器官中的內皮細胞死亡

在 COVID-19 死亡患者器官中發現：
- 廣泛的 VE-cadherin 流失、PECAM-1 表面標記消失
- Glycocalyx（糖萼）破壞
- EC 核型態改變：pyknosis（核濃縮）、karyorrhexis（核碎裂）、karyolysis（核溶解）
- EC detachment（10-50% 血管）

> **Pearl**: EC death 的程度與 tissue injury 的嚴重度呈正相關（p < 0.0001）。

### 4.2 RBC Haemolysis 在 COVID-19 血管中

在 EC death 部位發現廣泛沉積的 amorphous, proteinaceous acellular material：心臟微血管 30%、腎臟 29%、肝臟 27%、肺臟 5%。

| 特徵 | 完整 RBC (CD235^low) | 溶血 RBC (CD235^high) |
|------|---------------------|----------------------|
| CD235 螢光強度 | 低 | 高（約 5 倍增加） |
| 形態 | 正常雙凹圓盤狀 | 不規則、碎片化 |
| Haemoglobin | 保留 | 流失（Hb−） |
| 沉積位置 | 血管腔內 | 血管壁及 RBC 之間 |

> **Pearl**: COVID-19 microangiopathy 與 DIC 無關，而是與發生在 EC death 部位的 RBC haemolytic process 相關。

### 4.3 器官缺血誘導 RBC Haemolysis

| 器官/疾病 | 發現 |
|----------|------|
| Acute myocardial infarction | 廣泛的 RBC haemolysis 和 EC death，造成微血管阻塞 |
| Ischaemic gut | 同樣模式的 RBC haemolysis 和 EC death |
| Ischaemic stroke | 同樣模式 |
| Cardiogenic shock | 多器官 RBC haemolysis 和 EC death |
| Septic shock | 多器官 RBC haemolysis 和 EC death |

### 4.4 兩種不同的 EC 死亡途徑

| 特徵 | Necroptosis (PS^low) | Apoptosis (PS^high) |
|------|---------------------|---------------------|
| 位置 | Villus 尖端（apical region） | Villus 下部（lower body） |
| Annexin V 結合 | 弱 | 強 |
| PI/RedDot2 通透性 | 陽性（細胞膜破壞） | 陰性 |
| TUNEL | 陰性 | 陽性 |
| Caspase-3 | 陰性 | 陽性 |
| p-MLKL/p-RIPK3 | 陽性 | 陰性 |
| 導致的阻塞 | **RBC haemolysis** | Platelet + Fibrin |
| 佔 PS+ EC 比例 | 約 70% | 約 30% |

**關鍵概念：**
- Necroptotic EC → RBC haemolysis → RBC-dependent 微血管阻塞（佔 >65%）
- Apoptotic EC → Platelet + Fibrin 沉積 → 傳統微血栓
- 這兩種機制在空間上不重疊！

### 4.5 EC Necroptosis 和 Complement 介導 Haemolysis

**MLKL 的角色：**
- Mlkl−/− 小鼠：RBC haemolysis 明顯減少
- MlklEC−/− 小鼠：RBC haemolysis 減少
- MlklRBC−/− 小鼠：RBC haemolysis 正常

> **Pearl**: EC necroptosis（而非 RBC necroptosis）是促進 RBC haemolysis 的主要機制。

**Complement 的角色：**
- C9−/− 小鼠（無法形成 MAC, C5b-9）：RBC haemolysis 明顯減少
- C5b-9 沉積在溶血 RBC 表面，但不在完整 RBC 上

### 4.6 溶血 RBC 形成止血栓塞（Endo-seal）

Intravital microscopy 觀察到的動態過程：

1. RBC 黏附到垂死 EC 表面（PS+）
2. 黏附的 RBC 在血流拖曳力下變形伸長
3. RBC membrane 碎裂
4. Membrane fragments 沉積在血管壁
5. 沉積的 fragments 作為後續 RBC 的 docking sites
6. 重複循環 → 多層 RBC membrane 堆積
7. 形成連續的 endovascular sealant（"endo-seal"，厚度 5-10 μm）

> **Pearl**: Endo-seal 是一種全新的止血機制 — 獨立於 platelet 和 fibrin，可在低流速（<25 s⁻¹）和酸性環境（pH 6.0-7.0）下運作 — 這是傳統 platelet 和 coagulation 反應受抑制的環境。

### 4.7 微血管阻塞的兩種機制比較

| 參數 | RBC Microangiopathy | Platelet-Fibrin Microthrombi |
|------|--------------------|-----------------------------|
| Shear rate | <25 s⁻¹ | >50 s⁻¹ |
| 觸發條件 | Ischaemia, acidosis | Reperfusion |
| EC 死亡類型 | Necroptosis | Apoptosis |
| 主要成分 | 溶血 RBC membranes | Platelet + Fibrin |
| 需要 complement | 是（C5b-9/MAC） | 否 |
| 位置 | Villus tip | Villus base |

## 5. 機制總結

### 5.1 RBC Microangiopathy 的病理生理機制

```text
Ischaemia → 組織缺氧 + 代謝性酸中毒（pH 6.0-7.0）
    ↓
EC injury → MLKL phosphorylation → EC necroptosis
    ↓
Dying EC → 暴露 phosphatidylserine (PS) → Complement 活化
    ↓
C5b-9 (MAC) formation → RBC membrane damage → Haemolysis
    ↓
Haemolysed RBC membranes:
  ├── 沉積在 EC 表面 → 形成 endo-seal（止血）
  └── 在 RBC 之間形成 adhesive bridges → RBC aggregation（阻塞）
    ↓
過度反應 → 微血管阻塞 → 組織缺血加劇
```

### 5.2 為何傳統抗凝血治療效果有限？

- 傳統治療針對 platelet 和 fibrin
- 但 COVID-19 微血管阻塞主要由 RBC haemolysis 造成
- RBC microangiopathy 在低 shear rate 和 acidosis 條件下發生
- 這些條件下 platelet function 和 coagulation 反應本就受抑制
- 因此抗 platelet 和 anticoagulant 無法有效改善此類阻塞

## 6. 臨床意義

### 6.1 疾病相關性

| 疾病 | 相關性 |
|------|--------|
| COVID-19（急性） | 多器官微血管 RBC haemolysis 和阻塞 |
| Long COVID | 持續的 capillary dysfunction、循環 haem 升高與 terminal complement complex 形成相關 |
| Acute myocardial infarction | 缺血區域的 RBC haemolysis 和 no-reflow phenomenon |
| Ischaemic stroke | 腦微血管 RBC haemolysis |
| Septic shock / Cardiogenic shock | 多器官微血管 RBC microangiopathy |

### 6.2 潛在治療策略

- **Necroptosis 抑制劑：** 減少 EC necroptosis
- **Complement 抑制：** 防止 RBC haemolysis
- **Haemopexin：** 中和過量游離 haem
- **聯合治療：** 可能更有效

> **Pearl**: 治療考量 — 抑制 necroptosis 可能增加微血管出血風險（因 endo-seal 減少），類似於抗 platelet/anticoagulant 的出血風險，需仔細評估 benefit-risk ratio。

## 7. 名詞解釋

| 名詞 | 解釋 |
|------|------|
| Necroptosis | 一種可調控的細胞壞死形式，由 RIPK1-RIPK3-MLKL 途徑介導 |
| Apoptosis | 程序性細胞凋亡，由 caspase 介導，不引起發炎 |
| MLKL | Mixed Lineage Kinase Domain-Like pseudokinase，necroptosis 的終端執行者 |
| RIPK1/RIPK3 | Receptor-Interacting Protein Kinase 1/3，necroptosis 途徑的上游激酶 |
| Phosphatidylserine (PS) | 正常位於細胞膜內層的磷脂，細胞死亡時外翻暴露 |
| C5b-9 (MAC) | Membrane Attack Complex，complement 終末產物，在細胞膜上形成孔洞 |
| Glycophorin A (CD235) | RBC 表面的主要唾液酸糖蛋白 |
| VE-cadherin | Vascular endothelial cadherin，EC 間的黏附分子 |
| Endo-seal | 由溶血 RBC membrane 形成的內血管止血栓塞 |

---

## 8. 重點摘要

> **Pearl 1**: COVID-19 微血管阻塞的主要原因是 RBC haemolysis，而非傳統認為的 platelet-fibrin microthrombi
>
> **Pearl 2**: 機制：Ischaemia → EC necroptosis (MLKL) → Complement activation (C5b-9) → RBC haemolysis
>
> **Pearl 3**: 溶血 RBC membrane 具有雙重功能 — endo-seal（防止出血，有益）與 RBC aggregation（造成阻塞，有害）
>
> **Pearl 4**: 這是一種全新的止血機制，獨立於 platelet 和 fibrin
>
> **Pearl 5**: 此機制不僅見於 COVID-19，也見於 MI、stroke、sepsis、cardiogenic shock 等缺血再灌注相關疾病
>
> **Pearl 6**: 這解釋了為何傳統 anticoagulant/antiplatelet 治療對 COVID-19 微血管功能障礙效果有限
