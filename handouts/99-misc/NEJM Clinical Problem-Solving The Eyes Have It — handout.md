## 核心發現摘要

> **最終診斷**：食源性肉毒桿菌中毒 (Foodborne botulism)，血清 botulinum neurotoxin type A 陽性
>
> **臨床啟示**：當患者出現急性 ophthalmoplegia 合併 dysarthria 時，即使有酒精使用障礙等干擾因素，仍應將 botulism 納入鑑別診斷。Antitoxin 的早期給予至關重要，延遲治療將導致不可逆的神經損傷。

## 1. 病例摘要

### 患者資訊

- **年齡/性別**：47 歲男性
- **主訴**：呼吸困難 (dyspnea)，舌頭腫脹感 (globus sensation)，症狀持續 2 小時
- **發現方式**：EMS 發現患者在車內呼吸困難

### 病史

- **酒精使用障礙** (alcohol use disorder)：每日 4-6 杯酒精飲料
- 近 2 年多次因酒精中毒至急診
- 慢性血小板低下 (chronic thrombocytopenia)
- 曾有 delirium tremens 發作
- Penicillin allergy
- 無長期用藥
- 無業，與母親同住於美國東南部

### 初始生命徵象

| 項目 | 數值 |
|------|------|
| 血壓 | 139/102 mmHg |
| 心率 | 104 bpm |
| 呼吸速率 | 22 breaths/min |
| SpO₂ | 91% (room air) |
| 體溫 | 37.3°C |

### 身體檢查

- 流汗 (diaphoretic)，清醒但注意力間斷性不集中
- 言語含糊 (slurred, muffled speech)，但可說完整句子
- **無臉部水腫或舌頭腫脹**
- 肺部聽診清晰，腹部柔軟無壓痛
- 院前已給予 epinephrine、diphenhydramine、glucocorticoids，**症狀未改善**

## 2. 臨床推理過程

### 第一階段：急診初始評估

**臨床線索**：呼吸困難 + 言語含糊 → 上呼吸道問題

**排除的診斷**：

| 考量診斷 | 排除理由 |
|----------|----------|
| Anaphylaxis | 無舌頭腫脹、無蕁麻疹、無低血壓；epinephrine 無效 |
| Angioedema | 無口咽部水腫 |
| Peritonsillar abscess | 無發燒、無 odynophagia、無喉嚨痛 |
| Epiglottitis | 無發燒、無喉嚨痛；喉鏡檢查正常 glottis |
| 單純酒精中毒 | 可說完整句子，言語異常不符合單純中毒 |

### 第二階段：發現關鍵線索 — Ophthalmoplegia

**關鍵發現**：Extraocular movements 完全缺失 — 雙眼無法往任何方向移動

**喉鏡檢查**：Vocal cord abduction 受限（解釋言語含糊與吸入性肺炎風險）

**實驗室數據**：

| 檢驗 | 結果 | 正常值 |
|------|------|--------|
| WBC | 3,800 /mm³ (N 86%, L 5.5%) | |
| Hemoglobin | 15.8 g/dL | |
| Platelet | 58,000 /mm³ | |
| Total bilirubin | 1.6 mg/dL | < 1.2 |
| AST | 297 IU/L | < 34 |
| ALT | 168 IU/L | < 49 |
| ALP | 61 IU/L | < 116 |
| INR | 0.93 | |
| Blood ethanol | 209 mg/dL | < 10 |
| Lactate | 1.5 mmol/L | < 1.8 |

**影像學**：
- Chest CTA：雙側基底 atelectasis，無 pulmonary embolism
- Head CT/MRI：無顱內異常，無 venous sinus thrombosis，無 mass，DWI 正常

### 第三階段：ICU — 神經學評估

**ICU 神經學檢查** (無鎮靜下)：
- 可依指令張眼，可點頭搖頭回答問題
- 瞳孔反應正常且等大
- **雙眼完全無法移動** — 水平、垂直、passive head rotation 均無反應
- Corneal reflexes 完好，cough reflex 完好
- 雙前臂間歇性顫抖
- 四肢肌力至少 3/5
- 觸覺正常
- **Reflexes 2+**（此時仍正常）
- Babinski (-), Hoffmann's sign (-)

### 第四階段：Hospital Day 2 — 病情惡化

- **新發現**：不再對指令張眼，不再對疼痛刺激退縮
- EEG：持續性低振幅 alpha 活動 (正常後顯性節律，不符合 encephalopathy)
- 新發高燒 39.7°C、心搏過速 130 bpm、WBC 14,000、FiO₂ 需求升至 0.70
- Chest X-ray：雙下葉 consolidation → **吸入性肺炎** (aspiration pneumonia)
- 低血壓 86/60 mmHg（可能為 sepsis 或 autonomic dysfunction）

### 第五階段：Hospital Day 3 — 確立診斷

**新的神經學發現**：
- **Pupillary reflexes 消失** (新)
- **Corneal reflexes 消失** (新)
- **Cough/gag reflexes 消失** (新)
- 無痛覺退縮反應
- **Reflexes 全部消失** (新 — areflexia)
- 自發性腳趾活動仍在

**CSF 分析**：

| CSF 項目 | 結果 | 正常值 |
|----------|------|--------|
| Nucleated cells | 1 /μL | |
| RBC | 0 /μL | |
| Protein | 53 mg/dL | 20-59 |
| Glucose | 140 mg/dL | 48-79 |
| Treponemal Ab | Negative | |
| RPR | Negative | |

> **Pearl 1**: CSF 結果接近正常（**無** albuminocytologic dissociation），這是排除 Guillain-Barré syndrome 的關鍵線索，且支持 botulism 的診斷。

## 3. 鑑別診斷分析

### 對稱性下行運動麻痺的鑑別診斷

| 疾病 | 支持點 | 不支持點 |
|------|--------|----------|
| **Botulism** | 下行性麻痺、ophthalmoplegia、vocal cord paresis、autonomic dysfunction、CSF 正常、食物暴露史 | EMG 未顯示典型 incremental response |
| **Guillain-Barré syndrome (GBS)** | 進行性無力、areflexia | CSF 無 albuminocytologic dissociation、IVIG 無效、GM1 Ab (-)、cranial nerve 受累過重 |
| **AMAN (GBS subtype)** | 純運動軸索損傷 | IVIG 無效、GM1 Ab (-)、CSF 正常 |
| **Myasthenia gravis** | Ophthalmoplegia | 無 fluctuating weakness、IVIG/steroid 無效、RNS 無 decremental response |
| **Lambert-Eaton (LEMS)** | Autonomic dysfunction | RNS 無 incremental response、IVIG 無效 |
| **Wernicke's encephalopathy** | Ophthalmoplegia + 酒精使用障礙 | Thiamine 治療無改善、EEG 正常、下行性麻痺不典型 |
| **Bickerstaff's brainstem encephalitis** | Ophthalmoplegia、cranial nerve palsies | MRI 正常、CSF 正常 |

### 確診

- **Endopep-MS assay** (CDC)：**Botulinum neurotoxin type A 陽性**
- Stool sample：neurotoxin 未偵測（檢體運送溫度不合規格，可能導致偽陰性）
- Mouse bioassay：結果模糊 (equivocal)
- GM1 antibody：陰性（進一步排除 GBS）

## 4. 肉毒桿菌中毒總論

### 病因與機制

- **致病菌**：*Clostridium botulinum* — 產孢、厭氧、gram-positive 桿菌
- 存在於土壤、灰塵、水底沉積物
- **Botulinum neurotoxin** 有 7 種血清型 (type A-G)
- **機轉**：不可逆地抑制 neuromuscular junction 處的 acetylcholine 釋放
  - Neurotoxin 結合 presynaptic neuron 受體 → receptor-mediated endocytosis
  - Light chain 釋出 → 裂解 SNARE proteins
  - 阻斷 synaptic vesicle 與細胞膜融合 → **無 ACh 釋放 → 弛緩性麻痺**

### 感染途徑

| 類型 | 來源 | 機制 |
|------|------|------|
| **Foodborne** | 受汙染食物（罐頭、室溫放置過久的食物） | 攝入預形成毒素 |
| **Wound** | 傷口感染（IV drug use、外傷） | 傷口局部產毒 |
| **Infant** | 攝入孢子（蜂蜜等） | 腸道定殖產毒 |
| **Adult intestinal colonization** | 攝入孢子 | 腸道定殖產毒 |
| **Iatrogenic** | 美容/治療注射 | 注射預形成毒素 |

### 流行病學

- 美國 2001-2017 年間僅報告 **326 例** foodborne botulism
- 不當家庭醃製/罐頭僅佔約 1/3
- 曾有包裝在塑膠袋中不當保存的 chili 和 nacho cheese 引發的群聚事件

### 臨床表現

- **典型表現**：對稱性下行運動麻痺 (symmetric descending motor paralysis)
- 症狀出現於暴露後 **2 小時至 8 天**，高峰在 **48 小時**
- **初始症狀** (86% foodborne cases 系列)：
  - 視覺障礙：diplopia、ptosis、blurred vision、ocular palsy
  - Bulbar weakness：dysphagia、dysarthria、dysphonia
  - 全身無力、嘔吐、噁心、口乾、腹痛
  - **1/3 入院時即有呼吸窘迫**
- **不會有**：感覺喪失、意識喪失（若有發燒，通常為續發感染）

## 5. 診斷與治療

### 診斷方法

| 檢查 | 說明 |
|------|------|
| **Serum/stool neurotoxin** | Mouse bioassay + molecular tests (Endopep-MS)；高度特異性，但敏感度依時機、檢體品質而異 |
| **EMG/NCS** | 典型表現：small CMAP + high-frequency RNS 有 incremental response；但部分病例（尤其 type A 和重症）不具此特徵 |
| **CSF 分析** | 正常或接近正常（與 GBS 之 albuminocytologic dissociation 相區分） |
| **臨床診斷** | 群聚性對稱下行運動麻痺強烈提示 botulism |

> **Pearl 2**: Botulinum neurotoxin 僅在約 70% 的臨床 foodborne botulism 患者檢體中被偵測到，陰性結果不能排除診斷。

### 治療

#### Antitoxin

- **Heptavalent botulinum antitoxin** (equine-derived)：可中和所有 7 種血清型
- 在美國，非嬰兒 botulism 的 antitoxin 僅由 **CDC** 獨家供應
- **必須透過州衛生部門聯繫 CDC 申請**
- **Antitoxin 可阻止疾病進展，但無法逆轉已建立的麻痺**
  - 麻痺的恢復需要新的 nerve terminals 生長
- 症狀出現後 **> 48 小時**才給予 antitoxin → **更長的 ICU 住院時間、更高死亡率**
- 不良反應少見：anaphylaxis 約 1.4%；給藥時應備 epinephrine

#### 支持性治療

- **Mechanical ventilation**：可能持續數週至數月
- 呼吸器技術進步已降低死亡率
- 即使接受 antitoxin，死亡率仍高達 **10%**

### 預後

- 存活者通常在 **1 年內**恢復正常呼吸肌力量
- 可能持續的後遺症：疲倦、運動耐受不良、全身無力、呼吸困難、口乾

### 本案結局

- 接受 antitoxin（症狀出現後 **90 小時** — 顯著延遲）
- 接受 IVIG（經驗性治療 AMAN，無效）
- Tracheostomy → 轉長期照護機構
- **1 個月後**：上肢肌力 0/5（手指屈伸 1/5），下肢 1/5（足部 3/5）
- **3 個月後**：仍未恢復超出足部活動以外的自主功能

## 6. 教學重點與臨床啟示

### 臨床推理教學要點

> **Pearl 3**: 急性 ophthalmoplegia + dysarthria 的鑑別診斷，應永遠包含 botulism，即使患者有其他可解釋症狀的共病（如本案的酒精使用障礙）。

> **Pearl 4**: Wernicke's encephalopathy 是臨床診斷，早期 MRI 敏感度低。在有酒精使用障礙的患者，即使診斷不確定也應給予 thiamine 200-500 mg/day，因為副作用輕微但不治療可能導致不可逆的 Korsakoff's syndrome。

> **Pearl 5**: CSF 正常（無 albuminocytologic dissociation）有助於區分 botulism 和 GBS。GBS 通常呈現 CSF protein 升高但細胞數正常。

> **Pearl 6**: Antitoxin 延遲給予（>48 小時）顯著影響預後。當臨床懷疑 botulism 時，應立即聯繫公衛單位申請 antitoxin，不必等待確認性檢驗結果。

### 診斷陷阱

1. **酒精使用障礙掩蓋真正診斷**：本案中 ophthalmoplegia 和言語異常最初被歸因於 Wernicke's encephalopathy
2. **Penicillin allergy 的不當延伸**：Ceftriaxone 引起的低血壓被誤認為 anaphylaxis，但 3rd-generation cephalosporin 對 penicillin allergy 患者的過敏率極低
3. **EMG/NCS 不典型不能排除 botulism**：Type A neurotoxin 和重症患者可能缺乏 incremental response

### 處置流程

```text
急性 ophthalmoplegia + dysarthria
    |
排除 angioedema/anaphylaxis（無水腫、epinephrine 無效）
    |
排除 CNS 病變（CT/MRI 正常）
    |
給予 thiamine（考慮 Wernicke's）
    |
觀察是否有下行性麻痺 (descending paralysis)
    |
    ├── 有 → 考慮 Botulism, GBS, MG, LEMS
    |         |
    |         CSF 分析
    |         ├── Protein ↑↑ + 無細胞 → GBS (albuminocytologic dissociation)
    |         └── 正常 → Botulism, MG, LEMS
    |                    |
    |                    立即聯繫 CDC 申請 antitoxin
    |                    送 serum/stool botulinum toxin assay
    |
    └── 無 → 持續觀察，考慮其他診斷
```
