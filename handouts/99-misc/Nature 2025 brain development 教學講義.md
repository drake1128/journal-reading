# Nature 2025 brain development

**整理：謝慕揚 MD, PhD, FESC**
**日期：2025-11-06**

\maketitle


> **重大突破：**科學家創建了迄今最詳細的大腦發育圖譜，追蹤了胚胎發育和早期生命期間幹細胞如何分化為神經元。此研究由美國國立衛生研究院的BRAIN Initiative Cell Atlas Network (BICAN)計畫資助，投入5億美元經費，為神經科學研究提供了關鍵的參考工具。
> **研究意義：**這些圖譜提供了前所未有的分子層級細節，記錄了數十萬個早期大腦細胞的發育歷程，為研究自閉症和精神分裂症等神經系統疾病開闢了新途徑。

## 研究背景與重要性


### BICAN計畫簡介


本研究隸屬於BRAIN Initiative Cell Atlas Network (BICAN)計畫，該計畫於2022年啟動，由美國國立衛生研究院的Brain Research through Advancing Innovative Neurotechnologies (BRAIN) Initiative資助，總經費達5億美元，目標是建立哺乳動物大腦的參考圖譜。


### 研究突破


> 「這真的是任何發育中大腦『細胞圖譜』的初步第一稿。」—— Hongkui Zeng，艾倫腦科學研究所執行副所長


根據牛津大學發育神經科學家Zoltán Molnár的評論，這些圖譜為神經科學家提供了「非常有價值」的工具。研究人員現在可以「深入挖掘數據，找到在特定時間點對特定細胞類型的特定事件可能至關重要的基因」。


## 主要研究內容


此Nature collection包含五篇論文，於2025年11月6日發表，研究團隊追蹤了人類和小鼠大腦皮質中數十萬個早期大腦細胞，以前所未有的精確度捕捉了產生神經元和支持細胞混合物的分子事件。


### 小鼠視覺皮質發育研究


#### 研究設計與發現


Hongkui Zeng及其同事專注於小鼠視覺皮質從11.5天大的胚胎到56天大小鼠的發育過程。


| p{10cm}} 小鼠視覺皮質發育研究重點數據 |  |  |
| --- | --- | --- |
| **研究參數** | **數據** |  |
| 研究細胞總數 | 568,654個單細胞 |  |
| 識別細胞集群 | 148個細胞集群 |  |
| 識別細胞亞型 | 714個細胞亞型 |  |
| 研究時間跨度 | 胚胎期11.5天至出生後56天 |  |
| 研究範圍 | 包含產前和產後階段的完整高解析度圖譜 |  |
| 關鍵發現 | 許多神經元在出生後仍持續獲得專化特性，特別是在第11天（新生小鼠首次睜眼）到第21天之間 |  |


#### 意外發現


Zeng表示，令人驚訝的是，許多神經元在出生後仍持續獲得專化特性，特別是在第11天（新生小鼠首次睜眼）到第21天之間。這顯示大腦發育的複雜性超出了先前的預期。


### 人類胎兒大腦發育研究


#### 研究方法


加州大學舊金山分校的神經科學家Tomasz Nowakowski及其同事創建了人類皮質細胞的家譜圖。研究使用了來自八個不同發育階段人類胎兒的大腦組織。


> **方法：**研究團隊識別了這些組織中的6,402個祖細胞，並在每個細胞中插入獨特的DNA「條碼」。
> **原理：**當細胞分裂時，條碼被複製並傳遞給子細胞，使研究人員能夠追蹤哪些幹細胞產生了哪些類型的大腦細胞。


#### 關鍵發現


| p{9cm}} 人類胎兒大腦發育的時序特徵 |  |  |
| --- | --- | --- |
| **發育階段** | **細胞分化特徵** |  |
| 妊娠20週前 | 祖細胞主要發育為興奮性神經元（傳遞電信號以激活其他神經元） |  |
| 妊娠20週後 | 細胞轉向產生抑制性神經元（終止神經元通訊） |  |
| 星形膠質細胞和少突膠質細胞生成 | 在人類中似乎「比小鼠更連續和持久」 |  |


Nowakowski指出：「這顯示了這些祖細胞如何從一種類型轉變為另一種類型。」他補充說：「我們實際上不知道是什麼控制了所有這些發育程序和轉換，或者為什麼它們在發育過程中發生，特別是在如此精確的時刻發生。解決這些問題將是未來研究的關鍵目標。」


### 跨物種神經元比較研究


#### 研究背景


普林斯頓大學的神經科學家Fenna Krienen表示，細胞圖譜正在重塑神經科學家如何研究不同生命階段和物種的大腦。「這將整個科學事業推向了神經科學和基因組學的團隊科學體制，這非常令人興奮。」


#### 紋狀體抑制性神經元研究


Krienen及其同事研究了大腦紋狀體區域的一組抑制性神經元。紋狀體介導運動，並參與決策和獎勵行為。


| p{10cm}} 跨物種神經元保守性研究發現 |  |  |
| --- | --- | --- |
| **研究階段** | **發現** |  |
| 早期研究 | Krienen等人的早期研究顯示這些神經元似乎是靈長類動物特有的 |  |
| 最新發現 | 在多種哺乳動物中發現了這些神經元，包括豬、兔子和雪貂，跨越1.6億年的演化 |  |
| 小鼠和大鼠研究 | 使用不同的分子標記發現了這些細胞的版本 |  |
| 演化意義 | 這些神經元在不同物種的大腦中「移動到不同位置」，其「比例變化很大」 |  |


> Krienen回憶道：「收到一封主旨為『在豬身上找到了它們，該怎麼辦？』的電子郵件真是太美妙了。」這促使團隊在小鼠和大鼠中尋找這些神經元。


#### 演化啟示


研究共同作者、加州大學舊金山分校的神經科學家Alex Pollen表示：「至少在哺乳動物中，演化是通過教老細胞類型新把戲來發揮作用的。」

他補充說，下一步將是找出這些保守的神經元扮演什麼角色。「這些跨物種細胞圖譜為我們提供了一種通用的細胞類型語言，用於將生物醫學模型與人類聯繫起來，這非常有價值。」

## 臨床意義與未來應用


### 對神經系統疾病研究的啟示


這些圖譜可為研究自閉症和精神分裂症等神經系統疾病提供新途徑。研究人員現在可以：


1. **精確定位疾病相關基因**：在特定時間點、特定細胞類型中識別關鍵基因
2. **理解發育異常**：追蹤疾病發展過程中的細胞分化異常
3. **開發治療策略**：基於發育圖譜設計針對性治療方法
4. **建立疾病模型**：利用跨物種數據建立更準確的動物模型


### 研究的廣泛影響


> 普林斯頓大學的Fenna Krienen指出：「這將整個科學事業推向了神經科學和基因組學的團隊科學體制，這非常令人興奮。」
> 這標誌著神經科學研究從個別實驗室的小規模研究轉向大型協作計畫的重要轉變。


### 未來研究方向


1. **機制研究**：深入了解控制發育程序轉換的分子機制
2. **時序調控**：研究為何某些發育事件在特定時間點發生
3. **功能研究**：確定保守神經元在不同物種中的具體功能
4. **疾病應用**：將圖譜數據應用於神經系統疾病的診斷和治療
5. **擴展圖譜**：涵蓋更多大腦區域和發育階段

## 結論


> enumerate
>     \item **歷史性突破**：創建了首個完整的哺乳動物大腦發育細胞圖譜
>     \item **大規模數據**：追蹤了超過50萬個細胞，識別了數百種細胞類型和亞型
>     \item **時序精確性**：揭示了出生前後大腦發育的關鍵時間窗口
>     \item **演化保守性**：發現某些神經元類型在哺乳動物中跨越1.6億年演化歷史保持保守
>     \item **臨床應用潛力**：為神經系統疾病研究提供了寶貴的參考工具
>     \item **團隊科學典範**：展示了大型協作計畫在現代神經科學研究中的重要性
> enumerate


這些圖譜代表了神經科學領域的重大進展，為理解大腦發育的基本原理和研究神經系統疾病奠定了堅實基礎。正如Molnár所說：「我們即將迎來一個非常令人興奮的時代。」

## 補充說明：給非醫學背景讀者


### 為什麼這個研究重要？


想像大腦發育就像建造一座複雜的城市：


- **幹細胞**：像是城市的「建築工人」，可以變成各種專業人員
- **神經元**：像是城市中的「居民」，負責傳遞信息
- **支持細胞**：像是「基礎設施」，維持城市運作
- **發育程序**：像是「建設計畫」，確保城市按正確順序建造


這個研究就是創建了一份詳細的「建設藍圖」，告訴我們這座「城市」是如何一步步建成的。


### 實際應用價值


1. **疾病理解**：就像知道建築藍圖後，我們可以找出建築缺陷在哪裡
2. **治療開發**：了解正常發育過程，才能修復異常發育
3. **科學合作**：這種大型計畫展示了現代科學研究的新模式


### 未來展望


這只是開始。就像第一份人類基因組圖譜引發了基因組學革命，這些大腦發育圖譜也將為神經科學開啟新時代。
