# 重振床邊臨床診察的策略 (Strategies to Reinvigorate the Bedside Clinical Encounter)

**整理：謝慕揚 MD, PhD, FESC**
**日期：2025-11-12**
**原文連結：[NEJM — Strategies to Reinvigorate the Bedside Clinical Encounter](https://doi.org/10.1056/NEJMra2500226)**

---

## 1. 概述

本文為《新英格蘭醫學雜誌》醫學教育系列文章，探討現代醫療環境中床邊臨床技能衰退的問題，並提供六大策略協助臨床教育者重振床邊醫學文化。

> **Pearl**：當代醫學訓練者僅花費 13% 的時間與病人直接接觸，此趨勢導致基本床邊技能下降、診斷錯誤增加、臨床結果惡化，並增加醫療成本。

### 2.1 問題背景

| 現況問題 | 影響與後果 |
|----------|------------|
| 直接接觸時間減少 | 醫學生僅 13% 時間與病人直接接觸；基本床邊技能持續下降；診斷能力受到影響 |
| 診斷錯誤增加 | 超過 50% 門診診斷錯誤源於病史詢問不良；理學檢查執行錯誤；過度依賴科技檢查 |
| 醫療成本上升 | 過度依賴科技導致過度檢查；臨床技能衰退引發不必要調查；醫療資源浪費 |
| 醫病關係弱化 | 醫學生與住院醫師同理心下降；執業醫師壓力與職業倦怠增加；醫病關係信任感降低 |
| 健康不平等惡化 | 床邊時間不足影響弱勢族群；加劇醫療照護差異；邊緣化群體受到不成比例影響 |

## 3. 歷史演變：如何走到今天？

### 3.1 早期美國醫學教育

- 19 世紀初：個別師徒制（preceptorship）
- 19 世紀中：學位授予的講授式醫學院
- 20 世紀初：William Osler 於 Johns Hopkins Hospital 建立床邊教學典範
- Osler 教學法：「學生檢查病人、做出診斷、聽到病變肺部的囉音、感受腫瘤異常堅硬的質地」

### 3.2 現代醫學教育的轉變

| 因素 | 影響 |
|------|------|
| 科技為基礎的檢查 | 診斷過程移向實驗室與放射科；造成醫師所見、所感、所聞不再準確的錯誤印象；實體診察價值被低估 |
| 電子病歷系統 (EHR) | 工作流程迫使醫師花更多時間面對「iPatient」；實際與病人相處時間減少；臨床資料數位化優先於實體接觸 |
| 值班時數限制 | 經濟壓力下優先考量效率；病人處理量優先於床邊評估；教學時間被壓縮 |
| 時間碎片化 | 與病人相處的有限時間日益分散；難以進行完整評估；連續性照護受到挑戰 |
| 教學能力下降 | 具床邊教學能力的教師減少；形成惡性循環；技能傳承中斷 |

## 4. 重振床邊醫學文化的六大策略

### 4.1 策略概覽

| 策略 | 核心理念 |
|------|----------|
| 1. 前往床邊觀察 | 病人在哪裡，診斷線索就在哪裡 |
| 2. 實證理學檢查 | 假說驅動的診察流程 |
| 3. 刻意練習機會 | 晨間迴診與專門教學時段 |
| 4. 科技輔助教學 | 數位聽診器、床邊超音波、人工智慧 |
| 5. 回饋與評估 | 直接觀察與情境回饋 |
| 6. 診斷外的價值 | 處理不確定性、建立關係、減少健康不平等 |

## 5. 策略一：前往床邊觀察

### 5.1 Sutton 法則的應用

- 原始 Sutton 法則：「我搶銀行是因為錢在那裡」
- 修訂版 Sutton 法則：要改善床邊臨床技能，就必須前往病人所在之處——床邊

### 5.2 觀察技能的重要性

| 應用範疇 | 臨床價值 |
|----------|----------|
| 診斷建立 | James Parkinson 對震顫麻痺的描述幾乎完全依賴直接觀察；從床尾或走廊觀察即可發現關鍵診斷線索 |
| 預後判斷 | 整體外觀反映疾病嚴重度；活動能力提供預後資訊 |
| 個人狀況了解 | 觀察病人個人環境；理解社會支持系統；評估功能狀態 |
| 教學評估 | 觀察學習者進行臨床診察；提供技能回饋的豐富機會；評估臨床能力發展 |

### 5.3 提升觀察能力的方法

1. **臨床前期訓練**：在非醫學情境練習觀察（例如：觀賞藝術）；可改善臨床觀察能力；建立系統化觀察習慣
2. **教師能力發展**：透過練習提升直接觀察臨床技能的能力；增強教師信心；改善回饋品質

## 6. 策略二：實證理學檢查

### 6.1 假說驅動的理學檢查

| 步驟 | 內容與方法 |
|------|------------|
| 步驟 1：估計檢查前機率 | 運用病史線索；考量疾病盛行率；建立診斷假說 |
| 步驟 2：選擇檢查手法 | 針對疑似診斷選擇適當檢查；考慮檢查的診斷價值；如同選擇其他診斷工具般謹慎 |
| 步驟 3：執行檢查 | 某些發現具病理診斷性（pathognomonic）；例如：帶狀疱疹的皮疹、蜂窩性組織炎的外觀與溫度 |
| 步驟 4：與科技檢查比較 | 多數情況需與科技檢查結果比對；例如：心尖搏動外移與第三心音 vs. 心臟超音波；建立似然比（likelihood ratio） |
| 步驟 5：臨床決策 | 決定是否需要額外檢查；判斷資訊是否足以診斷；提供治療建議 |

### 6.2 理學檢查的診斷價值

- 某些情況下，理學檢查仍是診斷標準
- 其他情況下，可利用似然比選擇適當檢查手法
- 透過與科技檢查比對，建立理學檢查的證據基礎

## 7. 策略三：創造刻意練習機會

### 7.1 刻意練習的重要性

| 面向 | 效果 |
|------|------|
| 技能改善 | 專門臨床技能課程可改善理學檢查技能；結構化練習產生長期效果 |
| 價值認知 | 與病人相處教導床邊技能的價值；資料收集立即影響病人照護；建立技能與臨床結果的連結 |
| 克服障礙 | 賦予教師與學習者回到床邊的能力；打破「床邊沒有效率」的迷思 |

### 7.2 晨間迴診的優化

| 策略 | 實施方法 |
|------|----------|
| 事前準備 | 讓病人與學習者了解床邊迴診的期待；編排床邊互動流程；分配團隊成員任務 |
| 角色分配 | 指定成員負責病歷輸入；指定成員負責資料檢索；確保每位成員有明確職責 |
| 結構化呈現 | 聚焦臨床報告；保留時間澄清重要病史；將理學檢查細節整合進個案報告 |
| 教學機會 | 運用結構化時間提供教學；One-Minute Preceptor；Five-Minute Moment |
| 效益 | 辨識立即需處理的問題；建立共同醫療議程；加速檢查、治療與會診的安排；提升醫師滿意度 |

### 7.3 病人選擇策略

1. **高診斷價值情境**：病史或理學檢查可能有助於診斷的情況；例如：慢性呼吸困難鑑別診斷（COPD vs. 心衰竭）
2. **技能提升機會**：選擇可改善教師與學習者臨床技能的個案；針對「神經恐懼症」(neurophobia)：選擇神經症狀病人
3. **反應性學習**：檢查臨床狀況新發生或惡化的病人；即時展示床邊診察在診斷與決策中的價值

### 7.4 專門教學時段

| 時段類型 | 內容與方法 |
|----------|------------|
| 專門理學檢查時段 | 住院醫師提供欲探索的主訴或症狀；團隊至床邊進行理學檢查（包括超音波）；討論發現的臨床意義；與影像學等診斷結果比對 |
| 晨會後床邊教學 | 晨會報告後至床邊；進行聚焦的病史詢問與理學檢查；床邊發現可能大幅改變鑑別診斷 |
| 標準化病人教學 | 在傳統會議中使用標準化病人；示範與練習特定理學檢查手法；健康志願者協助 |

### 7.5 早期介入的重要性

- **習慣形成**：醫學生涯早期形成的習慣會影響整個職業生涯
- **臨床前教育**：參與觀察性床邊診察（真實與標準化病人）；第三年實習結束時臨床技能較佳
- **隱藏課程影響**：臨床實習時未見醫師使用病史與理學檢查；學生快速轉向電子病歷與科技檢查；早期良好訓練可能被抵消

## 8. 策略四：運用科技教學與強化臨床診察技能

### 8.1 數位聽診器

| 功能 | 臨床應用 |
|------|----------|
| 歷史重要性 | Laennec 發明聽診器：兩世紀前最重要的醫學科技進展；現代科技擴展診斷能力 |
| 即時視覺化 | 同時聽診與查看聲譜圖；即時顯示心電圖；增強教學效果 |
| 人工智慧輔助 | AI 演算法協助診斷心律不整；協助診斷瓣膜性心臟病；提高診斷準確性 |

### 8.2 床邊超音波 (POCUS)

| 面向 | 價值與應用 |
|------|------------|
| 診斷能力 | 建立傳統理學檢查難以發現的診斷；即時影像化病理生理特徵 |
| 技能校正 | 學習者校正理學檢查技能；連結檢查發現與即時影像；理解病理生理機轉 |
| AI 輔助 | 協助影像擷取；協助影像判讀；提升學習曲線 |
| 真正力量 | 將病人、醫師、學習者聚集在床邊；需要與病人互動；需要適當暴露檢查部位 |
| 附加發現 | 過程中發現重要病史線索；發現理學檢查徵象（如胸骨切開疤痕、心律調節器）；促進醫病溝通 |
| 共同決策 | 與病人建立連結；展示即時影像；促進參與式決策 |

### 8.3 遠距醫療

- **克服障礙**：克服個人防護裝備等物理障礙；限制觸診與聽診時提供即時診斷；病人臨床不穩定或感染管制需求時的替代方案
- **重新定義床邊**：包含遠距健康診察；病人自主進行醫師協助的理學檢查；提供醫師對病人居家環境的洞察

### 8.4 人工智慧的角色

| 應用領域 | 內容說明 |
|----------|----------|
| 未來潛力 | 多模態 AI 系統協助觀察與視診；大型語言模型顯示臨床推理潛力 |
| 立即影響 | 卸除電子病歷的行政負擔；釋放時間進行病人直接接觸；提升效率 |
| 關鍵原則 | AI 應告知而非取代：人類觀察、人類臨床決策、人類溝通 |
| 未來需求 | AI 驅動的臨床工作場所更需要床邊臨床技能；強調直接觀察；強調技能回饋 |

## 9. 策略五：尋求並提供臨床技能回饋

### 9.1 床邊回饋的挑戰

| 挑戰 | 考量因素 |
|------|----------|
| 情境複雜性 | 在病人面前提供回饋是複雜技能；可能涉及糾正理學檢查技巧錯誤；可能需要修改鑑別診斷或治療計畫 |
| 潛在風險 | 回饋不當可能破壞學習者與病人關係；需要謹慎處理 |
| 有效回饋 | 情境特定且周到的方式提供；可讓病人安心整個團隊投入照護；增強醫療團隊形象 |

### 9.2 床邊回饋的最佳實務

1. **事前設定期待**：提前告知病人與學習者將提供回饋；詢問學習者特定需求領域；建立心理準備
2. **避免醫學術語**：盡可能使用淺顯易懂的語言；改善溝通效果；增進教育成效
3. **事後檢討**：床邊診察後與學習者簡短檢討；強化關鍵要點；確認改善未來經驗的方法

### 9.3 直接觀察的現況

| 地區/方式 | 評估方法與效果 |
|-----------|----------------|
| 美國現況 | 真實病人的直接觀察與臨床技能評估罕見；依賴多選題考試評估醫學知識；判斷學習者發展中的能力 |
| 其他國家 | 內科住院醫師畢業需通過實際病人考試；真實病人呈現真實發現；教師觀察員評估 |
| 效果差異 | 高風險評估可驅動學習；非美國訓練醫師臨床檢查技能通常更熟練；對床邊診察價值有更廣泛認識；可能轉化為更佳病人結果 |

### 9.4 APECS 評估系統

| 特徵 | 內容 |
|------|------|
| 評估性質 | 內科住院醫師形成性評估經驗；包含真實病人的整合式診察、僅理學檢查診察、遠距醫療診察、POCUS 診察 |
| 評估領域 | 跨七個臨床領域評估住院醫師；經驗豐富的教師提供個別化實作回饋 |
| 重要發現 | 良好技巧與正確識別理學檢查發現顯著相關；良好技巧與正確鑑別診斷顯著相關；正確識別發現與正確鑑別診斷顯著相關 |
| 實際案例 | 嚴重主動脈瓣逆流病人檢查：僅半數實習醫師聽到特徵性舒張期雜音；常見錯誤：隔著衣服聽診；常見錯誤：聽診時摸橈動脈而非心尖搏動或頸動脈 |
| 教師發展 | 不同專科與經驗的醫師配對為指導教師；與學習者共同學習；提供教師發展機會 |

### 9.5 實務工具

| 工具 | 應用方式 |
|------|----------|
| Ten-Minute Moment | 床邊醫學學會開發；日常工作流程中使用；教師觀察學習者進行聚焦理學檢查；使用技能工作表提供即時回饋 |
| Mini-CEX | Mini-Clinical Evaluation Exercise；常用於指導工作場所觀察；結構化回饋工具；廣泛使用 |

## 10. 策略六：認識床邊診察超越診斷的力量

### 10.1 處理不確定性

| 面向 | 策略與價值 |
|------|------------|
| 不確定性本質 | 臨床醫學在不確定性背景下實踐；存在於診斷、治療決策、臨床追蹤各階段；是床邊病人照護的一部分 |
| 承認的價值 | 承認不確定性的存在；允許病人、臨床醫師、學習者共同協商；可強化醫病關係 |
| 好奇心的角色 | Fitzgerald (1999)：「好奇心是調查、發現的衝動」；應用於臨床醫學：將病人視為一個人；徵象與症狀是故事中的片段；減緩不確定性的重要屬性 |
| 夥伴關係 | 以好奇心接近床邊診察；有效與病人和學習者建立夥伴關係；共同面對臨床挑戰 |

### 10.2 完全在場的實證做法

| 做法 | 效果 |
|------|------|
| 有意圖的準備 (Preparing with intention) | 心理與實體準備；建立正念狀態 |
| 專注傾聽 (Listening intently) | 全神貫注於病人敘述；避免分心 |
| 共識重點 (Agreeing on what matters) | 與病人建立共同目標；確立優先事項 |
| 效益 | 超越臨床線索的價值：與病人故事連結；建立信任；尋找工作意義；對抗壓力與職業倦怠的有力支撐 |

### 10.3 理學檢查的療癒效果

- **傳達關懷**：適當執行的理學檢查可傳達關懷
- **安慰劑效應**：超越診斷發現的安慰劑效果
- **慶祝發現**：慶祝床邊發現產生重要臨床資訊的時刻；在臨床診察中建立興奮感；對抗「靈光乍現缺乏症」(eurekapenia)
- **Eurekapenia 定義**：缺乏「靈光乍現」時刻；學習者認識臨床發現與疾病病理生理基礎之間連結；許多學習者與執業醫師經歷此現象；原因：很少時間在床邊

### 10.4 解決健康照護差異

| 問題 | 床邊診察的貢獻 |
|------|----------------|
| 種族與族裔差異 | 少數族裔青少年從未接受常規體檢的比例高於白人同儕；即使檢查，系統性不平等可能惡化結果 |
| 教育資源不足 | 包含深色膚色的皮膚科教科書缺乏；脈搏血氧儀等裝置存在偏差；可能導致有色人種病人結果惡化 |
| 語言障礙 | 英語能力有限的病人可能接受不適當照護；溝通障礙影響診斷準確性 |
| 教育者的角色 | 帶團隊至床邊：以情境特定方式承認這些不平等；解決健康照護差異；提供教學示範 |

---

## 11. 核心要點總結

1. **現況問題**：21 世紀醫學學習者訓練期間與病人相處時間少於 20 世紀同儕，導致床邊臨床技能知識與實踐下降
2. **嚴重後果**：床邊臨床技能下降導致診斷錯誤、臨床結果不良、醫療成本增加、醫師職業倦怠
3. **觀察與練習**：帶學習者至床邊促進臨床觀察技能、創造練習機會、允許實證示範檢查技能
4. **科技整合**：床邊診察整合床邊科技與人工智慧，補充人類觀察、人類臨床決策、人類溝通
5. **情境回饋**：尋求機會以情境特定方式提供臨床技能回饋，改善床邊技術與從診察獲得資訊的詮釋
6. **多重價值**：超越床邊臨床診察獲得的診斷資料，理學檢查幫助學習者處理臨床不確定性、幫助教師示範與病人互動、改善醫病溝通、增加專業成就感、協助解決健康照護差異

---

## 12. 結論

在科技進步、與病人時間有限、臨床不確定性的背景下，迫切需要重振床邊診察以滿足 21 世紀病人、學習者與臨床教育者的需求。透過運用六大策略，臨床教育者可以幫助學習者認識床邊診察在診斷推理中的價值、強化醫病關係、對抗健康照護不平等、改善專業成就感並避免職業倦怠。

> Osler 一個多世紀前的話語至今仍然真實：「醫學是在床邊學習，而非在教室中學習。不要讓你對疾病表現的概念來自講堂聽到的話語或從書本讀到的文字。觀察，然後推理、比較與控制。但首先要觀察。」
