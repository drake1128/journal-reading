\maketitle


> **重大發現：**重症加護病房超音波（CCE）已成為血流動力學不穩定病患的首選影像工具。在心房顫動（AF）管理中，CCE可協助辨識心律不整危險因子、評估心律不整慢性化參數，並指導治療決策——選擇節律控制或心率控制策略。
> **臨床意義：**左心房二維靜態與功能參數，結合二尖瓣脈衝波都卜勒，可協助決定AF管理採取節律控制或心率控制策略。本文整合重症醫學數據與心臟科指引，為重症病患AF管理提供實證建議。

## 研究背景與目的


### 心房顫動之流行病學


心房顫動（AF）為最常見的上心室心律不整，其盛行率隨年齡、疾病嚴重度與手術侵入性而增加。

### 研究目的


本敘述性回顧旨在：

- 說明重症超音波（CCE）在AF管理中的角色
- 協助決定節律控制（rhythm control）與心率控制（rate control）策略
- 指導抗心律不整藥物選擇與電擊整流（DC cardioversion）
- 評估抗凝血治療之風險效益
- 監測整流後竇性心律之機械性恢復


## 相關心臟疾病評估


### 左心室收縮功能


左心室收縮功能降低會增加心房與心室心律不整的發生率。


> itemize
>     \item 非心臟重症病患中，新發AF與LVEF < 35
>     \item LVEF < 35
>     \item 此類病患建議使用amiodarone和/或digoxin治療
> itemize

### 舒張功能與左心房


左心室舒張功能障礙在ICU病患中相當常見，與年齡（中位數55--65歲）、高血壓、各類心衰竭、缺血性心臟病及敗血症相關。

### 瓣膜疾病


- 約三分之一AF病患合併瓣膜疾病
- 瓣膜性心律不整定義：風濕性心臟病（尤其二尖瓣狹窄）或機械瓣膜置換後
- 狹窄性瓣膜（尤其二尖瓣狹窄）增加心內血栓形成風險
- 中重度二尖瓣逆流因血液沖刷效應，反而降低左心房血栓風險


## 心房顫動期間之超音波評估


### 測量原則


> itemize
>     \item 心室收縮功能參數應平均3--5個心動週期
>     \item 或測量「指標心跳」（前兩個RR間期相等時）
>     \item 測量有效性與再現性取決於心率與心室反應規律性
>     \item 大多數已發表比較研究基於心率 < 100/min之病患
> itemize

### 輸液反應性評估之限制


- AF期間不規則LV每搏輸出量源自變異的舒張充盈時間
- 限制都卜勒分析用於輸液挑戰或被動抬腿試驗
- 機械通氣病患主動脈都卜勒速度呼吸變異不可用於評估輸液反應性
- 每搏輸出量或LVOT VTI仍為有意義終點，但須平均多個心動週期

### 治療流程建議


> **初步評估：**
> enumerate
>     \item 評估LV收縮功能——確保避免在LV功能受損病患使用心臟抑制性抗心律不整藥
>     \item 矯正低血容量（使用升壓劑病患）或降低LV充盈壓
>     \item 補充電解質（K+ > 4.0 mM/L、Mg2+ > 1.0 mM/L）
>     \item 維持正常血氧
> enumerate
> **節律控制優先情境：**
> itemize
>     \item 新發AF
>     \item 非過度擴張之左心房
>     \item 無顯著瓣膜疾病
>     \item 高劑量catecholamine需求
> itemize

## 整流後血流動力學變化


根據187位敗血性休克合併AF病患的研究數據：


> itemize
>     \item **心率（HR）：**顯著下降（p < 0.001）
>     \item **平均動脈壓（MAP）：**維持穩定（p = 0.5）
>     \item **每搏輸出量（SV）：**顯著增加（p < 0.001）
>     \item **心輸出量（CO）：**下降（p < 0.001）——儘管SV增加
>     \item **全身血管阻力：**顯著上升——恢復MAP
>     \item **升壓劑劑量（Norepinephrine）：**顯著減少（p < 0.001）
> itemize
> **臨床意義：**若心臟氧需求主要取決於心率與後負荷，則整流在血流動力學壓力與敗血性休克條件下是心臟保護步驟。


## 超音波導引抗凝血策略


### 整流前評估


- AF持續時間不明或已知超過24小時者，DC整流前應行經食道超音波（TOE）
- 目的：排除心內血栓，降低系統性栓塞風險
- TOE對心內淤泥或血栓檢測敏感度與特異度均 > 90
- 自發性回聲對比（spontaneous echo contrast）見於20
- 左心耳（LAA）都卜勒流速 < 0.4 m/s表示血栓形成風險；$\leq$ 0.3 m/s風險更高


### 血栓風險分層


| p{8cm}} 超音波導引血栓風險評估 |  |  |
| --- | --- | --- |
| **超音波發現** | **心內血栓風險** |  |
| **表格續接** |  |  |
| **超音波發現** | **心內血栓風險** |  |
| AF合併正常超音波發現 | 1.5 |  |
| AF + 左心房擴張 + LV收縮功能中重度降低 + 無MR | 20 |  |
| 瓣膜狹窄或機械瓣膜 | 風險進一步增加 |  |
| 恢復機械性竇性心律 + 適當LAEF與Avti | 可降低抗凝血需求 |  |


重症病患常有多種促凝血因子升高但同時有出血傾向。CHA$_2$DS$_2$-VASc $\geq$ 2建議抗凝血的心臟科指引，在重症病患可能帶來不可接受的出血風險。應個別評估風險效益比。


## 研究限制


1. 重症醫學數據稀少
2. 現有研究多為單中心、小樣本量
3. 大多數超音波與侵入性測量比較研究排除顯著瓣膜疾病病患
4. 已發表比較多基於心率 < 100/min之病患
5. 心房應變缺乏ICU與特定病理之參考值
6. 心動過速導致E波與A波融合，限制Avti測量


## 結論與臨床建議


> enumerate
>     \item **CCE角色確立：**重症超音波已成為ICU血流動力學不穩定病患的首選影像工具
>     \item **治療前評估：**
>     itemize
>         \item 收縮末期LAVI可預測整流成功機率
>         \item LAVI < 40 ml/m²偏向節律控制
>         \item LAVI > 48 ml/m²偏向心率控制
>     itemize
>     \item **整流後監測：**
>     itemize
>         \item LAEF可預測AF復發
>         \item LAEF > 38
>         \item Avti > 7.0 cm（4h）支持節律控制
>     itemize
>     \item **藥物選擇：**
>     itemize
>         \item LVEF $\geq$ 35
>         \item LVEF < 35
>     itemize
>     \item **抗凝血決策：**超音波導引風險分層可協助平衡出血與血栓風險
> enumerate
