## 引言


本文件整合台大新竹分院心血管中心歷年CPR急救訓練相關檔案，包括會議紀錄、演練指引及案例討論，旨在提供全面的參考資料，促進院內急救品質提升。內容涵蓋病房、心導管室、CT室等不同場景的CPR流程，基於ACLS指引，針對常見錯誤、設備操作及藥物使用提供詳細指導。


本檔案參考2013至2021年的紀錄，強調團隊合作、角色分配及通報流程的標準化。


## 急救訓練總覽


### 訓練目標


- 提升非預期CPR的應對能力，確保病患安全。
- 強化團隊協作，明確角色分工。
- 改善通報率，完善事件回顧與品質管理。


### 訓練場景


- **病房**：普通病房與加護病房的CPR流程。
- **心導管室**：結合介入性手術的急救挑戰。
- **CT室**：非預期心跳停止的應急處理。
- **急診**：快速反應與初始處置。


importantbox
**核心原則：**
快速確認心跳停止、立即啟動CPR、明確分工、及時通報。
importantbox


## 病房CPR訓練


### 流程概述


- **確認意識與脈搏**：檢查頸動脈脈搏（<10秒）。
- **啟動急救**：呼救，啟動急救鈴，開始胸外按壓（100-120次/分，深度5-6 cm）。
- **角色分配**：         itemize
- 按壓者：負責胸外按壓，2分鐘輪換。
- 通氣者：使用氣囊面罩（BVM），每6秒通氣1次。
- 設備負責人：準備AED、急救車。
- 記錄者：記錄時間、藥物劑量。


itemize


病房CPR需特別注意設備準備（如AED充電）及與急救團隊的交接。


### 常見錯誤


- 未明確角色分配，導致現場混亂。
- 按壓頻率過慢（<100次/分）或深度不足（<5 cm）。
- 氣管插管準備不熟練，如未測試氣囊或未使用stylet。
- AED操作延誤，貼片位置錯誤。


## 心導管室CPR訓練


### 特殊流程


- **環境調整**：移除X光管（lateral tube），確保按壓空間。
- **設備準備**：         itemize
- IABP（體內主動脈球囊幫浦）。
- ECMO（體外膜氧合）。
- A-line（動脈導管）。


    \item **緊急呼叫**：聯繫ICU主治醫師及護理師，準備超音波機。
    \item **手術並行**：若手術未完成，需同時進行導管手術及CPR。
itemize


| p{5cm}p{5cm}}  **項目** | **病房CPR** | **心導管室CPR** |
| --- | --- | --- |
| 環境 | 開放空間 | 有限空間，需移除X光管 |
| 設備 | AED、急救車 | IABP、ECMO、A-line |
| 角色 | 按壓、通氣、記錄 | 手術操作員、技術員 |
| 目標 | 恢復自發性循環 | 完成手術並恢復循環 |

心導管室CPR目標為100


### 案例討論（2021-11-26）


- **案例**：52歲女性，急性冠狀動脈症候群（ACS），於心導管室發生心跳停止。
- **流程**：         itemize
- 急診已置入氣管內管（ET）。
- 通知呼吸治療師（RT）：08:00-24:00由專師聯繫，00:00-08:00由急診護理長聯繫（專線53758）。
- 設備：2台幫浦、A-line、IABP、ECMO戰車。


    \item **結論**：需提前準備ECMO，確保手術與急救無縫銜接。
itemize


## CT室CPR訓練


### 案例分析（2013-04-12）


- **情境**：CT室發生非預期CPR，患者於檢查中意識下降（GCS由15降至11）。
- **問題**：         itemize
- 氣管內管置入時間過晚。
- 若患者已簽DNR，檢查前缺乏共識。


    \item **建議**：提前進行氣管插管，制定DNR患者的檢查流程。
itemize


CT室CPR需快速轉移至急救環境，確保氣道管理優先。


## 急診CPR訓練


### 流程與注意事項


- **初始處置**：快速確認心跳停止，啟動CPR，準備腎上腺素。
- **通報流程**：急診護理長負責聯繫RT及其他支援。
- **特殊情境**：COVID確診患者，需穿戴完整防護裝備（2021-08）。


importantbox
**挑戰：**
急診環境需平衡感染控制與急救效率，防護裝備可能延遲反應時間。
importantbox

### 設備問題


- **AED/除顫器**：         itemize
- 病房設備需定期檢查充電狀態。
- 導管室除顫器（SIEMENS）無自動充電，需手動操作。


    \item **體外心律調節器**：
        
- 操作不熟練，需在1分鐘內設定完成。
- 貼片需結合三導聯感測線，確保有效起搏。


    \item **氣管插管**：
        
- 未測試氣囊、未加潤滑劑或stylet過長。
- 建議準備標準化插管套件。


itemize


設備操作不熟練是CPR失敗的主要原因，需定期模擬訓練。


## 通報與品質管理


### 非預期CPR定義


- 當班及前一班無病況惡化記錄。
- 檢查室（如CT室、導管室）發生的CPR。
- 未簽署DNR的急救事件。


### 通報流程


- **方法一**：透過大同系統搜尋「Bosmine IV」或「呼吸器使用」代碼。
- **方法二**：追蹤腎上腺素（Adrenalin）使用記錄，篩選劑量≥2 amp的案例。


### 通報率調查（2013-04）


- **回收率**：96
- **參與者**：護理師60.4
- **認知正確率**：72.1
- **未通報原因**：         itemize
- 太忙（27人次）。
- 程序麻煩（12人次）。
- 不知道如何通報（4人次）。

## 品質提升策略


### 數據收集


- **挑戰**：腎上腺素記錄不完整，需使用商品名（Adrenalin）查詢。
- **解決方案**：與藥劑部合作，整合藥物使用數據（2013年1-2月：Adrenalin 288筆，呼吸器236筆）。


### 問卷設計


- **基本資料**：姓名、年齡、性別、入院日期、診斷。
- **事件詳情**：慢性疾病、危險因子、急救結果（ROSC與否）。
- **神經學狀態**：CPC分級（1-5級，1為正常，5為死亡）。
- **原因分析**：生理因素、工作流程、設備問題、溝通缺失。


| p{9cm}}  **CPC等級** | **描述** |  |
| --- | --- | --- |
| 1 | 大腦皮質功能良好，意識清楚，日常生活自理 |  |
| 2 | 中度功能障礙，部分活動受限（如偏癱） |  |
| 3 | 重度功能障礙，意識不清，需他人協助 |  |
| 4 | 深度昏迷，呈植物人狀態 |  |
| 5 | 死亡 |  |

### 持續改進


- 定期舉辦非預期CPR案例討論會。
