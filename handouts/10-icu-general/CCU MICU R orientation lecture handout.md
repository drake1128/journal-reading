## 前言

toc{section}{前言}

本講義旨在建立住院醫師對重症處置的信心與能力，內容涵蓋CCU與MICU常見急症的處置原則、緊急技能操作，以及醫院特殊流程。所有內容皆基於醫院實際流程與臨床實務經驗編寫。

**核心原則：**病人安全第一，團隊合作，持續學習。

### 敗血症與敗血性休克處置


criticalbox
**Sepsis-3定義：**感染 + SOFA ≥ 2分   **敗血性休克：**需血管加壓劑維持MAP ≥ 65mmHg + 乳酸 > 2 mmol/L
criticalbox


| |p{5cm}|p{7cm}|} 敗血症1小時Bundle與治療 |  |  |
| --- | --- | --- |
| **時間點** | **1小時Bundle** | **後續治療** |
| 立即 | **血液培養採檢** | 培養後72小時檢討 |
|  | **抗生素給予** | De-escalation策略 |
|  | **乳酸檢驗** | 乳酸清除追蹤 |
|  | **輸液復甦 30ml/kg** | 根據反應調整 |

| 第一線血管加壓劑 | Norepinephrine 0.1-3 μg/kg/min | 中央靜脈給予 |
