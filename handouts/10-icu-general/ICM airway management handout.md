### 研究背景

氣管插管是重症加護病房最常執行的高風險處置之一。根據INTUBE研究，在來自29國2964名重症成人患者中，至少發生一項主要圍插管期不良事件的比例高達45


## 重症氣道的特殊性：生理性困難氣道


### 定義與概念


> **定義：**患者的生理及病理生理改變增加了氣管插管及轉換至正壓通氣過程中發生併發症的風險。
> **核心概念：**
> itemize[leftmargin=*,nosep]
>     \item 重症患者的困難並非僅來自解剖結構
>     \item 生理儲備不足是主要風險因素
>     \item 需要系統性的準備與最佳化策略
> itemize


### 影響因素


造成重症患者插管高併發症率的因素包括：

- **複雜環境：**加護病房條件較手術室差，設備、燈光、團隊組成等皆有限制
- **操作者經驗差異：**不同專科、不同訓練程度的醫師執行插管
- **疾病嚴重度：**重症本身造成的生理改變，包括血流動力學不穩定、低血氧、代謝性酸中毒等


## 插管前準備


### 氣道評估：MACOCHA評分


MACOCHA評分是專為加護病房困難插管風險評估而設計的工具，包含生理改變、操作者經驗及解剖困難度等要素。


| p{3cm}p{5cm}} MACOCHA評分系統 |  |  |
| --- | --- | --- |
| **評估項目** | **分數** | **說明** |
| **與患者相關因素** |  |  |
| Mallampati III 或 IV | 5 | 口咽可視度評估 |
| 阻塞性睡眠呼吸中止症 | 2 | 病史詢問 |
| 頸椎活動度受限 | 1 | 理學檢查 |
| 張口受限 ($<$3 cm) | 1 | 理學檢查 |
| **與病理相關因素** |  |  |
| 昏迷 | 1 | GCS $<$ 8 |
| 嚴重低血氧 | 1 | SpO$_2$ $<$ 80 |
| **與操作者相關因素** |  |  |
| 非麻醉專科醫師 | 1 | 操作者背景 |
| **總分** | **0-12** | $\geq$3 預測困難插管 |


### 團隊準備


- 人為因素對安全執行氣管插管有重要影響
- 團隊準備是降低併發症風險的關鍵
- 插管前檢核表的使用尚未確立可降低併發症，但當包含生理最佳化介入措施時可能有效
- 對於經驗較少的團隊特別有幫助


## 呼吸系統最佳化


### 體位調整


> **建議：**床頭抬高30度
> **原理：**
> itemize[leftmargin=*,nosep]
>     \item 仰臥位時肺後側區域容易發生肺泡塌陷
>     \item 抬頭體位可增加功能性肺餘容量(FRC)
>     \item 減少插管過程中低血氧的發生率
>     \item 對肥胖患者特別重要
> itemize


### 預氧合策略比較


| p{4cm}p{4cm}p{3cm}} 預氧合策略比較 |  |  |
| --- | --- | --- |
| **方法** | **優點** | **缺點** |
| Bag-valve mask (BVM) | 簡單、廣泛可用 | 無法提供正壓支持、喉鏡檢查時需移除 |
| Non-rebreather mask | 可提供高濃度氧氣 | 無正壓支持、喉鏡檢查時需移除 |
| Non-invasive ventilation (NIV) | 提供正壓通氣、減少肺泡塌陷 | 喉鏡檢查時需移除、需要設備 |
| High-flow nasal oxygen (HFNO) | 可持續使用於喉鏡檢查期間、提供窒息期氧合 | 無正壓支持 |
| NIV + HFNO | 結合兩者優點 | 較複雜 |


### PREOXI試驗：關鍵證據


PREOXI試驗確立了NIV優於傳統氧氣面罩(non-rebreather mask或BVM)的預氧合效果，且未增加吸入性肺炎的發生率。


> **主要結果：**NIV組低血氧發生率顯著低於氧氣面罩組 (9.1
> **臨床意義：**
> itemize[leftmargin=*,nosep]
>     \item NNT = 11：每11位患者使用NIV預氧合，可預防1次嚴重低血氧事件
>     \item 對於所有氧氣治療層級的患者均有效益
>     \item BMI $\geq$ 30 的患者效益更為顯著
>     \item 未增加胃內容物吸入風險
> itemize
> **結論：**應考慮捨棄傳統氧氣面罩作為重症患者預氧合方式


### NIV vs HFNO比較


針對急性低血氧呼吸衰竭患者，比較NIV與HFNO(於喉鏡檢查期間持續使用)的試驗顯示：

- 整體嚴重低血氧發生率無顯著差異
- 次族群分析顯示：中重度低血氧患者中，NIV可能優於HFNO
- OPTINIV試驗(概念驗證研究)：NIV合併HFNO優於單獨NIV


### 特殊情況處理


| p{10cm}} 特殊情況的預氧合策略 |  |  |
| --- | --- | --- |
| **情況** | **建議策略** |  |
| 躁動患者 | 使用sub-anesthetic dose ketamine促進預氧合(延遲序列誘導, delayed sequence intubation)，可顯著減少圍插管期低血氧 |  |
| 改良式快速序列誘導 | 於窒息期間進行溫和面罩通氣，可減少圍術期低血氧且不增加吸入風險 |  |


## 血流動力學最佳化


### 血流動力學崩潰的重要性


考慮到圍插管期血流動力學崩潰的高發生率，血流動力學最佳化和監測至關重要。然而，執行血流動力學最佳化的精確方法在文獻中尚不明確。


### Montpellier插管流程


> Montpellier插管流程的使用已證實可減少插管相關併發症，其內容包括：
> itemize[leftmargin=*,nosep]
>     \item 插管前輸液(除非有禁忌)
>     \item NIV預氧合
>     \item 兩位操作者在場
>     \item 快速序列誘導
>     \item Capnography確認
>     \item 準備血管升壓劑
> itemize


### 輸液與血管升壓劑的角色


| p{5cm}p{6cm}} 血流動力學最佳化策略證據 |  |  |
| --- | --- | --- |
| **策略** | **研究結果** | **目前建議** |
| 插管前輸液 | Montpellier流程及觀察性研究支持；但PrePARE及另一RCT顯示無法減少心血管崩潰 | 除非有禁忌，可考慮使用 |
| 預防性血管升壓劑 | 兩項進行中的國際試驗：FLUVA Trial (NCT05318066)、PREVENTION Trial (NCT05014581) | 等待研究結果 |


## 執行氣管插管


### 改良式快速序列誘導 (Modified RSI)


應考慮使用改良式RSI技術以降低吸入風險：

- 給予快速起效的鎮靜催眠藥物
- 給予快速作用的神經肌肉阻斷劑
- 和/或溫和面罩通氣


### 環狀軟骨壓迫 (Cricoid Pressure)


- 若使用，應持續至氣管內管正確置入且氣囊充氣後
- 逆流可能發生於壓力傳遞至食道時
- 若造成聲門視野困難，應釋放壓迫
- 目前缺乏證據判斷哪些患者可從環狀軟骨壓迫中獲益


## 插管藥物選擇


### 誘導藥物


> INTUBE研究的事後分析顯示：使用propofol作為誘導藥物是這些患者心血管崩潰的**唯一可修正的獨立預測因子**。
> **建議：**使用血流動力學較穩定的藥物，如ketamine或etomidate


| p{5cm}p{6cm}} 誘導藥物比較 |  |  |
| --- | --- | --- |
| **藥物** | **特性** | **注意事項** |
| Propofol | 快速起效、恢復快 | 顯著血壓下降，為心血管崩潰的獨立預測因子 |
| Ketamine | 血流動力學較穩定、支氣管擴張 | 顱內壓升高疑慮(目前證據不支持)、可能增加分泌物 |
| Etomidate | 血流動力學最穩定 | 腎上腺抑制疑慮(單次使用臨床意義不明) |


### 神經肌肉阻斷劑


RSI期間使用神經肌肉阻斷劑與更高的首次插管成功率及較少併發症相關。


| p{5cm}p{6cm}} 神經肌肉阻斷劑比較 |  |  |
| --- | --- | --- |
| **藥物** | **特性** | **注意事項** |
| Succinylcholine | 最快速起效(45-60秒)、最短作用時間 | 禁忌：骨骼肌病變、已知過敏、惡性高熱病史、高血鉀、嚴重燒傷 |
| Rocuronium | 起效較慢、可用sugammadex逆轉 | 劑量效應：需要至少1.2 mg/kg以達到最快最佳插管條件 |


**Sugammadex的角色：**

- 可用於「無法插管、無法通氣」情況的快速逆轉rocuronium
- 重症患者的安全數據有限
- 角色極為有限，因為sugammadex可能逆轉麻痺，但無法逆轉對插管的需求


## 插管器械選擇


### 影像喉鏡 vs 直接喉鏡


> **主要結果：**
> itemize[leftmargin=*,nosep]
>     \item 影像喉鏡首次插管成功率：85.1
>     \item 直接喉鏡首次插管成功率：70.8
>     \item 絕對差異：14.3
>     \item 試驗因達到預設療效停止標準而提前終止
> itemize
> **結論：**影像喉鏡明顯提供優於直接喉鏡的插管條件


### 氣管內管輔助器械


| p{5cm}p{6cm}} 氣管內管輔助器械證據 |  |  |
| --- | --- | --- |
| **器械** | **研究結果** | **建議** |
| Stylet (管芯) | STYLETO試驗：使用直接喉鏡時，搭配管芯的首次插管成功率顯著高於單獨使用氣管內管 | 建議常規使用 |
| Bougie | 比較bougie與stylet的研究：首次插管成功率無差異 | 可作為替代選擇 |


**重要結論：**這些研究強調首次氣管插管嘗試時，常規使用stylet或bougie的重要性。


## 插管後照護


### 確認氣管內管位置


- 應使用持續波形capnography確認氣管內管位置（至少7次呼吸）
- 胸部聽診、胸部X光和/或支氣管鏡是確認氣管內管深度的可接受方法


### 後續管理


- 採用肺保護通氣策略
- 使用鎮靜藥物輸注
- 適當處理發生的併發症

## 臨床實務建議


> **插管前：**
> enumerate[leftmargin=*,nosep]
>     \item 使用MACOCHA評分評估困難氣道風險
>     \item 團隊準備與設備確認
>     \item 床頭抬高30度（HELP體位）
>     \item NIV預氧合3-5分鐘（首選方式）
>     \item 考慮合併HFNO於低血氧患者
>     \item 血流動力學最佳化（除非禁忌，考慮輸液）
> enumerate
> **誘導期：**
> enumerate[leftmargin=*,nosep]
>     \item 選擇血流動力學穩定的誘導藥物（ketamine或etomidate）
>     \item 使用適當劑量神經肌肉阻斷劑（rocuronium $\geq$1.2 mg/kg）
>     \item 改良式RSI：必要時可進行溫和面罩通氣
> enumerate
> **插管期：**
> enumerate[leftmargin=*,nosep]
>     \item 首選影像喉鏡
>     \item 常規使用stylet或bougie
>     \item SpO$_2$ $<$90
> enumerate
> **插管後：**
> enumerate[leftmargin=*,nosep]
>     \item Capnography確認（至少7次呼吸）
>     \item 確認氣管內管深度
>     \item 肺保護通氣策略
>     \item 鎮靜藥物輸注
> enumerate


## 結論


重症成人的氣管插管相關併發症仍然居高不下。雖然支持生理性困難氣道管理的證據仍在發展中，但實施上述已知策略可能有助於改善患者預後。關鍵要點包括：


1. **捨棄傳統氧氣面罩：**NIV應成為重症患者預氧合的標準方式
2. **優先使用影像喉鏡：**可提高首次插管成功率
3. **選擇血流動力學穩定的誘導藥物：**避免使用propofol，優先考慮ketamine或etomidate
4. **常規使用輔助器械：**stylet或bougie應於首次嘗試時即使用
5. **團隊準備與系統化流程：**遵循經驗證的插管流程如Montpellier protocol


**關鍵詞：**重症氣道管理、氣管插管、生理性困難氣道、預氧合、影像喉鏡、非侵襲性通氣
