### 病患基本資料


casebox
**病患背景：**

- 72歲女性
- 已知LVEF 20
- 目前使用BiPAP呼吸輔助
- 前兩天病況相對穩定

### 臨床惡化過程


| |p{12cm}|} 病況惡化時間軸 |  |  |
| --- | --- | --- |
| **時間點** | **病況變化與處置** |  |
| Day 1-2 | 病況相對穩定，BiPAP使用順利，生命徵象平穩 |  |
| Day 3 早上 | 護理師發現病人開始出現呼吸急促，SpO₂輕微下降 |  |
| 住院醫師到達 | 病人主訴喘，但意識清楚，可以說話 |  |
| 處置決策 | **住院醫師決定：先安排抽血與胸部X光檢查** |  |
| 20分鐘後 | 等待檢查結果期間，病人呼吸困難急遽惡化 |  |
| 惡化結果 | 病人「喘不動了」，意識模糊，緊急插管 |  |
| 插管過程 | 發生PEA (Pulseless Electrical Activity)，需要CPR |  |
| 最終結果 | 經CPR後恢復生命徵象，但整體預後受影響 |  |


## 問題分析：為什麼會發生PEA？


### 心衰患者的生理特殊性


criticalbox
**LVEF 20

- 心臟儲備功能極差
- 對前負荷和後負荷變化極度敏感
- 呼吸功的代償能力有限
- 一旦失代償，惡化速度極快


criticalbox


### BiPAP依賴患者的風險


| |p{11cm}|} BiPAP患者的特殊考量 |  |  |
| --- | --- | --- |
| **生理機轉** | **臨床意義** |  |
| 正壓通氣效應 | 減少venous return，降低前負荷，有助心衰 |  |
| 呼吸肌卸載 | 減少呼吸功，降低全身氧耗 |  |
| 依賴性風險 | 一旦移除BiPAP，前負荷驟增，呼吸功驟增 |  |
| 失效徵象 | SpO₂下降、呼吸次數增加、使用呼吸輔助肌 |  |


## 四種處理路徑分析


### 路徑一：積極床邊評估與預防性插管


tipbox
**最佳處理路徑：**
tipbox


| |p{5cm}|p{7cm}|} 路徑一：積極處理流程 |  |  |
| --- | --- | --- |
| **時間點** | **評估重點** | **處置決策** |
| 即時評估 | 床邊理學檢查：呼吸型態、血壓、心率、意識 | 立即使用Ambu bag評估呼吸驅動 |
| 1-2分鐘內 | **關鍵判斷：**病人需要更多支持 | 準備預防性插管，避免等到呼吸衰竭 |
| 5分鐘內 | 預防性插管完成 | 調整呼吸器設定，優化心肺功能 |
| 結果 | **成功避免PEA，維持血行動力學穩定** | 後續可逐步調整治療，預後較佳 |


#### 床邊評估重點


criticalbox
**5分鐘內必須評估的項目：**

- **呼吸型態：**是否使用呼吸輔助肌？
- **血行動力學：**血壓、心率變化趨勢
- **意識狀態：**是否仍能配合BiPAP？
- **Ambu bag test：**手動給予正壓，評估反應


criticalbox

warningbox
**實際發生的處理路徑（導致PEA）：**
warningbox


| |p{5cm}|p{7cm}|} 路徑二：延遲處理流程 |  |  |
| --- | --- | --- |
| **時間點** | **醫師思考** | **病況變化** |
| 病況惡化初期 | "先看數據再決定" | 呼吸困難持續加劇 |

| |p{11cm}|} 關鍵錯誤分析 |  |  |
| --- | --- | --- |
| **錯誤類型** | **說明** |  |
| 過度依賴檢查 | 忽略床邊理學檢查可提供的即時資訊 |  |
| 不了解病況急迫性 | LVEF 20 |  |

| 時間管理錯誤 | 心衰惡化是分鐘級的變化，不是小時級 |  |


### 路徑三：藥物優先的保守治療


| |p{5cm}|p{7cm}|} 路徑三：藥物治療路徑 |  |  |
| --- | --- | --- |
| **處置步驟** | **藥物選擇** | **預期效果與風險** |
| 利尿治療 | Furosemide 40-80mg IV | 減少前負荷，但需時間生效 |
| 血管擴張 | Nitroglycerin | 快速減少前負荷，但可能造成低血壓 |
| 強心劑 | Dobutamine考慮 | 增加心收縮力，但增加心律不整風險 |
| 風險評估 | 藥物生效需時間 | **LVEF 20 |
| 失敗結果 | 藥物效果不及時 | 仍可能進展到呼吸衰竭和PEA |

## 床邊理學檢查的關鍵指標


### 快速評估系統


criticalbox
**3分鐘床邊評估清單：**
criticalbox


| |p{5cm}|p{7cm}|} 床邊評估系統化檢查 |  |  |
| --- | --- | --- |
| **系統** | **檢查項目** | **紅色警訊** |
| 呼吸系統 | 呼吸頻率、型態、輔助肌使用 | RR >30, 矛盾呼吸, 呼吸輔助肌明顯 |
| 循環系統 | 血壓、心率、末梢灌流 | SBP <90, HR >120, 末梢冰冷 |
| 神經系統 | 意識狀態、配合度 | GCS下降、無法配合BiPAP |
| 整體評估 | 膚色、發汗、焦慮程度 | 發紺、冷汗、極度焦慮 |
| **任何一項紅色警訊 = 立即準備插管** |  |  |


### Ambu Bag測試技巧


tipbox
**Ambu Bag評估法：**

- 暫時移除BiPAP，使用Ambu bag手動給予正壓
- 觀察病人對手動正壓的反應
- 評估是否需要更積極的呼吸道控制
- 測試時間不超過1-2分鐘


tipbox


| |p{11cm}|} Ambu Bag測試結果判讀 |  |  |
| --- | --- | --- |
| **測試結果** | **臨床意義** |  |
| 病人明顯改善 | BiPAP可能不夠，但仍可嘗試調整參數 |  |
| 略有改善 | 考慮更積極的呼吸支持，準備插管 |  |
| 無明顯改善 | **立即準備插管，不要再等待** |  |
| 病況惡化 | **緊急插管，同時準備CPR** |  |


## 預防性插管 vs 緊急插管


### 時機選擇的關鍵


| |p{5cm}|p{6cm}|} 插管時機比較分析 |  |  |
| --- | --- | --- |
| **插管類型** | **優勢** | **劣勢** |
| **預防性插管** | 病人意識清楚，配合度佳 | 可能過度治療 |
| (早期決定) | 血行動力學相對穩定 | 需要醫師判斷經驗 |
|  | 技術難度較低 | 家屬可能不理解 |
|  | 避免PEA/心停風險 |  |
| **緊急插管** | "看起來"更有適應症 | 病人意識不清，不配合 |
| (等到衰竭) | 家屬較容易接受 | 血行動力學不穩定 |
|  |  | 技術困難度高 |
|  |  | PEA/心停風險高 |


### 心衰患者插管特殊考量


criticalbox
**LVEF 20

- 插管前後的血行動力學變化極大
- 插管藥物選擇要考慮心功能
- 插管後正壓通氣設定要小心
- 準備升壓劑和強心劑


criticalbox


## PEA的預防與處理


### PEA發生的機轉


| |p{11cm}|} 心衰患者PEA發生機轉 |  |  |
| --- | --- | --- |
| **階段** | **病理生理變化** |  |
| 初期代償失調 | 呼吸功增加，心臟負荷增加，血壓開始下降 |  |
| 代償機轉耗盡 | 交感神經過度刺激，心率過快，心臟充填不佳 |  |
| 循環惡性循環 | 心輸出量下降，冠狀動脈灌流不足 |  |
| PEA發生 | 心肌缺氧，有電活動但無有效收縮 |  |


### PEA的預防策略


protocolbox
**預防PEA的關鍵：**避免讓病人到達代償耗盡的階段
protocolbox

tipbox
**實用預防原則：**

- **早期識別：**不要等到病人"喘不動"
- **積極介入：**預防性插管優於緊急插管
- **藥物準備：**升壓劑和強心劑要隨時備用
- **團隊準備：**隨時準備CPR和ECMO


tipbox

## 特殊情境處理


### 肺栓塞緊急處理


| |p{11cm}|} 肺栓塞導致CPR的處理流程 |  |  |
| --- | --- | --- |
| **處理步驟** | **具體內容** |  |
| 病史評估 | 婦癌病史、前天已知DVT、因出血未用抗凝劑 |  |

| 後續處理 | 推超音波、給heparin、聯絡外科評估ECMO可能性 |  |
| 預後評估 | 癌症病人DVT+出血，需評估後續治療目標 |  |
