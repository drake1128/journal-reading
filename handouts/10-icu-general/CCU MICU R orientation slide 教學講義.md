## CCU常見急症


frame{急性冠心症處置流程}
criticalbox
**時間就是心肌：Door-to-balloon < 90分鐘**
criticalbox

columns
column{0.48\textwidth}
**STEMI處置：**

- **立即12導程ECG**
- **啟動心導管室**
- Aspirin 325mg 嚼碎
- Clopidogrel 600mg loading
- Heparin依體重給予
- Morphine疼痛控制


**追蹤重點：**

- Q6H cardiac enzymes至peak
- ECG變化追蹤


column
column{0.48\textwidth}
**NSTEMI處置：**

- 評估early intervention需求
- 症狀持續或enzymes持續上升
- Refractory angina
- Hemodynamic compromise


warningbox
**特殊ECG變化：**

- V1-V3 ST depression + R wave高
- →疑posterior STEMI
- →加做V7-V9
- Diffuse ST depression + aVR elevation
- →疑left main lesion

frame{心因性休克管理}
criticalbox
**定義：**收縮壓<90mmHg且心指數<2.2 L/min/m²
criticalbox

columns
column{0.58\textwidth}
**血行動力學監測：**

- 動脈導管置入
- 中央靜脈壓監測
- 尿液輸出量監測
- 乳酸值追蹤
- 考慮Swan-Ganz catheter


**藥物治療階梯：**

- **第一線：**Dopamine 5-20 μg/kg/min
- **第二線：**Norepinephrine 0.1-3 μg/kg/min
- **心收縮力：**Dobutamine 2.5-10 μg/kg/min


column
column{0.38\textwidth}
tipbox
**機械性支持：**

- IABP (intra-aortic balloon pump)
- ECMO評估
- 緊急血管再通


tipbox

**治療目標：**

- MAP ≥ 65 mmHg
- 尿量 > 0.5 ml/kg/hr
- 乳酸下降
- 中央靜脈血氧飽和度 > 70

frame{急性心律不整處置}
**心室性心律不整 (VT/VF)**

criticalbox
**有脈搏VT：**同步電擊100J → 150J → 200J  

**無脈搏VT/VF：**非同步電擊200J → CPR → Epinephrine 1mg
criticalbox

columns
column{0.48\textwidth}
**藥物治療：**

- Amiodarone 150mg IV (10分鐘)
- 接著 900mg/day infusion
- Lidocaine 1-1.5mg/kg IV bolus
- 接著 20-50 μg/kg/min infusion
- Magnesium 2g IV (Torsades de Pointes)


**電解質校正：**

- K⁺ > 4.0-4.5 mEq/L
- Mg²⁺ > 2.0 mg/dL
- 維持正常血氧


column
column{0.48\textwidth}
**心房纖維顫動：**

- Rate control: Metoprolol
- Rhythm control: Amiodarone
- 抗凝血評估 (CHA₂DS₂-VASc)


warningbox
**VT原因鑑別：**

- Ischemia (最常見)
- Cardiomyopathy
- Long QT syndrome
- Electrolyte imbalance
- Drug-induced

frame{急性心衰竭分級處置}
columns
column{0.48\textwidth}
**Killip分級：**

- **Class I：**無心衰竭徵象
- **Class II：**肺囉音<50
- **Class III：**急性肺水腫
- **Class IV：**心因性休克


**臨床評估：**

- 呼吸困難程度
- 肺部囉音範圍
- 頸靜脈怒張
- 下肢水腫程度
- BNP/NT-proBNP


column
column{0.48\textwidth}
criticalbox
**急性處置階梯：**

- **體位：**半坐，雙腳下垂
- **氧氣：**維持SpO₂>90
- **利尿：**Furosemide 40-80mg IV
- **血管擴張：**Nitroglycerin
- **後負荷：**ACE inhibitor (穩定後)


criticalbox

**監測指標：**

- 每日體重變化
- 水分平衡計算
- 腎功能追蹤
- 電解質變化

frame{Temporary Pacemaker操作}
columns
column{0.48\textwidth}
**置放技巧：**

- 20 cm → RA
- 30 cm → RV
- 40 cm → Pulmonary artery
- 放置時注意ECG monitor
- 看到VPC表示進入RV


**Threshold測試：**

- Pacing threshold < 1mA (理想)
- Output設定為threshold × 3
- Sensing threshold × 2作為setting


column
column{0.48\textwidth}
warningbox
**併發症預防：**

- 避免放置過深
- 注意VT風險
- 感染預防
- Lead移位監測


warningbox

**常見設定：**

- Rate: 依病人需求
- Output: 5mA (開始)
- Sensitivity: 依測試結果
- Mode: VVI或DDD

## MICU常見急症


frame{急性呼吸衰竭分類與處置}
criticalbox
**Type I：**PaO₂ < 60mmHg (氧合衰竭)  

**Type II：**PaCO₂ > 50mmHg (換氣衰竭)
criticalbox

columns
column{0.48\textwidth}
**氧氣治療階梯：**

1. 鼻導管 2-6 L/min
2. 面罩 6-10 L/min
3. Non-rebreather 10-15 L/min
4. High-flow nasal cannula
5. Non-invasive ventilation
6. 插管機械通氣


**NIV適應症：**

- COPD exacerbation
- Cardiogenic pulmonary edema
- Immunocompromised patients


column
column{0.48\textwidth}
**插管指標：**

- PaO₂ < 60mmHg (FiO₂ 60
- PaCO₂ > 50mmHg + pH < 7.25
- 呼吸頻率 > 35/min
- 意識變化
- 血行動力學不穩定
- NIV failure


warningbox
**插管前準備：**

- Pre-oxygenation充分
- 藥物準備就緒
- 困難插管設備
- 血壓支持準備


warningbox
column
columns
frame

frame{機械通氣設定與管理}
columns
column{0.48\textwidth}
**初始設定：**

- **模式：**Volume Control (AC/SIMV)
- **Tidal Volume：**6-8 ml/kg (預測體重)
- **PEEP：**5-8 cmH₂O
- **FiO₂：**100
- **呼吸頻率：**12-16/min


**ARDS保護性通氣：**

- Low tidal volume (6 ml/kg)
- Plateau pressure < 30 cmH₂O
- PEEP根據FiO₂調整
- Permissive hypercapnia


column
column{0.48\textwidth}
**Weaning評估：**

- RSBI < 105 breaths/min/L
- 原發疾病改善
- 血行動力學穩定
- 意識清楚，能配合
- 痰量適中


tipbox
**SBT Protocol：**

- T-piece trial 30-120分鐘
- 監測HR, RR, BP, SpO₂
- 觀察呼吸窘迫症狀
- 成功後考慮拔管

frame{敗血症與敗血性休克}
criticalbox
**Sepsis-3定義：**感染 + SOFA ≥ 2分  

**敗血性休克：**需血管加壓劑維持MAP ≥ 65mmHg + 乳酸 > 2 mmol/L
criticalbox

**1小時Bundle：**

1. ****血液培養採檢****
2. ****抗生素給予****
3. ****乳酸檢驗****
4. ****輸液復甦 30ml/kg****


columns
column{0.48\textwidth}
**抗生素選擇：**

- 社區感染：Ceftriaxone + Azithromycin
- 院內感染：Piperacillin/Tazobactam
- 抗藥性風險：Vancomycin + Carbapenem
- ICU acquired：考慮MRSA及gram negative coverage


column
column{0.48\textwidth}
**血管加壓劑：**

- **第一線：**Norepinephrine 0.1-3 μg/kg/min
- **第二線：**Vasopressin 0.01-0.04 U/min
- **心收縮力：**Dobutamine 2.5-20 μg/kg/min


**治療目標：**

- MAP ≥ 65 mmHg
- 尿量 > 0.5 ml/kg/hr
- 乳酸清除
- ScvO₂ > 70

columns

frame{急性腎衰竭與腎臟替代治療}
columns
column{0.48\textwidth}
**KDIGO分期：**

- **Stage 1：**Cr上升1.5-1.9倍或≥0.3mg/dL
- **Stage 2：**Cr上升2.0-2.9倍
- **Stage 3：**Cr上升≥3倍或≥4.0mg/dL


**原因分類：**

- **Pre-renal：**脫水、休克、藥物
- **Intrinsic：**ATN、腎絲球腎炎
- **Post-renal：**尿路阻塞


column
column{0.48\textwidth}
**透析指標：**

- BUN > 100 mg/dL
- 難治性肺水腫
- 嚴重代謝性酸中毒 (pH < 7.15)
- 高鉀血症 (K⁺ > 6.5)
- 尿毒症症狀


protocolbox
**CVVH vs IHD：**

- CVVH：血行動力學不穩定
- IHD：血行動力學穩定
- 床位：MICU 01-17, CCU 01-06

## 緊急處置技能


frame{中央靜脈導管置入}
columns
column{0.48\textwidth}
**適應症：**

- 血管加壓劑給予
- 中央靜脈壓監測
- 血液透析通路
- 週邊靜脈困難
- 大量輸液需求
- 高濃度藥物給予


**部位選擇優先順序：**

- **首選：**內頸靜脈 (超音波導引)
- **次選：**鎖骨下靜脈
- **感染時：**股靜脈


column
column{0.48\textwidth}
criticalbox
**操作步驟：**

1. 超音波導引定位血管
2. 嚴格無菌操作準備
3. 局部麻醉充分
4. 導引針穿刺確認
5. 導引線置入確認
6. 擴張器擴張
7. 導管置入確認回血
8. 胸部X光確認位置


criticalbox

warningbox
**併發症：**氣胸、血胸、血管摧傷、感染、空氣栓塞、心律不整

frame{困難插管處理}
criticalbox
**RSI (Rapid Sequence Intubation) 標準流程**
criticalbox

columns
column{0.48\textwidth}
**藥物選擇：**

- **鎮靜：**Propofol 1-2mg/kg
- **肌鬆：**Succinylcholine 1-1.5mg/kg
- **替代：**Rocuronium 1mg/kg
- **預處理：**Fentanyl 1-3μg/kg
- **血壓支持：**Ketamine 1mg/kg (shock病人)


**困難插管預測：**

- LEMON評估
- Mallampati分級
- 頸部活動度
- 牙齒狀況


column
column{0.48\textwidth}

**操作技巧：**

1. Pre-oxygenation (3-5分鐘)
2. 藥物給予順序正確
3. 等待肌鬆充分
4. 直接喉頭檢查
5. 氣管內管置入
6. 位置確認 (ETCO₂最可靠)
7. 固定氣管內管
8. 連接呼吸器


tipbox

**失敗處理：**

- 面罩通氣維持氧合
- 困難插管車
- 緊急外科呼吸道

columns
column{0.48\textwidth}

## 藥物使用原則


frame{血管加壓劑使用指引}
columns
column{0.6\textwidth}
\scriptsize


| **藥物** | **劑量** | **作用機轉** | **主要適應症** |
| --- | --- | --- | --- |
| Norepinephrine | 0.1-3 μg/kg/min | α₁ >> β₁ | 敗血性休克首選 |
| Epinephrine | 0.1-1 μg/kg/min | α₁ = β₁ | 心停、過敏性休克 |
| Dopamine | 5-20 μg/kg/min | 劑量依賴性 | 心因性休克 |
| Dobutamine | 2.5-20 μg/kg/min | β₁ > β₂ | 心衰竭、低心排 |
| Vasopressin | 0.01-0.04 U/min | V₁ receptor | 難治性休克輔助 |


column
column{0.35\textwidth}
tipbox

- 中央靜脈給予為佳
- 從低劑量開始
- 定期評估反應
- 避免突然停藥
- 注意藥物交互作用

## 醫院特殊流程


frame{APACHE II Score與TIMI Score}
columns
column{0.48\textwidth}
**APACHE II Score：**

- 轉入ICU時必須完成
- 轉出ICU時必須完成
- 病危出院或死亡免寫轉出分數
- 使用WebHIS系統書寫


**評估項目：**

- 生理指標 (12項)
- 年齡分數
- 慢性疾病分數
- 總分0-71分


column
column{0.48\textwidth}
**TIMI Score (急性冠心症)：**

- 年齡 ≥ 65歲
- 已知CAD (狹窄 ≥ 50
- 收縮壓 < 100 mmHg
- 心率 > 100 bpm
- Killip class II-IV
- 體重 < 67 kg
- 前壁ST elevation或LBBB
- 症狀開始至治療時間 > 4小時


tipbox
**總分越高，30天死亡率越高**

columns

**正常參考值：**

- pH: 7.35-7.45
- PaCO₂: 35-45 mmHg
- PaO₂: 80-100 mmHg
- HCO₃⁻: 22-26 mEq/L
- BE: -2 to +2 mEq/L
- SaO₂: >95


**判讀步驟：**

1. pH判斷酸鹼性
2. 確定原發異常
3. 評估代償反應
4. 計算anion gap
5. 評估氧合狀態


column
column{0.48\textwidth}
**常見酸鹼異常：**

- **代謝性酸中毒：**DKA、乳酸、腎衰竭
- **呼吸性酸中毒：**COPD、呼吸抑制
- **代謝性鹼中毒：**嘔吐、利尿劑
- **呼吸性鹼中毒：**過度換氣、焦慮

frame{檢驗值臨床意義}
columns
column{0.48\textwidth}
**心臟標記：**

- **Troponin I/T：**心肌摧傷最敏感
- **CK-MB：**急性期變化快
- **BNP/NT-proBNP：**心衰竭診斷
- **D-dimer：**血栓風險，但特異性差


**感染指標：**

- **WBC：**< 4,000 或 > 12,000 (SIRS)
- **CRP：**> 10 mg/L 顯著發炎
- **Procalcitonin：**> 0.25 ng/mL 細菌感染
- **Lactate：**> 2 mmol/L 組織灌流不足


column
column{0.48\textwidth}
**腎功能評估：**

- **Creatinine：**基準值變化重要
- **BUN：**脫水、上消化道出血
- **BUN/Cr ratio：**>20提示pre-renal
- **Estimated GFR：**腎功能評估

- K⁺ < 2.5 或 > 6.5 mEq/L
- Na⁺ < 120 或 > 160 mEq/L
- Glucose < 50 或 > 400 mg/dL
- Lactate > 4 mmol/L
- pH < 7.1 或 > 7.6
