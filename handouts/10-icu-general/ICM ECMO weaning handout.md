## 研究背景


### VV ECMO在重症ARDS的角色


靜脈-靜脈體外膜氧合（venovenous extracorporeal membrane oxygenation, VV ECMO）在嚴重急性呼吸窘迫症候群（ARDS）的應用已顯著增加。目前ESICM及ATS指引均建議在特定患者族群使用VV ECMO，以確保氣體交換的同時，減少呼吸器相關肺損傷（ventilator-induced lung injury, VILI）。


### VV ECMO脫離的挑戰


儘管VV ECMO使用日益普遍，其管理的關鍵面向仍缺乏標準化，尤其是脫離過程。脫離過程本質上極為複雜，涉及多重相互作用的生理與臨床因素。


| p{10cm}} VV ECMO脫離的主要挑戰 |  |  |
| --- | --- | --- |
| **挑戰類別** | **具體問題** |  |
| *續上表* |  |  |

| 證據基礎不足 | 目前實務主要依賴專家意見，缺乏證據導向的protocol |  |
| 時機選擇困難 | Sweep gas-off trial（SGOT）的最佳時機尚不確定 |  |
| 脫管標準不明 | Decannulation的客觀標準缺乏共識 |  |
| 呼吸器策略異質 | 脫離過渡期的呼吸器設定差異極大 |  |
| 評估指標多元 | 監測參數的選擇與權重因中心而異 |  |


### 研究目的


本國際調查旨在描述全球VV ECMO脫離的實際臨床實務，為未來制定標準化protocol提供基礎數據。

### 脫離失敗的認知因素


臨床醫師認為脫離失敗主要反映以下情況：

- 持續性氣體交換障礙
- 呼吸窘迫
- 臨床不穩定


## 討論


### 主要發現摘要


| p{10cm}} WEAN-ECMO調查主要發現 |  |  |
| --- | --- | --- |
| **面向** | **發現** |  |
| *續上表* |  |  |
| **面向** | **發現** |  |
| 實務異質性 | VV ECMO脫離實務存在顯著變異 |  |
| 主要依據 | 臨床醫師主要依賴氣體交換參數 |  |
| 呼吸力學地位 | 被視為相關但影響力較低 |  |
| 食道壓監測 | 仍不常用 |  |
| SGOT標準化 | 多數中心有protocol，但持續時間與評估標準差異大 |  |
| 呼吸器管理 | 脫離階段的通氣管理同樣異質 |  |


### 異質性的意涵


脫離實務的異質性反映了更廣泛的ECMO候選者選擇與管理的變異性。這種情況可能源於：


- 缺乏高品質證據支持特定策略
- 各中心經驗與資源差異
- 患者族群特性不同
- 缺乏明確的共識指引


### 「標準化」定義的問題


值得注意的是，調查中缺乏對「protocolized」的正式定義，可能導致某種程度的高報（over-reporting），並造成觀察到的變異性。

## 臨床意義與未來方向


### 標準化的重要性


脫離標準與protocol的標準化（包括監測參數與SGOT持續時間）可能有助於：


- 改善各中心間的可比較性
- 支持未來ECMO脫離建議的制定
- 減少不必要的變異
- 促進品質改善

## 實務建議摘要


> **脫離準備度評估（依重要性排序）：**
> enumerate[nosep]
>     \item 氣體交換參數：PaCO$_2$、PaO$_2$、pH
>     \item 呼吸力學：Tidal volume、Compliance、Driving pressure
>     \item 影像學：CXR、LUS（較邊緣性）
> enumerate
> **SGOT執行：**
> itemize[nosep]
>     \item 建議使用標準化protocol
>     \item 持續時間：中位數12小時（範圍6-24小時）
>     \item 成功標準應包含氣體交換與呼吸力學評估
> itemize
> **呼吸器管理：**
> itemize[nosep]
>     \item 多數患者維持intubation狀態脫離ECMO
>     \item Controlled與assisted ventilation均可選用
>     \item 早期tracheostomy在特定患者可考慮
> itemize
> **待解決問題：**
> itemize[nosep]
>     \item SGOT最佳持續時間
>     \item 食道壓監測的角色
>     \item 統一的decannulation標準
> itemize
