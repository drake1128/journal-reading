## 核心發現摘要

本文由三位專家（Baumgartner、Iung、Messika-Zeitoun）從 NEJM、JAMA、Lancet、Nature、EHJ、Circulation、JACC 中精選 2025 年瓣膜疾病領域最具影響力的十篇論文，涵蓋：

- 人工智慧（AI）在瓣膜疾病診斷與風險分層的應用
- 無症狀嚴重主動脈瓣狹窄（AS）的處置爭議
- TAVI vs 手術在低風險患者的長期追蹤
- TAVI 後藥物治療（Dapagliflozin）
- 二尖瓣與三尖瓣介入治療的最新證據

## 1. 人工智慧在瓣膜疾病的應用

### 1.1 ECHONEXT 研究：AI 偵測結構性心臟病

- **研究設計**：Deep learning ECG model，訓練資料超過一百萬筆心律與影像紀錄
- **目標**：偵測中重度瓣膜疾病（VHD）等結構性心臟病
- **結果**：
  - 內部與外部驗證均展現高診斷準確度
  - 在不同照護場域及種族群體表現一致
  - 臨床試驗中成功識別先前未診斷的心臟病患者
- **意義**：有潛力擴展大規模心臟病篩檢的可近性

### 1.2 DELINEATE-REGURGITATION 研究：AI 評估瓣膜逆流

- **資料**：71,660 份經胸心臟超音波，1,203,980 段 color Doppler 影像
- **準確度（Weighted kappa，內部/外部驗證）**：
  - Aortic regurgitation (AR)：0.81 / 0.76
  - Mitral regurgitation (MR)：0.76 / 0.72
  - Tricuspid regurgitation (TR)：0.73 / 0.64
- **特色**：多視角方法優於單視角；DELINEATE-MR-Progression 可預測 MR 進展

> **Pearl**：這些研究顯示 AI 即將在瓣膜疾病的臨床管理中產生重大影響。

## 2. 無症狀嚴重主動脈瓣狹窄的處置

### 2.1 四項 RCT 的 Meta-analysis

| 項目 | 內容 |
|------|------|
| 納入研究 | 4 項 RCT，共 1,427 位患者 |
| 比較 | 早期 AVR/TAVI vs 臨床監測 |
| 納入條件 | 無症狀嚴重高壓差 AS，左心室功能保留 |
| 平均追蹤 | 4.1 年 |
| 主要發現 | 早期 AVR 顯著降低：非計畫性 CV/HF 住院（14.6% vs 31.9%, P < .01）、中風（4.5% vs 7.2%, P = .03） |
| 死亡率 | 全因及心血管死亡：無顯著差異，但研究間存在高度異質性 |
| 爭議 | 最大的 RCT 中，監測組於 6 個月內交叉至介入被計為事件，此定義頗具爭議 |
| 需考量 | 需權衡短中期獲益與長期風險（人工瓣膜相關併發症、再介入） |

> **Pearl**：DANAVR 和 EASY-AS 等進行中的隨機試驗將提供更多證據。

## 3. 低風險患者：TAVI vs 外科手術

### 3.1 PARTNER-3 試驗：7 年追蹤

- **設計**：1,000 位低風險手術患者隨機分配至 TAVI 或 SAVR
- **7 年結果**：
  - 兩個主要複合終點（死亡、中風、再住院）無顯著差異
  - 血流動力學結果與瓣膜衰竭率相當
  - 值得注意：全因死亡的 Kaplan-Meier 曲線在 3 年後交叉
  - TAVI 組數值上較高死亡率（19.5% vs 16.8%）

### 3.2 NOTION-2 試驗：3 年追蹤

- **特色**：370 位低風險患者，平均年齡 71 歲（迄今最年輕）
- **首次納入**：雙葉主動脈瓣（Bicuspid aortic valve, BAV）
- **3 年結果**：
  - 整體主要終點：TAVI 16.1% vs SAVR 12.6%（P = .4）
  - 三葉瓣 AS：兩組相近（14.5% vs 14.4%, P = 1.0）
  - 雙葉瓣 AS：TAVI 風險較高（20.4% vs 7.8%, P = .08）

> **臨床建議**：
> - 三葉瓣低風險患者：TAVI 與手術結果相當
> - 雙葉瓣患者：應謹慎，需更多研究
> - 年輕患者需考量終生管理：預期再介入、TAVI 取出等

## 4. TAVI 後的藥物治療：DapaTAVI 試驗

| 項目 | 內容 |
|------|------|
| 設計 | 開放標籤 RCT，TAVI 後隨機分配至 Dapagliflozin vs 標準照護 |
| 患者 | 620 vs 637 位，平均年齡 82 歲，49% 女性 |
| 納入條件 | 心衰竭病史 + 至少一項：腎功能不全、糖尿病、LVEF ≤40% |
| 主要終點 | 1 年全因死亡或心衰竭惡化（住院或緊急就醫） |
| 結果 | Dapagliflozin 15.0% vs 標準照護 20.1%（P = .02） |
| 驅動因素 | 主要由心衰竭惡化降低驅動；死亡率無差異（7.8% vs 8.9%） |

> **Pearl**：此研究強調即使在瓣膜介入後，最佳化藥物治療仍然重要。

## 5. 二尖瓣逆流：手術與介入治療

### 5.1 MITRACURE Registry：二尖瓣手術真實世界數據

- **族群**：連續接受單獨或合併二尖瓣手術的 MR 患者
- **患者特徵**：
  - 最常見病因：Myxomatous（61%）
  - NYHA Class III/IV：43%
  - 心衰竭：30%
  - 心房顫動/flutter：22%
  - EF 下降：35%
  - 肺高壓：22%
- **早期介入**：僅 3% 接受早期介入
- **修補率**：整體 62%，退化性 MR 80%
- **院內死亡率**：整體 4.5%，退化性 MR 2.3%（單獨手術 1.8%）

> **Pearl**：此研究凸顯患者仍常被延遲轉介，早期介入比例低，需改善及時處置策略。

### 5.2 OCEAN-Mitral Registry：心房功能性 MR 的 TEER

- **背景**：心房功能性 MR（Atrial functional MR）被排除於三項現有 RCT 之外
- **比較**：TEER（441 人）vs 藥物治療（640 人，來自 REVEAL-AFMR registry）
- **結果**：
  - TEER 降低全因死亡 + HF 住院複合終點（HR 0.65, P = .044）
  - TEER 降低全因死亡（HR 0.58, P = .044）
  - **關鍵發現**：僅殘餘 MR ≤mild 者獲益
  - 殘餘 MR ≥moderate 者結果與藥物治療相近

### 5.3 SAPIEN M3 Trial：經導管二尖瓣置換

- **適應症**：不適合手術或 TEER 的嚴重 MR 患者
- **篩選**：1,171 人篩選，299 人植入（1/3 primary MR，2/3 secondary MR）
- **30 天結果**：
  - 死亡率 0.7%（STS score 6.6%）
  - 無術中死亡
  - 無 LVOT obstruction 導致血流動力學影響
  - 無轉為手術
- **主要終點**：全因死亡 + HF 住院 25.2%，顯著低於預設目標 45%
- **限制**：篩選失敗率高，無對照組

## 6. 三尖瓣逆流：TRILUMINATE Pivotal 2 年結果

| 項目 | 內容 |
|------|------|
| 設計 | 572 位嚴重續發性 TR 患者，TEER + 藥物 vs 單純藥物 |
| 1 年結果 | 生活品質：TEER 較佳；死亡或 HF 住院：無差異 |
| 2 年結果（允許交叉） | HF 住院年化率：TEER 0.19 vs 藥物 0.26（P = .02） |
| 全因死亡 | 無差異（17.9% vs 17.1%） |
| 三尖瓣手術率 | 無差異（2.3% vs 4.3%） |
| 限制 | 無 sham 對照的開放試驗；未提供 time-to-event 分析 |

> **Pearl**：對於嚴重續發性 TR，仍需進一步優化 TEER 的患者選擇標準。

## 8. 臨床要點總結

1. **AI**：ECG-based deep learning 可偵測結構性心臟病，AI 評估瓣膜逆流準確度高
2. **無症狀嚴重 AS**：早期介入可降低住院和中風，但死亡率無差異；需等待更多長期數據
3. **低風險 TAVI**：三葉瓣患者 TAVI vs 手術結果相當；雙葉瓣應謹慎
4. **TAVI 後藥物**：Dapagliflozin 可降低高風險患者的心衰竭惡化
5. **MR 手術**：患者常延遲轉介，早期介入比例僅 3%
6. **心房功能性 MR**：TEER 可能有益，但需達到良好修復（殘餘 MR ≤mild）
7. **經導管二尖瓣置換**：對不適合手術/TEER 者是可行選項
8. **TR TEER**：可降低 HF 住院，但對死亡率無影響
