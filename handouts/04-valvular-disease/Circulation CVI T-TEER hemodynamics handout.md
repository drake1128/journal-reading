> itemize

## 研究背景


### 三尖瓣逆流的臨床重要性


三尖瓣逆流（Tricuspid Regurgitation, TR）是老年人群中日益普遍的疾病，與功能狀態不佳及預後不良相關。對於重度TR患者，外科手術常因手術風險過高或禁忌而受限，因此經導管治療策略已成為較低侵入性的治療選擇。


### T-TEER的角色與挑戰


經導管三尖瓣緣對緣修復術（Transcatheter Tricuspid Valve Edge-to-Edge Repair, T-TEER）是目前最常採用的經導管治療方式，在經過謹慎篩選的高風險患者中已展現其可行性、安全性及症狀改善效果。然而，整體死亡率仍然顯著，且介入時機及患者選擇標準持續引發討論。


### 右心導管的重要性


由於嚴重TR時超音波估算肺動脈壓力的準確性受限，術前右心導管檢查（Right Heart Catheterization, RHC）在評估中扮演關鍵角色。先前研究已顯示肺高壓與T-TEER後死亡率增加相關。


> 評估侵入性測量的肺血流動力學參數對預後的影響，並確定特定閾值是否能增強風險分層及支持T-TEER患者的臨床決策。

## 研究方法


### 血流動力學評估


術前右心導管檢查由各參與中心自行決定執行，目標為在最佳臨床狀態下獲取最具代表性的血流動力學數據。


> **肺血管阻力（PVR）計算公式：**
> \[
> PVR = mPAP - PCWP{CO}
> \]
> itemize[leftmargin=*]
>     \item mPAP：平均肺動脈壓
>     \item PCWP：肺微血管楔壓
>     \item CO：心輸出量
> itemize

### 統計分析方法


- 類別變數：卡方檢定
- 連續變數：Student t檢定或Mann-Whitney U檢定
- 閾值確定：ROC曲線分析及Youden指數（早期終點）；最大選擇秩統計量（2年終點）
- 生存分析：Cox比例風險回歸及Kaplan-Meier曲線
- 非線性關係評估：限制性三次樣條（restricted cubic splines）
- 多重比較校正：Bonferroni校正
- 中心效應：以cluster effect建模，使用穩健Huber-White標準誤

> itemize[leftmargin=*]
>     \item 0 vs 1個風險因子：P=0.99
>     \item 0 vs 2個風險因子：P$<$0.01
>     \item 0 vs 3個風險因子：P$<$0.01
>     \item 1 vs 2個風險因子：P$<$0.01
>     \item 1 vs 3個風險因子：P$<$0.01
>     \item 2 vs 3個風險因子：P=0.02
> itemize
> **結論：**具有$\geq$2個參數超越閾值的患者預後最差。

### NYHA功能改善


T-TEER與追蹤期間NYHA功能分級顯著改善相關（P$<$0.01）。


> **主要發現：**
> itemize[leftmargin=*]
>     \item **mPAP$<$32 mmHg組：**基線與追蹤NYHA均顯著改善（P$<$0.01）
>     \item **mPAP$\geq$32 mmHg組：**NYHA仍顯著改善（P$<$0.01），但改善幅度較小（P=0.04）
>     \item **PCWP$<$20 mmHg組：**NYHA顯著改善（P$<$0.01）
>     \item **PCWP$\geq$20 mmHg組：**NYHA仍顯著改善（P$<$0.01），但改善幅度較小（P$<$0.01）
>     \item **PVR分組：**兩組NYHA均顯著改善（P$<$0.01 vs P=0.02），組間差異不顯著（P=0.38）
> itemize
> **關鍵結論：**儘管基線血流動力學狀態不佳的患者事件率較高且症狀改善較小，但T-TEER在所有亞組中均能帶來有意義的功能改善。


### 殘餘TR的預測因子


> **無侵入性血流動力學參數與殘餘TR$\geq$3+獨立相關。**
> 這意味著：
> enumerate[leftmargin=*]
>     \item 血流動力學參數不應用於預測手術成功率
>     \item 追求最佳技術結果在血流動力學不利的患者中同樣重要
>     \item 殘餘TR$\geq$3+本身是早期和晚期不良結局的獨立預測因子
> enumerate

## 臨床討論


### 主要發現總結


1. **閾值差異：**與不良結局相關的血流動力學閾值（mPAP$\geq$32 mmHg、PCWP$\geq$20 mmHg、PVR$\geq$5 WU）持續高於ESC肺高壓診斷閾值，提示T-TEER候選患者可能需要特定於手術的風險定義。
2. **綜合評估必要性：**單一侵入性參數無法充分反映右心病理生理，全面的血流動力學評估對於準確預後判斷至關重要。
3. **臨床獲益的普遍性：**儘管血流動力學不利的患者更可能出現早期不良事件且症狀反應減弱，T-TEER仍能帶來有意義的臨床獲益，即使在高風險患者中亦然。
4. **排除標準的適用性：**這些發現具有特殊相關性，因為具有顯著和毛細血管前肺高壓的患者通常被排除在主要隨機試驗之外。


### 與現行指引的關係


| p{5cm}p{5cm}} 本研究發現與ESC指引的比較 |  |  |
| --- | --- | --- |
| **方面** | **ESC指引** | **本研究發現** |
| 診斷閾值 | mPAP$>$20 mmHg    PCWP$>$15 mmHg    PVR$>$2 WU | 預後閾值更高   （+12/+5/+3） |
| 臨床價值 | 用於肺高壓診斷    已證實對左心疾病有預後價值 | 對T-TEER患者的預後價值需重新評估    建議採用結局導向的情境特定分層 |
| 治療決策 | 合併前後毛細血管肺高壓（PVR$>$5 WU）建議個人化處理 | 支持此建議    提供病理生理學依據支持進一步研究 |


### 機制解釋


> PCWP與早期及晚期結局的獨立相關性強調了雙心室心衰竭在此族群中的影響。此關聯進一步由以下發現支持：
> itemize[leftmargin=*]
>     \item **LVEDD：**反映左心室功能不全及擴大的貢獻
>     \item **RAP：**雖主要因TR升高，也反映肺壓力及右心室收縮/舒張功能
> itemize
> 這些發現突顯左右心室功能不全對此族群預後的共同貢獻。

| 風險分層 | 使用本研究確定的閾值（mPAP$\geq$32、PCWP$\geq$20、PVR$\geq$5）進行風險分層    評估超越閾值的參數數量 |  |
| 患者選擇 | 侵入性血流動力學參數**不應**用於拒絕治療    而應用於識別可能需要更密切監測、強化藥物治療或早期再評估的患者 |  |
| 技術目標 | 即使在血流動力學不利的患者中，仍應追求最佳技術結果    殘餘TR$\geq$3+與不良預後相關 |  |
| 術後管理 | 血流動力學不利患者可能需要更積極的心衰竭治療    考慮更頻繁的追蹤 |  |
| 未來方向 | 對於合併前後毛細血管肺高壓患者（PVR$>$5 WU），目前無針對毛細血管前成分的治療    需要進一步研究以釐清靶向藥物治療或器械卸載的治療潛力 |  |


## 研究限制


1. **研究設計：**回顧性觀察性研究，無法提供T-TEER與藥物治療比較的確定性結論。
2. **選擇偏差：**僅納入接受術前RHC的患者，可能富集更多有症狀或臨床複雜的患者；接受RHC的患者可能接受更仔細的評估或優化的藥物治療。
3. **測量變異：**血流動力學評估由各中心自行決定，未經中央判定，測量技術、時機或解讀可能存在變異。
4. **超音波評估：**未經core lab驗證，可能存在觀察者間變異。
5. **術後數據缺失：**無術後血流動力學數據，T-TEER是否及如何改變每位患者的血流動力學狀態，以及此類變化是否具有預後意義，仍屬未知。


## 結論


> enumerate[leftmargin=*]
>     \item 在接受T-TEER的患者中，侵入性血流動力學評估提供有價值的預後資訊。
>     \item **PCWP$\geq$20 mmHg**獨立預測早期以患者為中心的不良結局及2年死亡/HFH。
>     \item 儘管血流動力學不利的患者事件率較高，T-TEER仍能在所有亞組中帶來有意義的功能改善。
>     \item 侵入性血流動力學參數不應用於拒絕治療，而應用於識別需要更密切監測和更積極治療的患者。
>     \item 全面的右心導管檢查可增強術前風險分層並指導術後管理。
> enumerate
