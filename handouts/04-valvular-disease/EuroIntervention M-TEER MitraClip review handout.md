## 核心摘要

- **研究類型**：State-of-the-Art 綜述文章
- **核心主題**：續發性二尖瓣逆流（SMR）在心衰竭患者中的經導管緣對緣修復術（TEER）
- **關鍵發現**：
  - TEER 已成為症狀性 SMR 患者的指引建議治療選項
  - 2025 年 ESC 指引將 TEER 用於 V-SMR 升級為 Class I（證據等級 A）
  - 患者選擇是 TEER 成功的關鍵決定因素
  - COAPT 試驗是目前唯一顯示存活獲益的隨機對照試驗

## 2. 背景與病理生理學

### 2.1 二尖瓣逆流與心衰竭的關係

二尖瓣逆流（MR）與心衰竭（HF）之間存在複雜的雙向交互作用：

- 全球超過 6,400 萬人罹患心衰竭
- 急性心衰竭患者中，超過 50% 合併中度至重度 MR
- MR 的存在會加重血流動力學負擔、加劇症狀、增加住院率
- MR 嚴重程度與心衰竭預後密切相關

### 2.2 續發性二尖瓣逆流的分類

續發性二尖瓣逆流（SMR）是指在結構正常的二尖瓣上，因左心室或左心房重塑導致的瓣膜功能異常。

**兩大類型：**

1. **心室型 SMR（V-SMR）**：佔大多數，與缺血性或非缺血性心肌病變相關
2. **心房型 SMR（A-SMR）**：因心房擴大、瓣環擴張所致，常與心房顫動及 HFpEF 相關

### 2.3 SMR 表型對照表

| 特徵 | 比例性 V-SMR | 非比例性 V-SMR | 混合型 | 心房型 SMR |
|------|-------------|---------------|--------|-----------|
| 心衰竭類型 | HFrEF | HFrEF | HFmrEF | HFpEF |
| LVEF 範圍 | <40% | <40% | 40-49% | ≥50% |
| 左心室/左心房特徵 | 嚴重擴大的左心室 | 輕至中度擴大的左心室 | 輕至中度左心室/左心房擴大 | 正常左心室，嚴重擴大的左心房 |
| 主要機轉 | 整體左心室重塑、對稱性瓣葉牽扯 | 區域性左心室重塑、不對稱性瓣葉牽扯 | 瓣環與心室擴張合併（機轉重疊） | 二尖瓣環擴張 |
| 代表試驗 | MITRA-FR 表型 | COAPT 表型 | 研究較少 | EXPAND 研究 |
| TEER 獲益 | 無明確獲益 | 經選擇患者可獲益 | 需進一步研究 | 初步數據顯示可能獲益 |

## 3. SMR 的評估

### 3.1 心臟超音波評估

| 參數 | 重度 MR 標準 |
|------|-------------|
| EROA（有效逆流孔面積） | ≥0.40 cm² |
| 逆流容積 | ≥60 mL/beat |
| 逆流分數 | ≥50% |
| PISA 半徑 | ≥1 cm |
| **SMR 特殊考量（較低閾值可能具臨床意義）** | |
| SMR 之 EROA 閾值 | ≥0.30 cm² |
| SMR 之逆流容積閾值 | ≥45 mL/beat |

### 3.2 評估限制與多模態影像

- SMR 的動態特性導致逆流嚴重度隨血流動力學負荷條件變化
- PISA 計算在 SMR 中容易出現誤差（非半球形孔口、偏心噴流）
- 建議使用多模態影像整合評估
- 3D 心臟超音波可提供更精確的 vena contracta 測量
- 心臟磁振造影（CMR）可提供精確的逆流容積測量

MitraClip（Abbott）是目前使用最廣泛的 TEER 裝置：

| 特徵 | 說明 |

### 5.3 試驗結果差異分析

> **關鍵洞見：比例性 vs 非比例性 MR**
>
> - **非比例性 MR（Disproportionate MR）**：逆流嚴重程度超過左心室擴張程度所預期 — 代表 COAPT 表型
> - **比例性 MR（Proportionate MR）**：逆流嚴重程度與左心室擴張程度相符 — 代表 MITRA-FR 表型

**臨床意義：**

- **COAPT 患者**：較高 EROA（0.41 cm²）、較小 LVEDV（193 mL）= 非比例性 MR = TEER 獲益明顯
- **MITRA-FR 患者**：較低 EROA（0.31 cm²）、較大 LVEDV（250 mL）= 比例性 MR = TEER 無明確獲益
- **RESHAPE-HF2 患者**：中間特徵 = HF 住院減少但無存活獲益

> **注意**：此框架的實際臨床應用價值有限，因 SMR 具動態特性，同一患者在不同時間點可能呈現不同分類。

### 5.4 MATTERHORN 試驗：TEER vs 手術

MATTERHORN 是首個比較 TEER 與外科手術治療孤立性 SMR 的隨機對照試驗：

| 特徵 | TEER 組 | 手術組 |
|------|---------|--------|
| 平均年齡（歲） | 70.5 | 70.5 |
| 平均 LVEF (%) | 43.0 | 43.0 |
| 1 年主要終點* | 16.7% | 22.5% |
| 非劣性 P 值 | <0.001（達到非劣性） | — |
| 1 年 NYHA III/IV | 23.3% | 18.2% |
| MR ≤2+ 維持率 | 96% | 稍高 |

*主要終點：死亡、HF 住院、再介入、LVAD 植入或中風的複合終點

> **主要結論**：TEER 在孤立性 SMR 治療中達到與手術非劣性，且具有較佳的安全性特徵。

### 6.2 長期預後數據

**COAPT 試驗 5 年追蹤結果：**

- HF 住院年率：TEER 33.1% vs GDMT 57.2%（HR 0.53, 95% CI: 0.40-0.68）
- 全因死亡率：TEER 57.3% vs GDMT 67.2%（HR 0.72, 95% CI: 0.58-0.89）

**EuroSMR 註冊研究 5 年結果：**

- 5 年存活率：35.0%（較 COAPT 的 42.7% 低，反映真實世界較高風險族群）
- 5 年 MR ≤2+ 維持率：85.5%

## 7. GDMT、CRT 與患者選擇

### 7.1 指引導向藥物治療（GDMT）的角色

> **GDMT 優化的重要性**

**關鍵要點：**
- 目前指引建議在考慮 TEER 前應完成 GDMT 上調
- 長期 GDMT 可逆轉左心室重塑，減少 MR 嚴重度
- 近 60% 的 SMR 患者在 GDMT 後可見顯著 MR 改善

**新型藥物證據：**
- Sacubitril/valsartan：PRIME 研究顯示 EROA 減少 30%
- SGLT2 抑制劑：Dapagliflozin 對 MR 減少和心室重塑有益
- 注意：三大 TEER 試驗均在這些藥物廣泛使用前進行

**TEER 作為 GDMT 促進者：**
- TEER 可穩定血流動力學，創造 GDMT 上調的臨床窗口
- EuroSMR 註冊研究顯示：>1/3 患者在 TEER 後成功上調 GDMT
- GDMT 上調與較低死亡率和 HF 住院率獨立相關

### 7.2 心臟再同步化治療（CRT）

- 30-40% 的 CRT 候選患者合併中重度 SMR
- CRT 植入後 30-40% 患者出現持續或惡化的 SMR
- CRT 無反應者合併嚴重 SMR 者具有較高發病率和死亡率
- MITRA-CRT 試驗：TEER 在 CRT 無反應者中顯著減少心臟死亡、心臟移植和 HF 住院的複合終點（13% vs 67%, p=0.003）

### 7.3 COAPT 標準患者選擇

| 類別 | 標準 | 理由 |
|------|------|------|
| 二尖瓣逆流 | 重度 SMR（EROA ≥0.30 cm²，逆流容積 ≥45 mL/beat） | 確保足夠的 MR 負擔以從 TEER 獲益 |
| 左心室功能 | LVEF 20-50% | 避免過度重塑或保留射血分數的極端情況 |
| 左心室大小 | LVESD ≤70 mm | 過度左心室擴張（>70 mm）患者較難獲益 |
| 症狀狀態 | NYHA Class II-IV，儘管優化 GDMT ≥30 天 | 反映持續症狀和功能限制 |
| 心衰竭負擔 | 12 個月內 ≥1 次 HF 住院，或 BNP ≥300 pg/mL 或 NT-proBNP ≥1,500 pg/mL | 高 HF 負擔預測 MR 減少的獲益 |
| 肺動脈高壓 | 肺動脈收縮壓 ≤70 mmHg | 嚴重肺高壓與不良預後相關 |
| 右心室功能 | 無嚴重右心室功能障礙（TAPSE >1.5 cm） | 保留右心室功能對 TEER 後血流動力學改善至關重要 |
| 二尖瓣解剖 | 有利解剖（足夠瓣葉對合、最小牽扯、無顯著鈣化），瓣膜面積 >4.0 cm² | 最佳瓣葉形態確保手術成功和 MR 減少持久性 |
| 手術風險 | 禁忌或高風險進行二尖瓣手術 | TEER 對高風險或不適合手術患者最有益 |

## 8. 預後預測因子

### 8.1 良好預後預測因子

| 類別 | 因子 |
|------|------|
| 臨床因子 | 優化 GDMT（beta blocker, ARNI, MRA, SGLT2i）；CRT 植入；NYHA ≤III；需要時完全血運重建 |
| 急性手術成功 | 符合 COAPT 標準；LVEF ≥20%；LVESD ≤70 mm；PASP ≤70 mmHg；TAPSE ≥15 mm；無末期心衰竭；TR 嚴重度 <3 |
| 有利解剖 | MVA >4 cm²；單一中央 MR 噴流；抓取區 ≥10 mm；後瓣葉 >10 mm；連枷間隙 <10 mm；對合長度 >2 mm |

### 8.2 不良預後預測因子

| 類別 | 因子 |
|------|------|
| 不符合 COAPT 標準 | 血流動力學不穩定；中重度右心室功能障礙；合併嚴重瓣膜疾病；嚴重 COPD；肥厚型、限制型或浸潤性心肌病 |
| 不利解剖 | 環狀嚴重 MAC；MVA <3.0 cm²；多發/寬噴流；後瓣葉 <5 mm；二尖瓣狹窄（MG >5 mmHg）；抓取區瓣葉鈣化 |
| 臨床因子 | 心房顫動或心房撲動病史；CKD ≥III 期；TEER 後殘餘 MR ≥3+；糖尿病；嚴重左心房/左心室重塑（Indexed LA volume >34 mL/m²，LVEDV ≥159 mL）；NT-proBNP >10,000 ng/L |

### 8.3 SMR 分期系統

| 分期 | 定義 | 特徵 | 預後 |
|------|------|------|------|
| Stage 1 | 孤立左心室功能障礙 | 僅左心室受累 | 相對較好 |
| Stage 2 | 左心房受累 | 左心房擴大/功能障礙 | 中等 |
| Stage 3 | 右心室壓力/容積負荷 | 肺高壓、三尖瓣逆流 | 較差 |
| Stage 4 | 雙心室衰竭 | 右心室功能障礙 | 最差 |

| 2020 | AHA/ACC VHD 指引 | 升級至 Class IIa 用於 PMR；Class IIa 用於 SMR |
| 2021 | ESC VHD 指引 | Class IIb 用於 PMR；Class IIa 和 IIb 用於特定 SMR |
| 2025 | ESC VHD 指引 | 升級至 Class IIa 用於 PMR；**Class I（證據等級 A）用於 V-SMR**；Class IIb 用於症狀性 A-SMR |

> **2025 ESC 指引關鍵更新**
>
> **續發性二尖瓣逆流（V-SMR）：**
> - TEER 升級為 Class I, Level of Evidence A
> - 適用於：經謹慎選擇、優化藥物治療後仍持續 V-SMR 的患者
> - TEER 被認定為此族群的建議選項
>
> **心房型續發性二尖瓣逆流（A-SMR）：**
> - 新增建議：TEER 可考慮用於症狀性 A-SMR（Class IIb）
> - 這是 A-SMR 首次被納入 ESC 指引

## 12. 結論與臨床建議

### 12.1 主要結論

1. TEER 已成為 HFrEF 合併 SMR 患者的變革性治療，在適當選擇的患者中可顯著減少死亡率和 HF 住院
2. 患者選擇是 TEER 成功的關鍵決定因素，需要整合心臟超音波參數、臨床分期系統和多學科心臟團隊討論
3. 2025 年 ESC 指引將 V-SMR 的 TEER 升級為 Class I（證據等級 A），並首次納入 A-SMR 的 Class IIb 建議
4. COAPT 表型患者獲益最大：非比例性 MR、較高 EROA、較小左心室容積
5. GDMT 優化和 CRT 評估應在 TEER 前完成，但 TEER 也可作為 GDMT 上調的促進者
6. 未來研究方向包括：A-SMR、中度 SMR、無症狀嚴重 SMR、HFmrEF/HFpEF 合併 SMR

### 12.2 臨床實務建議

> **臨床決策流程**
>
> **步驟 1：GDMT 優化**
> - 確保 beta blocker、ARNI、MRA、SGLT2i 達最大耐受劑量
> - 評估 CRT 適應症並在符合條件時植入
>
> **步驟 2：重新評估（GDMT/CRT 後 2-3 個月）**
> - 重複心臟超音波評估 MR 嚴重度
> - 若 MR 改善且症狀緩解，繼續藥物治療
>
> **步驟 3：TEER 評估（若 MR 持續）**
> - 評估是否符合 COAPT 標準
> - 多學科心臟團隊討論
> - 評估解剖適合性
>
> **步驟 4：術後管理**
> - 繼續並嘗試上調 GDMT
> - 定期追蹤心臟超音波
> - 監測症狀和功能狀態
