2 1. 前言與學習目標

本教學文件旨在系統性介紹經導管主動脈瓣膜置換術 (Transcatheter Aortic Valve Replacement, TAVR) 前的電腦斷層 (CT) 評估要點。CT 影像已成為 TAVR 術前規劃的黃金標準，其重要性包括精確的瓣環測量、冠狀動脈阻塞風險評估、血管通路評估等。

2.1 1.1 學習目標

完成本文件學習後，學員應能夠：

  1. 識別並正確測量 aortic annulus 的各項參數
  2. 區分 tricuspid 與 bicuspid aortic valve 的 CT 特徵
  3. 評估冠狀動脈阻塞風險並計算 VTC 距離
  4. 判讀 Post-TAVR CT 中的 HALT 與其他併發症
  5. 掌握 Valve-in-Valve (ViV) TAVR 的特殊 CT 評估要點

3 2. CT 影像擷取技術

3.1 2.1 掃描參數與時機

項目 建議參數
ECG-gating 必須使用 retrospective ECG-gating 以獲得多時相重建
R-R interval Annulus 測量通常於 30-45% R-R interval（收縮期）進行；瓣膜形態與冠狀動脈評估於舒張期進行
對比劑使用 75-100 mL 非離子型對比劑，注射速率 4-5 mL/sec
掃描範圍 心臟 CT：從氣管分叉至心尖下方；血管 CT：從胸廓出口延伸至股動脈中段
輻射劑量優化 建議使用 dose modulation strategies，典型有效劑量：8-15 mSv

Pearl：對於嚴重主動脈瓣狹窄患者，不建議使用降心率藥物（如 beta-blockers）以避免血流動力學不穩定。若患者無法配合閉氣指令（如呼吸困難、聽力障礙），可考慮使用自由呼吸（free-breathing）模式配合寬偵測器 CT 進行掃描，研究顯示此方式不影響 annulus 測量的準確性。

4 3. Aortic Root 解剖與測量

4.1 3.1 Aortic Annulus 測量

Aortic annulus 是 TAVR sizing 最重要的參考依據，定義為三個 cusp 的 lowest insertion point (hinge point) 連成的虛擬平面 (virtual basal ring, VBR)。

Annulus 測量參數與 THV Sizing：

  • Balloon-expandable valve (BEV)：主要依據 annular area 進行 sizing
  • Self-expanding valve (SEV)：主要依據 perimeter 進行 sizing
  • 已淘汰的方法：單純使用短軸與長軸直徑測量，因 reproducibility 較差

4.1.1 3.1.1 Annulus 測量參數

測量參數 說明與臨床意義
Area (mm²) BEV sizing 的主要依據；例：Area 304 mm² → 約 23 mm SAPIEN 3
Perimeter (mm) SEV sizing 的主要依據；例：Perimeter 69 mm → Evolut valve sizing
Perimeter-derived diameter Perimeter / π；標準化比較不同形狀 annulus 的方法
Area-derived diameter √(Area/π)；另一種標準化直徑計算方式
Eccentricity index 1 – (短軸/長軸)；評估 annulus 橢圓度，影響 paravalvular leak 風險

4.1.2 3.1.2 測量時機選擇

R-R Interval 選擇原則：

  • 最大 annulus area 通常位於 30-45% R-R interval（收縮中期）
  • 若最大面積落在舒張期 → 提示可能存在 septal hypertrophy 或 diastolic dysfunction
  • 建議至少重建 5-10 個時相進行比較，選擇最大收縮期面積進行 sizing

4.1.3 3.1.3 測量誤差來源

常見的測量誤差來源包括：

  1. Cusp hinge point 判定困難（尤其在嚴重鈣化時）
  2. 含鈣化區塊影響邊界判定
  3. Motion artifact（心率過快或不規則）
  4. Partial volume effect
  5. Annulus 平面傾斜角度不正確

4.2 3.2 Left Ventricular Outflow Tract (LVOT)

LVOT 評估對於 TAVR 規劃至關重要，主要測量位置為距 annulus 3 mm 下方。

評估項目 臨床意義
LVOT 直徑與面積 若 LVOT > annulus → 可能允許較大 THV；若 LVOT < annulus → 可能需選擇較小 THV
LVOT 鈣化 顯著增加 BEV annular rupture 風險，尤其是 nodular calcification
Septal bulge 影響 SEV 定位與 conduction disturbance 風險
Membranous septum 長度 與 SEV 深度植入相關的 PPM 風險評估；短 membranous septum → 較高 PPM 風險

Pearl — LVOT 鈣化與 Annular Rupture：LVOT 嚴重鈣化（尤其是 nodular、asymmetric calcification）是 BEV annular rupture 的重要預測因子。在此類患者中，應考慮：(1) 使用 SEV 替代 BEV；(2) 選擇較保守的 sizing 策略；(3) 避免過度後擴張。

4.3 3.3 Sinus of Valsalva (SOV) 與冠狀動脈高度

4.3.1 3.3.1 低風險特徵參考值

冠狀動脈阻塞低風險特徵：

  • SOV 平均直徑 ≥30 mm
  • 冠狀動脈高度 (coronary height) ≥12 mm
  • 注意：這些為參考值，需結合 annulus 尺寸與 THV 選擇綜合判斷

4.3.2 3.3.2 冠狀動脈阻塞機轉

機轉 說明
Direct obstruction Native leaflet 直接被推移覆蓋冠狀動脈 ostium
Sinus sequestration Native leaflet 與 STJ 之間形成封閉空間，阻斷血流進入 sinus of Valsalva

4.3.3 3.3.3 VTC 距離測量

Virtual valve-to-coronary (VTC) 距離是評估冠狀動脈阻塞風險的關鍵參數。

VTC 距離 風險分級
< 3 mm High risk：強烈建議進行 BASILICA 或其他冠狀動脈保護措施
3-6 mm Intermediate risk：需綜合評估其他因素
> 6 mm Low risk：冠狀動脈阻塞風險較低

4.4 3.4 Sinotubular Junction (STJ)

評估項目 臨床意義
STJ 直徑 與 sinus sequestration 風險相關；窄 STJ 增加阻塞風險
STJ 高度 評估 leaflet 是否延伸超過 STJ
STJ 鈣化 增加 aortic dissection 風險；需特別注意 balloon post-dilation
VTSTJ 距離 Virtual valve-to-STJ 距離；< 3.5 mm 視為高風險

4.5 3.5 升主動脈評估

評估是否存在 ascending aortic aneurysm（直徑 > 40-45 mm），尤其在 bicuspid aortic valve 患者中更為重要。Horizontal aorta（aortic angle > 70°）可能影響 THV 定位與導管操作。

5 4. Bicuspid Aortic Valve (BAV) 評估

5.1 4.1 BAV 分類系統

Bicuspid aortic valve 在 TAVR 時代需要特殊的 CT 評估方法。

5.1.1 4.1.1 Sievers 分類

類型 描述
Type 0 無 raphe，真正的兩個 cusps；亞洲族群較常見
Type 1 一個 raphe 連接兩個 cusps；西方族群最常見，L-R raphe（左右冠狀動脈 cusp 融合）最多
Type 2 兩個 raphes；實質上接近 unicuspid valve，罕見

5.1.2 4.1.2 TAVR 時代 BAV 影像分類（BAVi Classification）

類型 描述與 TAVR 意義
Tricommissural (Functional BAV) 三個 commissures 但功能上呈 bicuspid pattern；又稱 acquired 或 functional bicuspid
Bicommissural raphe-type 兩個 commissures，有一個 raphe；Raphe 鈣化程度影響 TAVR 結果
Bicommissural non-raphe type 兩個 commissures，無 raphe；較常見於亞洲族群

5.2 4.2 BAV 的 CT 評估要點

BAV TAVR 的特殊考量：

  • Calcified raphe：增加 paravalvular regurgitation 與 annular rupture 風險
  • Leaflet asymmetry：影響 THV expansion 均勻度
  • Annulus eccentricity：BAV 通常比 tricuspid valve 更 elliptical
  • Aortopathy：需評估 ascending aorta 擴張情形
  • Coronary height：BAV 常伴隨異常冠狀動脈開口位置

5.2.1 4.2.1 BAV Sizing 策略

Sizing 方法 說明
傳統 Annulus-based 測量 virtual basal ring，與 tricuspid valve 相同；但可能低估 calcified raphe 的影響
Supra-annular sizing (LIRA method) 測量 Level of Implantation at the RAphe；適用於 raphe-type BAV 與 SEV
Inter-commissural distance 兩個 commissure 間的距離；是 PAR 的重要預測因子

Pearl — BAV TAVR 的臨床結果：研究顯示，使用新世代 THV 在 BAV 患者中可達到良好的 procedural success rate。然而，calcified raphe 合併 excessive leaflet calcium 的表型是最高風險群，與較高的 2 年 mortality 相關（約 3 倍風險）。此類患者需謹慎評估 TAVR vs SAVR 的選擇。

6 5. TAVR in SAVR (Valve-in-Valve)

Valve-in-Valve (ViV) TAVR 用於治療退化的 surgical bioprosthetic valve，其 CT 評估有特殊考量。

6.1 5.1 ViV TAVR 的關鍵評估項目

6.1.1 5.1.1 冠狀動脈阻塞風險 — VIVID Classification

VIVID 類型 描述與建議
Type I Leaflet 完全位於冠狀動脈 ostia 平面下方。低風險：可進行 conventional TAVR
Type IIA Leaflet 可能延伸至 ostium 部分高度，但 sinus 空間足夠。低風險：VTC ≥4 mm
Type IIB Leaflet 延伸至 ostium 高度，sinus 空間不足。高風險：VTC < 4 mm，建議 BASILICA
Type IIIA Leaflet 延伸至 STJ 附近或以上，VTC 與 VTSTJ 皆足夠。低風險：VTC ≥4 mm 且 VTSTJ ≥3.5 mm
Type IIIB Leaflet 延伸至 STJ 附近或以上，VTC 不足。高風險：VTC < 4 mm，建議 BASILICA
Type IIIC Leaflet 延伸至 STJ 附近或以上，VTC 足夠但 VTSTJ 不足。高風險：VTSTJ < 3.5 mm，建議 BASILICA

6.1.2 5.1.2 BASILICA 技術

BASILICA (Bioprosthetic or native Aortic Scallop Intentional Laceration to prevent Iatrogenic Coronary Artery obstruction) 是一種預防性冠狀動脈保護技術：

  • 使用電氣化導絲穿刺並撕裂可能阻塞冠狀動脈的 leaflet
  • 創造三角形空隙允許血流進入冠狀動脈
  • 成功率約 87-93%
  • 可使實際冠狀動脈事件風險降低約 29%

6.1.3 5.1.3 其他 ViV 特殊考量

評估項目 說明
Surgical valve ID（內徑） 需參考 reference chart 查詢特定 surgical valve 的 true ID；不同廠牌標示尺寸與實際內徑可能不同
Stent frame type Stented vs Stentless valve；Stentless valve 類似 native TAVR 評估
Leaflet mounting Internal vs External mounting；External mounting（如 Mitroflow, Trifecta）冠狀動脈阻塞風險較高
Patient-prosthesis mismatch (PPM) 可能需要 bioprosthetic valve fracture 以增加有效瓣口面積

Pearl — ViV TAVR 的 valve 選擇：ViV TAVR 傾向使用 self-expanding valve (SEV)，因為：(1) SEV 的 supra-annular 設計可提供較大的有效瓣口面積；(2) 較低的冠狀動脈阻塞風險；(3) 可 reposition 調整位置。若使用 BEV，需特別注意 depth of implantation，避免過深導致 valve opening 受限。

7 6. 血管通路評估

7.1 6.1 Iliofemoral Access 評估

評估項目 建議標準
最小血管直徑 ≥5.0-5.5 mm（依 THV delivery system 需求）
鈣化程度 環狀鈣化 (circumferential calcification) 增加血管併發症風險
Tortuosity 嚴重彎曲影響導管操控與 THV delivery
Aneurysm 髂動脈或股動脈瘤需特別評估
前次手術 Bypass graft、stent 等既往介入史

7.2 6.2 Alternative Access 評估

若 transfemoral approach 不適合，需評估 alternative access：

  • Transaxillary/Subclavian：需評估 subclavian artery 直徑與鈣化
  • Transcarotid：需評估 common carotid artery
  • Transapical：評估 LV apex 位置與心尖壁厚度
  • Direct aortic：評估 ascending aorta 鈣化與手術 approach

8 7. Post-TAVR CT 評估

Post-TAVR CT 用於評估 valve 功能、leaflet 形態、position 及是否有併發症。

8.1 7.1 Hypoattenuated Leaflet Thickening (HALT)

8.1.1 7.1.1 定義與診斷

HALT 是 subclinical leaflet thrombosis 的 CT 表現，定義為：

  • 在舒張期 CT 影像上，valve leaflet 基部出現 hypoattenuating opacity
  • 需與 leaflet coaptation 後的正常影像鑑別
  • Hypoattenuation Affecting Motion (HAM)：HALT 合併 leaflet motion 減少

8.1.2 7.1.2 HALT 的臨床意義

項目 說明
發生率 術後 30 天約 10-16%；1 年後約 24-30%
自然病程 Dynamic finding：約 50% 可自發緩解，但也有新發病例
臨床影響 部分研究顯示與較高 mean gradient 相關；長期追蹤顯示可能增加 symptomatic valve deterioration 風險
治療 Warfarin anticoagulation 可使 HALT 消退；但 routine prophylactic anticoagulation 仍有爭議

8.1.3 7.1.3 HALT 的預測因子

  • Valve-in-valve procedure
  • Larger prosthesis size
  • Balloon-expandable valve（相較於 self-expanding）
  • Single antiplatelet therapy (vs DAPT)
  • Intra-annular design
  • Obesity

8.2 7.2 Pannus 與 Thrombus 鑑別

Pannus vs Thrombus/HALT 鑑別： - Pannus：位於 leaflet 下方 (below leaflet)，通常呈 fibrous tissue density - Thrombus/HALT：位於 leaflet 上方 (aortic side)，呈 hypoattenuating appearance - Pannus 通常發展較慢，與 chronic inflammation 相關

8.3 7.3 THV Deformation 與 Under-expansion

發現 臨床意義
Elliptical deformation 常見於 calcified native valve 或 SAVR ViV；可能增加 PAR 風險
Under-expansion Non-fracturable surgical valve 限制 THV expansion；影響 hemodynamics 與 durability
Pin-wheeling Too-deep deployment 導致；Leaflet opening 延遲、有效口徑縮小
Frame fracture Post BVF (bioprosthetic valve fracture) 後的預期發現；需確認 expansion 是否改善

8.4 7.4 其他 Post-TAVR CT 評估項目

  • Paravalvular regurgitation (PVR)：評估 leak 位置與嚴重度
  • THV position：評估 implantation depth 與 canting
  • Coronary access：THV commissure 位置與未來 coronary intervention 可行性
  • Aortic root integrity：排除 contained rupture 或 dissection

10.2 9.1 考題詳解

  1. 正確答案：b) 30-45% R-R interval — 最大 annulus 面積通常出現在收縮中期。若最大面積出現在舒張期，可能提示 septal hypertrophy 或 diastolic dysfunction。

  2. 正確答案：b) Annular area — BEV 主要依據 annular area 進行 sizing，而 SEV 則主要依據 perimeter。

  3. 正確答案：c) SOV 直徑 ≥30 mm，冠狀動脈高度 ≥12 mm — 這些為低風險參考值，但需結合 annulus 尺寸與 THV 選擇綜合判斷。

  4. 正確答案：b) < 3 mm — VTC 距離 < 3 mm 為高風險，強烈建議進行 BASILICA 或其他冠狀動脈保護措施。

  5. 正確答案：b) 顯著增加 BEV annular rupture 風險 — LVOT 鈣化（尤其是 nodular、asymmetric calcification）是 BEV annular rupture 的重要預測因子。

  6. 正確答案：b) 一個 raphe 連接兩個 cusps — Type 0 無 raphe；Type 1 有一個 raphe，是西方族群最常見的類型；Type 2 有兩個 raphes。

  7. 正確答案：c) Type IIB — Type IIB（VTC < 4 mm）、Type IIIB、Type IIIC 被認為是高風險 anatomy，建議考慮 BASILICA。

  8. 正確答案：c) 是一種 dynamic finding，約 50% 可自發緩解 — HALT 應在舒張期評估；術後 30 天發生率約 10-16%；Pannus 位於 leaflet 下方，HALT 位於上方。

  9. 正確答案：b) Self-expanding valve — SEV 的 supra-annular 設計可提供較大的有效瓣口面積，且較低冠狀動脈阻塞風險。

  10. 正確答案：c) ≥5.0-5.5 mm — 依 THV delivery system 的需求而定，還需評估鈣化程度與 tortuosity。

11 10. 結語

Pre-TAVR CT 評估是確保 TAVR 成功的關鍵步驟。系統性的 CT 分析涵蓋了 aortic root anatomy、coronary obstruction risk、vascular access 等多個面向。隨著 TAVR 適應症擴展至低風險與年輕患者，以及 bicuspid aortic valve 等複雜 anatomy 的處理需求增加，精確的 CT 評估更顯重要。

關鍵學習要點： 1. Annulus 測量是 THV sizing 的基礎，需掌握 area-based 與 perimeter-based sizing 的適用時機 2. 冠狀動脈阻塞風險評估需綜合考量 coronary height、SOV dimensions、VTC 與 VTSTJ 距離 3. Bicuspid aortic valve 需特殊 sizing 策略，calcified raphe 是重要的風險因子 4. Valve-in-Valve TAVR 的 CT 評估需熟悉 VIVID classification 與 BASILICA 適應症 5. Post-TAVR CT 可評估 HALT、valve position 與其他併發症

謹記：TAVR 的成功建立在精確的術前規劃之上，CT 影像評估是 Heart Team 決策不可或缺的一環。
