# 2026 ACC/AHA/ASE/HRS/STS Expert Consensus
## [O'Gara PT et al., *JACC* 2026](https://doi.org/10.1016/j.jacc.2026.02.5069)

**Tricuspid Interventions 的 Operator 與 Institutional Requirements**

- **Operator**：結構性心臟病訓練 + proctored T-TEER cases + 年度最低量
- **Institution**：Heart team、TEE imaging、心臟外科備援
- **Quality**：Procedural success 定義、追蹤架構、Registry 參與

> **意義**：T-TEER 進入規範化時代，亞洲地區 program development 重要參考

- M-TEER 在 **secondary MR** 獲 Class I 推薦
  - 受 RESHAPE-HF2 + MATTERHORN 影響

- 多學科 Heart Team 決策角色強化

# RESHAPE-HF2：打破僵局或混淆視聽？
## [Kaul S, *JACC Cardiovasc Interv* 2026](https://doi.org/10.1016/j.jcin.2026.01.301)

**核心問題**：
- RESHAPE-HF2 是否解決 **COAPT vs MITRA-FR** 爭議？
- Secondary MR 中 M-TEER 適應症的精準定義
- **GDMT 優化 vs 早期介入**的時機抉擇

**RESHAPE-HF2 + MATTERHORN 影響**（[Iliadis C, *Herz* 2026](https://doi.org/10.1007/s00059-026-05372-y)）：
- 歐洲指引更新的主要驅動力
- SMR 治療策略從「是否介入」轉向「何時介入」

---

# M-TEER 在極高齡病患
## [Price MJ et al., *JAHA* 2026](https://doi.org/10.1161/JAHA.125.044794)

**≥80 歲 Primary MR 病患接受 contemporary M-TEER**

- MR reduction 與年輕病患 **相當**
- NYHA class、6MWD、KCCQ 均顯著改善
- 高齡本身不應為 M-TEER 禁忌

> **Pearl**：重點在 frailty 評估，而非 chronological age

---

# M-TEER 在 Advanced Heart Failure
## OCEAN-Mitral Registry & Single-Center Data

**OCEAN-Mitral**（[Tanaka S et al., *JAHA* 2026](https://doi.org/10.1161/JAHA.125.046638)）：
- M-TEER 在 LVAD favorable candidates 中的良好短期結果
- 作為 advanced HF **bridge therapy** 的潛力

**Anatomical Determinants**（[Crivelaro PCF et al., *CCI* 2026](https://doi.org/10.1002/ccd.70574)）：
- Tenting height、coaptation depth 影響 procedural complexity
- Advanced HF 的解剖特徵常增加技術挑戰

---

# LA Strain：TEER 新預後指標
## [Shechter A et al., *EHJCI* 2026](https://doi.org/10.1093/ehjci/jeaf267)

**Degenerative MR 接受 TEER 的 LA Strain 分析**

- Pre-procedural **LA reservoir strain** 為獨立預後因子
- 可識別最能從 TEER 獲益的病患
- 建議納入術前標準評估

> **Pearl**：LA strain 提供超越 LA volume 的預後資訊

# EuroTR Risk Score：AI 驅動風險評估
## [Hausleiter J et al., *JACC Cardiovasc Interv* 2026](https://doi.org/10.1016/j.jcin.2025.11.034)

| 特徵 | 數值 |
|------|------|
| 研究人數 | **1,826** (derivation 1,225 + validation 601) |
| Algorithm | Extreme Gradient Boosting (XGBoost) |
| 參數 | 18 項 (clinical + lab + echo + hemodynamic) |
| **C-index** | **0.741** (validation) |
| Low-risk 不良事件率 | 30.6% |
| High-risk 不良事件率 | **85.5%** |

- 顯著優於 EuroScore 和 TRI-SCORE
- 首個 **AI-derived, externally validated** T-TEER 專用 risk score

# FATE Registry：T-TEER Device Failure
## [De Marco F et al., *JACC Cardiovasc Interv* 2026](https://doi.org/10.1016/j.jcin.2026.01.285)

**31 centers, 2,278 consecutive T-TEER procedures (2017–2024)**

**Failure 類型**：
- Loss of leaflet insertion
- Single-leaflet device attachment (SLDA)
- Device embolization

**首次系統性報告**：
- Device failure 的發生率與機轉
- Management 策略與 outcomes
- Bail-out 方案的成功率

# TriValve Registry：Bleeding 後果嚴重
## [Dykun I et al., *JACC Cardiovasc Interv* 2026;19(6):711-722](https://doi.org/10.1016/j.jcin.2026.01.288)

| 參數 | 結果 |
|------|------|
| N | 440 |
| BARC Major Bleeding | **11.4%** |
| 院內死亡 adj OR | **106** |
| 1-yr 全因死亡 adj HR | **3.55** (1.75–7.21) |
| 1-yr CV 死亡 adj HR | **3.72** (1.62–8.52) |

**獨立預測因子**：Residual TR (OR 1.83)、sPAP (OR 1.61)、**Procedure duration** (OR 1.49)
口服抗凝藥 **非**獨立因子 (OR 1.51, P=0.30)

# TR Proportionality：新篩選概念
## [Russo G et al., *Circ Cardiovasc Interv* 2026](https://doi.org/10.1161/CIRCINTERVENTIONS.125.016269)

**首次將 Proportionality 概念應用於 TR**

- **Disproportionate** secondary TR：TR severity 超出預期
  - 術後改善 **更為顯著**
- 類似 COAPT 中 disproportionate MR 的成功經驗

> **Pearl**：Proportionality assessment 可能成為 T-TEER 病患篩選的新標準

# RV-Arterial Coupling 與 Liver Stiffness
## 新的 T-TEER 預後與監測工具

**RV-Arterial Coupling (VAC)**（Angulo Lara B et al., *EHJCI* 2026）：
- TAPSE/PASP ratio ≤0.39 → 較差預後 (p=0.038)
- 比傳統 RV function 參數 **更精準**

**Liver Stiffness**（Angulo Lara B et al., *EHJCI* 2026）：
- T-TEER 後肝纖維化顯著改善：**8.9 → 6.7 kPa** (p=0.013)
- Fibrosis stage **F3 → F1** (p=0.003)
- Elastography 比實驗室指標更敏感

> **Pearl**：VAC 和 liver elastography 應納入 T-TEER 術前/術後評估

# T-TEER 在亞洲與先天性心臟病

**亞洲經驗**（[Chan K et al., *JACC: Asia* 2026](https://pubmed.ncbi.nlm.nih.gov/41548860/)）：
- 香港單中心：TriClip + MitraClip "miskey" technique
- 亞洲地區寶貴的 real-world data

**先天性心臟病 TEER**（Papadopoulos G et al., *EHJ Suppl* 2026）：
- 4 cases：ccTGA、Mustard atrial switch、interrupted IVC
- **100% procedural success**
- 最長追蹤 **7 年**（longest FU of SAVV TEER in ACHD）
- Devices：PASCAL ACE (3) + TriClip (1)

# 總結：2026 年 TEER 趨勢

1. **規範化**：ACC/AHA Expert Consensus 建立 tricuspid intervention 標準
2. **精準化**：AI risk score (EuroTR)、proportionality、LA strain
3. **擴展化**：M-TEER 適應症持續拓展（HCM、acute MR、moderate MR）
4. **真實世界**：大型 Registry 數據（FATE、TriValve、EuroTR、OCEAN）
5. **併發症意識**：Bleeding (11.4%)、device failure、SORI 現象
6. **亞洲數據**：香港 T-TEER、papillary muscle rupture TEER
