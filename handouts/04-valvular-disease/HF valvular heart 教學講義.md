## 前言


瓣膜性心臟病是臨床常見的心血管疾病，包括瓣膜狹窄（stenosis）與瓣膜閉鎖不全（regurgitation）。本教材旨在協助心臟科護理師及非心臟專科內科醫師了解瓣膜性心臟病的基本概念、診斷與處置原則。


## 心臟瓣膜解剖與功能


### 四大瓣膜


心臟有四個主要瓣膜，分別位於心房與心室、心室與大血管之間：


| llll@} 心臟瓣膜概覽 |  |
| --- | --- |
| **瓣膜** | **英文名稱** |
| 二尖瓣 | Mitral valve |
| 主動脈瓣 | Aortic valve |
| 三尖瓣 | Tricuspid valve |
| 肺動脈瓣 | Pulmonary valve |


### 瓣膜功能


- **房室瓣**（二尖瓣、三尖瓣）：心室收縮時關閉，防止血液逆流回心房
- **半月瓣**（主動脈瓣、肺動脈瓣）：心室舒張時關閉，防止血液逆流回心室


## 主動脈瓣疾病


### 主動脈瓣狹窄（Aortic Stenosis, AS）


#### 病因


| ll@} 主動脈瓣狹窄常見病因 |  |
| --- | --- |
| **病因** | **特點** |
| 退化性鈣化（Degenerative calcification） | 最常見，好發於老年人 |
| 先天性雙葉瓣膜（Bicuspid aortic valve） | 提早發生狹窄，約50-60歲 |
| 風濕性心臟病（Rheumatic heart disease） | 較少見，常合併二尖瓣病變 |


#### 臨床表現——經典三症狀


importantbox
**主動脈瓣狹窄三大症狀（出現代表預後不佳）**

1. **心絞痛**（Angina）：心肌需氧量增加但供應不足
2. **暈厥**（Syncope）：運動時腦部灌流不足
3. **心衰竭**（Heart failure）：左心室功能衰退


importantbox


#### 身體檢查


- **心雜音**：收縮期噴射性雜音（Systolic ejection murmur），於右上胸骨緣最響
- **傳導**：雜音傳至頸動脈
- **脈搏**：脈搏緩慢上升且振幅小（Pulsus parvus et tardus）
- **心尖搏動**：左心室肥厚導致心尖搏動增強且持續時間延長


#### 嚴重度分類


| llll@} 主動脈瓣狹窄嚴重度（Echocardiography評估） |  |
| --- | --- |
| **參數** | **輕度** |
| 瓣膜面積（Valve area） | $>$ 1.5 cm² |
| 平均壓力差（Mean gradient） | $<$ 25 mmHg |
| 最大流速（Peak velocity） | $<$ 3.0 m/s |


### 主動脈瓣閉鎖不全（Aortic Regurgitation, AR）


#### 病因


- **急性**：感染性心內膜炎（Infective endocarditis）、主動脈剝離（Aortic dissection）、外傷
- **慢性**：二葉主動脈瓣、風濕性心臟病、主動脈根部擴張、結締組織疾病


#### 臨床表現


**慢性AR：**

- 初期可能無症狀
- 運動耐受力下降
- 心悸（尤其平躺時）
- 後期出現心衰竭症狀


**急性AR：**

- 急性肺水腫
- 心因性休克
- 為內科急症，需緊急處理


#### 身體檢查——經典徵象


| ll@} 主動脈瓣閉鎖不全之身體檢查發現 |  |
| --- | --- |
| **徵象名稱** | **表現** |
| 心雜音 | 舒張早期高音調遞減性雜音（左胸骨緣） |
| 脈壓增大 | 收縮壓升高，舒張壓降低 |
| 水錘脈（Water-hammer pulse） | 脈搏快速上升又快速下降 |
| Hill's sign | 下肢血壓較上肢高 $>$ 20 mmHg |
| Quincke's sign | 甲床微血管搏動 |
| Duroziez's sign | 股動脈加壓時可聞雙期雜音 |


## 二尖瓣疾病


### 二尖瓣狹窄（Mitral Stenosis, MS）


#### 病因


- **風濕性心臟病**：最主要原因（約90
- 先天性
- 二尖瓣環鈣化（老年人）


風濕性心臟病是因A群β溶血性鏈球菌（Group A β-hemolytic streptococcus）感染後的自體免疫反應。台灣雖然已少見，但在開發中國家仍是重要病因。


#### 臨床表現


- **呼吸困難**：勞動性呼吸困難、端坐呼吸、陣發性夜間呼吸困難
- **血痰**（Hemoptysis）：肺靜脈壓力升高導致
- **心房顫動**（Atrial fibrillation）：左心房擴大所致
- **栓塞事件**：血栓形成風險增加


#### 身體檢查


- **心雜音**：舒張期隆隆聲（Diastolic rumbling murmur），心尖處最明顯
- **Opening snap**：二尖瓣開放時的彈響聲
- **S1增強**：第一心音增強
- **二尖瓣面容**（Mitral facies）：雙頰潮紅（因周邊血管收縮）


#### 嚴重度分類


| llll@} 二尖瓣狹窄嚴重度 |  |
| --- | --- |
| **參數** | **輕度** |
| 瓣膜面積（Valve area） | $>$ 1.5 cm² |
| 平均壓力差（Mean gradient） | $<$ 5 mmHg |
| 肺動脈壓（PA systolic pressure） | $<$ 30 mmHg |


### 二尖瓣閉鎖不全（Mitral Regurgitation, MR）


#### 病因


| p{8cm}} 二尖瓣閉鎖不全病因分類 |  |  |
| --- | --- | --- |
| **類型** | **原因** |  |
| 原發性（Primary） | 二尖瓣脫垂（Mitral valve prolapse, MVP）    風濕性心臟病    感染性心內膜炎    腱索斷裂（Chordae rupture） |  |
| 續發性（Secondary/Functional） | 缺血性心臟病導致乳頭肌功能異常    擴張性心肌病變導致瓣環擴張 |  |


#### 臨床表現


**慢性MR：**

- 長期可能無症狀（代償期）
- 漸進性運動耐受力下降
- 心房顫動
- 最終出現心衰竭症狀


**急性MR：**

- 急性肺水腫
- 心因性休克
- 常見原因：乳頭肌斷裂（心肌梗塞後）、腱索斷裂、感染性心內膜炎


#### 身體檢查


- **心雜音**：全收縮期高音調雜音（Holosystolic murmur），心尖處最明顯，傳至腋下
- **S3**：第三心音可能出現（快速充盈音）
- **心尖搏動**：向外下移位且增強


### 二尖瓣脫垂（Mitral Valve Prolapse, MVP）


#### 特點


- 最常見的瓣膜異常，盛行率約2-3
- 好發於年輕女性
- 多數為良性，預後良好
- 與結締組織疾病相關（如Marfan syndrome）


#### 身體檢查


- **收縮中期喀答聲**（Mid-systolic click）
- **收縮晚期雜音**（Late systolic murmur）
- 站立或Valsalva maneuver使喀答聲提前
- 蹲下使喀答聲延後


## 右側心臟瓣膜疾病


### 三尖瓣閉鎖不全（Tricuspid Regurgitation, TR）


#### 病因


- **功能性**：最常見，因右心室擴張或肺高壓導致瓣環擴張
- **器質性**：風濕性心臟病、感染性心內膜炎、心臟節律器導線、類癌症候群（Carcinoid syndrome）


#### 臨床表現


- 周邊水腫
- 腹水
- 肝腫大、搏動性肝腫大
- 頸靜脈怒張且明顯搏動


#### 身體檢查


- **心雜音**：全收縮期雜音，胸骨左下緣
- **吸氣時加強**（Carvallo's sign）
- 頸靜脈可見明顯V波


### 肺動脈瓣狹窄（Pulmonary Stenosis, PS）


#### 特點


- 多為先天性
- 成人後才發現者通常為輕度
- 常與其他先天性心臟病合併（如Fallot氏四重症）


## 診斷工具


### 心電圖（Electrocardiogram, ECG）


| ll@} 瓣膜疾病之ECG變化 |  |
| --- | --- |
| **瓣膜疾病** | **ECG表現** |
| 主動脈瓣狹窄 | 左心室肥厚（LVH）、左心軸偏移 |
| 主動脈瓣閉鎖不全 | LVH、可能有ST-T變化 |
| 二尖瓣狹窄 | P wave呈雙峰（P mitrale）、右心室肥厚、心房顫動 |
| 二尖瓣閉鎖不全 | 左心房擴大、LVH、心房顫動 |


### 胸部X光（Chest X-ray, CXR）


可評估：

- 心臟大小與輪廓
- 肺血管充血情形
- 特定心房心室擴大
- 瓣膜鈣化（透視下較明顯）


### 心臟超音波（Echocardiography）


Echocardiography是診斷瓣膜疾病最重要的工具！

- **經胸超音波**（Transthoracic echocardiography, TTE）：首選檢查
- **經食道超音波**（Transesophageal echocardiography, TEE）：影像品質更佳，特別適合評估二尖瓣及感染性心內膜炎
- **壓力超音波**（Stress echocardiography）：評估嚴重無症狀AS患者


### 心導管檢查（Cardiac catheterization）


適應症：

- 超音波與臨床表現不符時
- 術前評估冠狀動脈疾病
- 侵入性血行動力學評估

### 藥物治療


| lll@} 瓣膜疾病藥物治療原則 |  |
| --- | --- |
| **瓣膜疾病** | **藥物** |
| AS | **避免Vasodilators** |
| AR（慢性） | ACE inhibitors/ARBs |

**重要提醒：**

- 主動脈瓣狹窄患者****避免****使用血管擴張劑（Vasodilators），可能導致嚴重低血壓
- 藥物僅能緩解症狀，無法改變疾病進程
- 有症狀或嚴重瓣膜疾病需考慮手術治療

### 外科手術


#### 手術適應症（簡化原則）


importantbox
**手術時機考量：**

1. **有症狀**的重度瓣膜疾病
2. **無症狀**但有：      - 左心室功能惡化（LVEF $<$ 50-60 - 左心室明顯擴大 - 肺高壓 - 心房顫動新發生
3. 需接受其他心臟手術時（如CABG）


importantbox


#### 手術方式


- **瓣膜修補**（Valve repair）：優先考慮，尤其二尖瓣
- **瓣膜置換**（Valve replacement）：     itemize
- 機械瓣膜（Mechanical valve）：耐用但需終身抗凝血
- 生物瓣膜（Bioprosthetic valve）：無需長期抗凝血但壽命較短

## 感染性心內膜炎預防


### 需要預防性抗生素的情況


importantbox
**高風險病患（需預防性抗生素）：**

1. 人工瓣膜（Prosthetic valve）
2. 曾有感染性心內膜炎病史
3. 先天性心臟病：      - 未矯正的發紺型先天性心臟病 - 手術後6個月內仍有殘餘缺損 - 修補處鄰近有人工材料
4. 心臟移植後瓣膜病變


importantbox


### 需要預防的處置


- 牙科：拔牙、牙周手術、根管治療等
- 呼吸道：扁桃腺或腺樣體手術
- **不需要**：一般腸胃道或泌尿道檢查


### 預防性抗生素處方


- **標準處方**：Amoxicillin 2g PO，處置前30-60分鐘
- **Penicillin過敏**：Clindamycin 600mg PO 或 Azithromycin 500mg PO


## 護理照護重點


### 評估重點


1. **症狀評估**：      - 呼吸困難程度（NYHA分級） - 運動耐受力變化 - 胸痛、暈厥、心悸
2. **生命徵象**：      - 血壓（注意脈壓） - 心跳（心律不整） - 呼吸速率與血氧
3. **身體檢查**：      - 心雜音特性變化 - 肺部囉音 - 周邊水腫 - 頸靜脈怒張
4. **功能狀態**：      - 日常活動能力 - 體重變化（水分滯留）


### 用藥護理


| lll@} 常用藥物護理注意事項 |  |
| --- | --- |
| **藥物類別** | **監測項目** |
| Diuretics | 體重、電解質    腎功能、尿量 |
| ACE inhibitors/ARBs | 血壓、腎功能    血鉀 |
| Beta-blockers | 心跳、血壓 |
| Warfarin | INR、出血徵象 |
| Digoxin | 心跳、血鉀    Digoxin濃度 |


### 術後照護重點


#### 瓣膜置換術後


1. **機械瓣膜**：      - 聽診瓣膜機械音是否正常 - 確保抗凝血藥物按時服用 - 教育INR監測重要性 - 避免服用影響凝血功能的藥物或食物
2. **生物瓣膜**：      - 術後3-6個月可能仍需抗凝血（尤其二尖瓣位置） - 監測瓣膜功能退化徵象
3. **所有人工瓣膜**：      - 預防感染性心內膜炎 - 定期追蹤Echocardiography - 觀察瓣膜功能異常徵象：新發心雜音、心衰竭惡化


### 病患衛教


importantbox
**重要衛教內容：**

1. **認識警示症狀**：      - 呼吸困難加劇 - 新發或加劇的胸痛 - 暈厥 - 體重快速增加（$>$ 2kg/3天） - 發燒（可能為心內膜炎）
2. **規律服藥**：      - 了解藥物名稱、劑量、作用 - 不可自行停藥 - 注意藥物副作用
3. **定期追蹤**：      - 門診複診 - 定期Echocardiography檢查
4. **生活型態調整**：      - 限制鈉攝取（心衰竭患者） - 適度運動（依醫師建議） - 避免劇烈運動（重度AS） - 戒菸
5. **牙科及其他侵入性處置**：      - 告知醫師有瓣膜疾病 - 必要時使用預防性抗生素

## 臨床案例討論


### 案例一：主動脈瓣狹窄


**病史：**75歲男性，近半年活動時出現胸悶及呼吸困難，上週爬樓梯時曾暈倒。

**身體檢查：**收縮期噴射性雜音於右上胸骨緣，傳至頸動脈，血壓110/80 mmHg。

**Echocardiography：**主動脈瓣嚴重鈣化，瓣膜面積0.8 cm²，平均壓力差45 mmHg，LVEF 55

**討論：**

- 診斷：症狀性重度主動脈瓣狹窄
- 已出現經典三症狀（胸痛、暈厥）
- 需緊急評估瓣膜置換術
- 考慮年齡，可評估TAVR適合性


### 案例二：急性二尖瓣閉鎖不全


**病史：**60歲男性，急性前壁心肌梗塞後第3天突然出現嚴重呼吸困難。

**身體檢查：**血壓90/60 mmHg，心跳120/分，呼吸急促，雙肺囉音，心尖處新出現全收縮期雜音。

**討論：**

- 懷疑：急性二尖瓣閉鎖不全（可能為乳頭肌斷裂）
- 需緊急Echocardiography確診
- 穩定血行動力學：Inotropic agents, Vasodilators（小心使用），IABP
- 緊急心臟外科會診

## 測驗題詳解


1. **答案：(B) 退化性鈣化**  **詳解：**主動脈瓣狹窄最常見的病因在不同年齡層有所不同。在老年人（$>$ 70歲），退化性鈣化（Degenerative/Senile calcification）是最主要原因，約占70
2. **答案：(C) 血痰（Hemoptysis）**  **詳解：**主動脈瓣狹窄的經典三症狀為：心絞痛（Angina）、暈厥（Syncope）、心衰竭（Heart failure），簡記為「ASF」。這三個症狀的出現代表病情嚴重且預後不佳。心絞痛是因心肌肥厚導致需氧量增加但供應不足；暈厥常發生於運動時，因心輸出量固定無法增加導致腦部灌流不足；心衰竭則是左心室長期後負荷過重導致功能衰退。血痰（Hemoptysis）是二尖瓣狹窄的表現，因肺靜脈壓力升高所致。
3. **答案：(B) 右上胸骨緣**  **詳解：**主動脈瓣狹窄的收縮期噴射性雞音（Systolic ejection murmur）在右上胸骨緣第2肋間（主動脈瓣區）最為明顯，且會傳導至頸動脈。這是因為主動脈瓣位於右上方，血流通過狹窄的瓣膜時產生湍流。雜音在收縮中期最強，呈現漸強-漸弱的特性（Crescendo-decrescendo）。嚴重狹窄時，雜音可能反而減弱（因心輸出量下降）。
4. **答案：(C) $<$ 1.0 cm²**  **詳解：**根據超音波評估，主動脈瓣狹窄的嚴重度分類如下：  - 輕度：瓣膜面積 $>$ 1.5 cm² - 中度：瓣膜面積 1.0-1.5 cm² - 重度：瓣膜面積 $<$ 1.0 cm²（正常約3.0-4.0 cm²）   其他參考指標包括平均壓力差（重度 $>$ 40 mmHg）和最大流速（重度 $>$ 4.0 m/s）。當瓣膜面積 $<$ 1.0 cm²且有症狀時，強烈建議進行瓣膜置換。
5. **答案：(B) Water-hammer pulse（水錘脈）**  **詳解：**主動脈瓣閉鎖不全有多個特徵性身體檢查發現：  - Water-hammer pulse（Corrigan's pulse）：脈搏快速上升又快速下降 - 脈壓增大（Wide pulse pressure）：收縮壓升高，舒張壓降低 - Hill's sign：下肢血壓較上肢高 $>$ 20 mmHg - Quincke's sign：甲床微血管搏動 - Duroziez's sign：股動脈加壓時可聞雙期雜音   Opening snap是二尖瓣狹窄的特徵。Pulsus paradoxus見於心包填塞。
6. **答案：(B) 風濕性心臟病**  **詳解：**二尖瓣狹窄約90
7. **答案：(B) 周邊血管收縮**  **詳解：**二尖瓣面容（Mitral facies）的特徵是雙頰潮紅，呈現類似「蝴蝶斑」的外觀。這是因為慢性低心輸出量狀態導致周邊血管代償性收縮，使得面部血管擴張。雖然這是二尖瓣狹窄的經典表徵，但在現代因早期診斷治療，已較少見到。這與中央發紺（如發紺型先天性心臟病）或肝功能異常引起的黃疸不同。
8. **答案：(B) 舒張期隆隆聲**  **詳解：**二尖瓣狹窄的心雜音為舒張期隆隆聲（Diastolic rumbling murmur），在心尖處最明顯，不會傳導。雜音產生是因血流通過狹窄的二尖瓣時產生湍流。特徵包括：  - 低頻率（需用聽診器的鐘型面） - 左側臥位最明顯 - 運動後加強 - 常伴隨Opening snap（瓣膜開放時的彈響聲） - S1（第一心音）增強
9. **答案：(C) Opening snap**  **詳解：**二尖瓣脫垂（MVP）的典型身體檢查發現包括：  - 收縮中期喀答聲（Mid-systolic click）：瓣葉脫垂時產生 - 收縮晚期雜音（Late systolic murmur）：若有閉鎖不全 - 姿勢改變的影響：     itemize - 站立或Valsalva maneuver：左心室容積減少，喀答聲提前 - 蹲下：左心室容積增加，喀答聲延後   itemize Opening snap是二尖瓣狹窄的特徵，與MVP無關。
10. **答案：(B) 全收縮期高音調雜音，傳至腋下**  **詳解：**慢性二尖瓣閉鎖不全的心雜音特性為：  - 全收縮期（Holosystolic/Pansystolic）高音調雜音 - 心尖處最明顯 - 傳導至腋下或左肩胛下區 - 音調恆定，不隨收縮期變化 - 可能伴隨S3（第三心音，快速充盈音）   急性MR的雜音可能較短且軟，因左心房順應性尚未增加。
11. **答案：(A) 吸氣時（Carvallo's sign）**  **詳解：**三尖瓣閉鎖不全的雜音特性：  - 全收縮期雜音 - 胸骨左下緣最明顯 - **吸氣時加強**（Carvallo's sign）：這是診斷右側心臟雜音的重要特徵   原理：吸氣時靜脈回流增加，右心室充盈增加，通過三尖瓣的血流量增加，雜音因此加強。這與左側心臟的雜音相反（通常吸氣時減弱）。其他右側心臟病變的雜音也會有類似現象。
12. **答案：(C) 二尖瓣狹窄**  **詳解：**P mitrale是指心電圖上P波呈現雙峰（M型），兩個峰之間間隔 $>$ 0.04秒，總寬度 $>$ 0.12秒，最常見於導程II。這代表左心房擴大，左心房去極化時間延長。二尖瓣狹窄因左心房壓力長期升高導致左心房擴大，因此最常出現P mitrale。相對地，肺心症或右側心臟病變會出現「P pulmonale」（P波高聳，代表右心房擴大）。
13. **答案：(C) 心臟超音波（Echocardiography）**  **詳解：**Echocardiography是診斷瓣膜性心臟病最重要的工具，因為它可以：  - 直接視覺化瓣膜構造 - 評估瓣膜開合功能 - 測量瓣膜面積、壓力差、流速 - 評估心室功能（LVEF） - 估計肺動脈壓力 - 評估其他心臟構造   經胸超音波（TTE）是首選，經食道超音波（TEE）影像品質更佳，特別適合評估二尖瓣及診斷感染性心內膜炎。ECG和CXR提供輔助資訊但無法確診。心導管檢查為侵入性，僅在特殊情況下使用。
14. **答案：(C) Vasodilators（血管擴張劑）**  **詳解：**嚴重主動脈瓣狹窄患者應**避免使用Vasodilators**，原因包括：  - AS患者心輸出量固定（無法透過增加來代償） - 血管擴張會降低全身血管阻力 - 導致嚴重低血壓和器官灌流不足 - 可能引發心因性休克   其他注意事項：  - Diuretics需小心使用（避免前負荷過度降低） - Beta-blockers和Digoxin可在心衰竭時小心使用 - **唯一根本治療是瓣膜置換**
15. **答案：(B) 二尖瓣狹窄**  **詳解：**經皮二尖瓣整形術（Percutaneous transvenous mitral commissurotomy, PTMC；或稱Percutaneous balloon mitral valvuloplasty, PBMV）是治療二尖瓣狹窄的有效方法。適應症：  - 症狀性中度至重度二尖瓣狹窄 - 瓣膜形態適合（柔軟、鈣化少、腱索未嚴重纖維化） - 無或僅有輕度二尖瓣閉鎖不全 - 無左心房血栓   優點：成功率高（90
16. **答案：(C) 人工瓣膜置換術後**  **詳解：**根據AHA/ACC指引，需要預防性抗生素的**高風險**情況包括：  - 人工瓣膜（Prosthetic valve） - 曾有感染性心內膜炎病史 - 特定先天性心臟病 - 心臟移植後出現瓣膜病變   **不需要**預防性抗生素的情況：  - 單純二尖瓣脫垂（無閉鎖不全） - 輕度瓣膜狹窄或閉鎖不全 - 心房中膈缺損（ASD）修補術後 - 冠狀動脈繞道手術後
17. **答案：(B) Amoxicillin 2g PO，處置前30-60分鐘**  **詳解：**感染性心內膜炎預防性抗生素處方：  - **標準處方**：Amoxicillin 2g PO，處置前30-60分鐘 - **無法口服**：Ampicillin 2g IV 或 IM - **Penicillin過敏**：     itemize - Clindamycin 600mg PO - Azithromycin 500mg PO - Cephalexin 2g PO（非嚴重過敏者）   itemize 注意：僅需單次給藥，不需後續劑量。
18. **答案：(B) 2.0-3.0**  **詳解：**機械瓣膜置換術後需終身抗凝血治療，使用Warfarin的目標INR：  - **主動脈瓣位置**：INR 2.0-3.0 - **二尖瓣位置**：INR 2.5-3.5（因血栓風險較高） - **合併其他危險因子**（如心房顫動、左心房血栓、栓塞病史）：可能需要更高目標值   過度抗凝血（INR $>$ 4.0）會增加出血風險；抗凝血不足則增加血栓及栓塞風險。需定期監測INR，通常每4-6週一次（穩定後）。

## 結語


瓣膜性心臟病是複雜但重要的心血管疾病。作為心臟科護理師或內科醫師，了解基本的病理生理、臨床表現、診斷與治療原則，對於提供優質照護至關重要。定期複習相關知識，並在臨床實務中不斷精進，將有助於改善病患預後。
