\section*{\textcolor{emergency_red}{\Large 一、認識您的疾病}}
\addcontentsline{toc}{section}{一、認識您的疾病}

\begin{infobox}[什麼是主動脈瓣膜狹窄？]
主動脈瓣膜位於心臟與主動脈之間，就像一扇「門」，讓血液從心臟流向全身。

當這扇「門」因為老化、�ite化而變得狹窄、打不開時，心臟必須更用力才能把血液送出去，這就是\textbf{「主動脈瓣膜狹窄」}。
\end{infobox}

\section*{\textcolor{emergency_red}{\Large 二、治療方式介紹}}
\addcontentsline{toc}{section}{二、治療方式介紹}

\begin{center}
{\large 目前有\textbf{兩種}主要的治療方式可以選擇：}
\end{center}

\vspace{0.5cm}

%--- TAVR ---
\begin{tcolorbox}[
    colback=blue!5,
    colframe=emergency_blue,
    title={\Large\bfseries 選項 A：經導管主動脈瓣膜置換術 (TAVR)},
    fonttitle=\bfseries\color{white},
    coltitle=white,
    colbacktitle=emergency_blue,
    arc=10pt,
    boxrule=2pt
]

\textbf{\large 什麼是 TAVR？}\\[0.3cm]
醫師從您的\textbf{大腿血管}（鼠蹊部）放入一根細管（導管），將新的人工瓣膜送到心臟，撐開並取代原本狹窄的瓣膜。

\vspace{0.5cm}

\begin{minipage}[t]{0.48\textwidth}
\textbf{\textcolor{pda_green}{優點：}}
\begin{itemize}[leftmargin=*, itemsep=3pt]
    \item 不需要開胸手術
    \item 不需要體外循環（心肺機）
    \item 傷口小（約 0.5-1 公分）
    \item 恢復較快（通常 3-5 天可出院）
    \item 疼痛較少
\end{itemize}
\end{minipage}
\hfill
\begin{minipage}[t]{0.48\textwidth}
\textbf{\textcolor{emergency_red}{需要考慮的事：}}
\begin{itemize}[leftmargin=*, itemsep=3pt]
    \item 人工瓣膜的長期耐用度資料較少（目前約 7-10 年數據）
    \item 可能需要裝置永久心律調節器的機率較高
    \item 血管太小或太彎曲可能無法進行
\end{itemize}
\end{minipage}

\end{tcolorbox}

\vspace{0.8cm}

%--- SAVR ---
\begin{tcolorbox}[
    colback=green!5,
    colframe=pda_green,
    title={\Large\bfseries 選項 B：外科主動脈瓣膜置換術 (SAVR)},
    fonttitle=\bfseries\color{white},
    coltitle=white,
    colbacktitle=pda_green,
    arc=10pt,
    boxrule=2pt
]

\textbf{\large 什麼是 SAVR？}\\[0.3cm]
心臟外科醫師會在您的胸口做一個切口，打開胸骨，在心臟停止跳動的狀態下（使用心肺機），將舊的瓣膜切除，縫上新的人工瓣膜。

\vspace{0.5cm}

\begin{minipage}[t]{0.48\textwidth}
\textbf{\textcolor{pda_green}{優點：}}
\begin{itemize}[leftmargin=*, itemsep=3pt]
    \item 長期耐用度已被證實（15-20 年）
    \item 可以同時處理其他心臟問題（如冠狀動脈繞道）
    \item 可以做瓣膜環擴大手術
    \item 未來如果瓣膜退化，還可以選擇 TAVR
\end{itemize}
\end{minipage}
\hfill
\begin{minipage}[t]{0.48\textwidth}
\textbf{\textcolor{emergency_red}{需要考慮的事：}}
\begin{itemize}[leftmargin=*, itemsep=3pt]
    \item 需要全身麻醉、開胸手術
    \item 傷口較大（約 15-20 公分）
    \item 恢復時間較長（住院約 7-10 天）
    \item 術後疼痛較明顯
    \item 需要 2-3 個月才能完全恢復
\end{itemize}
\end{minipage}

\end{tcolorbox}

    title={\Large\bfseries\color{white} 您需要記住的重點}, colbacktitle=pda_orange]

\begin{enumerate}[leftmargin=*, itemsep=12pt, label=\textcolor{emergency_red}{\arabic*.}]

    \item \textbf{嚴重主動脈瓣膜狹窄需要治療}\\
    藥物無法治癒瓣膜狹窄，只有換瓣膜才能根本解決問題。

    \item \textbf{兩種治療方式各有優缺點}
    \begin{itemize}[leftmargin=*]
        \item \textbf{TAVR}：微創、恢復快，但長期耐用度資料較少
        \item \textbf{SAVR}：開胸手術、恢復較慢，但長期耐用度已證實
    \end{itemize}

    \item \textbf{中期（5-7 年）結果兩者相似}\\
    大型臨床試驗顯示，兩種治療在中期的存活率和主要併發症相當。

    \item \textbf{選擇取決於您的個人情況}
    \begin{itemize}[leftmargin=*]
        \item 年齡與預期壽命
        \item 身體狀況與手術風險
        \item 血管與心臟的解剖構造
        \item 您的價值觀與偏好
    \end{itemize}

    \item \textbf{由心臟團隊評估，與您共同決定}\\
    我們的心臟內科、心臟外科、影像科醫師會一起討論，給您最適合的建議。

\end{enumerate}
