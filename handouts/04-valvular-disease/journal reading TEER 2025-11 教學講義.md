### 研究概述


這是首個評估SQ-Kyrin-M系統（新型TEER裝置）治療功能性二尖瓣逆流的多中心前瞻性研究，針對中國患者群體，為亞洲地區TEER治療提供重要循證依據。


#### 研究設計


- **研究類型：**多中心、前瞻性、單臂研究
- **研究對象：**125例重度症狀性功能性二尖瓣逆流患者
- **平均年齡：**65.5±8.1歲
- **入選條件：**     itemize[nosep]
- 心衰合併中重度或重度功能性MR
- 儘管接受最大劑量指導性藥物治療(GDMT)仍有症狀


    \item **主要終點：**1年全因死亡率與心衰住院複合事件
itemize


### 主要結果


| p{9cm}}  **結果指標** | **數據** |  |
| --- | --- | --- |
| 裝置成功率 | 良好的技術成功率 |  |
| 1年複合終點 | 全因死亡率與心衰住院複合事件達標 |  |
| 安全性 | 主要不良事件發生率低 |  |
| MR改善程度 | 顯著減少二尖瓣逆流 |  |
| 症狀改善 | NYHA功能分級顯著改善 |  |


### 臨床意義


1. ****亞洲數據：****首次提供中國患者群體使用SQ-Kyrin-M系統的循證數據
2. ****新型裝置：****驗證新一代TEER裝置的安全性與有效性
3. ****功能性MR：****擴展TEER在功能性二尖瓣逆流的應用證據
4. ****GDMT優化：****強調在最佳藥物治療基礎上的介入治療價值

### 文章概述


這篇綜述系統性回顧了TEER時代繼發性二尖瓣逆流的診斷挑戰，強調精準診斷與患者選擇對治療成功的關鍵性。


### 繼發性MR的分類


#### 三種主要類型


1. **非缺血性sMR：**      - 與擴張性心肌病相關 - 左心室整體性擴張 - 對稱性瓣環擴大
2. **缺血性sMR：**      - 心肌梗塞後 - 局部室壁運動異常 - 乳頭肌功能不良
3. **心房功能性sMR：**      - 左心房顯著擴大 - 瓣環擴張為主 - 左心室射出分率正常


### 主要臨床試驗回顧


| p{3cm}p{3cm}p{5.5cm}}  **試驗** | **EROA** | **LVEDV** |
| --- | --- | --- |
| COAPT | 較大 | 較小 |
| MITRA-FR | 較小 | 較大 |
| RESHAPE-HF2 | 中等 | 中等 |
| MATTERHORN | 中等 | 中等 |


### 診斷挑戰與解決方案


#### 關鍵診斷參數


1. **EROA測量：**      - 精確評估有效反流孔面積 - 考慮左心室大小的比例關係 - Modified Haller Index輔助評估
2. **反流分數(RF)：**      - RF≥50 - 中度MR（EROA 20-29 mm²）需評估RF - 影響TEER適應症判斷
3. **多瓣膜疾病：**      - 主動脈瓣狹窄合併MR - AVR可改善MR嚴重度 - 需評估patient-prosthesis mismatch

### 簡答題


**6. 比較二尖瓣TEER與三尖瓣TEER在亞洲地區發展的異同，並說明三尖瓣TEER面臨的特殊挑戰。**

**參考答案：**

- **相同點：**都採用經導管微創技術、需要多學科團隊、強調學習曲線重要性
- **不同點：**二尖瓣TEER已有成熟數據支持，三尖瓣TEER仍在累積經驗階段
- **三尖瓣特殊挑戰：**解剖結構更複雜、常合併心房顫動、患者多為高齡且心衰晚期、右心功能評估困難


**7. 根據繼發性MR診斷綜述，說明心房功能性sMR與心室功能性sMR的主要鑑別要點。**

**參考答案：**

- **心房功能性sMR：**左心房顯著擴大、瓣環擴張為主要機制、左心室EF正常或輕度下降、常合併心房顫動
- **心室功能性sMR：**左心室EF明顯下降、左心室整體或局部擴張、乳頭肌位移、瓣葉本身結構正常但tethering明顯
- **鑑別意義：**影響TEER適應症判斷與預後評估


**8. SQ-Kyrin-M系統研究的臨床意義為何？對未來TEER裝置發展有何啟示？**
