## 卵圓孔未閉概述


### 定義與流行病學


卵圓孔未閉(Patent Foramen Ovale, PFO)是心房中膈第一隔膜(septum primum)與第二隔膜(septum secundum)之間未完全融合所形成的潛在性通道。這是一種相當常見的解剖變異,在一般人群中的盛行率約為25


PFO並非真正的心房中膈缺損,而是胚胎發育過程中正常結構未完全閉合的結果。在正常情況下,出生後隨著左心房壓力增加,第一隔膜應該與第二隔膜融合閉合。


### 臨床意義


PFO的臨床重要性在於其可能導致以下病理狀況:


- **矛盾性栓塞 (Paradoxical Embolism)**: 當右心房壓力短暫性升高時,靜脈系統的血栓可能經由PFO進入左心房,導致腦中風、心肌梗塞或周邊動脈栓塞
- **偏頭痛**: 部分研究顯示PFO與伴隨視覺前兆的偏頭痛有關聯
- **減壓症 (Decompression Sickness)**: 潛水員因氮氣泡經PFO進入動脈循環
- **直立性呼吸困難症候群 (Platypnea-Orthodeoxia Syndrome)**: 站立時出現呼吸困難與血氧下降


### 風險因子評估


根據歐洲心臟學會立場聲明,以下特徵被認為是高風險PFO的指標:


| p{11cm}}  **特徵** | **說明** |  |
| --- | --- | --- |
| 大型PFO | 分流量大,靜止狀態即可偵測到大量右至左分流 |  |
| 心房中膈瘤 (ASA) | 心房中膈活動度≥10 mm,瘤體直徑≥15 mm |  |
| 隧道型PFO | 第一隔膜與第二隔膜之間的距離≥10 mm |  |
| Eustachian valve | 突出的Eustachian valve可將下腔靜脈血流導向PFO |  |
| Chiari network | 網狀結構可能影響血流方向 |  |
| 長度 | PFO通道的長度越長,解剖上的風險越高 |  |


## 心臟超音波診斷方法


### 診斷工具比較


PFO的診斷需要結合多種影像技術。目前並無單一黃金標準,各種方法各有優缺點:


| p{4cm}p{3.5cm}p{3.5cm}}  **診斷方法** | **優點** | **缺點** |
| --- | --- | --- |
| Contrast TTE   (c-TTE) | 非侵入性    可重複檢查    病人接受度高 | 影像品質受限    Valsalva困難    敏感度較低 |
| Contrast TOE   (c-TOE) | 最佳解剖視野    可評估結構特徵    可見分流本身 | 侵入性檢查    需麻醉或鎮靜    Valsalva執行困難 |
| Contrast TCD   (c-TCD) | 高敏感度    Valsalva易執行    可定量分流 | 無法視覺化解剖    無法排除肺內分流    操作者依賴性高 |
| 3D TOE | 精確解剖評估    多平面重組    手術規劃最佳 | 設備要求高    需專業訓練    時間較長 |


根據meta-analysis結果,c-TTE相較於c-TOE的診斷準確度較低(AUC 0.91),且研究間存在顯著異質性。因此在臨床實務上,通常需要合併使用不同技術以提高診斷準確度。


### 經胸超音波檢查 (Transthoracic Echocardiography, TTE)


#### 標準切面與掃描技術


進行PFO評估的TTE檢查應包含以下標準切面:


1. **心尖四腔室切面 (Apical 4-chamber view)**      - 最常用的評估切面 - 可清楚顯示心房中膈結構 - 適合進行對比劑注射評估
2. **心尖五腔室切面 (Apical 5-chamber view)**      - 有助於排除主動脈下緣的中膈缺損 - 評估主動脈緣(aortic rim)的存在
3. **心下四腔室切面 (Subcostal 4-chamber view)**      - 提供心房中膈的正面觀察 - 對心房中膈瘤的評估特別重要 - 可觀察中膈的活動幅度
4. **胸骨旁短軸切面 (Parasternal short-axis view)**      - 主動脈根部層面可評估中膈上緣 - 評估心房中膈的厚度


#### 對比劑微氣泡檢查 (Contrast Bubble Study)


對比劑微氣泡檢查是診斷PFO的關鍵技術:

準備與執行


| p{12cm}}  **步驟** | **詳細說明** |  |
| --- | --- | --- |
| 對比劑製備 | 混合9 mL生理食鹽水 + 1 mL空氣 + 0.5-1 mL病人自體血液    使用三向接頭連接兩支10 mL空針    快速來回推注至少10次,產生均勻微氣泡 |  |
| 注射途徑 | 建議使用18-20 gauge靜脈留置針    優先選擇右側手臂靜脈(避免經左無名靜脈的稀釋效應)    若使用左側,應考慮氣泡可能通過left innominate vein被稀釋 |  |
| 基礎注射 | 快速推注對比劑(建議<4秒)    隨即推注5-10 mL生理食鹽水沖洗    觀察心臟4-6個週期 |  |
| Valsalva操作 | 在注射對比劑前5-10秒開始執行Valsalva動作    要求病人用力閉氣,如同排便用力    維持此動作直到對比劑完全注射完畢    突然釋放Valsalva(此時右心房壓力會瞬間升高) |  |
| 觀察時機 | 持續觀察釋放Valsalva後的5-10個心跳週期    PFO通常在釋放Valsalva後的第2-6個心跳出現氣泡 |  |
| 重複檢查 | 建議至少進行2-3次完整的對比劑注射    包括靜止狀態與Valsalva操作    確保檢查的可靠性 |  |


判讀標準

右至左分流的分級系統(依據左心房出現的微氣泡數量):

importantbox
**PFO分流分級 (Bubble Grading System)**


- **Grade 0**: 左心房未出現微氣泡
- **Grade 1 (少量)**: 1-10個微氣泡
- **Grade 2 (中量)**: 11-30個微氣泡
- **Grade 3 (大量)**: >30個微氣泡或左心房完全充滿氣泡


**重要時間點**:

- PFO: 氣泡在右心房顯影後的3-5個心跳週期內出現於左心房
- 肺內分流: 氣泡在右心房顯影後的>5個心跳週期才出現於左心房


importantbox


Valsalva maneuver的品質是影響c-TTE敏感度的主要因素。研究顯示,無法正確執行Valsalva是導致false negative的主要原因。對於無法配合的病人,可考慮改用c-TCD或c-TOE。


### 經食道超音波檢查 (Transesophageal Echocardiography, TOE)


TOE提供最佳的心房中膈解剖視野,是PFO特徵評估的首選方法。


#### 標準切面


1. **Mid-esophageal 四腔室切面 (0-20°)**      - 評估心房中膈的整體結構 - 觀察心房中膈瘤的存在與活動度
2. **Mid-esophageal 雙腔室切面 (60-90°)**      - 最佳的PFO觀察角度 - 可直接觀察第一隔膜與第二隔膜之間的通道 - 評估PFO的長度與形態
3. **Mid-esophageal 主動脈短軸切面 (30-60°)**      - 評估主動脈緣(aortic rim) - 對於計劃關閉術的病人特別重要
4. **Mid-esophageal 上下腔靜脈長軸切面 (90-110°)**      - 評估Eustachian valve - 觀察下腔靜脈血流方向


#### 3D TOE評估


三維TOE提供PFO解剖結構的全面評估:


| p{11cm}}  **評估項目** | **3D TOE的優勢** |  |
| --- | --- | --- |
| PFO形態 | PFO實際上呈半月形或裂隙狀(slit-like),而非圓形通道    3D影像可完整呈現真實形態,2D切面可能低估其大小 |  |
| 多平面重組 | 可同時顯示多個角度的影像    有助於理解複雜的解剖關係    提供更準確的測量 |  |
| 手術規劃 | 精確測量PFO大小與形態    評估周圍組織的充足性(rim assessment)    預測關閉裝置的尺寸選擇 |  |
| 即時導引 | 可在經皮關閉術中提供即時3D影像    確認導管與裝置的正確位置    即時評估關閉效果 |  |


#### TOE評估的系統性報告參數


根據歐洲立場聲明,TOE報告應包含以下標準化參數:


| p{5cm}p{5cm}}  **參數** | **測量方法** | **臨床意義** |
| --- | --- | --- |
| PFO大小 | 2D最大開口直徑    彩色都卜勒分流束寬度 | 分流量的預測因子    裝置尺寸選擇依據 |
| PFO隧道長度 | septum primum與septum secundum間的距離    測量通道的長軸長度 | ≥10 mm定義為tunnel PFO    影響栓塞風險 |
| 心房中膈瘤(ASA) | 中膈基部寬度(≥15 mm)    最大活動幅度(≥10 mm) | 高風險PFO特徵    增加栓塞風險 |
| 分流程度 | 對比劑氣泡分級(0-3級)    彩色都卜勒定性評估 | Grade 2-3為顯著分流    影響治療決策 |
| 分流方向 | 彩色都卜勒觀察    記錄雙向或單向 | 雙向分流較常見    主要為L→R,Valsalva時R→L |
| Eustachian valve | 存在與否    突出程度 | 突出的瓣膜可引導IVC血流向PFO    增加矛盾性栓塞風險 |
| Chiari network | 存在與否    形態描述 | 網狀結構可能影響血流    需注意與ASA鑑別 |
| Rim評估 | 測量PFO周圍各方向的組織    特別注意aortic rim | Aortic rim <5 mm為deficient    影響裝置選擇與成功率 |
| 右心室擴大 | RV/LV比值    目視評估 | 評估分流的血流動力學影響    少見於單純PFO |
| 合併異常 | ASD、肺靜脈異常引流等 | 排除其他心房中膈病變    影響治療策略 |


系統性評估與標準化報告有助於跨專科團隊溝通(神經科、心臟內科、心臟外科),並為治療決策提供完整資訊。建議所有執行PFO-TOE的單位採用標準化的報告模板。


## 鑑別診斷


### PFO與其他心房中膈病變的鑑別


| p{3.5cm}p{4cm}p{4cm}}  **診斷** | **解剖特徵** | **超音波表現** |
| --- | --- | --- |
| PFO | 瓣膜狀結構    正常胚胎殘留 | 通道狹窄    通常需Valsalva才顯現 |
| Secundum ASD | 卵圓窩區域缺損    真正的組織缺損 | 缺損邊緣清楚    持續性分流 |
| Primum ASD | 靠近AV valve    胚胎墊缺損 | 位置較低    常合併MV/TV異常 |
| Sinus venosus ASD | 上腔或下腔靜脈附近    靜脈竇型缺損 | 位置特殊    常合併PAPVR |
| 心房中膈瘤(ASA) | 中膈過度活動    可合併或不合併PFO | 中膈擺動幅度≥10 mm    基部寬度≥15 mm |


### 肺內分流(Intrapulmonary Shunt)的排除


在對比劑檢查中,區分PFO與肺內分流相當重要:

importantbox
**時間差異法則**


- **心內分流(包括PFO)**: 微氣泡在右心房充滿後的**3-5個心跳週期內**出現於左心房     itemize
- 機制:直接經由心房間通道進入左心房,無需經過肺循環
- 特徵:氣泡出現早且密集


    
    \item **肺內分流**: 微氣泡在右心房充滿後的**>5個心跳週期**才出現於左心房
    
- 機制:經由肺部異常血管通道(如AVM、hepatopulmonary syndrome)
- 特徵:氣泡出現晚,通常在6-8個或更多心跳週期後
- 氣泡較小(經過肺微血管的壓縮)


itemize
importantbox


| p{5.5cm}p{5.5cm}}  **特徵** | **PFO** | **肺內分流** |
| --- | --- | --- |
| 氣泡出現時間 | 3-5個心跳週期 | >5個心跳週期(通常6-10個) |
| Valsalva效應 | 明顯增加氣泡數量    是診斷關鍵 | 影響較小或無影響 |
| 氣泡大小 | 較大且均勻 | 較小(經肺微血管篩選) |
| 分布特性 | 可能集中在特定區域(如LA free wall) | 通常瀰漫性分布 |
| 體位影響 | 直立時可能增加(POS情況) | 直立或臥位差異小 |
| 常見病因 | 胚胎發育殘留    家族傾向 | Pulmonary AVM    Hepatopulmonary syndrome    ARDS    肝硬化 |
| 輔助檢查 | TOE可直接看到通道    TCD定量分流 | 胸部CT angiography    肺灌注掃描    肺血管攝影 |


對於late-appearing bubbles(>5心跳週期),應強烈懷疑肺內分流。此時應進行進一步檢查如胸部CT血管攝影或99mTc-MAA核醫掃描以確認診斷。單純依靠超音波無法完全排除兩者並存的可能性。


## 臨床應用與治療決策


### PFO-相關性腦中風的評估


當病人發生隱源性腦中風(cryptogenic stroke)時,需系統性評估PFO的致病角色:


#### PASCAL評分系統


PASCAL評分(Points, Age at event, Size of Shunt on contrast TTE, Anatomy of PFO on TOE, Clinical history)是評估PFO與腦中風相關性的工具:


| p{5cm}p{2cm}p{3cm}}  **評估項目** | **細項** | **分數** |
| --- | --- | --- |
| P (Points) | 年齡 <65歲 | +1 |
|  | 年齡 ≥65歲 | 0 |
| A (Age) | 年齡 <65歲 | +1 |
|  | 年齡 ≥65歲 | 0 |
| S (Size of shunt) | c-TTE Grade 0-1 | 0 |
|  | c-TTE Grade 2-3 | +1 |
| C (Cortical infarct) | 皮質梗塞 | +1 |
|  | 非皮質梗塞 | 0 |
| A (Anatomy) | ASA存在 | +1 |
|  | Large PFO或Hypermobile septum | +1 |
|  | 無上述特徵 | 0 |
| L (thrombophiLia) | 血栓形成傾向 | +1 |
|  | 無血栓形成傾向 | 0 |
| **總分範圍** | **0-7分** |  |


**PASCAL分數解讀**:

- 0-3分: PFO與腦中風相關性較低,考慮內科治療
- 4-5分: 中等相關性,需多專科討論
- 6-7分: 高度相關性,強烈建議考慮PFO關閉術


PASCAL評分雖然有助於臨床決策,但並非絕對標準。仍需考慮其他臨床因素,如:發病時是否有Valsalva動作、是否有深部靜脈血栓或肺栓塞病史、是否有長時間不動(如長途飛行)等。


#### 其他重要臨床考量因子


以下因素雖未納入PASCAL評分,但在評估PFO致病性時應予考慮:


| p{11cm}}  **臨床特徵** | **與矛盾性栓塞的相關性** |  |
| --- | --- | --- |
| Valsalva發作時機 | 中風發生時正在進行Valsalva動作(排便、咳嗽、舉重等)    強烈提示矛盾性栓塞機制 |  |
| 深部靜脈血栓史 | 近期或過去有DVT或PE病史    提供栓子來源的證據 |  |
| 長期不動 | 近期長途飛行(>4小時)、臥床、或長時間固定姿勢    增加DVT形成風險 |  |
| 凝血功能異常 | Factor V Leiden、Prothrombin G20210A突變    Protein C/S deficiency、Antiphospholipid syndrome    增加血栓形成傾向 |  |
| 其他非腦部栓塞 | 心肌梗塞(特別是年輕無危險因子者)    脾梗塞、腎梗塞、周邊動脈栓塞    提示系統性栓塞問題 |  |
| 睡眠呼吸中止症 | 反覆的呼吸暫停導致右心房壓力變化    增加R→L分流機會 |  |
| Platypnea-orthodeoxia | 站立時呼吸困難與血氧下降    PFO分流的特異性表現 |  |
| 跨PFO血栓 | TOE直接看到血栓卡在PFO(straddling thrombus)    極罕見但高度特異性 |  |

大多數中心使用螢光鏡或合併TOE/ICE進行PFO關閉術:

| --- | --- | --- |

| 螢光鏡 + TOE | 最佳軟組織影像    即時評估裝置位置    可確認無殘餘分流    評估相鄰結構 | 需全身麻醉或深度鎮靜    增加程序複雜度與成本    需額外靜脈通路    可能的食道併發症 |

根據歐洲專家共識,對於解剖結構單純的PFO,單純螢光鏡導引即可安全完成關閉術,無需常規使用TOE。然而對於複雜解剖(大型ASA、deficient rim、tunnel PFO)或高風險情況,建議使用影像導引。

## 特殊臨床情境


### Platypnea-Orthodeoxia Syndrome (POS)


#### 定義與機制


POS是一種罕見症候群,特徵為:

- **Platypnea**: 站立時出現呼吸困難,平躺時改善
- **Orthodeoxia**: 站立時血氧飽和度下降(通常≥5


#### 超音波評估要點


importantbox
**POS的超音波檢查關鍵**


1. **體位相關性評估**      - 必須在平躺與直立(或半坐臥)兩種姿勢下進行對比劑檢查 - 直立時分流顯著增加是診斷關鍵
2. **Eustachian valve評估**      - 突出的Eustachian valve將IVC血流導向PFO - 站立時效應更明顯(IVC血流增加) - 需使用bi-caval view或hepatic vein view觀察
3. **特殊對比劑注射技術**      - 若上肢靜脈注射為陰性,應考慮從**股靜脈注射** - 股靜脈注射可避免SVC稀釋效應 - 更能反映IVC血流經Eustachian valve導向PFO的情況


importantbox


| p{11cm}}  **檢查步驟** | **技術細節** |  |
| --- | --- | --- |
| 平躺位c-TTE | 標準上肢靜脈注射    記錄分流程度(可能為陰性或輕微) |  |
| 直立或半坐臥c-TTE | 同樣上肢靜脈注射    觀察分流是否顯著增加    記錄血氧飽和度變化 |  |

| 股靜脈對比劑注射 | 若上述檢查陰性但臨床高度懷疑    從股靜脈注射對比劑    直接評估IVC→PFO的分流 |  |

### 偏頭痛


PFO與偏頭痛(特別是伴隨視覺前兆的偏頭痛)的關聯仍有爭議:


- 有視覺前兆的偏頭痛病人中,PFO盛行率較高(約40-50
- 機制可能與微小栓子或血管活性物質經PFO進入腦循環有關
- 部分隨機對照試驗(如PREMIUM, PRIMA)顯示PFO關閉可減少偏頭痛發作


根據最新證據,PFO關閉術對於難治型偏頭痛可能有益,但仍非常規治療。適合考慮的病人包括:對藥物治療反應不佳、有大型PFO合併ASA、偏頭痛嚴重影響生活品質者。決策應由神經科與心臟科共同討論。

## 常見問題與臨床困境


### Q&A常見問題


1. **Q: 發現PFO就一定要關閉嗎?**          A: 不一定。約25      - 隱源性腦中風且證據顯示PFO可能相關 - 反覆減壓症的潛水員 - 嚴重的Platypnea-Orthodeoxia Syndrome - 難治型偏頭痛(仍有爭議)
2. **Q: c-TTE陰性但臨床高度懷疑PFO,下一步該怎麼做?**          A: 可考慮:      - 改用c-TOE(敏感度較高,且可直接看到解剖結構) - 使用c-TCD(敏感度最高,且Valsalva較易執行) - 確認Valsalva執行是否確實(這是最常見的false negative原因) - 考慮其他診斷(如肺內分流)
3. **Q: 如何區分PFO與小型secundum ASD?**          A: 關鍵差異:      - PFO: 瓣膜狀,通常需Valsalva才有明顯分流,無RV擴大 - Secundum ASD: 真正缺損,持續性L→R分流,常有RV擴大 - TOE可直接觀察:PFO是兩層中膈間的通道,ASD是真正的組織缺損
4. **Q: PFO關閉術後多久可以停止抗血小板藥物?**          A: 一般建議:      - Dual antiplatelet (Aspirin + Clopidogrel): 1-3個月 - Single antiplatelet (通常Aspirin): 至少6個月 - 之後根據其他心血管危險因子決定是否繼續 - 若術後出現心房顫動,需改用抗凝血劑
5. **Q: 關閉術後仍有小量殘餘分流(Grade 1),需要再次介入嗎?**          A: 通常不需要。Grade 0-1被視為"臨床有效關閉",栓塞復發率很低。只有在以下情況才考慮再次介入:      - 出現臨床事件復發(如再次中風) - Grade 2-3的顯著殘餘分流 - 病人有特殊需求(如職業潛水員)


### 臨床決策困境


#### 年輕病人的首次隱源性中風


**案例情境**: 35歲女性,無心血管危險因子,發生腦中風,影像顯示皮質梗塞,c-TTE顯示Grade 2 PFO,TOE確認有ASA。

**決策考量**:


| p{5.5cm}p{5.5cm}}  **治療選項** | **優點** | **缺點** |
| --- | --- | --- |
| 抗血小板治療 | 非侵入性    成本低    無手術風險 | 中風復發率約2  需長期服藥    預防效果可能不足 |
| PFO關閉術 | 顯著降低復發率    (相對風險降低約60  年輕病人長期獲益大 | 侵入性手術    有併發症風險    可能出現心房顫動 |
| 抗凝血治療 | 預防效果可能較好 | 出血風險    需長期監測    影響生活品質    證據不如其他兩者 |


**建議**: 根據PASCAL評分(此案例約5-6分)與隨機對照試驗證據,年輕、無其他危險因子、有高風險PFO特徵(ASA、大型PFO)的病人,強烈建議考慮PFO關閉術合併抗血小板治療。


#### 老年病人的PFO與中風


**案例情境**: 68歲男性,有高血壓、糖尿病,發生腦中風,c-TTE顯示Grade 1 PFO,無ASA。

**決策考量**:

- 年齡>60歲,PFO致病性相對較低
- 有其他心血管危險因子,可能有其他中風原因
- PASCAL評分較低(約2-3分)
- RCT亞組分析顯示年齡較大者PFO關閉獲益較小


**建議**: 此類病人優先選擇內科治療(抗血小板或抗凝血,視其他適應症而定)。只有在復發性中風且確認無其他原因時,才考慮PFO關閉術。


## 最新進展與未來方向


### 新技術發展


1. **Deviceless closure (縫合系統)**      - NobleStitch系統:直接縫合PFO,無需永久植入物 - 優點:未來若需左心房介入較容易,無長期異物 - 缺點:技術要求高,殘餘分流率較傳統裝置高 - 超音波評估:術後追蹤較困難(無明顯異物可供辨識)
2. **生物可吸收關閉裝置**      - Carag CBSO:支架為PLGA可吸收材料 - 理論優勢:數年後完全吸收,只剩內皮化組織 - 目前仍在早期研究階段
3. **Fluoroscopy-free closure**      - 單純使用ICE或TOE導引,完全不用螢光鏡 - 優點:零輻射暴露,對病人與術者都更安全 - 適合年輕病人或懷孕婦女 - 技術要求:需要優秀的影像判讀與手術技巧
4. **人工智慧輔助診斷**      - AI自動辨識與測量PFO - AI預測關閉術成功率與併發症風險 - 仍在研究階段,未來可能輔助決策


### 未解決的問題


- PFO關閉對偏頭痛的效果仍有爭議,需要更大規模RCT
- 最佳的抗血小板療程仍未確立
- 殘餘分流的長期影響需要更長期追蹤
- PFO與其他疾病(如睡眠呼吸中止症)的關聯需要更多研究


## 總結與臨床實踐要點


importantbox
**臨床實踐關鍵要點**


1. **診斷**      - PFO診斷需要對比劑檢查,單純2D超音波不足 - Valsalva maneuver的品質是診斷成功的關鍵 - 無單一黃金標準,通常需要合併使用不同技術 - 注意區分PFO與肺內分流(時間差異法則)
2. **評估**      - 系統性評估PFO的解剖特徵(大小、長度、ASA等) - 使用標準化報告模板促進跨專科溝通 - TOE提供最佳解剖資訊,對手術規劃至關重要
3. **治療決策**      - 並非所有PFO都需要關閉,需個別化評估 - 年輕、高風險PFO特徵、隱源性中風是關閉術的主要適應症 - PASCAL評分可輔助決策但非絕對標準 - 多專科團隊討論(神經科、心臟科)極為重要
4. **術後追蹤**      - Grade 0-1分流視為成功,無需過度擔心 - 定期追蹤評估裝置穩定性與併發症 - 注意晚期心房顫動的發生
