## 概述


本研究分析了43例接受MitraClip®治療的重度二尖瓣逆流(MR)高危患者，提出並驗證了「中央夾子概念」(central clip concept)的治療策略。研究發現，**血管收縮處(vena contracta)≥7.5mm**可作為預測需要第二個夾子的重要指標（敏感度83

| p{11cm}}  **項目** | **標準** |  |
| --- | --- | --- |
| **納入標準** | minipage[t]{11cm} • 重度MR (3+或4+) |  |
| • 高手術風險（根據logistic EuroSCORE評估） |  |  |
| • 拒絕或不適合傳統二尖瓣修復手術 |  |  |
| • 有症狀的慢性MR |  |  |
| **排除標準** | minipage[t]{11cm} • 活動性心內膜炎 |  |

| **特殊考量** | minipage[t]{11cm} 本研究的排除標準與EVEREST試驗不同，以下情況**非**絕對禁忌症： |  |
| • 部分鈣化瓣葉 |  |  |
| • 對合長度和深度不足 |  |  |
| • 脫垂瓣葉的寬度和最大間隙 |  |  |
| • 絕對瓣膜面積 <4.0 cm² |  |  |
| • 射出分數不佳 |  |  |
| • 噴射起源位置 |  |  |


### 中央夾子概念(Central Clip Concept)


| p{11cm}}  **步驟** | **策略** |  |
| --- | --- | --- |
| **初始定位** | minipage[t]{11cm} • 將第一個夾子置於噴射中心位置 |  |
| • 將MR分成兩個對稱的噴射 |  |  |
| • 部分閉合夾子確認MR減少效果 |  |  |
| **評估決策** | minipage[t]{11cm} **情況A：**如果產生兩個小噴射 |  |
| • 維持中央位置 |  |  |
| • 完全部署夾子 |  |  |
| • 僅使用一個夾子 |  |  |
| [0.5em] **情況B：**如果產生兩個寬噴射 |  |  |
| • 將第一個夾子重新定位至內側 |  |  |
| • 切換為兩夾子策略 |  |  |
| • 部署第一個內側夾子後，植入第二個外側夾子 |  |  |
| • 確認無明顯二尖瓣壓力梯度 |  |  |
| **特殊情況** | minipage[t]{11cm} • 如果噴射起源非常寬，從一開始就考慮兩個夾子策略 |  |
| • 在P2脫垂伴有寬噴射的情況下，應從手術開始就計劃兩夾子策略 |  |  |

| • 過夜監測血流動力學狀態 |  |  |

## 研究限制


### 方法學限制


1. **樣本大小相對較小：**      - 總共43例患者 - ≥2夾子組僅19例 - 可能限制統計檢定力
2. **單中心經驗：**      - 僅來自Royal Brompton Hospital - 可能存在選擇偏差 - 結果可能不完全適用於其他中心
3. **追蹤期間中等：**      - 中位數追蹤385天 - 缺乏長期(>2年)結果 - 夾子耐久性尚待評估
4. **非隨機對照研究：**      - 觀察性研究設計 - 無對照組比較 - 患者選擇基於臨床判斷
5. **藥物治療未標準化：**      - 最佳藥物治療未在研究中標準化 - 可能影響臨床結果的解釋


### 技術與評估限制


1. **TOE評估的主觀性：**      - MR程度評估依賴操作者經驗 - VCW測量可能存在變異性 - 噴射數量和寬度的判斷帶有主觀性
2. **學習曲線效應：**      - 研究橫跨2009-2012年 - 技術和經驗隨時間改進 - 早期vs晚期病例可能存在系統性差異
3. **裝置技術進展：**      - 使用較早期的MitraClip®系統 - 新一代裝置可能有不同的性能特徵

### 簡答題


**7. 解釋中央夾子概念的工作流程，包括如何決定是否需要第二個夾子。**

**參考答案：**

1. **初始定位：**將第一個MitraClip®置於MR噴射中心位置，將逆流分成兩個對稱噴射
2. **部分閉合評估：**在部分閉合夾子狀態下，用TOE評估MR減少效果和產生的兩個噴射大小
3. **決策點：**      - 如果產生兩個**小**噴射：維持中央位置，完全部署夾子，僅使用一個夾子 - 如果產生兩個**寬**噴射：將第一個夾子重新定位至內側，切換為兩夾子策略，部署第一個夾子後植入第二個外側夾子
4. **特殊情況：**如果術前評估顯示噴射起源非常寬(如VCW≥7.5mm或存在兩個寬噴射)，從手術開始就計劃兩夾子策略
5. **安全確認：**植入每個夾子前，確認不會產生顯著的跨二尖瓣壓力梯度


**8. 列舉並解釋本研究發現的三個主要預測需要第二個MitraClip®的因子。**

**參考答案：**

1. **VCW(Vena Contracta)≥7.5mm：**      - 多變量分析中唯一的獨立預測因子(OR 2.5, P=0.013) - 敏感度83 - 反映MR噴射的寬度，較寬的噴射難以用單一夾子完全覆蓋 - ROC曲線下面積0.896，具有優秀的預測能力
2. **存在≥2個寬MR噴射：**      - 與需要第二個夾子顯著相關(P<0.001) - ≥2夾子組中57.9 - 表示MR起源於多個位置或範圍廣泛 - 單一夾子無法同時處理兩個分離的噴射源
3. **受限後瓣葉(Restricted PML)的缺乏：**      - 與需要多個夾子呈負相關(P=0.02) - 1夾子組中61.9 - 受限後瓣葉可能限制多夾子的使用，因為可能增加產生顯著二尖瓣狹窄的風險 - 提示瓣葉活動度在決定夾子數量中的重要性
