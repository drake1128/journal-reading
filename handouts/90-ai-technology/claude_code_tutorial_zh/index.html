Claude Code 教學手冊 終端機介面基礎 簡介 Claude Code 是Anthropic 開發的終端機介面AI 程式助手。與其他支援多種模型的AI 程式工 具不同，Claude Code 專為Claude 模型設計，實現工具與底層AI 的深度整合與最佳化。 資訊 Claude Code 直接與Anthropic 的模型配合運作，省去中間環節，以具競爭力的價格提 供最佳效能。 1.1 為什麼選擇Claude Code？ Claude Code 的主要優勢包括： • 大型檔案處理：可靠地更新超過18,000 行程式碼的檔案 • 複雜程式庫導覽：擅長搜尋模式並理解各元件之間的關係 • 穩定性：相較其他AI 程式工具，極少發生卡住的情況 • 直接整合：由開發底層模型的同一團隊打造 • 成本效益：無中間商加價，直接享有API 成本優惠 安裝與設定 2.1 VS Code 擴充套件 Claude Code 官方擴充套件支援以下編輯器： • Visual Studio Code • Cursor • Windsurf • 其他VS Code 分支版本 提示 此擴充套件讓您能直接在IDE 中啟動Claude Code。您可以同時開啟多個Claude Code 視窗，平行處理程式庫的不同部分。 2.2 自動載入檔案內容 當您在編輯器中開啟檔案時，Claude Code 會自動將其載入對話內容中。這種無縫整合減少了 手動指定檔案的麻煩。 終端機介面基礎 Claude Code 採用終端機使用者介面，初次使用可能需要適應，但熟悉後將能發揮強大功能。
