2 1. 核心概念

Pearl：常見的誤解認為缺血性肢體壞死僅來自肢體動脈的血栓或血栓栓塞，伴隨動脈脈搏消失。然而，缺血性肢體壞疽也可因微循環血栓 (microthrombosis) 引起——包括小靜脈的血栓。在此情況下，動脈脈搏是可觸及的或可用 Doppler 偵測到的。

本文聚焦於因 disseminated intravascular coagulation (DIC) 和天然抗凝機制喪失所導致的微血栓性肢體壞疽。

3 2. 微血栓相關肢體缺血的兩種症候群

基礎 DIC 病因 Heparin-induced thrombocytopenia (HIT)、metastatic adenocarcinoma、antiphospholipid syndrome Septic shock（如 meningococcemia）、cardiogenic shock

Warfarin 是否涉及 經常 通常無
先天性高凝狀態 通常無 通常無
血小板低下 是 是

共同病理生理特徵：微血栓伴隨促凝-抗凝平衡紊亂 (disturbed procoagulant-anticoagulant balance)

4 3. DIC 與天然抗凝機制失效

4.1 3.1 DIC 的特徵

  • 全身性凝血活化 (pathologic thrombin generation)
  • 纖溶功能受損 (impaired fibrinolysis)
  • 血管內 fibrin 形成與沉積 → 微血管血栓性閉塞

4.2 3.2 兩大天然抗凝系統

4.2.1 Protein C 系統

Thrombin + Thrombomodulin (endothelial cell)
        ↓
Protein C → Activated Protein C (APC)
        ↓
降解 Factor Va 和 Factor VIIIa → 下調 thrombin generation

4.2.2 Antithrombin 系統

Thrombin + Antithrombin → Thrombin-Antithrombin Complex（不可逆失活）
        ↑
催化：endothelial heparan sulfate 或 pharmacologic heparin

4.3 3.3 為何 Warfarin 加重微血栓

  • Warfarin 耗竭 vitamin K-dependent factors，包括 protein C（抗凝因子）
  • Factor VII 和 protein C 均有短半衰期（分別 5 小時和 9 小時）及低濃度
  • Supratherapeutic INR（通常 > 4.0）是 protein C 嚴重耗竭的替代指標
  • 矛盾：儘管 INR 升高，thrombin generation 仍持續，微血栓仍在發生

5 4. 靜脈性肢體壞疽 (Venous Limb Gangrene)

5.1 4.1 定義

有 deep-vein thrombosis 的肢體出現肢端缺血性壞死 (acral ischemic necrosis)。可逆的前驅狀態為 phlegmasia cerulea dolens（腫脹、發紺、疼痛的肢體）。

5.2 4.2 癌症相關的靜脈性肢體壞疽

  • 至少 50% 的靜脈壞疽患者有基礎癌症（通常為 metastatic adenocarcinoma）
  • 典型臨床情境：表面上「特發性」DVT 患者，在 heparin-warfarin 重疊期結束後不久出現 phlegmasia 或靜脈壞疽
  • 實驗室特徵：
    • Thrombin-antithrombin complexes 顯著升高（體內 thrombin generation 標記）
    • Protein C activity 顯著降低
    • Warfarin 無法抑制癌症相關的高凝狀態，同時耗竭 protein C 促進微血栓

5.3 4.3 HIT 相關的靜脈性肢體壞疽

  • 血小板下降通常在免疫性暴露（heparin）後 5–10 天開始
  • Warfarin 涉及大多數發生靜脈壞疽的 HIT 患者
  • INR > 4.0 為特徵
  • 少數在無 warfarin 的情況下發生：通常有異常嚴重的血小板低下（< 20,000/mm³）

5.4 4.4 靜脈壞疽 vs. 經典 Warfarin 誘發的皮膚壞死

特徵 靜脈性肢體壞疽 Warfarin-induced skin necrosis
壞死部位 肢端 (acral)，可深至骨骼 非肢端（乳房、腹部、大腿、小腿）
開始時間 Warfarin 開始後 2–6 天 Warfarin 開始後 2–6 天
先天 protein C 異常 通常無 常有（protein C deficiency, Factor V Leiden）
基礎疾病 HIT 或癌症相關消耗性凝血病 + DVT 通常無嚴重消耗性凝血病

5.5 4.5 預防與治療

預防： - 在有急性 DVT + 血小板低下或凝血異常的患者中，避免使用 warfarin - 指引建議：在 HIT 的急性（血小板低下）期避免 warfarin - LMWH 在癌症相關 DVT 中優於 warfarin - 避免在 HIT 或癌症等高凝狀態患者中使用 IVC filter

治療（兩項原則）： 1. Vitamin K 10 mg slow IV（針對 warfarin 導致的 INR 延長），12–24 小時後可重複 2. 治療劑量抗凝血

6 5. 對稱性周邊壞疽與紫斑性暴發症

6.1 5.1 臨床特徵

  • 對稱性周邊壞疽 (Symmetric Peripheral Gangrene)：主要為肢端壞死，對稱性影響 2 或 4 肢
  • 紫斑性暴發症 (Purpura Fulminans)：有廣泛非肢端皮膚壞死時使用此術語
  • 患者通常病危：cardiogenic shock 或 septic shock
  • 死亡率 > 50%；約 1/4 患者需要四肢截肢

6.2 5.2 臨床表現

  • 發燒、低血壓、瘀點皮疹 → 演變為融合的紫癜性缺血壞死區域
  • 早期徵兆：肢端明顯冰冷、蒼白、遠端肢體疼痛
  • 水皰（常為出血性）表示組織壞死
  • 壞死區域常有清晰分界且驚人地對稱
  • 下肢先於上肢受影響

6.3 5.3 病理特徵

  • 皮膚微血栓涉及小靜脈和毛細血管
  • 多重器官衰竭常見（腎臟、肺臟、肝臟）

6.4 5.4 常見相關微生物

族群 常見病原
兒童及青少年 Neisseria meningitidis（腦膜炎球菌）
成人 Streptococcus pneumoniae（肺炎球菌）
脾切除或功能性無脾 Encapsulated bacteria（meningococcus, H. influenzae, pneumococcus）
其他 E. coli、rickettsia、malaria、Capnocytophaga（犬咬傷）

6.5 5.5 急性缺血性肝炎（Shock Liver）

  • 約 90% 發展為肢端缺血壞死的危重病人有先前 shock liver
  • 肢體壞死通常在肝酵素升高後 2–5 天開始
  • 時程類似 warfarin-induced skin necrosis（反映 protein C 合成受損後耗竭至臨界低水平所需時間）

7 6. 治療原則

7.1 6.1 一般原則

治療基於理論考量和病例觀察（非 RCT）：

  1. 藥物中斷 thrombin generation：Heparin anticoagulation
  2. 補充天然抗凝因子：Frozen plasma 或 protein C / antithrombin 濃縮液
  3. 減少降低肢體灌流的風險因素：矯正低血壓、減少或避免 vasopressors

7.2 6.2 抗凝選擇

Pearl：許多專家偏好使用 heparin，因為： - 抗凝效果可直接用 anti-factor Xa 監測 - 避免 APTT confounding（DIC 患者基線 APTT 已升高，使用 APTT nomogram 可能導致系統性抗凝不足） - 即使肝腎衰竭，heparin clearance 仍正常 - 具有獨立於抗凝作用的抗發炎特性 - 注意：Heparin 需要其輔因子 antithrombin，而 DIC + 肝功能障礙患者 antithrombin 可能降低

7.3 6.3 特殊治療

  • Protein C concentrates：有些專家建議在嚴重腦膜炎球菌血症中早期使用
  • Recombinant activated protein C：已退出市場（大型 RCT 未顯示改善 septic shock 存活）
  • High-dose antithrombin concentrates：在嚴重 sepsis 中未改善整體死亡率

7.4 6.4 外科考量

  • Fasciotomy：減少腔室壓力，但破壞皮膚屏障且中斷抗凝治療
  • 早期截肢應避免：難以區分存活組織和壞死組織；耐心等待自然截肢可能減少最終組織損失
  • 建議多學科團隊（整形外科 / 傷口護理、內科 / 感染科、足部醫學），可在燒傷單位處理

8 7. Key Points 重點整理

  1. 缺血性肢體壞疽可在有脈搏的情況下發生——由微血管血栓而非大血管阻塞引起
  2. 兩大症候群：靜脈性肢體壞疽（合併 DVT）和對稱性周邊壞疽（通常無 DVT）
  3. 共同機制：DIC + 天然抗凝系統失效（protein C 和 antithrombin）
  4. Warfarin 在 DIC 中是危險的：耗竭 protein C 而無法有效抑制 thrombin generation
  5. Supratherapeutic INR（> 4.0）是 protein C 嚴重耗竭的替代標記
  6. 預防：DIC 患者避免 warfarin；HIT 急性期避免 warfarin；癌症 DVT 用 LMWH
  7. 治療：Vitamin K（逆轉 warfarin）+ 治療劑量抗凝 + 補充天然抗凝因子 + 矯正低血壓
