## 概述


本研究為ANGEL-REBOOT隨機對照試驗的一年追蹤結果,評估在急性缺血性腦中風患者接受血栓移除術失敗後,補救性顱內血管成形術或支架置放術的長期療效。這是首個針對血栓移除術失敗患者進行補救性介入治療的前瞻性隨機對照試驗,提供了重要的長期追蹤數據。

### 研究背景


**急性缺血性腦中風的治療策略:**

- **化學溶栓療法:**使用纖溶酶原活化劑(如alteplase和tenecteplase)
- **機械性血栓移除術:**透過血管內接觸抽吸或支架取栓
- **成功率:**約80
- **挑戰:**約20


**血栓移除術失敗的原因:**

- 顱內血管本身存在病變(動脈粥狀硬化狹窄)
- 血栓形成於破裂的動脈粥狀硬化斑塊上
- 導管操作困難,可能造成血管剝離
- 殘留高度狹窄(>70


**傳統補救策略:**

- 動脈內纖溶治療
- 靜脈抗血小板藥物(如tirofiban)
- 補救性血管成形術或支架置放(缺乏隨機對照試驗證據)


## 研究方法


### 研究設計


- **研究類型:**多中心、開放性、隨機對照試驗
- **研究地點:**中國36家三級醫學中心
- **研究期間:**招募期至2024年,追蹤至一年
- **主要終點:**一年時的Modified Rankin Score(mRS)順序量表分析
- **次要終點:**再發腦中風率、死亡率、功能獨立性

### 長期效益的關鍵機轉


**為何一年結果優於90天結果?**

1. **預防再發腦中風:**支架置放移除了後續血栓栓塞的來源
2. **維持血管通暢:**減少殘留狹窄導致的再閉塞風險
3. **改善腦部灌流:**長期維持足夠的腦血流供應
4. **克服早期併發症:**早期風險被長期效益所抵銷


## 研究討論與意義


### 關鍵發現


1. ****長期效益顯著:****儘管90天無差異,一年追蹤顯示補救性介入顯著改善功能預後
2. ****再發中風大幅減少:****再發缺血性腦中風風險降低70
3. ****功能獨立性提升:****完全功能獨立比例從47
4. ****風險效益平衡:****早期併發症風險被長期效益所超越


### 與現有文獻的關聯


**支持性證據:**

- **頸動脈狹窄治療:**頸動脈內膜切除術或支架置放術在預防再發中風的長期效益
- **顱內動脈狹窄:**次最大球囊血管成形術(submaximal angioplasty)顯示早期併發症但長期效益
- **觀察性研究:**先前觀察性研究提示補救性介入可能有益,但存在選擇偏差

## 研究限制


### 內在效度限制


1. **開放性設計:**非盲性試驗,可能影響評估
2. **樣本數有限:**總收案348例,結果事件數相對較少
3. **隨機誤差:**小樣本可能存在隨機誤差
4. **早期併發症高:**14


### 外在效度限制


1. **單一種族:**僅限中國患者,ICAD在亞洲較常見
2. **專家中心:**36家三級醫學中心,操作者經驗豐富
3. **特定病因:**94
4. **設備與技術:**需要特定的介入設備與技術


### 其他考量


1. **追蹤時間:**僅一年,更長期結果未知
2. **成本效益:**未評估經濟效益分析
3. **併發症管理:**早期併發症的最佳處理策略
4. **患者選擇:**如何識別最適合的患者族群

## 結論


### 主要結論


1. ****長期效益確立:****ANGEL-REBOOT試驗顯示,儘管早期有併發症風險,補救性顱內血管成形術或支架置放在一年追蹤時顯著改善功能預後
2. ****再發中風顯著減少:****補救性介入使再發缺血性腦中風風險降低70
3. ****需要複製研究:****在其他族群、中心和病因類型中驗證這些發現
4. ****專業中心實施:****此技術應在具備充分經驗和設備的介入中心進行


### 臨床實務建議


1. 對於血栓移除術後不完全再灌注或高度殘留狹窄的患者,可考慮補救性介入
2. 需在經驗豐富的介入中心,由專業團隊執行
3. 與患者充分討論早期風險與長期效益
4. 術後需密切監測併發症並積極進行二級預防
5. 持續追蹤以評估長期預後


### 未來研究方向


1. 在非亞洲族群中進行驗證性研究
2. 評估在非ICAD相關大血管閉塞中的療效
3. 研究降低早期併發症的策略
4. 進行更長期(>1年)的追蹤研究
5. 評估成本效益與醫療資源配置
6. 發展最佳患者選擇標準

## DAPA-ACT HF-TIMI 68試驗詳細分析


### 試驗背景與重要性


**研究重要性:**

- 首個在**住院急性心衰竭患者**中評估SGLT2抑制劑療效的心血管結果試驗
- SGLT2抑制劑已是慢性心衰竭的標準治療
- 但在急性住院期間啟動治療的安全性與療效尚未確立
- 本試驗由TIMI研究小組執行,發表於2025年ESC大會


**SGLT2抑制劑背景知識:**

- 全名:鈉-葡萄糖協同轉運蛋白2抑制劑
- 原為糖尿病藥物,發現有心腎保護作用
- 已證實可降低慢性心衰竭患者的心血管死亡與住院率
- Dapagliflozin為其中一種SGLT2抑制劑

### 研究討論


#### 主要發現


1. ****單一試驗未達統計顯著:****DAPA-ACT HF-TIMI 68單獨分析顯示風險降低14
2. ****趨勢一致:****各次要終點均顯示有利於dapagliflozin的趨勢
3. ****統合分析顯著:****整合三個試驗後顯示SGLT2抑制劑顯著降低心血管死亡或心衰竭惡化風險29
4. ****全因死亡大幅降低:****統合分析顯示全因死亡降低43
5. ****安全性可接受:****低血壓風險輕微增加(3.6


#### 臨床意義


**支持早期啟動治療:**

- 住院期間啟動SGLT2抑制劑是安全的
- 可能提供早期保護效應
- 改善出院後持續用藥依從性
- 避免門診追蹤時才啟動的延遲


**機轉考量:**

- **利尿作用:**溫和的滲透性利尿,減輕容量負荷
- **代謝效應:**改善心肌能量代謝
- **神經荷爾蒙:**抑制過度活化的神經荷爾蒙系統
- **抗發炎:**減少系統性發炎反應
- **心肌重塑:**改善不良心室重塑


#### 與指引建議的關聯


**現行指引建議:**

- 2021 ESC心衰竭指引:SGLT2抑制劑為HFrEF的Class I建議
- 2022 AHA/ACC/HFSA指引:建議用於所有符合條件的HFrEF患者
- 但啟動時機多建議在血流動力學穩定後


**本試驗的貢獻:**

- 提供住院期間啟動的安全性數據
- 支持更早啟動SGLT2抑制劑
- 可能影響未來指引更新


### 研究限制


1. **檢定力不足:**單一試驗可能檢定力不足以檢測14
2. **追蹤時間短:**僅2個月追蹤,長期效益未知
3. **早期啟動時機:**中位啟動時間在住院後多久未明確報告
4. **出院後依從性:**出院後持續用藥比例未詳細報告
5. **特定族群:**可能不適用於心因性休克或嚴重腎功能不全患者

### 未來研究方向


1. **長期追蹤:**評估6個月、1年甚至更長期的療效
2. **最佳時機:**確定住院期間啟動的最佳時間點
3. **HFpEF族群:**在射出分率保留的心衰竭患者中驗證
4. **合併用藥:**與ARNI、SGLT2抑制劑等同時啟動的安全性
5. **高危險群:**在更高風險患者(如心因性休克恢復期)中評估
6. **成本效益:**評估早期啟動的經濟效益
7. **真實世界:**在常規臨床實務中的應用與成效


### 結論


1. DAPA-ACT HF-TIMI 68試驗單獨分析未達主要終點的統計顯著性
2. 但整合EMPULSE和SOLOIST-WHF的統合分析顯示SGLT2抑制劑在急性心衰竭住院患者中顯著降低心血管死亡或心衰竭惡化風險29
3. 安全性良好,低血壓風險輕微增加但可接受
4. 支持在急性心衰竭住院期間,血流動力學穩定後即可考慮啟動SGLT2抑制劑
5. 可改善短期預後並可能提升出院後用藥依從性
