### 鈣化結節的臨床挑戰


鈣化結節（Calcified Nodules, CNs）是經皮冠狀動脈介入治療（PCI）中具有挑戰性的病變類型。本綜述探討了CNs的識別、分類及其不同亞型，並介紹了目前可供介入心臟科醫師使用的治療策略。


> **為何鈣化結節值得關注？**
> itemize
>     \item 鈣化結節是急性冠心症的重要病因之一，特別是在老年患者和透析患者中
>     \item 與斑塊破裂和斑塊侵蝕不同，鈣化結節具有獨特的病理生理學特徵
>     \item 傳統PCI技術對嚴重鈣化病變的效果有限
>     \item 需要特殊的血管內影像學評估和斑塊修飾技術
> itemize

### 研究背景


心肌梗塞後乙型阻斷劑的使用一直是心臟學界的重要議題。傳統上，乙型阻斷劑被認為是心肌梗塞後的標準治療，但隨著再灌注治療和二級預防藥物的進步，其在左心室功能保留患者中的必要性受到質疑。


### 研究目的與方法


> **研究目的：**評估心肌梗塞後左心室射血分數（LVEF）>40
> **研究對象：**
> itemize
>     \item 心肌梗塞患者
>     \item 左心室射血分數 >40
>     \item 來自REBOOT試驗
> itemize
> **評估重點：**
> itemize
>     \item 停用乙型阻斷劑（Withdrawal）的安全性
>     \item 不使用乙型阻斷劑（Withholding）的安全性
>     \item 短期及復發性缺血事件的發生率
> itemize


### 主要研究發現


| p{10cm}} REBOOT試驗事後分析主要發現 |  |  |
| --- | --- | --- |
| **評估項目** | **結果** |  |
| **評估項目** | **結果** |  |
| **停用乙型阻斷劑** | 未增加短期或復發性缺血事件的發生率 |  |
| **不使用乙型阻斷劑** | 未增加短期或復發性缺血事件的發生率 |  |
| **整體安全性** | 在LVEF >40 |  |


### 臨床意義


> **研究影響：**
> itemize
>     \item 本研究結果為近期關於心肌梗塞後乙型阻斷劑使用的熱烈討論提供了重要證據
>     \item 支持在左心室功能保留的心肌梗塞患者中，可以考慮個人化的乙型阻斷劑處方策略
>     \item 需要更多前瞻性隨機對照試驗來確認長期安全性
> itemize
> **注意事項：**
> itemize
>     \item 本研究為事後分析，證據等級有限
>     \item 僅適用於LVEF >40
>     \item 評估的是短期安全性，長期效果需進一步研究
> itemize

### 研究背景


高出血風險（High Bleeding Risk, HBR）患者合併慢性腎臟病（Chronic Kidney Disease, CKD）是臨床上常見且具挑戰性的族群。這類患者同時面臨較高的缺血風險和出血風險，最佳的雙重抗血小板治療（Dual Antiplatelet Therapy, DAPT）療程一直存在爭議。


### 研究設計與目的


> **研究目的：**比較高出血風險合併慢性腎臟病患者接受PCI後，1個月DAPT與≥3個月DAPT的療效。
> **研究特點：**
> itemize
>     \item 預設分析（Prespecified analysis）
>     \item 針對HBR患者設計
>     \item 評估腎功能對DAPT療程選擇的影響
> itemize
> **主要終點：**
> itemize
>     \item 淨不良臨床事件（Net Adverse Clinical Events, NACE）
>     \item 主要心腦血管不良事件（Major Adverse Cardiac or Cerebral Events, MACCE）
>     \item 出血事件
> itemize


### 研究對象


| p{11cm}} MASTER DAPT試驗CKD分析納入標準 |  |  |
| --- | --- | --- |
| **條件** | **說明** |  |
| **條件** | **說明** |  |
| **高出血風險定義** | 符合ARC-HBR標準的患者 |  |
| **慢性腎臟病** | 依據腎功能進行分層分析 |  |
| **介入治療** | 接受PCI治療的患者 |  |
| **隨機分組** | 1個月DAPT vs ≥3個月DAPT |  |


### 主要研究發現


| p{5cm}p{5.5cm}} MASTER DAPT試驗CKD亞組分析結果 |  |  |
| --- | --- | --- |
| **終點** | **1個月DAPT** | **≥3個月DAPT** |
| **終點** | **1個月DAPT** | **≥3個月DAPT** |
| **淨不良臨床事件** | 相似 | 相似 |
| **主要心腦血管不良事件** | 相似 | 相似 |
| **出血事件** | 減少 | 較高 |
| **腎功能亞組一致性** | 無論腎功能如何，結果一致 |  |


### 臨床意義


> **主要結論：**
> enumerate
>     \item 在高出血風險合併慢性腎臟病的PCI患者中，1個月縮短DAPT療程：
>     itemize
>         \item 與≥3個月標準療程相比，淨不良臨床事件發生率相似
>         \item 主要心腦血管不良事件發生率相似
>         \item 出血事件減少
>     itemize
>     \item 無論腎功能狀態如何，縮短DAPT療程的效益一致
>     \item 支持在這一高風險族群中採用個人化的抗血小板策略
> enumerate
> **臨床啟示：**
> itemize
>     \item 對於HBR合併CKD的患者，縮短DAPT療程是安全可行的選擇
>     \item 可有效降低出血風險而不增加缺血事件
>     \item 支持以患者為中心的抗血小板治療決策
> itemize

## 詞彙表


description[leftmargin=4.5cm,style=nextline]
    \item[NSTE-ACS] Non-ST-Segment Elevation Acute Coronary Syndrome，非ST段上升型急性冠心症。包括不穩定型心絞痛和非ST段上升型心肌梗塞。
    
    \item[CN] Calcified Nodule，鈣化結節。冠狀動脈粥狀硬化斑塊的一種類型，以突出的鈣化沉積為特徵。
    
    \item[PCI] Percutaneous Coronary Intervention，經皮冠狀動脈介入治療。
    
    \item[LVEF] Left Ventricular Ejection Fraction，左心室射血分數。評估心臟收縮功能的指標。
    
    \item[DAPT] Dual Antiplatelet Therapy，雙重抗血小板治療。通常指阿斯匹靈加上一種P2Y12抑制劑。
    
    \item[HBR] High Bleeding Risk，高出血風險。根據ARC-HBR標準定義。
    
    \item[CKD] Chronic Kidney Disease，慢性腎臟病。
    
    \item[NACE] Net Adverse Clinical Events，淨不良臨床事件。綜合缺血和出血事件的複合終點。
    
    \item[MACCE] Major Adverse Cardiac or Cerebral Events，主要心腦血管不良事件。
    
    \item[IVL] Intravascular Lithotripsy，震波血管內碎石術。用於處理嚴重鈣化病變的新型斑塊修飾技術。
    
    \item[IVUS] Intravascular Ultrasound，血管內超音波。用於評估冠狀動脈病變的影像學技術。
    
    \item[OCT] Optical Coherence Tomography，光學同調斷層掃描。高解析度血管內影像學技術。
    
    \item[MINOCA] Myocardial Infarction with Non-Obstructive Coronary Arteries，心肌梗塞伴非阻塞性冠狀動脈。在女性患者中較常見。
    
    \item[SCAD] Spontaneous Coronary Artery Dissection，自發性冠狀動脈剝離。
    
    \item[ARC-HBR] Academic Research Consortium for High Bleeding Risk，學術研究聯盟高出血風險標準。用於定義PCI後高出血風險患者的標準化標準。
