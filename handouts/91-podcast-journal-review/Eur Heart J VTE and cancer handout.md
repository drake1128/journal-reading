> **研究重點：**本研究為AVERT試驗的事後分析，探討發炎相關標記（CRP、GDF-15）與心臟標記（NT-proBNP、hs-TnT）在預測癌症患者靜脈血栓栓塞（VTE）與臨床相關出血風險中的角色。
> **關鍵發現：**
> itemize[leftmargin=*]
>     \item 基線GDF-15升高與VTE風險增加相關（SHR 1.36）
>     \item 基線NT-proBNP與CRP升高與出血風險增加相關
>     \item hs-TnT從基線至第1個月的上升與VTE風險增加相關（SHR 1.89）
>     \item 研究開發了四個列線圖用於預測VTE與出血風險
> itemize
> **臨床意義：**這些生物標記可能有助於優化正在接受化療的癌症患者VTE與出血風險評估，潛在協助初級血栓預防的決策制定。

## 研究背景


### 癌症與血栓/出血風險


癌症患者同時面臨靜脈血栓栓塞（VTE）與出血風險增加的雙重挑戰。了解發炎與凝血之間的相互關係對於風險評估至關重要：


> **發炎促進凝血的機制：**
> enumerate
>     \item 發炎誘導的凝血活化
>     \item 發炎誘導的天然抗凝血劑下調
>     \item 纖溶作用抑制
> enumerate
> **支持證據：**發炎性疾病（如類風濕性關節炎、系統性紅斑狼瘡、系統性硬化症、發炎性腸病）患者的VTE風險均增加。


### 研究動機與目的


- 先前研究顯示C反應蛋白（CRP）與VTE風險相關，但在癌症患者中的結果不一致
- GDF-15與VTE在癌症患者中的關聯仍有爭議
- 心臟生物標記（NT-proBNP、hs-TnT）在一般人群中與血栓風險相關，但在癌症患者中的角色尚不清楚
- 需要評估生物標記隨時間變化與臨床結果的關係

### 統計分析方法


- 生物標記數據經對數轉換並標準化（減去平均值後除以標準差）
- 使用Fine and Gray競爭風險迴歸模型，將死亡作為競爭風險
- 使用限制性三次樣條（restricted cubic splines）評估非線性關係
- 多變量模型調整年齡和晚期/轉移性癌症
- 缺失數據使用單次插補預測平均配對法

## 機制探討


### GDF-15與VTE的關聯機制


| p{11cm}}  **方面** | **說明** |  |
| --- | --- | --- |
| **方面** | **說明** |  |
| **GDF-15特性** | 壓力反應細胞因子，對發炎、氧化壓力、缺氧、端粒侵蝕和癌基因活化有反應 |  |
| **與CRP的區別** | GDF-15在發炎或壓力反應中的角色比CRP更廣泛（CRP主要對感染和發炎有反應） |  |
| **可能機制** | GDF-15表達通過激活Smad2/pSmad2/Snail通路，促進內皮-間質轉化，降低內皮細胞的抗血栓能力 |  |


### NT-proBNP與出血的關聯機制


| p{11cm}}  **方面** | **說明** |  |
| --- | --- | --- |
| **方面** | **說明** |  |
| **NT-proBNP特性** | 心臟標記，對心肌細胞伸展、缺氧和內分泌活化有反應 |  |
| **可能機制** | BNP抑制腎素-血管緊張素系統和交感神經系統，可能影響纖溶和止血 |  |
| **相關出血風險因素** | 升高的NT-proBNP與多個出血風險因素相關：低血紅蛋白、高血壓、轉移性疾病、系統性發炎 |  |


### hs-TnT變化與VTE的關聯


- hs-TnT是心肌損傷的標記
- 化療開始後hs-TnT上升可能反映化療對心血管系統的損害效應
- 先前研究已發現第1個月hs-TnT>14 pg/mL可預測未來VTE事件
- 本研究進一步證實hs-TnT的變化趨勢可作為VTE風險監測指標

### 風險評估優化


> **基線評估：**
> itemize

> itemize
> **血栓預防決策：**
> itemize
>     \item 預測VTE風險≥10
>     \item 這些生物標記可協助個體化治療決策
> itemize

## 研究限制


### 方法學限制


1. **事後分析：**無法建立因果關係，結果需要前瞻性研究驗證
2. **樣本量有限：**事件數量限制了控制所有潛在混淆因素的能力
3. **選擇性人群：**僅納入Khorana評分≥2的患者，結果可能不適用於所有癌症類型
4. **排除的癌症類型：**急性白血病和骨髓增殖性腫瘤被排除


### 測量相關限制


1. 生物標記在不同實驗室測量，可能存在數值變異
2. 測量時間點限於基線、第1個月和第3個月，可能無法捕捉其他時間點的變化
3. 缺乏詳細的共病資料，限制了評估這些因素對生物標記變異性的獨立貢獻


### 臨床應用限制


1. 這些生物標記目前並非常規測量項目
2. 研究未評估生物標記檢測的實用性和成本效益
3. 列線圖需要外部驗證


## 結論


> enumerate
>     \item **VTE預測：**基線GDF-15升高與VTE風險增加相關；hs-TnT和CRP的縱向變化也可預測VTE風險
>     \item **出血預測：**基線NT-proBNP和CRP升高與出血風險增加相關
>     \item **實用工具：**研究開發了四個列線圖，可用於癌症患者的風險評估和監測
>     \item **臨床價值：**這些生物標記可協助優化正在接受化療的癌症患者VTE和出血風險評估，潛在促進血栓預防的個體化決策
>     \item **未來方向：**需要更大規模的前瞻性研究來驗證這些發現和評估臨床應用價值
> enumerate

## 詞彙表


description[leftmargin=5cm,style=nextline]
    \item[VTE] Venous Thromboembolism，靜脈血栓栓塞症。包括深靜脈血栓（DVT）和肺栓塞（PE）。
    
    \item[DVT] Deep Vein Thrombosis，深靜脈血栓。靜脈內形成的血栓，常見於下肢。
    
    \item[PE] Pulmonary Embolism，肺栓塞。血栓脫落後阻塞肺動脈。
    
    \item[GDF-15] Growth Differentiation Factor-15，生長分化因子-15。壓力反應細胞因子，對多種壓力刺激（發炎、缺氧、氧化壓力）有反應。
    
    \item[CRP] C-Reactive Protein，C反應蛋白。急性期反應蛋白，主要對感染和發炎有反應。
    
    \item[NT-proBNP] N-terminal pro-B-type Natriuretic Peptide，N端腦鈉肽前體。心臟標記，反映心肌壓力和伸展。
    
    \item[hs-TnT] High-sensitivity Troponin T，高敏感度肌鈣蛋白T。心臟標記，反映心肌損傷。
    
    \item[Khorana評分] 用於評估癌症患者VTE風險的臨床評分系統，基於癌症類型、血小板計數、血紅蛋白等因素。
    
    \item[SHR] Subdistribution Hazard Ratio，次分布風險比。Fine and Gray競爭風險模型中的效應估計值。
    
    \item[AVERT試驗] Apixaban for the Prevention of Venous Thromboembolism in High-Risk Ambulatory Cancer Patients試驗。評估apixaban預防癌症相關VTE的隨機對照試驗。
    
    \item[CRNMB] Clinically Relevant Non-Major Bleeding，臨床相關非大出血。不符合大出血標準但具有臨床意義的出血事件。
    
    \item[ISTH] International Society on Thrombosis and Haemostasis，國際血栓與止血學會。
    
    \item[DOAC] Direct Oral Anticoagulant，直接口服抗凝劑。如apixaban、rivaroxaban等。
    
    \item[AUC] Area Under the Curve，曲線下面積。用於評估預測模型的區辨能力。
    
    \item[列線圖] Nomogram。圖形化的風險預測工具，將多個預測因子整合為單一預測概率。
