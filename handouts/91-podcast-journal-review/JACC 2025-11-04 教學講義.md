## 2025高血壓指引：核心內容


### 指引的重大更新

| --- | --- | --- | --- | --- | --- |
| **項目** | **內容** |  |  |  |  |
| 發布時間 | 2017年指引後的首次完整更新，    反映近**十年**的新證據 |  |  |  |  |
| ****血壓測量**** | 確認****診間外血壓測量****(out-of-office blood    pressure measurement)的核心角色：    • 居家血壓監測(home blood pressure monitoring,    HBPM)    • 24小時動態血壓監測(ambulatory blood pressure    monitoring, ABPM) |  |  |  |  |
| ****團隊照護**** | 提倡****團隊照護****(team-based care)作為改善    血壓控制率的關鍵要素：    • 醫師(physicians)    • 藥師(pharmacists)    • 護理師(nurse practitioners, nurses)    • 醫師助理(physician assistants/associates)    • 營養師(dietitians)    • 社區衛生工作者(community health workers) |  |  |  |  |


### 治療目標：關鍵數字

| --- | --- | --- | --- | --- | --- |
| **項目** | **內容** |  |  |  |  |
| 建議治療目標 | ****收縮壓至少<130 mmHg****    (systolic blood pressure at least below 130 mmHg) |  |  |  |  |
| ****更積極目標**** | ****鼓勵在可行時達到<120 mmHg****    (encouragement to reach below 120 mmHg when feasible) |  |  |  |  |

| 支持證據 | 基於強化降壓治療試驗(intensive treatment trials)：    • **SPRINT試驗**(Systolic Blood Pressure Intervention    Trial)    • **STEP試驗**(Strategy of Blood Pressure Intervention    in the Elderly Hypertensive Patients)    • **FREE試驗** |  |  |  |  |
| 重申意義 | ****指引再次確認****：我們應該治療到比以前認為    可能或期望的**更低的血壓水平** |  |  |  |  |


### 其他重要更新


| p{4cm}>{\arraybackslash}p{11cm}} 2025指引的創新內容 |  |  |  |  |  |

| **新風險評估工具** | 引入****PREVENT calculator****：    (Predicting Risk of Cardiovascular Disease EVENTs)        **取代**：Pooled Cohort Equations (PCE)        **優勢**：    • 納入身體質量指數(BMI)    • 更寬的年齡範圍(30-79歲)    • 更詳細的腎功能評估    • **不包含種族作為決定因素**        **使用**：    用於第一期高血壓患者決定是否需要藥物治療：    • 10年心血管風險<7.5  • 10年心血管風險≥7.5 |  |  |  |  |
| **大腦健康** | 強調****血壓控制作為保護大腦健康和降低    失智風險的手段****    (preserve brain health and reduce dementia risk) |  |  |  |  |
| **次發性高血壓** | 擴大對次發性高血壓的評估，    特別是****原發性醛固酮增多症****    (primary aldosteronism)：        • 所有**頑固性高血壓**(resistant hypertension)患者    都應篩檢，**不論血鉀濃度**    • 高達1/3頑固性高血壓患者有醛固酮過多    • 可能受益於特定介入(手術或藥物) |  |  |  |  |
| **居家血壓監測** | 支持居家血壓監測(home blood pressure monitoring)：    • 用於**診斷**(diagnosis)    • 用於**管理**(management) |  |  |  |  |
| **初始治療策略** | 建議大多數需要藥物治療的患者使用    ****單一複方藥物****(single-pill combinations)作為    初始治療 |  |  |  |  |
| **整合性觀點** | 將高血壓連結到更廣泛的    ****心腎代謝疾病****(cardio-kidney-metabolic disease,    CKM)架構 |  |  |  |  |
| 前瞻性 | ****非常前瞻性的文件****，強調迫切需要    將證據轉化為**人群層級的影響**    (population-level impact) |  |  |  |  |

| --- | --- | --- | --- | --- | --- |
| **概念** | **內容** |  |  |  |  |
| 編輯頁標題 | ****「嚴重高血壓：下一個永不事件」****    (Severe Hypertension: The Next Never Event) |  |  |  |  |
| 核心論點 | 應該將****收縮壓>160 mmHg****視為    ****永遠不應該在沒有偵測和治療的情況下發生****    的事件 |  |  |  |  |
| 類比 | 正如我們透過問責制度系統(systems of accountability)    消除了**錯誤部位手術**(wrong-site surgery)，    我們也可以建立系統讓未控制的嚴重高血壓同樣罕見 |  |  |  |  |

| 願景效益 | 使嚴重高血壓成為永不事件不僅會：    • **拯救生命**    • **強化全面高血壓控制的基礎** |  |  |  |  |

| ****團隊照護**** | • 與藥師、護理師合作    • 建立高血壓照護團隊    • 利用遠距醫療和數位工具 |  |  |  |  |
| ****篩檢次發性    高血壓**** | • 頑固性高血壓患者應篩檢原發性醛固酮增多症    • 不需等到有低血鉀才篩檢    • 測量aldosterone、renin和鉀離子 |  |  |  |  |
| ****生活方式介入**** | • 重視DASH飲食    • 評估睡眠品質    • 壓力管理    • 運動處方 |  |  |  |  |
| ****藥物治療**** | • 考慮使用複方單一藥片作為初始治療    • 簡化治療方案以提高順從性    • 注意GLP-1 RA在肥胖高血壓患者的雙重效益 |  |  |  |  |
| ****風險評估**** | • 熟悉PREVENT calculator    • 用於決定第一期高血壓患者是否需要立即藥物治療    • 整合CKM風險概念 |  |  |  |  |
| ****嚴重高血壓**** | • 將收縮壓>160 mmHg視為需要緊急處理    • 建立系統性篩檢和追蹤機制    • 避免任何嚴重高血壓患者流失追蹤 |  |  |  |  |
