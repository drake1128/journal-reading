# EuroIntervention 2025-12-01 calcified nodules

**整理：謝慕揚 MD, PhD, FESC**
**日期：2025-01-01**

---

\maketitle


> **重要發現：**鈣化結節（Calcified Nodules, CNs）是經皮冠狀動脈介入術（PCI）中最具挑戰性的病變類型之一，其複雜的形態學特徵、不同的治療反應以及與支架失敗的相關性（包括結節再突出或支架擴張不良）使其成為臨床關注的焦點。
> **分類意義：**CNs可分為噴發型（eruptive）和非噴發型（non-eruptive）兩種亞型，各具獨特的組織學特徵和預後意義。噴發型CNs具有生物活性，與纖維帽破裂、表面血栓和斑塊內出血相關；非噴發型CNs則相對穩定，纖維帽完整。
> **臨床啟示：**理解CNs的可變形性對優化治療結果至關重要。增強血管造影和血管內成像技術大幅提升了我們對CNs的認識，有助於準確識別結節亞型並指導治療策略。

## 病理生理學


### 噴發型鈣化結節的形成機制


> **核心特徵：**
> enumerate
>     \item 多個緻密鈣化結節碎片穿透覆蓋的纖維帽
>     \item 內皮細胞丟失
>     \item 向管腔突出，表面覆蓋纖維蛋白/血小板血栓
>     \item 壞死核心鈣化（缺乏膠原）主要位於CNs的罪犯節段
>     \item 近端和遠端側翼節段為富含膠原的鈣化
> enumerate


| p{11cm}} CNs發展的假說機制 |  |  |
| --- | --- | --- |
| **機制** | **說明** |  |
| **機制** | **說明** |  |
| **壞死核心鈣化崩解** | 噴發型CN可能源於缺乏膠原的壞死核心鈣化的崩解，這些鈣化被剛性、富含膠原的鈣化板夾在中間 |  |
| **鉸鏈運動理論** | 動脈運動可能阻止鉸鏈處鈣化板的膠原化，導致形成無法融合但易於因動脈運動而噴發的鈣化碎片 |  |
| **機械應力作用** | 冠狀動脈鉸鏈運動產生的機械力在噴發型CNs的演變中起關鍵作用，缺乏膠原的壞死核心鈣化比側翼富含膠原的鈣化板更易受機械應力影響 |  |
| **血管新生作用** | CNs內的血管新生可能促進其演變，不同時期的纖維蛋白沉積提示反覆的毛細血管損傷或滲漏 |  |
| **循環過程** | 毛細血管破壞和纖維蛋白沉積的循環過程可能驅動結節鈣化的進行性生長和突出，最終導致其向管腔內突出 |  |

| **特徵** | **說明** |  |
| **特徵** | **說明** |  |

### 可變形性概念


> **定義：**在OCT研究中，如果治療後管腔突出可見減少，伴隨更對稱的支架擴張（不對稱指數和偏心指數>0.7），則認為CN是可變形的。
> **臨床意義：**
> itemize
>     \item ****噴發型CNs：****幾乎普遍可變形，與其碎裂的壞死核心鈣化成分一致
>     \item ****非噴發型CNs（可變形）：****與主要軟性殘餘脂質相關，可被壓入較軟的中心核心
>     \item ****非噴發型CNs（不可變形）：****與底層鈣化成分相關，無法被壓縮抵抗剛性鈣化結構
> itemize

## 治療挑戰


### PCI後不良預後的因素


> enumerate
>     \item **支架擴張不良風險：**不可變形CNs導致的支架擴張不足
>     \item **矛盾的長期預後：**噴發型與非噴發型CNs有不同的即時治療反應但相反的長期結果
>     \item **早期再突出風險：**CNs通過支架網孔再突出，可能導致支架內再狹窄
>     \item **支架斷裂風險：**由鉸鏈運動引起，因CNs常見的解剖位置
>     \item **鄰近重度鈣化：**CNs旁常有重度鈣化板，導致最小支架面積（MSA）常在CN位點以外
>     \item **患者層面風險因素：**與CNs常相關的慢性腎病等心血管危險因素
> enumerate

## 建議治療策略


### 治療流程


> **第一步：血管內成像**  
> 考慮到血管內成像引導PCI在複雜冠狀動脈病變（包括嚴重鈣化病變）中的益處證據日益增加，在血管造影發現中度至重度鈣化的情況下應考慮使用血管內成像。
> **第二步：可變形性評估**  
> 對於噴發型和非噴發型CNs，使用1:1 NC或特殊球囊進行高壓球囊擴張評估可變形性是合理且實用的初始方法：
> itemize
>     \item 噴發型CNs更可能可變形
>     \item 非噴發型CNs可能不可變形
> itemize
> **第三步：進階病變準備**  
> 如果在兩個正交視角使用增強透視無法達到完全球囊擴張：
> itemize
>     \item CNs很可能不可變形
>     \item 需要進階病變準備策略
>     \item 策略選擇應基於操作者偏好、CN位點管腔大小、導絲偏移情況
> itemize
> **特殊情況：**
> itemize
>     \item 無法輸送裝置或病變嚴重程度和鈣化負荷高（依鈣化評分系統）時，可考慮前期進階病變準備
>     \item 無論使用何種方式，理想情況下應在支架植入前達到血管造影上的完全球囊擴張
> itemize

## 結論


### 主要結論


1. ****病變類型：****CN是一種獨特的鈣化冠狀動脈病變類型，代表PCI中最具挑戰性的病變亞群之一
2. ****診斷技術：****增強血管造影和血管內成像提供了識別CNs並區分其亞型（噴發型和非噴發型）的能力
3. ****亞型差異：****兩種亞型展示了不同的形態學特徵、治療反應和PCI後長期預後
4. ****治療指導：****深入理解這些特徵，結合當前證據，對指導操作者選擇最有效的治療策略至關重要
5. ****持續挑戰：****儘管使用進階病變準備技術，CNs的PCI後結果仍不理想
6. ****未來方向：****需要進一步努力增強和改進治療方法以改善臨床結果


### 臨床實務要點


> **識別要點：**
> itemize
>     \item 血管造影嚴重鈣化，特別是在鉸鏈點或左主幹分叉處，應提高對CNs的警覺
>     \item 透析和慢性腎病ACS患者約40
>     \item 血管內成像對區分亞型至關重要
> itemize
> **治療要點：**
> itemize
>     \item 球囊擴張可評估可變形性，指導後續策略
>     \item 不可變形CNs需要進階病變準備
>     \item IVL不受導絲偏移影響，對整個鈣化病變準備有利
>     \item 旋切術效果受導絲偏移和管腔大小影響
>     \item 支架植入前應追求完全球囊擴張
> itemize
> **預後考量：**
> itemize
>     \item 噴發型CNs儘管可變形性好，長期預後較差
>     \item 需警惕CN再突出和支架斷裂風險
>     \item MSA常在CN位點以外，需全面處理鈣化病變
> itemize

## 詞彙表


description[leftmargin=5cm,style=nextline]
    \item[Calcified Nodule (CN)] 鈣化結節。一種獨特的鈣化冠狀動脈病變，以凸面向管腔突出，常伴有鄰近重度鈣化板。
    
    \item[Eruptive CN] 噴發型鈣化結節。具有生物活性的CN亞型，特徵為纖維帽破裂、表面血栓和斑塊內出血，常與ACS相關。
    
    \item[Non-eruptive CN] 非噴發型鈣化結節，或稱結節鈣化（nodular calcification）。較穩定的CN亞型，纖維帽完整，常表現為慢性冠狀動脈症候群或無症狀。
    
    \item[Deformability] 可變形性。指CN在治療後管腔突出減少的能力，與支架擴張對稱性相關。
    
    \item[Hinge Motion] 鉸鏈運動。冠狀動脈在彎曲點的機械運動，被認為在噴發型CN的形成中起關鍵作用。
    
    \item[Reprotrusion] 再突出。指植入支架後CN物質通過支架網孔再次突入管腔的現象。
    
    \item[Necrotic Core Calcification] 壞死核心鈣化。缺乏膠原的鈣化類型，主要位於CN的罪犯節段，較易受機械應力影響。
    
    \item[Collagen-rich Calcification] 富含膠原的鈣化。主要位於CN側翼節段的鈣化類型，形成剛性鈣化板。
    
    \item[Intravascular Lithotripsy (IVL)] 血管內震波術。利用聲波能量斷裂鈣化的治療技術，不受導絲偏移影響。
    
    \item[Guidewire Bias] 導絲偏移。在使用旋切裝置時，導絲位置對裝置軌跡和治療效果的影響。
    
    \item[Angiographic Radiolucent Mass (ARM)] 血管造影透亮腫塊。血管造影上管腔內的透亮充盈缺損，提示CN存在但也可能代表血栓。
