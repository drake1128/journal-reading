# GRSF1 與心臟衰竭代謝調控

# GRSF1：心臟衰竭的代謝新標靶

- **GRSF1** (G-rich RNA sequence binding factor 1) 是一種 RNA 結合蛋白
- 在**衰竭心臟**中表達顯著降低
- 調控支鏈胺基酸 (branched-chain amino acids, **BCAAs**) 的分解代謝
- 來自北京大學許明教授團隊

> **BCAA 堆積**已知與心臟衰竭進展相關

---

# GRSF1 研究：主要發現

| 實驗 | 結果 |
|------|------|
| 衰竭心臟 GRSF1 表達 | **顯著降低** |
| GRSF1 缺失 | BCAA 代謝酶 mRNA 不穩定 → BCAA 堆積 |
| 實驗性恢復 GRSF1 | 心臟耐受壓力↑、心功能維持、結構損傷↓ |

> **Pearl**: 透過支持心肌代謝 (metabolic support) 而非單純抑制神經激素系統，可能開啟 HF 治療新方向

# PVB19 兒童心肌炎：研究設計

- **多中心回顧性研究**：11 個義大利兒童心臟中心
- 通訊作者：Dr. Enrico Ammirati (Milan)
- 收案期間：2022 年 1 月至 2024 年 10 月
- **66 例** PVB19 陽性 vs **82 例** PVB19 陰性對照
- 另含 Lombardy 地區 2004-2024 年 population-based 趨勢

---

# PVB19 心肌炎：病人特徵

| 特徵 | 數據 |
|------|------|
| 中位年齡 | **32 個月** |
| ≤7 歲 | **77%** |
| 男性 | 55% |
| 2024 年收治 | **58/66 (88%)** — 爆發年 |
| 組織學：lymphocytic myocarditis | 23/26 (**88%**) |

---

# PVB19 心肌炎：令人警醒的預後

| 結局 | PVB19 陽性 | PVB19 陰性 |
|------|-----------|-----------|
| **死亡/心臟移植/LVAD** | **13/66 (20%)** | **1/82 (1%)** |

> **Pearl**: PVB19 心肌炎並非良性！兒童 PVB19 感染合併心臟症狀 → 高度警覺

> 88% 病例集中於 2024 年爆發期，提示流行期間需系統性心臟篩檢

# ICI 心肌炎：Anti-LAG-3 的新風險

- **研究者**：Dr. Javid Moslehi (UCSF)
- 國際藥物警戒資料分析 + 基礎科學研究
- **Anti-LAG-3 + anti-PD-1** 組合 vs anti-PD-1 單藥

| 發現 | 說明 |
|------|------|
| 心肌炎風險 | Anti-LAG-3 + anti-PD-1 **顯著高於** anti-PD-1 alone |
| 關鍵致病細胞 | **CXCR6+ T cells** |
| 病理機轉 | CXCR6+ T cells → 心肌發炎 → 損傷 + 心律不整 |
| 治療展望 | 標靶 CXCR6 → 可能預防 ICI 心肌炎 |

---

# ICI 心肌炎：臨床啟示

> **Cardio-oncology 重點**：任何 ICI 治療病人出現新發 troponin 上升、心律不整或心功能下降 → 高度懷疑 ICI 心肌炎

- 處方 anti-LAG-3 + anti-PD-1 前：
  - Baseline **troponin**
  - Baseline **ECG**
  - 考慮 baseline **echocardiography**
- CXCR6 抑制劑：未來可能的預防策略

# 肥胖與 CV 結局：研究規模

- **研究者**：Dr. Michael Blaha (Johns Hopkins)
- **24 個 NIH 世代研究**合併分析
  - Framingham, MESA, ERiC, CHS, CARDIA 等
- **~300,000-400,000** 參與者
- **~20 年**追蹤

| WHO 肥胖分級 | BMI (kg/m²) |
|-------------|-------------|
| Class 1 | 30.0 - 34.9 |
| Class 2 | 35.0 - 39.9 |
| Class 3 | **≥40.0** |

---

# 九項心血管結局

分析涵蓋：

| 結局 | 與肥胖的關聯強度 |
|------|-----------------|
| **Heart Failure** | **最強** |
| **Atrial Fibrillation** | **最強** |
| Fatal CV Events | 強 |
| Myocardial Infarction | 中等 |
| All-cause Mortality | 中等 |
| Stroke | **較弱** |
| Composite endpoints | 視組成而定 |

---

# 關鍵發現

- **劑量效應**：肥胖等級越高 → CV 風險越高
- **Class 2-3 obesity**：風險信號最強
- 肥胖與**致死性**事件的關聯 > 非致死性
- 目前**每 10 人中有 1 人 BMI >40**

> **性別差異**：相同 BMI 下，**女性的心血管風險高於男性**

---

# 肥胖研究：臨床與公衛啟示

### GLP-1 RA 時代的意義
- Semaglutide, tirzepatide 等藥物的心血管獲益有流行病學基礎
- **Class 2-3 obesity** 病人積極減重 → 最大心血管風險降低

### 公共衛生
- 早期介入 (early intervention)：生活型態 + 藥物 + 必要時 bariatric surgery
- 人口層面政策改變刻不容緩

> **Pearl**: HF 與 AF 是與肥胖關聯最密切的結局 — 反映 volume overload、left atrial enlargement、myocardial fibrosis 的病理機轉

---

# 本期 Take-Home Messages

1. **GRSF1**：RNA 結合蛋白調控 BCAA 代謝，為 HF 代謝治療開新路
2. **PVB19 心肌炎**：兒童預後極差 (20% 死亡/移植/LVAD)，非良性疾病
3. **ICI 心肌炎**：Anti-LAG-3 + anti-PD-1 風險↑，CXCR6 為治療標靶
4. **肥胖 CV 風險**：劑量效應明確，HF/AF 關聯最強，女性風險更高
