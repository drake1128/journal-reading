## 本期焦點：運動RV-PA耦合在原發性二尖瓣逆流之預後價值

請參考期刊 [Prognostic Value of Exercise Right Ventricular–Pulmonary Arterial Coupling in Primary Mitral Regurgitation, Circulation
Volume 152, Number 23, 2025. ](https://www.ahajournals.org/doi/10.1161/CIRCULATIONAHA.125.073778)


### 研究背景與重要性


> **原發性二尖瓣逆流（Primary Mitral Regurgitation, PMR）的風險分層困難：**
> itemize[leftmargin=*]
>     \item 患者可能血流動力學顯著但仍無症狀或僅輕微症狀
>     \item 即使無介入指徵，仍可能發生不良心血管事件
>     \item 需要可靠的風險分層標記以指導臨床決策
>     \item 關鍵問題：何時介入？何時密切追蹤？
> itemize

### 研究限制


1. 排除具第一類介入指徵患者，可能影響對疾病晚期的推廣性
2. 排除持續性/永久性心房顫動患者
3. 主要為二尖瓣脫垂患者，對更廣泛PMR族群的推廣性需驗證
4. 測量變異性（已透過標準化方法和振盪膠體減輕）
5. 已透過外部驗證世代確認結果可重複性


## CMR引導心室性心搏過速消融之幾何特徵分析


### 研究背景


> **研究團隊：**Dr. Saman Nazarian，賓州大學醫學院
> **目標：**描述缺血性心肌病變患者中，從延遲釓增強心臟磁振造影（LGE-CMR）衍生之可行走廊的幾何特徵，以及這些走廊如何參與心室性心搏過速（VT）迴路。


### 研究方法


| p{11cm}} 研究設計 |  |  |
| --- | --- | --- |
| **項目** | **內容** |  |
| **項目** | **內容** |  |
| **研究對象** | 缺血性心肌病變且VT消融前有CMR檢查之患者 |  |
| **影像處理** | minipage[t]{11cm} • 從LGE影像計算穿越梗塞組織的可行走廊 |  |
| • 延遲增強：白色（疤痕）對比黑色（正常心肌） |  |  |
| **電生理整合** | 與電解剖圖共同配準（Co-registration），以拖曳（Entrainment）或起搏標定（Pace mapping）確認VT走廊入口/出口位點 |  |
| **評估參數** | 走廊形狀、入口角度、寬度、長度、厚度、體積、腔室可及性 |  |


### 主要發現


| p{11cm}} VT走廊幾何特徵 |  |  |
| --- | --- | --- |
| **形狀類型** | **特徵** |  |
| **形狀類型** | **特徵** |  |
| **雙曲面型（Hyperboloid）** | 大多數VT走廊與此模型最吻合 |  |
| **漏斗型（Funnel）** | 部分走廊呈此形態 |  |
| **圓柱型（Cylinder）** | 部分走廊呈此形態 |  |
| **入口/出口特徵** | 中央雙曲線在出口口部的走廊角度較入口口部大 |  |


### 臨床意義


> **潛在價值：**
> itemize[leftmargin=*]
>     \item 結合先進CMR影像與VT消融之解剖-電生理整合
>     \item 未來可能利用CMR影像引導消融位置選擇
>     \item 提升VT消融的精準度和成功率
>     \item 減少消融時間和併發症
> itemize

#### 血壓與事件關係之差異


> **無心臟壓力組：**
> itemize[leftmargin=*]
>     \item 收縮壓與CVD事件呈U型關係

> itemize[leftmargin=*]

### 臨床意義


1. NT-proBNP定義之心臟壓力可能有助於指導個人化血壓管理
2. 有心臟壓力者可能受益於更積極的血壓控制
3. 無心臟壓力者過低血壓可能增加風險（J曲線效應）
4. 需要前瞻性研究驗證此策略是否改善預後
