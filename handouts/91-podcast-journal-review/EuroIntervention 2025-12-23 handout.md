### 主要內容


#### 次發性二尖瓣逆流的病理生理學


次發性二尖瓣逆流（Secondary Mitral Regurgitation, SMR）常見於心衰竭合併射血分數降低（HFrEF）患者，與死亡率增加、住院頻率上升及生活品質下降密切相關。其發生機制為多因素性，主要涉及：


- 左心室重塑導致瓣膜牽拉（Ventricular SMR, V-SMR）
- 心房擴大導致瓣環擴張（Atrial SMR, A-SMR）
- 兩種機制之混合型態


#### 指引建議現況


> itemize[leftmargin=*]
>     \item **ACC/AHA指引**：主要參考 COAPT 及 MITRA-FR 試驗
>     \item **2025年ESC指引**：納入 RESHAPE-HF2 試驗之新證據，建議症狀性 A-SMR 患者可考慮 TEER（Class IIb）
>     \item 區分 A-SMR 與 V-SMR 之標準化診斷準則仍待建立
> itemize


#### 三大標誌性隨機試驗比較


| p{3.5cm}p{3.5cm}p{3.5cm}} COAPT、MITRA-FR 及 RESHAPE-HF2 試驗比較 |  |  |
| --- | --- | --- |
| **特徵** | **COAPT** | **MITRA-FR** |
| **特徵** | **COAPT** | **MITRA-FR** |

| 主要終點 | 2年心衰住院 | 1年死亡或心衰住院 |

| 病患特徵 | 較嚴重 SMR、較小 LV 容積 | 較不嚴重 SMR、較大 LV 容積 |
| 臨床意義 | 定義適合族群標準 | 強調患者選擇重要性 |


### 臨床實務要點


1. 患者選擇是 TEER 成功的關鍵因素
2. 優化藥物治療仍為 HFrEF 管理的基石
3. 新裝置的發展持續擴展 TEER 的適應症範圍
4. 多專科團隊評估對治療決策至關重要

### 研究背景


左心耳封堵術（Left Atrial Appendage Closure, LAAC）已成為心房顫動高出血風險患者預防中風的重要替代療法。Watchman FLX 為新一代裝置，相較於前代 Watchman 2.5 具有多項設計改良。


### 研究方法與結果


| p{5cm}p{5cm}} Watchman FLX vs 2.5 主要結果比較 |  |  |
| --- | --- | --- |
| **指標** | **Watchman FLX** | **Watchman 2.5** |
| **指標** | **Watchman FLX** | **Watchman 2.5** |
| 裝置相關血栓 | 較低 | 較高 |
| 裝置周圍滲漏 | 較低 | 較高 |
| 植入成功率 | 相當 | 相當 |
| 手術相關併發症 | 相當 | 相當 |


> 經propensity score matched analysis分析後，新一代 Watchman FLX 在影像追蹤時展現：
> itemize[leftmargin=*]
>     \item **裝置相關血栓發生率較低**
>     \item **裝置周圍滲漏發生率較低**
>     \item 臨床結果與前代裝置相當
> itemize


### 臨床意義


此研究為 Watchman FLX 在真實世界中的效能提供了重要證據，支持其作為左心耳封堵術的優先選擇。

### 研究背景


ACURATE neo/neo2 為自膨式經導管主動脈瓣膜，瓣膜擴張不全（underexpansion）會影響血流動力學表現及臨床結果。了解其預測因子對於改善手術結果至關重要。


### 主要發現


| p{8cm}} 瓣膜擴張不全之預測因子 |  |  |
| --- | --- | --- |
| **因子** | **與擴張不全之關聯** |  |
| **因子** | **與擴張不全之關聯** |  |
| 瓣環鈣化量過多 | 增加擴張不全風險 |  |
| 後擴張（post-dilatation） | 降低擴張不全風險 |  |
| 植入深度 | 影響瓣膜展開 |  |
| 瓣環尺寸選擇 | 影響最終擴張程度 |  |


> itemize[leftmargin=*]
>     \item 過量的瓣環鈣化與較高的擴張不全風險相關
>     \item 後擴張（post-dilatation）可有效降低擴張不全風險
>     \item 需進一步研究以確認其他經導管瓣膜是否有類似的擴張不全率
>     \item 術前 CT 評估鈣化分布對於預測和預防擴張不全至關重要
> itemize

### 研究背景


曾接受 Bio-Bentall 手術或瓣膜保留主動脈根部修復術（Valve-Sparing Root Repair, VSRR）的患者，日後可能需要二次介入。Valve-in-root TAVI 的可行性取決於多項解剖條件。


### 終身管理之解剖學考量


| p{10cm}} 影響 Valve-in-root TAVI 可行性之解剖因素 |  |  |
| --- | --- | --- |
| **解剖條件** | **考量重點** |  |
| **解剖條件** | **考量重點** |  |
| 瓣環面積 | 需確保足夠空間容納經導管瓣膜 |  |
| 冠狀動脈開口高度 | 決定冠狀動脈阻塞風險 |  |
| 瓣膜至冠狀動脈距離 | 影響經導管瓣膜植入後的冠狀動脈通暢性 |  |
| 竇管交界形態 | 影響瓣膜定位及固定 |  |


> **首次手術的前瞻性規劃：**
> itemize[leftmargin=*]
>     \item 在首次主動脈根部手術時即考量未來 TAVI 的可行性
>     \item 確保足夠的冠狀動脈開口高度
>     \item 選擇適當的生物瓣膜類型以利未來 valve-in-valve 介入
>     \item 詳細記錄手術解剖細節以供日後參考
> itemize

### RADIANCE-HTN TRIO：降壓藥物調整分析


| p{11cm}}  **項目** | **內容** |  |
| --- | --- | --- |
| **項目** | **內容** |  |
| 作者 | Victor J.M. Zeijen, Michel Azizi, Ajay J. Kirtane, Joost Daemen 等 |  |
| DOI | [10.4244/EIJ-D-25-00482](https://doi.org/10.4244/EIJ-D-25-00482) |  |
| 研究主題 | 分析超音波腎臟去神經術之效果，並校正檢測到的降壓藥物變化 |  |


> **研究設計：**
> itemize[leftmargin=*]
>     \item 頑固性高血壓患者在接受標準化三合一藥物治療後隨機分配至超音波腎臟去神經術（uRDN）或假手術
>     \item 使用尿液化學依從性測試監測降壓藥物
> itemize
> **主要發現：**
> itemize[leftmargin=*]
>     \item 校正降壓藥物變化後，uRDN 組日間動態收縮壓下降幅度較假手術組顯著
>     \item 基線校正後組間差異為 −6.9 mmHg（p=0.007）
>     \item 進一步校正檢測到的降壓藥物數量後，組間差異為 −9.3 mmHg（p<0.001）
> itemize
