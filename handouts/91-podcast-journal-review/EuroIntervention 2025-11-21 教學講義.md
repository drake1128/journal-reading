## 概述


本期EuroIntervention期刊收錄多項重要的介入性心臟病學研究，涵蓋範圍包括機械循環支持、冠狀動脈介入治療、抗血小板治療、光學同調斷層掃描(OCT)影像導引、腎交感神經去除術等前沿領域。這些研究為臨床實務提供了重要的循證醫學證據，並為未來的治療策略發展指引方向。


## 主要研究內容


### PROTECT IV試驗：機械循環支持研究


#### 研究背景與目的

Norman Mangner、Gregg W. Stone及研究團隊正在進行的PROTECT IV試驗，旨在評估在高風險心衰竭患者接受經皮冠狀動脈介入治療(PCI)時，使用Impella CP進行血流動力學支持的有效性。


#### 研究設計

| --- | --- | --- |
| **項目** | **內容** |  |
| **研究類型** | 隨機對照試驗 |  |
| **收案對象** | minipage[t]{11cm} - 射出分率降低之心衰竭患者    - 複雜冠狀動脈疾病    - 需接受經皮冠狀動脈介入治療    - 高風險族群 |  |

#### 臨床意義

此研究將為複雜PCI患者的機械循環支持策略提供重要證據，有助於確立Impella CP在此類高風險患者中的角色定位。


### 急性冠心症患者：OCT與血管攝影導引之比較


#### 研究背景

Yong-Joon Lee、Byeong-Keuk Kim及研究團隊對OCCUPI Trial進行事後分析，專注於急性冠心症(ACS)患者在複雜病灶接受藥物洗脫支架置放時的影像導引策略。


#### 主要發現


| p{11cm}}  **比較項目** | **研究結果** |  |
| --- | --- | --- |
| **比較項目** | **研究結果** |  |
| **1年MACE發生率** | OCT導引組顯著低於血管攝影導引組 |  |
| **獲益最大族群** | minipage[t]{11cm} 透過OCT達到支架最佳化的患者獲益最為顯著    - 支架貼壁良好    - 支架擴張充分    - 無邊緣夾層或組織脫垂 |  |
| **機轉探討** | minipage[t]{11cm} Tom Adriaenssens與Pascal Frederiks在隨附社論中討論可能機轉：    - OCT提供更精確的病灶評估    - 能即時偵測支架置放問題    - 允許術中優化處理策略 |  |
| **文獻來源** | [EuroIntervention 2025;21:e1269](https://eurointervention.pcronline.com) |  |


#### 臨床應用建議


1. ACS患者接受複雜病灶PCI時，應考慮使用OCT導引
2. 術中應追求達到OCT定義的支架最佳化標準
3. 特別適用於高風險病灶（分叉病變、長病灶、鈣化病灶等）


### 丹麥心肌梗塞患者抗血小板藥物使用趨勢變化


#### 研究背景與目的

Mia Ravn Jacobsen、Rikke Sørensen及研究團隊檢視丹麥在心肌梗塞患者接受PCI治療時，從常規使用ticagrelor轉變為prasugrel的政策變化及其臨床影響。


#### 研究發現


| p{10cm}}  **評估項目** | **研究結果** |  |
| --- | --- | --- |
| **評估項目** | **研究結果** |  |
| **用藥趨勢變化** | minipage[t]{10cm} 雙重抗血小板治療策略的轉變：    - 從ticagrelor為主    - 轉為prasugrel為主    - 政策性改變影響全國實務 |  |
| **心血管事件** | minipage[t]{10cm} Prasugrel組相較於ticagrelor組：    - 主要心血管不良事件(MACE)發生率降低    - 心肌梗塞復發率降低    - 具統計學顯著差異 |  |
| **出血併發症** | 兩組間出血發生率無顯著差異 |  |
| **專家評論** | minipage[t]{10cm} Adnan Kastrati與Fiorenzo Simonetti在隨附社論中指出：    - 確認prasugrel在MI患者的療效優勢    - 未增加出血風險的發現具重要意義    - 支持prasugrel作為首選P2Y12抑制劑 |  |
| **文獻來源** | [EuroIntervention 2025;21:e1259](https://eurointervention.pcronline.com) |  |


#### 臨床實務建議


1. 急性心肌梗塞患者接受PCI後，prasugrel可作為優先選擇
2. 相較於ticagrelor，prasugrel在降低缺血事件方面可能更具優勢
3. 兩種藥物的出血風險相當，不應成為選擇障礙
4. 仍需依個別患者特性（年齡、體重、中風病史等）進行個人化評估


### 腎交感神經去除術後血壓變化預測因子


#### 研究背景

Roland E. Schmieder、Michael Böhm及研究團隊透過整合五項臨床研究的資料，探討腎交感神經去除術(RDN)後血壓降低的預測因子。


#### 研究方法與結果


| p{11cm}}  **研究面向** | **內容描述** |  |
| --- | --- | --- |
| **研究面向** | **內容描述** |  |
| **研究設計** | minipage[t]{11cm} - 整合分析五項臨床研究    - 評估基準收縮壓(SBP)與預期收縮壓變化之關聯    - 追蹤期間：術後6個月 |  |
| **主要發現** | minipage[t]{11cm} ****基準收縮壓越高，RDN術後降壓幅度越大****    - 呈現正相關    - 具統計學顯著性    - 可作為療效預測參考 |  |
| **臨床應用** | minipage[t]{11cm} 雖無法提供精確預測，但有助於：    - 介入醫師與患者的溝通    - 治療決策制定    - 療效期待值設定    - 患者篩選參考 |  |
| **侷限性** | minipage[t]{11cm} - 無法精確預測個別患者降壓幅度    - 仍有其他未知影響因子    - 需更多研究確認其他預測因子 |  |
| **文獻來源** | [EuroIntervention 2025;21:e1281](https://eurointervention.pcronline.com/article/long-term-effects-of-radiofrequency-based-renal-denervation-on-blood-pressure-and-renal-function-by-degree-of-renal-dysfunction) |  |


#### 臨床意義


1. 基準血壓可作為RDN療效的初步預測指標
2. 有助於患者篩選與諮詢
3. 基準血壓較高的患者可能獲益更多
4. 需結合其他臨床因素進行綜合評估


### 腎交感神經去除術與慢性腎臟病


#### 研究問題

Verdiana Galli、Flavio Ribichini及研究團隊在研究通訊中探討：RDN對慢性腎臟病(CKD)患者的血壓及腎功能在中長期的影響為何？


#### 研究發現


| p{11cm}}  **評估項目** | **研究結果** |  |
| --- | --- | --- |
| **評估項目** | **研究結果** |  |
| **安全性評估** | minipage[t]{11cm} 在篩選後的CKD患者中：    - RDN手術安全性良好    - 未觀察到明顯腎功能惡化    - 無嚴重併發症發生 |  |
| **療效評估** | minipage[t]{11cm} - 有效降低血壓    - 降壓效果持續存在    - 中長期追蹤顯示療效維持 |  |
| **額外效益** | minipage[t]{11cm} ****可能具有超越降壓的額外好處：****    - 改善腎臟血流動力學    - 減少交感神經過度活化對腎臟的傷害    - 可能延緩腎功能惡化進程    - 需更多研究證實 |  |
| **適用族群** | minipage[t]{11cm} 精選的CKD患者：    - 高血壓控制不佳    - 藥物治療效果有限或耐受性差    - 腎動脈解剖結構適合    - 無禁忌症 |  |
| **文獻來源** | [EuroIntervention 2025;21:e1288](https://eurointervention.pcronline.com) |  |


#### 臨床啟示


1. RDN在精選CKD患者中是安全可行的治療選項
2. 可能提供超越降壓的腎臟保護效益
3. 需建立明確的患者篩選標準
4. 應納入多專科團隊評估（心臟科、腎臟科）
5. 需要更長期的追蹤研究確認腎功能保護效果


## 其他重要研究議題


### OCT影像特殊發現：手風琴效應模擬斑塊破裂


本期亦收錄關於OCT影像判讀的重要病例報告，提醒臨床醫師注意「手風琴效應」(concertina effect)可能在OCT影像上模擬斑塊破裂的表現，此類影像偽影可能導致誤判，影響治療決策。


#### 鑑別要點


1. 仔細評估病灶整體形態
2. 注意血管壁的連續性
3. 結合血管攝影與臨床表現
4. 必要時使用其他影像模式確認


## 綜合討論


### 機械循環支持的發展

PROTECT IV試驗代表機械循環支持領域的重要進展，為高風險PCI患者提供更積極的血流動力學支持策略。隨著裝置技術進步與臨床證據累積，預期此領域將持續發展。


### 影像導引PCI的重要性

OCT導引PCI在ACS患者中展現的療效優勢，再次強調精確影像導引在複雜冠狀動脈介入治療中的重要性。未來需要更多研究確立最佳使用時機與標準化操作流程。


### 抗血小板治療策略優化

丹麥的prasugrel使用經驗提供重要的真實世界證據，顯示在適當患者選擇下，prasugrel可能提供較ticagrelor更佳的缺血事件預防效果，且不增加出血風險。


### 腎交感神經去除術的應用拓展

RDN在CKD患者中的安全性與有效性數據，為此技術在特殊族群的應用提供支持。基準血壓作為療效預測因子的發現，有助於改善患者篩選與諮詢品質。


## 臨床實務建議


### 高風險PCI患者管理


1. 評估是否需要預防性機械循環支持
2. 根據患者血流動力學狀態與病灶複雜度決定支持策略
3. 考慮參與相關臨床試驗以累積經驗


### 影像導引策略


1. ACS患者複雜病灶PCI應考慮OCT導引
2. 建立OCT支架最佳化標準操作流程
3. 加強OCT影像判讀訓練，避免偽影誤判


### 抗血小板藥物選擇


1. MI患者PCI後優先考慮prasugrel（無禁忌症時）
2. 個別化評估出血與缺血風險
3. 建立機構內用藥指引與監測機制


### 腎交感神經去除術應用


1. 在精選的頑固性高血壓合併CKD患者中考慮RDN
2. 建立多專科團隊評估機制
3. 術前充分溝通降壓預期效果
4. 術後持續監測血壓與腎功能變化


## 未來展望

## 結論


本期EuroIntervention收錄的研究涵蓋介入性心臟病學多個重要領域，為臨床實務提供寶貴證據。從機械循環支持、影像導引PCI、抗血小板治療到腎交感神經去除術，這些研究共同推進我們對心血管疾病治療的理解與能力。臨床醫師應持續關注最新研究進展，並將證據轉化為改善患者照護的實際行動。
