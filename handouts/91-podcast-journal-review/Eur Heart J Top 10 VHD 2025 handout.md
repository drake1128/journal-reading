## 人工智慧在瓣膜疾病的應用


### ECHONEXT 研究：AI 偵測結構性心臟病


- **研究設計**：Deep learning ECG model，訓練資料超過一百萬筆心律與影像紀錄
- **目標**：偵測中重度瓣膜疾病（VHD）等結構性心臟病
- **結果**：     itemize
- 內部與外部驗證均展現高診斷準確度
- 在不同照護場域及種族群體表現一致
- 臨床試驗中成功識別先前未診斷的心臟病患者


    \item **意義**：有潛力擴展大規模心臟病篩檢的可近性
itemize


### DELINEATE-REGURGITATION 研究：AI 評估瓣膜逆流


- **資料**：71,660 份經胸心臟超音波，1,203,980 段 color Doppler 影像
- **準確度**（Weighted kappa，內部/外部驗證）：     itemize
- Aortic regurgitation (AR)：0.81 / 0.76
- Mitral regurgitation (MR)：0.76 / 0.72
- Tricuspid regurgitation (TR)：0.73 / 0.64


    \item **特色**：多視角方法優於單視角；DELINEATE-MR-Progression 可預測 MR 進展
itemize

這些研究顯示 AI 即將在瓣膜疾病的臨床管理中產生重大影響。

## 低風險患者：TAVI vs 外科手術


### PARTNER-3 試驗：7 年追蹤


- **設計**：1,000 位低風險手術患者隨機分配至 TAVI 或 SAVR
- **7 年結果**：     itemize
- 兩個主要複合終點（死亡、中風、再住院）無顯著差異
- 血流動力學結果與瓣膜衰竭率相當
- 值得注意：全因死亡的 Kaplan-Meier 曲線在 3 年後交叉
- TAVI 組數值上較高死亡率（19.5


itemize


### NOTION-2 試驗：3 年追蹤


- **特色**：370 位低風險患者，平均年齡 71 歲（迄今最年輕）
- **首次納入**：雙葉主動脈瓣（Bicuspid aortic valve, BAV）
- **3 年結果**：     itemize
- 整體主要終點：TAVI 16.1
- **三葉瓣 AS**：兩組相近（14.5

## 二尖瓣逆流：手術與介入治療


### MITRACURE Registry：二尖瓣手術真實世界數據


- **族群**：連續接受單獨或合併二尖瓣手術的 MR 患者
- **患者特徵**：     itemize
- 最常見病因：Myxomatous（61
- NYHA Class III/IV：43
- 心衰竭：30
- 心房顫動/flutter：22
- EF 下降：35
- 肺高壓：22


    \item **早期介入**：僅 3
    \item **修補率**：整體 62
    \item **院內死亡率**：整體 4.5
itemize

此研究凸顯患者仍常被延遲轉介，早期介入比例低，需改善及時處置策略。


### OCEAN-Mitral Registry：心房功能性 MR 的 TEER


- **背景**：心房功能性 MR（Atrial functional MR）被排除於三項現有 RCT 之外
- **比較**：TEER（441 人）vs 藥物治療（640 人，來自 REVEAL-AFMR registry）
- **結果**：     itemize
- TEER 降低全因死亡 + HF 住院複合終點（HR 0.65, P = .044）
- TEER 降低全因死亡（HR 0.58, P = .044）
- **關鍵發現**：僅殘餘 MR $\leq$ mild 者獲益
- 殘餘 MR $\geq$ moderate 者結果與藥物治療相近


itemize


### SAPIEN M3 Trial：經導管二尖瓣置換


- **適應症**：不適合手術或 TEER 的嚴重 MR 患者
- **篩選**：1,171 人篩選，299 人植入（1/3 primary MR，2/3 secondary MR）
- **30 天結果**：     itemize
- 死亡率 0.7
- 無術中死亡
- 無 LVOT obstruction 導致血流動力學影響
- 無轉為手術


    \item **主要終點**：全因死亡 + HF 住院 25.2
    \item **限制**：篩選失敗率高，無對照組
itemize

## 臨床要點總結


1. **AI**：ECG-based deep learning 可偵測結構性心臟病，AI 評估瓣膜逆流準確度高
2. **無症狀嚴重 AS**：早期介入可降低住院和中風，但死亡率無差異；需等待更多長期數據
3. **低風險 TAVI**：三葉瓣患者 TAVI vs 手術結果相當；**雙葉瓣應謹慎**
4. **TAVI 後藥物**：Dapagliflozin 可降低高風險患者的心衰竭惡化
5. **MR 手術**：患者常延遲轉介，早期介入比例僅 3
6. **心房功能性 MR**：TEER 可能有益，但需達到良好修復（殘餘 MR $\leq$ mild）
7. **經導管二尖瓣置換**：對不適合手術/TEER 者是可行選項
8. **TR TEER**：可降低 HF 住院，但對死亡率無影響
