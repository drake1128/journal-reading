## 心血管基因體學教育：行動的時刻


### 背景


2024年12月，歐洲心臟學會（ESC）心臟基因體學委員會召開會議，討論當代心臟病學的關鍵問題：是否應將心血管遺傳學和基因體學教育納入心臟科醫師的核心訓練？


### 共識結論


答案是明確的「是」，因為基因體學具有改善心血管疾病診斷和治療的潛力。然而，挑戰在於如何最有效地達成此目標。


> **教育需求對象：**
> itemize[leftmargin=*]
>     \item 醫學生
>     \item 科學研究人員
>     \item 臨床醫師
>     \item 不同專業的醫療人員
> itemize
> **關鍵議題：**
> itemize[leftmargin=*]
>     \item 心肌病變診斷路徑中的基因檢測整合
>     \item 早發性心房顫動的基因檢測
>     \item 肥厚型心肌病變的遺傳學臨床應用
>     \item 主動脈擴張患者的遺傳資料應用
> itemize

> **臨床意義：**
> itemize[leftmargin=*]
>     \item 支持對此類患者採取差異化治療策略
>     \item 治療目標應超越單純的猝死預防
>     \item 需要開發整合心律不整風險與心衰竭進展風險的評分系統
>     \item 未來研究應探討多基因風險評分在DCM預後預測中的應用
> itemize

## Y染色體缺失：心血管疾病的蛋白質體學特徵


### 研究背景


白血球中的Y染色體嵌合性缺失（mLOY）是男性最常見的後天性克隆嵌合形式，與多種心血管疾病的發生相關，包括心衰竭、心房顫動、冠狀動脈疾病患者的致命性心肌梗塞，以及慢性腎臟病患者的心衰竭相關死亡。

## Titin相關家族性擴張型心肌病變：疾病發作的相關因素


### 研究背景


Titin基因截斷變異（TTNtv）是擴張型心肌病變（DCM）最常見的遺傳原因，但也會作為一般人群中的偶然發現。本研究探討與TTNtv臨床表現相關的因素。


### 研究資訊


| p{10cm}}  **項目** | **內容** |  |
| --- | --- | --- |
| **項目** | **內容** |  |
| 標題 | Titin-Related Familial Dilated Cardiomyopathy: Factors Associated with Disease Onset |  |
| 文章類型 | Clinical Research Article |  |
| 第一作者 | Renee Johnson |  |
| 機構 | Victor Chang Cardiac Research Institute, Darlinghurst, Australia |  |
| 期刊 | European Heart Journal (2025) |  |
| DOI | [10.1093/eurheartj/ehaf380](https://doi.org/10.1093/eurheartj/ehaf380) |  |


### 研究方法


- 國際多中心回顧性觀察研究
- 研究對象：TTNtv相關DCM家族
- 統計方法：共享脆弱性模型（shared frailty models）估計變異特徵與DCM終生風險的關聯；邏輯迴歸估計臨床風險因子與DCM的勝算比
- 總計約3000名受試者，來自1043個TTNtv相關DCM家族


### 主要結果


| p{4cm}p{4cm}}  **因素** | **OR** | **備註** |
| --- | --- | --- |
| **因素** | **OR** | **備註** |
| TTNtv陽性 vs 陰性 | 21.21 | DCM發病風險 |
| 臨床風險因子存在 | 3.41 | 較早發病 |
| 心房顫動病史 | 2.05 | DCM風險增加 |
| 年輕發病 (<30歲) | 4.75 | 風險因子關聯性最強 |
| 預防性BB/RASi治療 | 0.13 | DCM風險降低87 |


### 其他發現


- 男性發病較早，但不同類型和位置的TTNtv發病時間相似
- 臨床風險因子的盛行率隨年齡增加
- 在顯性DCM發病前給予beta-blocker或RAS阻斷劑與DCM風險降低87


### 結論


> TTNtv相關家族性DCM的疾病發作取決於個別患者的臨床背景，且可能透過風險因子管理和預防性治療介入加以改變。
> **精準醫療觀點：**
> itemize[leftmargin=*]
>     \item Titin是精準醫療挑戰與機會的典範
>     \item 強調遺傳、臨床和生活型態風險因子的複雜性
>     \item TTNtv是DCM和心衰竭的主要原因，但在一般人群中也很常見
>     \item 需要在臨床介入和避免過度診斷之間取得平衡
> itemize

## 家族性高膽固醇血症兒童的LDL-C篩檢


### 研究背景


對兒童進行膽固醇篩檢，並對最高百分位者進行後續基因檢測，已被建議作為家族性高膽固醇血症（FH）的有效全面篩檢方法。本研究調查新生兒和1-12歲兒童在挪威全國家族級聯篩檢計畫中測量的膽固醇水平。

## LIPA基因座與冠狀動脈疾病：解碼變異-功能關係


### 研究背景


全基因體關聯研究（GWAS）已一致將LIPA（lipase A, lysosomal acid type）識別為冠狀動脈疾病（CAD）的風險位點。然而，闡明因果變異、調控機制、靶細胞類型及其對動脈粥狀硬化因果影響的功能研究一直缺乏。


### 研究資訊


| p{10cm}}  **項目** | **內容** |  |
| --- | --- | --- |
| **項目** | **內容** |  |
| 標題 | LIPA, A Risk Locus for Coronary Artery Disease: Decoding the Variant-to-Function Relationship |  |
| 文章類型 | Translational Science Article |  |
| 第一作者 | Fang Li |  |
| 機構 | Columbia University Irving Medical Center, New York, NY, USA |  |
| 期刊 | European Heart Journal (2025) |  |
| DOI | [10.1093/eurheartj/ehaf581](https://doi.org/10.1093/eurheartj/ehaf581) |  |


### 研究方法


使用的post-GWAS分析工具和分子生物學技術：

- Expression quantitative trait loci (eQTL) 分析
- Tri-HiC
- Luciferase assay
- CRISPRi
- Allele-specific binding分析
- Motif分析
- Electrophoretic mobility shift assay (EMSA)
- 骨髓特異性LIPA過度表現小鼠模型（Ldlr$^{-/-}$背景）


### 主要發現


> **人類功能基因體學發現：**
> itemize[leftmargin=*]
>     \item CAD風險等位基因（rs1320496 T, rs1412444 T）增加單核球/巨噬細胞中的LIPA表現和酵素活性
>     \item 機制：增強PU.1與內含子增強子區域的結合，該區域與LIPA啟動子相互作用
> itemize
> **小鼠模型驗證：**
> 骨髓特異性LIPA過度表現的Ldlr$^{-/-}$小鼠餵食西方飲食後出現：
> itemize[leftmargin=*]
>     \item 較大的動脈粥狀硬化病灶
>     \item 病灶內巨噬細胞累積增加（來自循環單核球）
>     \item 中性脂質含量減少
>     \item 游離膽固醇增加
>     \item Integrin和細胞外基質（ECM）路徑基因上調
> itemize


### 結論


本研究建立了LIPA風險等位基因與單核球/巨噬細胞LIPA增加之間的直接因果關係，後者會加劇動脈粥狀硬化，將人類功能基因體學證據與CAD的機制理解連結起來。


### 編輯評論


Philipp Wild和Tanja Zeller在評論中指出：

- 本研究展示了post-GWAS研究的典範
- 為靶向治療開闢了新途徑
- 強調了深入功能和實驗分析對於實現GWAS發現潛力的重要性
- 作為最初透過GWAS發現LIPA為CAD相關位點的研究者，看到該位點從統計關聯進展到詳細機制洞見令人欣慰

## IL-17受體A訊號：JAK2克隆性造血的動脈粥狀硬化保護作用


### 研究背景


不確定潛能克隆性造血（CHIP）涉及與年齡相關的血液惡性腫瘤常見突變基因（如DNMT3A、TET2或JAK2）的獲得和擴增。JAK2-CHIP個體的心血管疾病風險增加四倍。

### 主要發現


> **流行病學發現：**
> itemize[leftmargin=*]
>     \item 遺傳預測的IL-17RA水平僅在JAK2-CHIP攜帶者中降低心血管疾病風險
>     \item 只有遺傳預測IL-17RA水平較低的JAK2-CHIP攜帶者有較高的心血管疾病發生風險
> itemize
> **機制研究（動脈粥狀硬化易感JAK2-V617F小鼠）：**
> 炎症體活化和Ninjurin-1介導的細胞膜破裂可能創造一個保護性反饋機制：
> itemize[leftmargin=*]
>     \item Th17細胞透過IL-17RA向TREM2巨噬細胞發出訊號
>     \item 從而抑制發炎反應
> itemize


### 臨床意義


> **接受IL-17阻斷治療的JAK2-CHIP患者可能經歷升高的心血管疾病風險。**
> 這對於同時患有自體免疫疾病（如乾癬、類風濕性關節炎）而正在使用IL-17抑制劑的JAK2-CHIP患者具有重要臨床意義。
