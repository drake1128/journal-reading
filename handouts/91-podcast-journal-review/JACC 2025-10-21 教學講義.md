## HFpEF 的臨床困境


### 長期挫折


數十年來,射出分率保留型心衰竭(Heart Failure with Preserved Ejection Fraction, HFpEF)一直是心臟科醫師的巨大挫折:


- **盛行率持續上升**
- **多年來沒有治療有效**
- **病因機制不明確**
- 傳統歸因於高血壓、糖尿病、慢性腎病和肥胖,但沒有真正破解治療的密碼


### 治療的突破


情況在 SGLT2 抑制劑和 GLP-1 受體促效劑開始顯示療效後改變:


> **這些藥物為什麼有效?**


Milton Packer 認為答案的起點不在心臟,而在**脂肪組織**。


## 脂肪激素假說的核心概念


### 主要論點


| |p{10cm}|} 脂肪激素假說的關鍵要素 |  |  |
| --- | --- | --- |
| **概念** | **說明** |  |
| **起源點** | HFpEF 主要源自**內臟脂肪組織的擴張與生物學轉變** |  |
| **脂肪組織角色** | 不再是單純的儲存組織,而是成為「失控的內分泌器官」 |  |
| **病理機制** | 失調的脂肪組織分泌一系列**脂肪激素**(adipokines),這些訊號分子驅動:    • 發炎反應    • 鈉滯留    • 血管僵硬    • 心臟纖維化 |  |
| **疾病本質重新定義** | HFpEF 可能根本**不是從心臟問題開始**,而是**全身性脂肪驅動的過程**在心臟表現出來 |  |
| **內分泌與旁分泌效應** | • **內分泌效應**:脂肪激素經血液循環作用於全身    • **旁分泌效應**:內臟脂肪直接包圍心臟,局部分泌訊號分子導致心臟纖維化和僵硬 |  |
| **治療機制解釋** | 縮小內臟脂肪或重新平衡脂肪激素訊號的療法(如 GLP-1 受體促效劑、SGLT2 抑制劑、MRA)可能都是透過**相同的生物學路徑**發揮作用 |  |


### 脂肪組織的雙重身份


> 「脂肪不是被動的儲存器官,它實際上是活躍的,它產生訊號蛋白,這些訊號蛋白對健康有影響。隨著脂肪累積,會發生從滋養性訊號蛋白(他稱之為 domain 1)到這些 domain 3 的轉變。」
> ——Harlan Krumholz


## 脂肪激素的三域分類


Packer 將脂肪激素組織成三個領域(domains),這個框架既優雅又具教育意義:


| |p{4cm}|p{7cm}|} 脂肪激素的三域框架 |  |  |
| --- | --- | --- |
| **領域** | **主要脂肪激素** | **功能與特徵** |
| **Domain 1**     **心臟保護型** | • Adiponectin    • Omentin-1 | **在瘦體型個體中豐富**    **在肥胖時被抑制**        功能:    • 抗發炎    • 改善胰島素敏感性    • 血管保護    • 抗纖維化 |
| **Domain 2**     **代償型** | • FGF-21    • GDF-15    • Irisin | **在肥胖時升高以對抗壓力**    **但常因身體產生抗性而失效**        功能:    • 試圖改善代謝    • 對抗氧化壓力    • 促進能量消耗        **問題**:持續壓力下,受體變得不敏感 |
| **Domain 3**     **適應不良型** | • Leptin    • Resistin    • Chemerin    • 其他促發炎因子 | **在肥胖時大量增加**    **促進疾病進展**        作用:    • 促進發炎    • 促進纖維化    • 增加血管僵硬    • 鈉滯留    • 促進心臟重塑 |


> **核心概念:**這三個領域之間的平衡決定了脂肪組織是滋養還是傷害心臟。
> **肥胖時的轉變:**
> itemize
> \item Domain 1 $\downarrow$ (保護因子減少)
> \item Domain 2 $\uparrow$ 但失效(代償機制衰竭)
> \item Domain 3 $\uparrow\uparrow\uparrow$ (病理因子大增)
> itemize
> 結果:脂肪組織從「朋友」變成「敵人」

## 臨床思維的改變


### 重新看待患者


> 「當我閱讀這篇文章時,它讓我對患者的思考有所不同。我長期以來一直在思考,肥胖不僅僅是關於你的外表,它真的是關於**心臟代謝健康**。
> 他將此延伸為:這是心臟代謝健康,但也可以是**脂肪的直接毒性**——內臟脂肪通過這些訊號蛋白對心血管結構(如心臟)的直接毒性。它通過內分泌和**旁分泌效應**被介導。
> 旁分泌效應發生在你有內臟脂肪包圍心臟時,直接與心臟並列,實際上產生的訊號蛋白不是在全身傳播,而是**就在那裡局部造成纖維化、僵硬和伴隨 HFpEF 的那些事情**。」
> ——Harlan Krumholz


### 對肥胖的重新理解


1. **外觀 vs. 健康**:肥胖不是美學問題,而是活躍的病理生理過程
2. **被動 vs. 主動**:脂肪組織不是惰性旁觀者,而是主動訊號器官
3. **儲存 vs. 分泌**:脂肪細胞不僅儲存能量,更分泌影響全身的分子
4. **局部 vs. 全身**:內臟脂肪的局部(旁分泌)效應可能與全身(內分泌)效應同樣重要


## 治療意涵


### 現有藥物的機制重新詮釋


| |p{10cm}|} HFpEF 治療藥物的脂肪激素機制解釋 |  |  |
| --- | --- | --- |
| **藥物類別** | **可能的脂肪激素調節機制** |  |
| **GLP-1 受體促效劑**     (如 Semaglutide) | • 減少內臟脂肪    • 恢復 Domain 1 保護性脂肪激素(adiponectin $\uparrow$)    • 減少 Domain 3 適應不良型脂肪激素(leptin $\downarrow$)    • 改善脂肪組織的內分泌功能 |  |
| **SGLT2 抑制劑**     (如 Empagliflozin, Dapagliflozin) | • 促進內臟脂肪減少    • 改善脂肪組織胰島素敏感性    • 可能調節脂肪激素分泌    • 減少脂肪組織發炎 |  |
| **MRA**     (Mineralocorticoid Receptor Antagonists) | • 可能透過減少脂肪組織鈉滯留    • 調節脂肪激素訊號 |  |
| **未來可能標靶** | • 直接重塑 Domain 3 脂肪激素的拮抗劑    • 增強 Domain 1 保護性脂肪激素的促效劑    • 改善 Domain 2 脂肪激素抗性的藥物    • 調節脂肪組織發炎的藥物 |  |


### 治療範式的可能轉變


> 「也許最具挑釁性的是,這個框架表明,**縮小內臟脂肪或重新平衡脂肪激素訊號的療法**——像 GLP-1 受體促效劑、SGLT2 抑制劑或 MRA——**可能都是透過相同的生物學通路起作用**。」
> 這意味著 HFpEF 的不同治療可能有共同的上游作用機制。

## 對臨床醫師的啟示


### 改變臨床思維


1. **重新評估內臟脂肪**     - 不只是體重數字 - 關注內臟脂肪分布(腰圍、影像學) - 理解其為活躍的內分泌器官
2. **整合代謝與心臟健康**     - HFpEF 不能孤立看待心臟 - 需要全面評估代謝狀態 - 重視減重和代謝控制
3. **個體化治療選擇**     - 考慮患者的肥胖表型 - GLP-1、SGLT2i 的使用可能在肥胖型 HFpEF 特別重要 - 重視生活方式介入
4. **未來生物標記**     - 脂肪激素可能成為診斷和預後標記 - 可能用於指導治療選擇 - 監測治療反應


## 結語:值得測試的想法


> 「這個困難的假說可能有一天會重塑這個領域,或者可能催生出新的框架。無論哪種方式,**這是一個值得測試的想法**。這就是我在這期專刊中編輯頁面的標題:**An Idea Worth Testing**(值得測試的想法)。」
