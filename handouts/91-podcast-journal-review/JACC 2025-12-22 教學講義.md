### 研究背景


傳統觀念認為 ventricular secondary TR（V-STR）主要見於 HFrEF，而 atrial secondary TR（A-STR）則較常見於 HFpEF。然而，此研究挑戰了這個概念，發現在 HFpEF 族群中 V-STR 反而是主要的 TR 型態。

| 主要機轉 | 右心室功能障礙及重塑 | 右心房擴大及三尖瓣環擴張 |

### 臨床意義與治療建議


#### 關鍵臨床訊息


1. **TR 是 HFpEF 的常見表現：**即使在早期 HFpEF 患者中，TR 也相當常見
2. **V-STR 代表更嚴重的疾病狀態：**看到 V-STR 時，應考慮患者可能處於較嚴重的 HFpEF 階段
3. **TR 與肺血管疾病密切相關：**TR 的惡化強烈關聯於肺血管疾病的存在
4. **不應忽視 TR 的根本原因：**TR 很少是獨立疾病，常為 HFpEF 的次發表現


#### 治療策略


| p{10cm}}  **策略** | **說明** |  |
| --- | --- | --- |
| **策略** | **說明** |  |
| 積極診斷 | 發現 TR 時應積極排除 HFpEF 作為潛在原因 |  |
| Guideline-directed medical therapy（GDMT） | 確診 HFpEF 後應儘速啟動 GDMT，V-STR 患者尤其需要優先處理 |  |
| 風險告知 | V-STR 患者為高風險族群，應告知疾病嚴重度 |  |
| 追蹤 TR 進展 | 定期評估 TR 嚴重度，考慮介入時機 |  |
| 介入治療考量 | Transcatheter tricuspid intervention 雖尚未在 HFpEF 進行隨機試驗，但在嚴重 TR 時可考慮 |  |


### 未來展望


- 目前針對 HFpEF 相關肺血管疾病尚無特定治療，但正積極研究中
- 多種藥物及裝置正在評估中，包括抗重塑療法（anti-remodeling therapies）
- V-STR 與 A-STR 的區分可能有助於精準醫療策略的發展

## Danicamtiv 治療擴張型心肌病：Phase 2A 開放標籤試驗


> **研究重點：**Danicamtiv 是一種選擇性心肌肌凝蛋白活化劑（cardiac myosin activator），在 Phase 2a 試驗中顯示可改善擴張型心肌病（DCM）患者的心臟收縮功能。帶有 MYH7 或 TTN 基因變異的患者獲益最大，LVEF 改善達 8.8

## 綜合討論與臨床要點


### 兩篇研究的共同主題


| p{10cm}}  **主題** | **臨床意義** |  |

| **主題** | **臨床意義** |  |

| 次分類的重要性 | TR 的 V-STR vs A-STR 區分、DCM 的基因分型皆影響預後及治療反應 |  |

### 臨床實務重點


1. **HFpEF 與 TR：**      - 發現 TR 時應積極考慮 HFpEF 作為潛在病因 - V-STR 代表較嚴重的疾病狀態，需要更積極的治療 - 不應僅以心房顫動病史來定義 A-STR
2. **DCM 與基因檢測：**      - 基因檢測在 DCM 中使用不足，應更積極推廣 - MYH7、TTN 變異的患者可能對 myosin activators 有較好的反應 - 基因治療正在快速發展，某些基因（如 BAG3、PKP2）已有臨床試驗
3. **未來方向：**      - HFpEF 相關肺血管疾病的特定治療正在研發中 - Danicamtiv 的 KINSHIP-DCM Phase 2B 試驗將提供更多療效資訊 - 精準醫療策略在心血管疾病中的應用將持續擴展
