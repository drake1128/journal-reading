### 研究概述


這是西班牙與義大利進行的開放標籤隨機試驗，評估心肌梗塞後LVEF >40


#### 研究設計


- **研究類型：**多中心、開放標籤、隨機試驗
- **研究對象：**8505例心肌梗塞患者（LVEF >40
- **平均年齡：**約65歲
- **入選條件：**     itemize[nosep]
- 急性心肌梗塞（STEMI或NSTEMI）
- 接受侵入性治療（冠狀動脈造影）
- LVEF >40


    \item **主要終點：**全因死亡、再梗塞或心衰住院的複合事件
itemize


### 主要結果


| p{9cm}}  **結果指標** | **數據** |  |
| --- | --- | --- |
| 主要終點 | beta-blocker組：22.5 events/1000 patient-years     no-beta-blocker組：21.7 events/1000 patient-years     (HR 1.04; 95 |  |
| 全因死亡 | 11.2 vs. 10.5 events/1000 patient-years (HR 1.06; 95 |  |
| 再梗塞 | 10.2 vs. 10.1 events/1000 patient-years (HR 1.01; 95 |  |
| 心衰住院 | 2.7 vs. 3.0 events/1000 patient-years (HR 0.89; 95 |  |
| 安全性 | 無顯著差異 |  |


### 臨床意義


1. ****無益處：****在LVEF >40
2. ****現代治療：****反映當代再灌注與藥物治療下的結果
3. ****指南影響：****可能挑戰現行指南推薦
4. ****長期追蹤：****中位追蹤3.7年，提供可靠證據

### 研究概述


這是丹麥與挪威合併的開放標籤隨機試驗，評估心肌梗塞後LVEF ≥40


#### 研究設計


- **研究類型：**多中心、開放標籤、隨機試驗
- **研究對象：**5574例心肌梗塞患者（LVEF ≥40
- **平均年齡：**約65歲
- **入選條件：**     itemize[nosep]
- 心肌梗塞後14天內
- LVEF ≥40
- 無心衰徵象


    \item **主要終點：**全因死亡或主要心血管事件（新心肌梗塞、非計劃冠狀動脈再血管化、缺血性中風、心衰或惡性心室心律不整）的複合事件
itemize


### 主要結果


| p{9cm}}  **結果指標** | **數據** |  |
| --- | --- | --- |
| 主要終點 | beta-blocker組：14.2  no-beta-blocker組：16.3  (HR 0.85; 95 |  |
| 全因死亡 | 4.2 |  |
| 新心肌梗塞 | 5.0 |  |
| 安全性 | 無顯著差異 |  |


### 臨床意義


1. ****益處證據：****降低死亡或主要心血管事件風險
2. ****廣泛終點：****包含更多心血管事件，提供全面評估
3. ****區域數據：****北歐患者群體的真實世界證據
4. ****長期追蹤：****中位追蹤3.5年


## 綜合討論與臨床啟示


### 兩試驗比較


| p{11cm}}  **比較項目** | **REBOOT-CNIC vs. BETAMI-DANBLOCK** |  |
| --- | --- | --- |
| 患者數 | 8505 vs. 5574 |  |
| LVEF cutoff | >40 |  |
| 主要終點 | 死亡、再梗塞、心衰住院 vs. 死亡或主要心血管事件（含新梗塞、非計劃再血管化、中風、心衰、惡性心律不整） |  |
| 事件率 | 較低（21-22 events/1000 patient-years） vs. 較高（14-16 |  |
| 追蹤時間 | 中位3.7年 vs. 中位3.5年 |  |
| 區域 | 西班牙/義大利 vs. 丹麥/挪威 |  |
| 結論 | 無益處 (HR 1.04) vs. 有益處 (HR 0.85) |  |


### 結論差異原因分析


1. **終點定義差異：**BETAMI-DANBLOCK的複合終點更廣泛，包含更多事件，可能更容易偵測差異。[參考原文](https://www.nejm.org/doi/full/10.1056/NEJMoa2505985)
2. **患者族群：**REBOOT-CNIC包含更多STEMI患者，可能有不同基線風險。
3. **事件率與功率：**REBOOT-CNIC事件率較低，可能統計功率不足偵測小差異。
4. **區域與治療差異：**南歐 vs. 北歐的醫療實務可能影響結果。
5. **藥物使用：**兩試驗均開放標籤，可能影響順從性。
