## 主要研究：DOUBLE-CHOICE試驗


### 研究背景


經導管主動脈瓣置換術（Transcatheter Aortic Valve Implantation, TAVI）在過去20年經歷了重大進展。早期TAVI時代，所有患者均需接受全身麻醉並進行插管。隨著技術進步，現已無需全身麻醉即可完成手術，逐步朝向更簡化的流程發展。


> **SOLVE-TAVI試驗（先前研究）：**
> itemize[leftmargin=*,noitemsep]
>     \item 比較全身麻醉與清醒鎮靜
>     \item 兩種策略等效
>     \item 但清醒鎮靜組譫妄發生率仍偏高
>     \item 譫妄為重要臨床終點，與死亡率相關
> itemize
> **DOUBLE-CHOICE試驗動機：**
> itemize[leftmargin=*,noitemsep]
>     \item 進一步簡化手術流程
>     \item 降低譫妄發生率
>     \item 比較僅局部麻醉與清醒鎮靜策略
> itemize

### 主要結果


> **主要終點結果：**
> itemize[leftmargin=*,noitemsep]
>     \item 極簡方法組與標準照護組事件發生率相似
>     \item **極簡方法非劣效於標準照護**
>     \item 雙側檢定亦未達統計學顯著差異
> itemize

> itemize[leftmargin=*,noitemsep]
>     \item 術中壓力程度：極簡方法組較高（符合預期）
>     \item 焦慮程度：術前最高，術中次之
>     \item 麻醉師在場對患者安撫有幫助
> itemize

## 側支循環與穩定型心絞痛：ORBITA-STAR研究

## IRF5 siRNA奈米免疫治療：恢復動脈粥樣硬化中巨噬細胞吞噬功能


### 研究背景


巨噬細胞是動脈粥樣硬化斑塊中最豐富的吞噬細胞，在進階斑塊中，巨噬細胞的吞噬清除功能（efferocytosis）受損，導致凋亡細胞堆積、壞死核心擴大及斑塊破裂。


### 研究資訊


| p{11cm}}  **項目** | **內容** |  |
| --- | --- | --- |
| **項目** | **內容** |  |
| **標題** | IRF5 siRNA Nanoimmunotherapy: Restoring Macrophage Efferocytosis in Atherosclerosis |  |
| **通訊作者** | Wei Chen, PhD（Harvard Medical School, Boston） |  |
| **期刊** | Circulation (2025) |  |
| **DOI** | [10.1161/CIRCULATIONAHA.125.075352](https://doi.org/10.1161/CIRCULATIONAHA.125.075352) |  |


### 研究方法與結果


| p{11cm}}  **項目** | **內容** |  |
| --- | --- | --- |
| **項目** | **內容** |  |
| **治療平台** | 工程化小干擾RNA（siRNA）奈米粒子平台 |  |
| **靶點** | IRF5（Interferon Regulatory Factor 5，干擾素調節因子5）——斑塊不穩定性相關巨噬細胞分子 |  |
| **靶向細胞** | 斑塊內巨噬細胞，特別是Cd11c+和Trem2hi巨噬細胞 |  |
| **作用機制** | 高效沉默斑塊內巨噬細胞的IRF5表達，增強凋亡細胞的吞噬清除能力 |  |
| **療效** | minipage[t]{10cm}  • 顯著抑制斑塊進展和不穩定化 |  |
| • 無可觀察到的副作用 |  |  |
| • 在兩種獨立ApoE−/−小鼠模型中證實 |  |  |
| **適用模型** | 慢性及加速發炎動脈粥樣硬化模型 |  |


> itemize[leftmargin=*,noitemsep]
>     \item 此奈米免疫治療方法可跨多種動脈粥樣硬化模型展現治療潛力
>     \item IRF5是RNA干預的有前景靶點
>     \item 為恢復斑塊內巨噬細胞吞噬功能提供新治療策略
> itemize

## 詞彙表


description[leftmargin=5cm,style=nextline]
    \item[TAVI/TAVR] Transcatheter Aortic Valve Implantation/Replacement，經導管主動脈瓣置換術。一種微創心臟瓣膜置換手術，透過導管將人工瓣膜送至心臟。
    
    \item[MA (Minimalist Approach)] 極簡方法。僅使用局部麻醉，不使用鎮靜劑、中央靜脈導管或尿管的TAVI策略。
    
    \item[SoC (Standard of Care)] 標準照護。使用局部麻醉合併清醒鎮靜的傳統TAVI策略。
    
    \item[FFR] Fractional Flow Reserve，血流儲備分數。評估冠狀動脈狹窄功能意義的指標，<0.80表示具血流動力學意義。
    
    \item[iFR] Instantaneous Wave-Free Ratio，瞬時無波比。另一種評估冠狀動脈狹窄功能意義的指標。
    
    \item[Collateral Flow Index] 側支血流指數。評估冠狀動脈側支循環程度的指標。
    
    \item[IRF5] Interferon Regulatory Factor 5，干擾素調節因子5。調控巨噬細胞極化的轉錄因子，促進M1型促炎巨噬細胞表型。
    
    \item[siRNA] Small Interfering RNA，小干擾RNA。可特異性沉默目標基因表達的短RNA分子。
    
    \item[Efferocytosis] 吞噬清除功能。巨噬細胞清除凋亡細胞的過程，對維持組織穩態和減少發炎至關重要。
    
    \item[Xenotransplantation] 異種移植。將器官從一個物種移植到另一個物種的手術。
    
    \item[EuroSCORE] European System for Cardiac Operative Risk Evaluation。評估心臟手術風險的評分系統。
    
    \item[STS Score] Society of Thoracic Surgeons Score。另一種心臟手術風險評估工具。
    
    \item[NeuroARC] Neurologic Academic Research Consortium。神經學事件標準化分類系統。
