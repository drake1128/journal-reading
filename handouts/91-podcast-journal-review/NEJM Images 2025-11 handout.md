## 急性主動脈症候群 Acute Aortic Syndrome


### 定義與分類


急性主動脈症候群 (Acute Aortic Syndrome, AAS) 是一組涉及主動脈壁急性病理變化的緊急狀況，包括:


| p{9.5cm}}  **類型** | **定義** |  |
| --- | --- | --- |
| **典型主動脈剝離**    (Classic Aortic Dissection) | 內膜撕裂導致血液進入中膜層，形成真腔與假腔 |  |
| **壁內血腫**    (Intramural Hematoma, IMH) | 血液滲入主動脈中膜層，**影像上無可見內膜撕裂**，形成血栓將主動脈壁向外推 |  |
| **穿透性動脈粥瘤潰瘍**    (Penetrating Atherosclerotic Ulcer, PAU) | 動脈粥瘤斑塊潰瘍穿透內彈性層進入中膜 |  |
| **外傷性主動脈損傷**    (Traumatic Aortic Injury) | 外傷導致的主動脈壁損傷 |  |


### 壁內血腫的病理生理


> **英文原文:** "An intramural hematoma is a type of acute aortic syndrome in which blood leaks into the aortic media (in the absence of a detectable intimal tear on imaging), forming a thrombus that pushes the aortic wall outward."
> **中文翻譯:** 壁內血腫是一種急性主動脈症候群，血液滲入主動脈中膜層（影像上無可偵測的內膜撕裂），形成血栓將主動脈壁向外推。


#### 可能機轉


1. **Vasa vasorum破裂:** 主動脈壁滋養血管自發性破裂出血
2. **微小內膜撕裂:** 影像無法偵測的小撕裂導致血液進入中膜
3. **動脈粥瘤斑塊內出血:** 斑塊內出血延伸至中膜層


### Stanford分類與處置原則


| p{5cm} p{6cm}}  **分類** | **定義** | **處置原則** |
| --- | --- | --- |
| **Type A** | 累及升主動脈    (不論是否延伸至降主動脈) | ****緊急手術****    高死亡風險 (破裂、心包填塞、冠狀動脈阻塞) |
| **Type B** | 僅累及降主動脈    (起始於左鎖骨下動脈遠端) | **藥物治療為主**    若有併發症則考慮介入治療    (TEVAR或手術) |


**本案例分類:**

- 病灶位於升主動脈並延伸至主動脈弓及近端降主動脈
- 屬於 **Stanford Type A** 壁內血腫
- Type A IMH有進展為Type A dissection的風險

### 考題詳解


1. **正確答案: b) 血液滲入主動脈中膜層，影像上無可偵測的內膜撕裂**          **詳解:** 根據原文定義，壁內血腫是「血液滲入主動脈中膜層（影像上無可偵測的內膜撕裂），形成血栓將主動脈壁向外推」。這與典型剝離不同，後者有可見的內膜撕裂及內膜瓣。
2. **正確答案: a) Type A，因累及升主動脈**          **詳解:** Stanford分類中，Type A定義為累及升主動脈（不論是否延伸至降主動脈）。本案例病灶位於升主動脈並延伸至主動脈弓及近端降主動脈，屬於Type A。分類依據是病灶位置而非是否有內膜撕裂。
3. **正確答案: d) D-dimer升高**          **詳解:** Type A IMH的高風險特徵包括潰瘍樣突出 (ULP)、最大主動脈直徑>50mm、血腫厚度>11mm。D-dimer升高雖然在診斷時有用，但並非決定預後或手術時機的高風險特徵。
4. **正確答案: b) 肺栓塞**          **詳解:** 原文明確指出病患因胸膜性胸痛及D-dimer升高而接受CT pulmonary angiogram，目的是排除肺栓塞。壁內血腫是意外發現。
5. **正確答案: b) 心率 60-80 bpm，收縮壓 100-120 mmHg**          **詳解:** Anti-impulse therapy的目標是降低主動脈壁應力，標準目標為心率60-80 bpm及收縮壓100-120 mmHg。過低的心率或血壓可能導致器官灌流不足。
6. **正確答案: b) 追蹤CT顯示潰瘍樣突出增大**          **詳解:** 原文明確指出初始經共同決策延後手術，但「3天後追蹤CT angiogram顯示潰瘍樣突出增大」，因此進行手術。這展示了密切影像追蹤的重要性。
7. **正確答案: b) 是否有可見的內膜瓣及假腔血流**          **詳解:** 典型主動脈剝離有可見的內膜瓣 (intimal flap) 及假腔血流，而壁內血腫影像上無可見內膜瓣且無假腔血流。這是區分兩者最重要的影像特徵。
8. **正確答案: b) Beta-blockers**          **詳解:** Beta-blockers是急性主動脈症候群的第一線治療，可同時降低心率及血壓，減少主動脈壁應力 (dP/dt)。Nitroprusside單獨使用可能引起反射性心跳加速，需先給予beta-blocker。
9. **正確答案: b) 內膜撕裂及升主動脈剝離，合併壁內血腫**          **詳解:** 原文明確指出「術中發現內膜撕裂 (intimal tear) 及升主動脈剝離 (dissection)，合併壁內血腫」。這證實了壁內血腫可能進展為典型剝離。
