## 保留射血分數心衰竭的風險分層工具


### 基本資訊


**標題：**HFpEF-ABA Score Is a More Useful Enrichment Tool Than Natriuretic Peptides in Heart Failure with Preserved Ejection Fraction  

**主要作者：**Barry Borlaug(美國梅奧診所,明尼蘇達州羅徹斯特)等  

**期刊：***European Heart Journal*, 2025;46:4607-9  

**DOI：**[10.1093/eurheartj/ehaf663](https://doi.org/10.1093/eurheartj/ehaf663)  

**文章類型：**Fast Track Congress Article


### 研究背景與動機


| p{10cm}}  **項目** | **說明** |  |
| --- | --- | --- |
| **項目** | **說明** |  |
| NT-proBNP與風險 | minipage[t]{10cm} 升高的利鈉肽(NP)水平與HFpEF患者心衰竭住院或死亡的更大風險相關,支持作為臨床試驗的富集標準 |  |
| 肥胖相關問題 | minipage[t]{10cm} 肥胖相關HFpEF患者顯示較低的NT-proBNP水平,遠低於當代試驗使用的截斷點 |  |
| 代表性問題 | minipage[t]{10cm} 由於大多數HFpEF患者合併肥胖,基於NP的納入標準導致大量可能受益於治療的患者代表性不足 |  |
| HFpEF-ABA評分 | minipage[t]{10cm} 開發用於促進HFpEF診斷,無需NT-proBNP,使用三個普遍可用的標準：年齡(Age)、體重指數(BMI)和心房顫動病史(Atrial fibrillation) |  |


### SUMMIT試驗


- **介入措施：**tirzepatide(葡萄糖依賴性胰島素促泌多肽和類升糖素胜肽-1受體促效劑)
- **主要結果：**降低肥胖相關HFpEF患者的心血管死亡或心衰竭惡化風險
- **隨機分配：**1:1分配至安慰劑或tirzepatide,耐受劑量滴定至15.0 mg/週


### 共同主要終點


1. 首次經裁定的心血管死亡或心衰竭惡化的時間
2. 52週時堪薩斯城心肌病問卷臨床摘要評分(KCCQ-CSS)的變化


### 本次事後分析目的


1. 確定HFpEF-ABA評分在表徵HFpEF患者的影響
2. 評估患者對tirzepatide治療的反應
3. 特別評估是否可作為NT-proBNP的替代方案來富集心衰竭事件風險


### 主要研究發現


****HFpEF-ABA評分較高的患者：****


| p{10cm}}  **指標** | **表現** |  |
| --- | --- | --- |
| **指標** | **表現** |  |
| 6分鐘步行距離 | 較差 |  |
| NYHA功能分級 | 更差(更多III-IV級) |  |
| 估計腎絲球過濾率 | 較低 |  |
| 心肌損傷(肌鈣蛋白T) | 更嚴重 |  |
| NT-proBNP | 較高 |  |
| 心衰竭事件風險 | 顯著增加 |  |
| tirzepatide絕對效益 | 更大的效益信號 |  |


### 臨床意義


****重要結論：****


1. HFpEF-ABA評分識別肥胖相關HFpEF患者,這些患者在多個領域具有更大的疾病嚴重度
2. 心衰竭事件風險顯著增加
3. tirzepatide帶來更大的絕對效益
4. 對於NT-proBNP水平被抑制的肥胖患者,HFpEF-ABA評分提供優於NT-proBNP閾值的富集標準
5. 更具包容性和全面性
6. 可用於未來試驗以提高招募速度
7. 改善試驗結果對社區HFpEF患者的普遍適用性


### 研究限制


本研究結果僅適用於肥胖相關HFpEF患者,需要進一步研究了解HFpEF-ABA評分在較低BMI患者中的影響。


## FLNC截短變異的CRISPR活化治療


### 基本資訊


**標題：**CRISPR Activation to Repair Electrocardiogram Abnormalities Caused by a FLNC Truncating Variant in Mice  

**主要作者：**Rodrigo Cañas-Alvaro(西班牙馬德里國家心血管研究中心)等  

**期刊：***European Heart Journal*, 2025;46:4610-3  

**DOI：**[10.1093/eurheartj/ehaf703](https://doi.org/10.1093/eurheartj/ehaf703)  

**文章類型：**Fast Track Congress Article


### 研究背景


| p{10cm}}  **項目** | **說明** |  |
| --- | --- | --- |
| **項目** | **說明** |  |
| Filamin C基因 | FLNC基因截短變異(FLNCtv)是遺傳性DCM的常見原因 |  |
| 疾病特徵 | minipage[t]{10cm} - 遺傳性擴張性心肌病(DCM) |  |
| - 非擴張型左心室心肌病(NDLVC) |  |  |
| - 均以心律不整併發症為特徵 |  |  |
| - 猝死風險增加 |  |  |
| 治療缺口 | 目前不存在針對FLNCtv誘發心肌病的基因特異性療法 |  |
| CRISPR活化技術 | minipage[t]{10cm} - 通過轉錄活化上調基因表達 |  |
| - 不切割基因組 |  |  |
| - 提供有前景的策略 |  |  |
| - 特別適用於FLNC等大型基因(排除傳統AAV基因置換) |  |  |
| 研究現狀 | CRISPRa尚未在體內測試用於心肌病治療 |  |


### 研究方法


1. **小鼠模型：**產生FLNC外顯子15組成型異型合子缺失小鼠(FLNC-Ex15del/wt),模擬致病性人類變異
2. **心臟表達評估：**定量逆轉錄PCR(qRT-PCR)和西方墨點法評估FLNC的心臟表達
3. **電生理功能：**通過心電圖評估,包括氟卡尼挑戰試驗
4. **AAV-CRISPRa系統設計：**      - 心臟特異性全合一AAV-CRISPRa系統 - 編碼連接至VP64轉錄活化域的死Cas9(dSaCas9-VP64) - 在cTnT啟動子下 - 單導向RNA(sgRNAs)在HL-1心肌細胞中預先優化
5. **病毒包裝：**最終構建物包裝在向心性AAVMYO囊殼中並全身給藥
6. **評估時間：**注射後5週進行qRT-PCR、心電圖和表達分析


### 主要研究結果


| p{10cm}}  **發現** | **結果** |  |
| --- | --- | --- |
| **發現** | **結果** |  |
| **小鼠表型** |  |  |
| 收縮功能 | 無明顯收縮功能障礙 |  |
| 心電圖異常 | minipage[t]{10cm} 顯著心電圖異常： |  |
| - QRS時程延長 |  |  |
| - 振幅降低 |  |  |
| 氟卡尼誘發 | 約40 |  |
| 分子機制 | FLNC mRNA和蛋白質水平降低,符合單倍體不足 |  |
| **HL-1細胞研究** |  |  |
| sgRNA效果 | 發現高效sgRNA和支架組合,實現內源性FLNC表達上調達1.8倍 |  |
| **體內治療效果** |  |  |
| FLNC恢復 | 32週齡突變小鼠全身AAV遞送CRISPRa構建物後,FLNC mRNA恢復至野生型水平 |  |
| 心電圖改善 | minipage[t]{10cm} 治療後心電圖顯示： |  |
| - QRS振幅增加 |  |  |
| - 氟卡尼誘發的心律不整在治療動物中完全預防 |  |  |


### 研究結論


****重要發現：****


1. 這個異型合子FLNCtv模型部分重現FLNCtv攜帶者DCM/NDLVC中觀察到的電異常,為疾病發病機制提供寶貴見解
2. ****最重要的是：****本研究提供首次體內證明,CRISPRa-AAV介導的基因活化可以有效治療由單倍體不足驅動的遺傳性心肌病

### 不死時間偏差(Immortal Time Bias)詳解


#### 定義


****不死時間偏差****是觀察性研究中常見但容易被忽視的系統性偏差。根據流行病學定義：


> **不死時間(Immortal Time)：**隊列研究追蹤期間的一段時間,在此期間,由於暴露定義的方式,研究結果(如死亡)在設計上不可能發生。
> **偏差產生：**當不死時間被錯誤分類或在分析中被排除時,會導致偏差的關聯性估計,通常使治療效果看起來比實際更有效。

### 臨床意義與啟示


#### 對AS-CA患者管理的影響


1. **治療決策：**儘管方法學爭議,主動脈瓣置換術在AS-CA患者中確實有益
2. **類澱粉沉積症治療：**ATTR特異性治療可能提供額外效益,但需要更嚴謹的研究證實
3. **研究設計：**未來研究應更仔細處理不死時間偏差


#### 對觀察性研究的啟示


****重要教訓：****


1. **不死時間偏差普遍存在：**在藥物流行病學、臨床流行病學和環境流行病學中都很常見
2. **可造成重大偏差：**可能完全逆轉研究結論
3. **容易預防：**使用適當的設計和分析策略即可避免
4. **需要提高警覺：**審稿人和讀者需要警覺此偏差的存在
5. **報告透明度：**研究應清楚說明如何處理潛在的不死時間偏差


#### 經典錯誤案例


- **奧斯卡得主長壽研究：**2000年代初期錯誤發現奧斯卡獎得主生存期較長
- **高影響力期刊撤稿：**2018-2019年多篇論文因未處理不死時間偏差而撤回
- 使用適當方法重新分析後,治療實際上**沒有效果**


### 結論


****學術討論的價值：****


1. Watanabe等人的批評促使研究者更仔細思考方法學問題
2. Nitsche等人的回應提供了研究設計的詳細說明
3. 這種學術對話有助於提高觀察性研究的品質
4. 強調在缺乏RCT的情況下,嚴謹的觀察性研究設計的重要性


****臨床實務建議：****


- AS-CA患者應接受主動脈瓣置換術
- ATTR特異性治療可能有益,但證據仍需加強
- 臨床決策應基於最佳可得證據,同時認識其方法學限制
- 未來需要更嚴謹設計的前瞻性研究
