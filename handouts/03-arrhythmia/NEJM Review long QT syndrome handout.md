# NEJM Review long QT syndrome

**整理：謝慕揚 MD, PhD, FESC**
**日期：2025-11-20**

---

\maketitle


> **疾病概述：**長QT症候群（LQTS）是一種罕見的遺傳性心律不整疾病，盛行率超過1/2000，是年輕人猝死的主要原因之一。該症候群以心電圖QT間期延長及室性心律不整為特徵，主要在交感神經激活時發生。
> **臨床重要性：**及時診斷與適當治療極為重要，因為猝死往往是首發症狀。現有治療方法（包括乙型阻斷劑、左心交感神經去除術、mexiletine）療效顯著，大多數患者可維持正常生活品質。
> **治療進展：**基因型特異性治療與風險分層策略的應用，使得植入式心臟去顫器（ICD）的使用可限制在少數高風險患者，基因治療也展現出前景。

## 研究背景與歷史


### 疾病發現歷程


長QT症候群的認識經歷了漫長的發展過程：


- **1957年：**Jervell與Lange-Nielsen首次報告伴有先天性耳聾的家族中，QT間期延長與猝死之間的關聯
- **1963-1964年：**Romano及Ward分別報告聽力正常患者的類似發現
- **1975年：**Schwartz等人將Romano-Ward症候群與Jervell-Lange-Nielsen症候群統稱為「長QT症候群」
- **1995-1996年：**三個主要致病基因（KCNQ1、KCNH2、SCN5A）被發現，開啟基因時代

### 離子通道功能


> **正常心臟電生理：**
> itemize
>     \item **IKs**與IKr鉀通道：負責心臟復極化的外向電流
>     \item **交感激活時：**IKs成為主要復極化電流
>     \item **QT間期縮短：**心跳加快時應適當縮短
> itemize
> **病理狀態：**
> itemize
>     \item KCNQ1/KCNH2變異：減少外向鉀電流
>     \item SCN5A變異：增加內向鈉電流持續時間
>     \item 結果：復極化延長，QT間期延長
>     \item 運動時風險：QT間期未能適當縮短，可能引發心室顫動
> itemize

### 修飾基因


> **發現來源：**南非創始人群體研究，發現攜帶相同變異（KCNQ1-A341V）的數百名患者，QTc表現差異極大。
> **修飾基因定義：**能夠增強或減弱致病變異後果的遺傳因素，通常為常見變異。
> **臨床意義：**
> enumerate
>     \item **風險分層優化：**幫助更精準評估個體風險，制定更積極或保守的治療策略
>     \item **治療靶點：**修飾基因的細胞機制為開發針對特定分子路徑的新療法鋪路
>     \item **已發現的修飾基因：**NOS1AP、MTMR4等

## 臨床表現與診斷


### 心電圖特徵


長QT症候群的心電圖特徵包括：


| p{10cm}} 長QT症候群心電圖特徵 |  |  |
| --- | --- | --- |
| **特徵** | **臨床意義** |  |
| QT間期延長 | 男性>440 msec，女性>460 msec為異常；>500 msec提示中至高度風險 |  |
| T波形態異常 | 寬基底T波、雙相T波、T波切跡（尤其LQT2型） |  |
| T波交替 | 心臟電不穩定的重要指標，是心室顫動的前兆 |  |
| 運動恢復期QT延長 | 運動後第4分鐘QT明顯延長，診斷特異性高 |  |
| T-P波融合 | 運動峰值時T波與P波完全融合 |  |


### QT間期測量


> **測量方法：**
> itemize
>     \item 從Q波起點測量至T波返回基線點
>     \item 切線法常低估真實復極化長度
>     \item 應測量**最長的QT間期**
> itemize
> **Bazett校正公式：**
> itemize
>     \item QTc = QT / $RR-interval$
>     \item 儘管有局限性，仍能有效區分正常與異常值
>     \item 適用於嬰兒
> itemize
> **風險分層：**
> itemize
>     \item QTc >500 msec：中至高度心律不整風險
>     \item 測量誤差約10 msec
> itemize

### 診斷輔助檢查


> **推薦檢查：**
> itemize
>     \item **運動壓力測試：**唯一真正有用的測試
>     itemize
>         \item 恢復期第4分鐘QT明顯延長，對LQTS高度特異
>         \item 運動峰值T-P波融合現象
>     itemize
>     \item **12導程24小時Holter監測：**可發現典型變化，尤其夜間
>     \item **基因檢測：**在邊緣病例中協助診斷
> itemize
> **不推薦檢查：**
> itemize
>     \item **立位測試：**價值有限
>     \item **腎上腺素激發試驗：**具有危險性，可能引發心律不整，且可能在正常人中引起QT延長導致誤診
>     \item 歐洲心臟病學會指引明確指出不應使用腎上腺素測試進行診斷
> itemize

## 治療策略


### 治療四大支柱


長QT症候群的治療建立在四大支柱之上：


1. **乙型阻斷劑（Beta-blockers）**
2. **Mexiletine（鈉通道阻斷劑）**
3. **左心交感神經去除術（LCSD）**
4. **植入式心臟去顫器（ICD）**


### 乙型阻斷劑治療


> **歷史地位：**自1970年代中期以來，一直是LQTS治療的基石。
> **療效證據：**
> itemize
>     \item 療效已被反覆證實，不受基因型影響
>     \item 大幅降低心律不整事件發生率
> itemize
> **推薦藥物與劑量：**
> itemize
>     \item **Propranolol：**2.0-3.5 mg/kg/day
>     \item **Nadolol：**1.0-1.5 mg/kg/day
>     \item **不建議：**Metoprolol療效不佳，不應使用
> itemize
> **適應症：**
> itemize
>     \item 所有確診LQTS患者
>     \item 基因型陽性-表型陰性者（少數例外）
>     \item 例外：25歲以上仍無症狀的LQT1男性患者
> itemize


| p{9cm}} 乙型阻斷劑治療失敗的常見原因 |  |  |
| --- | --- | --- |
| **原因** | **說明** |  |
| 服藥順從性不佳 | 未按時或未按劑量服藥，是最常見原因 |  |
| 使用QT延長藥物 | 併用其他會延長QT間期的藥物 |  |
| 劑量不足 | 使用非治療劑量（「安慰劑劑量」） |  |
| 藥物選擇錯誤 | 使用療效較差的乙型阻斷劑（如metoprolol） |  |


### 左心交感神經去除術（LCSD）


> **手術方式：**
> itemize
>     \item 目前主要採用胸腔鏡手術
>     \item 切除星狀神經節下半部（避免Horner症候群）及T1-T4胸交感神經節
> itemize
> **作用機制：**
> itemize
>     \item 強大的抗心室顫動效應
>     \item 大幅減少心室層面的正腎上腺素釋放
>     \item 無去神經後敏感性增加
>     \item 不會降低心跳
> itemize
> **臨床療效：**
> itemize
>     \item 成功率極高
>     \item 對電風暴患者，ICD電擊年發生率降低90
>     \item 大多數患者QTc臨床顯著縮短
>     \item QTc縮短程度與長期保護相關
> itemize


| p{9cm}} 左心交感神經去除術適應症 |  |  |
| --- | --- | --- |
| **適應症** | **臨床情境** |  |
| 首選適應症 | 接受足量乙型阻斷劑治療後仍有暈厥復發 |  |
| 電風暴 | 多次心室心搏過速-心室顫動發作，導致多次ICD電擊 |  |
| 術前考量 | 應在考慮植入ICD前，告知患者LCSD的利弊 |  |
| 全球可及性 | 越來越多中心能夠進行此手術 |  |
| 生活品質 | 顯著改善生活品質，特別是對電風暴患者 |  |


### Mexiletine治療


> **發展歷史：**
> itemize
>     \item 1995年提出：發現SCN5A變異增加鈉電流後，提出mexiletine作為LQT3的首個基因特異性治療
>     \item 目前廣泛應用於LQT3患者
> itemize
> **作用機制：**
> itemize
>     \item 鈉通道阻斷劑
>     \item 縮短QTc，降低心律不整風險
>     \item 並非所有LQT3變異都有反應
> itemize
> **新發現：**近期數據顯示約70
> **急性口服藥物測試：**
> itemize
>     \item 劑量：6-8 mg/kg
>     \item 評估時間：2小時內
>     \item 有效標準：QTc縮短>40 msec
>     \item 優點：僅對有反應患者啟動長期治療
> itemize


| p{3cm}p{7cm}} Mexiletine在不同基因型中的應用 |  |  |
| --- | --- | --- |
| **基因型** | **療效** | **臨床考量** |
| LQT1 | 未知 | 缺乏充分證據 |
| LQT2 | 約70 | 急性測試評估QTc縮短>40 msec |
| LQT3 | 大多數有效 | 首個基因特異性治療，主要目標縮短QTc |


### 植入式心臟去顫器（ICD）


> **全球使用差異：**
> itemize
>     \item 美國部分中心：近50
>     \item Mayo Clinic與Istituto Auxologico：僅約5
> itemize
> **過度使用問題：**
> itemize
>     \item 許多植入ICD患者未經歷心臟停止
>     \item 許多患者未出現乙型阻斷劑治療失敗
>     \item LQT3組：45
> itemize
> **研究數據：**
> itemize
>     \item 233名ICD患者研究：平均追蹤5年，25
>     \item 近1000名患者數據：三聯療法（乙型阻斷劑+mexiletine+LCSD）與每年治療優化，幾乎所有患者可存活，ICD使用極少
> itemize


| p{8cm}} 植入式心臟去顫器適應症建議 |  |  |
| --- | --- | --- |
| **適應症類別** | **具體情況** |  |
| **強烈建議** | 接受適當藥物治療期間倖存心臟停止 |  |
| **條件性建議** | 接受足量乙型阻斷劑後仍有暈厥，且LCSD與mexiletine不可用或不適用 |  |
| **最後手段** | 接受足量乙型阻斷劑+LCSD後仍有暈厥 |  |
| **裝置選擇** | 優先選擇經靜脈ICD（非皮下），允許起搏功能 |  |
| **起搏重要性** | 需要增加乙型阻斷劑劑量時，或電風暴期間，起搏功能至關重要 |  |

## 臨床管理要點


### 基因型特異性管理


| p{3.5cm}p{3.5cm}p{4cm}} 不同基因型的管理策略 |  |  |
| --- | --- | --- |
| **管理面向** | **LQT1** | **LQT2** |
| 乙型阻斷劑 | +++ | ++ |
| LCSD | +++ | ++ |
| Mexiletine | 未知 | ++ |
| 誘發因素 | 交感激活-劇烈運動、游泳、強烈情緒 | 驚嚇（突發巨響、鬧鐘、電話鈴聲）、低血鉀、產後期 |
| 特殊建議 | 限制劇烈運動（游泳允許但需成人監督）、避免言語或肢體衝突、每年評估風險 | 維持血鉀$\geq$4 mmol/L、避免臥室鬧鐘和電話、乙型阻斷劑早晚服用、產後臥室共享以保護睡眠、每年評估風險 |


### 風險再評估的重要性


> **核心概念：**長QT症候群的心律不整風險會隨時間變化，需要治療優化。
> **定期評估：**
> itemize
>     \item 患者應至少每年追蹤一次
>     \item 允許治療優化（三聯療法或ICD）
> itemize
> **風險分層工具的局限性：**
> itemize
>     \item 不應在首次就診或治療開始前應用風險評分
>     \item 治療本身會改變心律不整傾向
>     \item 在治療優化前決定植入ICD並不合理
> itemize
> **近期證據：**
> itemize
>     \item 946名患者研究：若嚴格按風險評分，應植入142例ICD，實際僅植入22例
>     \item 僅3例接受適當ICD電擊，無患者死亡或心臟停止
>     \item 追蹤期間，部分患者風險增加，強化治療後預防了心律不整發作
> itemize
> **2861名患者數據：**僅少數符合指引ICD植入標準的患者實際需要ICD。


### 基因型陰性-表型陽性患者


| p{9cm}} 基因型陰性患者的管理 |  |  |
| --- | --- | --- |
| **項目** | **說明** |  |
| 發生率 | 10-15 |  |
| 風險評估 | 832名患者數據顯示，基因型陰性-表型陽性與基因型陽性-表型陽性患者心律不整風險相似 |  |

## 新興治療方向


### 新藥理學治療


| p{10cm}} 正在研究的新藥理學治療 |  |  |
| --- | --- | --- |
| **治療策略** | **機制與進展** |  |
| Lumacaftor-Ivacaftor聯合療法 | 已用於囊性纖維化治療，基於初步觀察，正在評估用於治療具有trafficking缺陷的LQT2患者 |  |
| SGK1抑制劑 | 血清/糖皮質激素調節激酶1調節心臟鈉通道，其抑制已在LQT2和LQT3模型中縮短心室復極化 |  |
| 臨床相關性要求 | 任何新療法的潛在臨床相關性需要：QTc變化不僅方向正確，且具有臨床意義（即縮短>40 msec） |  |


### 基因治療


> **理論基礎：**
> LQTS主要涉及單核苷酸變異，影響離子通道功能，基因治療應能：
> itemize
>     \item 沈默變異等位基因，或
>     \item 通過直接編輯方法矯正特定變異
> itemize
> **早期策略的局限性：**
> enumerate
>     \item **基因沈默：**使用小RNA靶向病原性變異區域並阻斷變異等位基因表達
>     itemize
>         \item 在LQT1和LQT2細胞模型中成功應用
>         \item 主要限制：變異特異性，數百種LQTS變異限制臨床應用
>     itemize
>     \item **直接編輯：**同樣受變異特異性限制
> enumerate
> **新策略：抑制-替換療法**
> itemize
>     \item 成功矯正多種LQT1和LQT2細胞模型的LQTS表型
>     \item 克服了其他方法的主要限制（不依賴特定變異）
>     \item 在LQTS兔模型中得到驗證
>     \item 最近用於鈣調蛋白病治療
> itemize


| p{9cm}} 基因治療面臨的主要挑戰 |  |  |
| --- | --- | --- |
| **挑戰** | **說明** |  |
| 劑量確定 | 需確定正確劑量：治療不足無法矯正LQTS表型，過度治療可能致心律不整 |  |
| 細胞轉導異質性 | 基因治療很少產生均勻轉導的細胞群，可能因復極化異質性增加而致心律不整 |  |
| 安全性考量 | 基因治療曾出現不良事件，包括致命性事件 |  |
| LQTS特殊性 | LQTS心律不整風險不隨時間增加，現有治療死亡率極低 |  |
| 適用人群 | 現有治療效果極佳，大大減少需要實驗性方法的患者數量 |  |
| 潛在適應症 | 可能更適合極高風險患者（如第一年內心臟事件的嬰兒、某些鈣調蛋白變異或嚴重疾病變異如SCN5A p.R1623Q患者、儘管全面治療仍有ICD電擊者） |  |

### 後天性LQTS的病因


| p{9cm}} 後天性長QT症候群的常見原因 |  |  |
| --- | --- | --- |
| **原因類別** | **具體因素** |  |
| 電解質異常 | 低血鉀 |  |
| 心律異常 | 心搏過緩、心臟傳導阻滯 |  |
| 藥物因素 | 具有IKr阻斷活性的藥物 |  |
| 臨床意義 | 攜帶顯著torsades de pointes與猝死風險 |  |
| 預防策略 | 矯正致病因素可預防復發 |  |


### 遺傳易感性


> **復極化儲備概念：**
> 後天性LQTS的發生概率取決於：
> enumerate
>     \item 特定藥物的內在風險（主要取決於IKr阻斷強度）
>     \item 個體復極化儲備水平（由遺傳因素調節）
> enumerate
> **遺傳易感性：**
> itemize
>     \item 超罕見變異
>     \item 罕見變異
>     \item 常見遺傳變異
> itemize
> **識別高風險患者的變數：**
> enumerate
>     \item 年齡<40歲
>     \item 基線QTc >440 msec
>     \item 有心律不整發作史
> enumerate
> 這些因素提示後天性LQTS有時可能揭示低外顯率的潛在先天性LQTS（1982年假設）。


| p{9cm}} 與後天性LQTS相關的特定遺傳變異 |  |  |
| --- | --- | --- |
| **變異類型** | **具體變異與解釋程度** |  |
| 罕見功能變異 | KCNE1 p.D85N與SCN5A p.S1103Y，持續與後天性LQTS相關 |  |
| 常見遺傳變異組合 | 61個常見變異（均影響QT間期），可解釋後天性LQTS高達30 |  |
| 常見變異檢測 | 目前不建議在研究設置外檢測常見變異 |  |
| 分子機制 | 後天性LQTS例證遺傳與後天因素組合如何損害復極化儲備並引發心律不整事件 |  |


### 運動誘發的QT延長


> **臨床現象：**
> itemize
>     \item 過度體能訓練可誘發明顯QT延長，模擬LQTS
>     \item 尤其常見於青少年
>     \item 可能導致長期後果的診斷錯誤
> itemize
> **典型特徵：**
> itemize
>     \item 無LQTS家族史
>     \item 無症狀
>     \item 基因型陰性
>     \item 心室復極化異常可通過3-4個月減訓練逆轉
> itemize
> **臨床建議：**
> itemize
>     \item 心律不整潛力尚不清楚
>     \item 需降低體能訓練強度以預防QT延長
>     \item 運動醫學醫師應警覺此現象，避免過早將年輕人標記為LQTS
> itemize

## 研究限制與未來方向


### 當前研究的局限性


1. **基因型陰性病例：**10-15
2. **外顯率變異：**即使攜帶相同變異，表型差異仍很大
3. **修飾基因研究：**需更多研究確定修飾基因及其臨床應用
4. **風險分層工具：**現有評分系統可能導致過度治療，需改進
5. **新療法驗證：**新藥理學治療與基因治療需更多臨床試驗
6. **長期預後數據：**需要更長期追蹤研究評估不同治療策略的長期效果

## 結論


### 主要結論


> **疾病認識：**
> itemize
>     \item 長QT症候群是年輕人猝死的主要遺傳性原因
>     \item 盛行率至少1/2000，實際可能更高
>     \item 以QT間期延長、T波形態異常和致命性心律不整為特徵
> itemize
> **遺傳基礎：**
> itemize
>     \item 主要由KCNQ1、KCNH2、SCN5A三個基因變異引起（約90
>     \item 修飾基因影響疾病嚴重程度和治療反應
>     \item 基因型決定誘發因素和最佳治療策略
> itemize
> **診斷要點：**
> itemize
>     \item 心電圖模式識別極為重要
>     \item 運動壓力測試是最有用的輔助診斷工具
>     \item 基因檢測確認診斷並指導治療
>     \item 避免腎上腺素激發試驗（危險且可能誤導）
> itemize
> **治療進展：**
> itemize
>     \item 乙型阻斷劑仍是治療基石
>     \item LCSD極度有效，應在考慮ICD前使用
>     \item Mexiletine對LQT2和LQT3有效
>     \item ICD應限用於少數高風險患者
>     \item 三聯療法與每年治療優化使幾乎所有患者可存活
> itemize
> **管理原則：**
> itemize
>     \item 基因型特異性治療與風險評估
>     \item 定期重新評估風險與治療優化
>     \item 避免在治療優化前進行風險評分
>     \item 生活方式調整與避免QT延長藥物同等重要
> itemize
> **未來展望：**
> itemize
>     \item 新藥理學治療正在評估中
>     \item 基因治療顯示前景但仍需克服挑戰
>     \item 現有治療已極為有效，減少實驗性治療需求
> itemize


### 臨床實務啟示


1. **早期識別至關重要：**猝死常為首發症狀，強調及時診斷的重要性
2. **避免漏診：**對於QT延長的兒童或青少年，排除次發原因後應懷疑LQTS
3. **轉診專科中心：**不確定診斷時應轉介有LQTS經驗的中心
4. **基因型指導治療：**了解基因型有助於識別誘發因素和選擇最佳治療
5. **避免過度治療：**特別是ICD的使用，應基於治療優化後的風險再評估
6. **強調依從性：**藥物依從性不佳是治療失敗的主要原因
7. **家族篩檢：**確診患者的一等親應接受評估
8. **新生兒篩檢：**考慮心電圖篩檢以識別高風險嬰兒
9. **避免錯誤診斷：**過度訓練可引起QT延長，需與先天性LQTS區分
10. **多學科團隊：**需要心臟科醫師、遺傳諮詢師、電生理學家等共同管理
