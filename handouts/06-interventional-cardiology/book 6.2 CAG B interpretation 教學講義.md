## 前言

冠狀動脈造影 (Coronary Angiography) 是經皮冠狀動脈介入治療 (PCI) 14 個步驟中的第 6 步，於導管成功嵌入後執行。本講義將詳細說明如何系統性地解讀冠狀動脈造影影像。

## 1. 影像品質評估

在進行解讀之前，必須先確認影像品質是否足以進行準確判讀。

### 1.1 影像品質不佳的常見原因

- 導管嵌入 (engagement) 不佳導致冠狀動脈顯影不完全
- 冠狀動脈管腔未完全不透明化 (opacification)（顯影不清楚）
- 缺少對比劑回流至主動脈
- 未以正交投影 (orthogonal projections) 評估所有冠狀動脈節段

> **Pearl**: 每條冠狀動脈必須在至少兩個正交投影中評估。必須有對比劑回流至主動脈（小心是否有開口狹窄）。所有冠狀動脈節段都應清楚顯影。

## 2. 冠狀動脈造影的 13 項評估要素

### 2.1 冠狀動脈血流 — TIMI 分級系統

TIMI (Thrombolysis In Myocardial Infarction) 分級用於評估冠狀動脈血流：

| 分級 | 定義 |
|------|------|
| TIMI 0 | 完全閉塞，狹窄處無任何順向血流 |
| TIMI 1 | 對比劑可通過狹窄處，但未到達冠狀動脈末端 |
| TIMI 2 | 對比劑可到達冠狀動脈末端，但流速比對側正常冠狀動脈慢 |
| TIMI 3 | 正常順向血流，流速與未受影響的動脈相同 |

### 2.2 狹窄嚴重程度

**狹窄程度評估**
- 以直徑百分比狹窄 (% Diameter Stenosis) 報告
- 計算公式：狹窄百分比 = (1 − 最小管腔直徑 / 參考直徑) × 100%
- 參考直徑 = (近端參考直徑 + 遠端參考直徑) / 2

**狹窄嚴重程度分類**
- ≥70%：通常視為顯著狹窄
- 50-69%：中度狹窄
- < 50%：輕度狹窄

> **Pearl**: 直徑百分比狹窄與血流動力學顯著性並非完全相關！71-90% 狹窄中，部分 FFR > 0.80（非血流動力學顯著）；< 70% 狹窄中，部分 FFR < 0.80（血流動力學顯著）。不確定時可使用 FFR 或 iFR 進行評估。

**正交投影的重要性**
- 偏心性病變在某些投影角度可能完全看不到
- 必須在最嚴重投影角度報告狹窄程度
- 至少需要兩個正交投影以避免低估或高估

### 2.3 硝化甘油的使用

> **Pearl**: 每次造影前應給予硝化甘油（新竹：Millisrol 100 ug IC）。目的是避免將多處血管痙攣誤判為多處狹窄。未給予硝化甘油可能導致假陽性狹窄診斷。

### 2.4 血栓

**血栓的特徵**
- 冠狀動脈內的充盈缺損 (filling defect)
- 通常呈圓形或橢圓形
- 常見於急性心肌梗塞患者

**血栓與慢性完全閉塞的鑑別**
- 對應心電圖 ST 段抬高區域
- 對應心臟壁運動異常區域
- 充盈缺損提示血栓存在
- 對比劑在血栓內滯留

### 2.5 鈣化

- 正常情況：注射對比劑前，冠狀動脈在 X 光下不可見
- 鈣化：注射對比劑前即可見冠狀動脈輪廓
- 嚴重鈣化：動脈壁兩側均有明顯不透射線的影像

> **Pearl**: 冠狀動脈造影對鈣化的偵測敏感度較低，血管內影像 (IVUS/OCT) 更為敏感。

### 2.6 夾層 (Dissection)

**NHLBI 夾層分級**

| 分級 | 描述 |
|------|------|
| A | 小範圍管腔模糊 |
| B | 線狀夾層，未影響血流 |
| C | 夾層伴隨對比劑外滲 |
| D | 螺旋狀夾層 |
| E | 新的持續性充盈缺損 (filling defect) |
| F | 血管完全閉塞 |

**夾層的類型**
- **醫源性夾層**：介入治療過程中造成
- **自發性冠狀動脈夾層 (SCAD)**：
  - 常見於女性
  - 產後期間風險增加
  - 可能多處血管同時受影響

> **Pearl**: 注射對比劑前必須仔細評估壓力波形，避免在壓力波形衰減的情況下注射對比劑。

### 2.7 分叉病變 (Bifurcation Lesions)

**定義**：冠狀動脈分支處的病變，涉及主血管與分支血管。

**Medina 分類系統**

使用三個數字（0 或 1）描述分叉病變：
- 第一個數字：近端主血管
- 第二個數字：遠端主血管
- 第三個數字：分支血管
- 1 = 有顯著狹窄（≥50%），0 = 無顯著狹窄

例如：Medina (1,1,1) 表示三處均有顯著狹窄

### 2.8 開口病變 (Ostial Lesions)

**開口病變的類型**
- 主動脈開口病變：冠狀動脈或繞道血管起源處
- 分支開口病變：如 LAD - 對角支分叉處

**診斷要點**
- 必須有對比劑回流至主動脈才能評估
- 可能造成壓力波形衰減
- 注射對比劑時需特別小心，避免造成開口夾層

### 2.9 血管迂曲度 (Tortuosity)

- 冠狀動脈出現多處彎曲
- 增加導線通過難度
- 設備輸送困難
- 可能需要特殊技術和設備

### 2.10 動脈瘤 (Aneurysm)

- **定義**：冠狀動脈直徑 > 1.5 倍參考直徑
- **治療**：可能需要使用覆膜支架 (covered stent)，需評估血栓形成風險

### 2.11 側支循環 (Collateral Flow)

**發生時機**
- 慢性完全閉塞 (CTO) 患者常見
- 身體建立替代血流通路
- 提供閉塞處遠端組織血流

**臨床意義**
- CTO 介入治療的評估重點
- 決定是否適合逆行性技術
- 評估側支循環可通過性

**特殊情況：鎖骨下動脈竊血**
- 近端鎖骨下動脈顯著狹窄
- 造成內乳動脈 (IMA) 逆向血流
- 影響 IMA-LAD 繞道血管功能

### 2.12 穿孔 (Perforation)

**定義**：對比劑從冠狀動脈管腔流出至心包腔或周圍組織。

**穿孔的類型**
1. 大血管穿孔
2. 遠端導線穿孔
3. 側支循環穿孔（CTO 介入相關）
4. 微穿孔

> **Pearl**: 穿孔必須立即識別！延遲處理可能導致心包填塞，可能致命，需要快速反應與適當處置。

### 2.13 瘺管 (Fistula)

- **定義**：冠狀動脈與其他結構（如肺動脈、心室、心房）之間的異常連接
- 較為少見
- 顯著血流量可能成為心內膜炎病灶
- 可能需要經皮或手術閉合

## 3. 冠狀動脈造影解讀檢查表

### 3.1 術前檢查表

**I. 影像品質確認**

- □ 導管嵌入良好
- □ 所有冠狀動脈節段完全不透明化
- □ 對比劑回流至主動脈（評估開口病變）
- □ 每條血管至少有兩個正交投影
- □ 已給予硝化甘油
- □ 壓力波形正常（避免夾層風險）

### 3.2 系統性評估檢查表

| 項目 | 評估要素 | 檢查內容 |
|------|----------|----------|
| □ | 1. 冠狀動脈血流 | TIMI 0/1/2/3（圈選），評估所有主要血管 |
| □ | 2. 狹窄程度 | LM: ___%、LAD: ___% (近端/中段/遠端)、LCX: ___% (近端/中段/遠端)、RCA: ___% (近端/中段/遠端)。報告最嚴重投影角度。是否需要 FFR/iFR？ |
| □ | 3. 硝化甘油 | 已給予：是/否。給予後病變變化：有/無 |
| □ | 4. 血栓 | 有充盈缺損：是/否。位置、大小、是否需要血栓抽吸 |
| □ | 5. 鈣化 | 無/輕度/中度/嚴重。位置。是否需要旋磨術或 IVUS/OCT 評估 |
| □ | 6. 夾層 | NHLBI 分級：A/B/C/D/E/F。類型：醫源性/自發性。是否影響血流 |
| □ | 7. 分叉病變 | Medina 分類：(_,_,_)。分支血管重要性。是否需要雙支架 |
| □ | 8. 開口病變 | 類型：主動脈開口/分支開口。有對比劑回流至主動脈：是/否 |
| □ | 9. 血管迂曲度 | 無/輕度/中度/嚴重。可能影響設備輸送：是/否 |
| □ | 10. 動脈瘤 | 位置、大小。是否需要覆膜支架 |
| □ | 11. 側支循環 | Rentrop 分級：0/1/2/3。Werner 分級 (CTO)：CC0/CC1/CC2。適合逆行性技術：是/否。鎖骨下動脈竊血：是/否 |

## 4. 臨床建議與最佳實務

### 4.1 解讀冠狀動脈造影的黃金法則

1. **系統性評估**：使用檢查表確保不遺漏任何重要發現
2. **正交投影**：每條血管至少兩個角度
3. **動態評估**：觀察整個心動週期的血管變化
4. **對照評估**：比較對側或其他血管的血流
5. **整合資訊**：結合臨床表現、心電圖、心臟超音波

### 4.2 常見錯誤與避免方法

| 常見錯誤 | 避免方法 |
|----------|----------|
| 未給予硝化甘油 | 每次造影前常規給予 (radial, PO, IC) |
| 僅單一投影評估 | 至少兩個正交投影 |
| 低估鈣化程度 | 考慮使用 IVUS/OCT |
| 遺漏開口病變 | 確保對比劑回流至主動脈 |
| 忽視壓力波形 | 注射前必須檢查波形 |
| 過度依賴視覺估計 | 不確定時使用 FFR/iFR |

### 4.3 何時需要額外檢查

- **FFR/iFR**：50-90% 中度狹窄，不確定血流動力學顯著性
- **IVUS**：
  - 左主幹病變評估
  - 嚴重鈣化評估
  - 支架植入後最佳化
  - 不明原因支架內再狹窄
  - Dissection (夾層) 程度評估
- **OCT**：
  - 血栓評估
  - 嚴重鈣化評估
  - 支架貼壁評估

## 5. 總結

冠狀動脈造影的準確解讀是成功執行 PCI 的基礎。通過系統性評估這 13 項要素，並使用標準化的檢查表，可以：

- 確保不遺漏重要病變
- 準確評估病變複雜度
- 制定適當的治療策略
- 預測並預防併發症
- 提高手術成功率

> **核心原則**:
> 1. 良好的影像品質是準確解讀的前提
> 2. 每條血管需要多角度評估
> 3. 系統性檢查所有 13 項要素
> 4. 必要時使用輔助檢查工具
> 5. 整合所有資訊制定治療計劃
