## 前言


心導管手術雖然是微創手術，但仍然可能發生嚴重併發症。本教材整理了七個真實案例，涵蓋導管室最常見且最具挑戰性的併發症，包括冠狀動脈剝離、導管糾結、支架脫落、心包填塞等。每個案例都著重於併發症的早期辨識、正確診斷與緊急處置策略。

[七個案例的來源- Retrievals, Foreign Bodies, and Entrapments](https://www.pcronline.com/Cases-resources-images/Resources/Course-videos-slides/2025/AICT-AsiaPCR/Retrievals-foreign-bodies-and-entrapments)，是EuroPCR官方網站在2025-11-30推播的教學資料。共有七份投影片，我是ESC也是EuroPCR會員，login後再製作成教學講義，請同仁有空參考投影片閱讀自學。

### 導管室併發症的重要性


- 早期辨識可以降低死亡率
- 正確處置可以避免緊急手術
- 團隊協作是成功的關鍵
- 學習前人經驗可以減少錯誤


importantbox
****核心原則****


1. **預防勝於治療**：了解高風險因素，預先準備
2. **早期辨識**：持續監測生命徵象與影像
3. **果斷處置**：不要猶豫，該介入就介入
4. **團隊合作**：立即啟動支援系統

## 案例一：主動脈冠狀動脈剝離


### 案例摘要


**作者：**Sugawara Yu


| p{10cm}}   **項目** | **內容** |  |
| --- | --- | --- |
| **病人資料** | 79歲女性，勞力型心絞痛 |  |
| **病史** | Cardiac CT顯示兩側冠狀動脈嚴重鈣化 |  |
| **處置** | 接受coronary angiography，發現distal RCA狹窄，決定執行PCI |  |
| **併發症發生** | 使用guide extension catheter時發生醫源性主動脈冠狀動脈剝離 |  |
| **發生機轉** | Guide extension catheter tip損傷內膜，後續contrast injection使情況惡化 |  |


### 併發症辨識


importantbox
****關鍵辨識點****


1. **影像學發現**：造影時發現contrast進入主動脈壁層
2. **血流動力學**：可能出現血壓下降或心律不整
3. **症狀**：胸痛加劇、呼吸困難
4. **心電圖**：可能出現新的ST段變化


importantbox


### 緊急診斷流程


| p{12cm}}   **步驟** | **內容** |  |
| --- | --- | --- |
| **第一步** | 立即停止所有操作，避免再次注射contrast |  |
| **第二步** | 使用IVUS確認dissection entry point位置 |  |
| **第三步** | 評估剝離範圍：是否延伸至ascending aorta |  |
| **第四步** | 確認true lumen血流是否維持 |  |
| **第五步** | 決定bail-out stenting策略 |  |


### 緊急處置


**處置策略**


1. **Early detection**：使用IVUS guidance找出entry point
2. **Seal the entry**：置放stent封閉剝離入口
3. **Support circulation**：維持systemic circulation與true lumen血流
4. **本案例處置**：Deployed 3.5 × 23mm drug-eluting stent覆蓋flap


### 預後與追蹤


- Final angiography顯示coronary flow完全恢復
- PCI後CT確認dissection未延伸至ascending aorta
- 選擇保守治療
- Bail-out stenting後無進一步併發症
- 病人安全出院
- 10個月後追蹤CT顯示aortocoronary dissection已癒合


### 學習重點


importantbox
****Take Home Messages****


- 主動脈冠狀動脈剝離發生率約0.09
- Guide extension catheter tip可能造成內膜損傷
- Contrast injection會使剝離惡化，**避免重複注射**
- Early detection與prompt stent placement是關鍵
- IVUS guidance對於找出entry point與確認治療效果至關重要

## 案例二：股動脈導管糾結


### 案例摘要


**作者：**A Ashika


| p{10cm}}   **項目** | **內容** |  |
| --- | --- | --- |
| **路徑** | 經右股動脈 |  |
| **併發症** | 診斷導管在寬闊的股動脈內糾結(kinked) |  |
| **初步嘗試** | 反向torquing失敗，導管無法解開 |  |
| **風險評估** | En bloc拉出可能造成股動脈破裂    強行拉出導管可能造成導管斷裂與栓塞 |  |
| **最終處置** | 使用左橈動脈路徑進行snaring成功取出 |  |


### 併發症辨識


importantbox
****辨識關鍵****


1. **影像發現**：透視下看到導管明顯糾結
2. **操作困難**：導管無法前進或後退
3. **壓力波形**：可能出現damping
4. **導管阻力**：感覺到異常阻力


importantbox


### 處置策略演進


| p{5.5cm}p{5.5cm}}   **方法** | **技術** | **結果/風險** |
| --- | --- | --- |
| **方法一** | Gentle traction與opposite torque | 失敗    風險：導管斷裂與糾結惡化 |
| **方法二** | Over a wire straightening    (試用0.014"與0.035") | 失敗    風險：強行嘗試可能斷裂 |
| **方法三** | Snaring via contralateral left femoral | 未嘗試    理由：因主動脈tortuosity，左股動脈可能有類似問題 |
| **方法四** | Inflate balloon near/inside kink | 未嘗試    理由：糾結太緊，連coronary wire都無法通過 |
| **方法五** | 手術取出 | 列為最後選項 |
| **最終方法** | **經左橈動脈snaring**    1. 左橈動脈取得access    2. Snare導管尖端    3. 同時在兩端alternately loosening與torquing    4. 導管拉出through right femoral route with wire inside | **成功**    Straightened the kink    無血管損傷 |


### 緊急處置原則


**處置決策樹**


1. **評估糾結位置**：確認在哪一段血管
2. **評估傳統方法可行性**：Gentle torquing, wire straightening
3. **創意思考**：考慮非傳統路徑(如橈動脈)
4. **準備backup plan**：必要時外科會診
5. **避免**：不要強行torquing造成導管斷裂


### 學習重點


importantbox
****Take Home Messages****


- **Don't Fight the Kink**：強行torquing可能造成斷裂
- **Endovascular creativity**：創意思考可以避免手術
- **Radial access versatility**：橈動脈路徑在複雜情況中的應用價值
- **Persistence and innovation**：大多數導管併發症可經皮處理
- 此案例展示在寬闊股動脈內糾結的罕見情況，以及橈動脈路徑的救援價值

## 案例三：經橈動脈導管糾纏


### 案例摘要


**作者：**Thanabalasingam Nitharshan

**研究目的：**分享兩例經橈動脈Tiger catheter糾纏的percutaneous retrieval技術


| p{10cm}}   **項目** | **內容** |  |
| --- | --- | --- |
| **背景** | 經橈動脈途徑因安全性與病人舒適度而日益受青睞 |  |
| **問題** | 罕見併發症如torquing導致的導管糾纏常被低報 |  |
| **挑戰** | 需要創新、適應性策略進行導管取出 |  |
| **目的** | 分享實用的管理方案，文獻報導有限 |  |


### 案例三A：經股動脈snaring


| p{10cm}}   **項目** | **內容** |  |
| --- | --- | --- |
| **病人** | 65歲男性，stable angina |  |
| **路徑** | 右橈動脈，使用5F Tiger catheter |  |
| **問題** | 多次嘗試engage RCA失敗    突然arterial waveform damping    透視顯示導管在腋動脈內kinked and looped |  |
| **失敗嘗試** | 標準maneuvers無法untoque導管 |  |
| **處置** | 1. 經股動脈取得access    2. Snare導管    3. 成功取出entire assembly |  |
| **結果** | 無血管損傷，病人恢復良好 |  |


### 案例三B：Sheathless guide catheter技術


| p{10cm}}   **項目** | **內容** |  |
| --- | --- | --- |
| **病人** | 72歲男性，stable angina |  |
| **路徑** | 右橈動脈，使用5F Tiger catheter |  |
| **問題** | 多次嘗試engage RCA失敗    突然arterial waveform damping    透視顯示導管在腋動脈內kinked and looped |  |
| **失敗嘗試** | 標準maneuvers無法untoque導管    經股動脈snaring失敗(因鎖骨下動脈顯著tortuosity) |  |
| **創新處置** | 1. Cut and remove導管distal hub    2. 7F guide catheter經橈動脈advanced over entrapped catheter (functioning sheathless)    3. Catheter and guide assembly一起取出 |  |
| **結果** | 成功達成止血，無血管損傷 |  |


### 技術細節比較


| p{5cm}p{5cm}}   **項目** | **方法A：Transfemoral snaring** | **方法B：Sheathless guide** |
| --- | --- | --- |
| **適應症** | 導管糾纏在近端較直的血管段 | 鎖骨下動脈tortuosity使snaring困難 |
| **優點** | 傳統方法，較為熟悉 | 不需額外femoral access |
| **技術要求** | 需要額外femoral puncture與snare技術 | 需要切除catheter hub    需要較大的guide catheter |
| **成功關鍵** | Snare要能捕捉到導管tip | Guide catheter要能順利advance over kinked catheter |


### 學習重點


importantbox
****The Essentials to Remember****


- **Why?** 導管torquing與糾纏是罕見但嚴重的經橈動脈併發症，需要有效的經皮解決方案
- **What?** 兩例coronary angiography時Tiger catheter糾纏案例
- **How?**        Case 1使用transfemoral snaring        Case 2使用sheathless guide catheter advancement over trapped catheter
- **Results?** 兩例均成功non-surgical retrieval，無血管損傷
- **Why important?** 分享可適應、可重現的retrieval技術，增強經橈動脈介入的準備度與手術安全性


importantbox


### 預防與處置原則


**關鍵原則**


1. **Prompt identification**：及早辨識糾纏
2. **Flexible approach**：靈活的retrieval策略
3. **Anatomical consideration**：考慮解剖變異
4. **Multiple techniques**：準備多種備用方法
5. **Avoid force**：避免強行操作造成血管或導管損傷

## 案例四：冠狀動脈剝離與假腔支架置放


### 案例摘要


**作者：**Fathima Zoubia


| p{10cm}}   **項目** | **內容** |  |
| --- | --- | --- |
| **病人** | 44歲男性，高血壓，吸菸者 |  |
| **主訴** | 胸痛45分鐘 |  |
| **初步處理** | Aspirin、clopidogrel    BP 134/95 mmHg, HR 111 bpm, SpO2 99 |  |
| **診斷** | 初始ECG：STEMI    給予tenecteplase    90分鐘後ECG：持續胸痛    Rescue PCI |  |
| **CAG發現** | Proximal LAD：severe stenosis    Proximal RCA：moderate to severe stenosis with unstable plaque involving ostium |  |
| **LAD PCI** | 1 DES置放，良好結果 |  |
| **併發症** | RCA intervention時發生**右冠狀動脈剝離** |  |


### 併發症發展與處置


| p{12cm}}   **階段** | **內容** |  |
| --- | --- | --- |
| **剝離發生** | Coronary angiography顯示RCA dissection |  |
| **第一次支架** | 3.5 × 48 mm DES implanted in proximal RCA    Coronary injection顯示血流不佳 |  |
| **IVUS評估** | **重大發現：**    Wire位於false lumen    Stent distal end位於false lumen |  |
| **Wire重新定位** | 使用microcatheter確認wire位置在true lumen    用guidewires securing access to true lumen |  |
| **第二次支架** | 3.0 × 38 mm DES implanted in mid-RCA    覆蓋剝離區域 |  |
| **Post-dilation** | 3.5 × 20 mm balloon |  |
| **最終結果** | 無residual stenosis，TIMI 3 flow |  |


### 後續追蹤


- 術後6小時：病人再次出現嚴重胸痛
- Re-look CAG：**LAD stent patent**
- Bedside echo：術後無pericardial effusion
- 次日echo：LVEF 35-40
- 住院第三天：血流動力學穩定、無痛
- 以optimal medical treatment出院


### 關鍵學習點


importantbox
****冠狀動脈剝離處理原則****


1. **Early identification**：及早辨識至關重要
2. **Avoid repeat contrast injections**：避免重複注射contrast以防止剝離延伸
3. **Hydrophilic wire**：可協助在剝離血管中適當定位wire
4. **Two-wire technique**：如懷疑wire在false lumen，留在原位，用第二條wire嘗試進入true lumen
5. **Microcatheter confirmation**：使用microcatheter確認wire位於true lumen    Distal injection要gentle，用small 2.5 mL syringe with 50/50 contrast
6. **Stenting strategy**：通常從distal到proximal    本案例為seal dissection並防止延伸，從proximal到distal
7. **Intravascular imaging**：IVUS或OCT對管理冠狀動脈剝離至關重要


importantbox


**Microcatheter的應用**

雖然冠狀動脈microcatheters傳統上用於複雜PCI導航，本案例展示了一個well-described approach：使用microcatheter確認wire在true lumen的位置。這是處理dissected vessel時的重要技術。

**避免支架脫落的策略**

- 遇阻力時，停下來重新評估，不要強行推送
- 準備retrieval techniques：snaring, wire wrapping
- Larger sheath may help但不保證成功

## 案例七：IVC-RA血栓與肺栓塞


### 案例摘要


**作者：**Khalil Hisham


| p{10cm}}   **項目** | **內容** |  |
| --- | --- | --- |
| **病人** | 41歲男性 |  |
| **受傷機轉** | 胸部鈍挫傷(重物墜落於胸部) |  |
| **入急診狀況** | Hypovolemic shock: BP 75/55 mmHg    Tachycardia, faint distal pulse    GCS 4/15    Intubated, mechanically ventilated, fluid resuscitation |  |
| **CT胸部** | Bilateral hemopneumothorax    → Bilateral chest tube insertion |  |
| **CT腹部** | Splenic laceration    → Urgent laparotomy + splenectomy    + Repair of right hepatic vein and IVC |  |
| **Day 4併發症** | Sudden severe dyspnea, tachypnea    Desaturation <90  BP 110/70, HR 100 bpm    Suspected PE → therapeutic anticoagulation initiated |  |
| **CTPA** | Segmental PE, no RV strain    HS Troponin 2.1 |  |
| **Echo surprise** | **Large lobulated highly mobile mass**    At junction of IVC and RA    Very likely to complicate to massive PE |  |


### 治療決策


importantbox
****複雜臨床情境****

**病人特性：**

- Polytrauma patient with recent splenectomy
- **High bleeding risk**
- Evidence of acute PE with segmental emboli
- Relatively stable hemodynamics
- TTE: large lobulated highly mobile mass at IVC-RA junction (proven by TEE)
- Very likely to complicate to massive PE


**Guideline困境：**

- No guideline consensus about catheter-in-transit (CIT) management


importantbox


### 治療選項分析


| p{5.5cm}p{6.5cm}}   **選項** | **方法** | **考量** |
| --- | --- | --- |
| **選項一** | 2nd surgery for RA-IVC mass embolectomy | 高風險：Recent laparotomy    Bleeding risk    Hemodynamically unstable |
| **選項二** | Percutaneous suction using large pore catheter | 相對侵入性較低    需要專業設備與技術    可能無法完全清除 |
| **選項三** | Anticoagulation vs systemic thrombolysis | Anticoagulation: 效果慢，血栓可能embolize    Systemic thrombolysis: **禁忌**(recent surgery, high bleeding risk) |


### 學習重點


importantbox
****關鍵教訓****


- **Catheter-in-transit**是危急狀況：high risk of massive PE
- **Polytrauma patients**有multiple competing risks
- **Recent surgery**限制治療選項
- **Heart team decision making**至關重要
- **Echo surveillance**對診斷與追蹤不可或缺
- **No guideline consensus**：需要個別化決策
- **平衡bleeding vs thrombosis risk**：最大挑戰


importantbox


**Ticking Bomb in Transit**

此案例標題"The ticking bomb in transit"生動描述了CIT的危急性。這種高度mobile的血栓隨時可能造成massive PE，但病人的recent surgery與high bleeding risk又限制了aggressive treatment options。這是真正的臨床兩難(clinical dilemma)。

## 綜合學習：併發症管理總論


### 共同原則


| p{10.5cm}}   **原則** | **說明** |  |
| --- | --- | --- |
| **Early recognition** | 所有案例的成功處置都源於早期辨識    持續監測：vitals, hemodynamics, ECG, fluoroscopy |  |
| **Prompt action** | 一旦辨識出併發症，立即行動    不要等待或猶豫 |  |
| **Avoid making it worse** | 停止可能惡化情況的操作    例如：剝離時避免重複注射contrast |  |
| **Use imaging wisely** | IVUS, OCT, TEE在診斷與指引治療上不可或缺    適時使用可以改變結果 |  |
| **Creative problem-solving** | 準備多種backup strategies    傳統方法失敗時，創意思考    例如：橈動脈snaring, sheathless guide |  |
| **Team approach** | 複雜併發症需要heart team decision making    及時會診外科、心臟超音波、麻醉科 |  |
| **Know your limitations** | 了解何時需要外援    準備轉換為手術治療 |  |


### 併發症分類與對策


| p{4.5cm}p{6cm}}   **併發症類型** | **代表案例** | **關鍵對策** |
| --- | --- | --- |
| **血管損傷** | Aortocoronary dissection    Coronary dissection | IVUS guidance    Seal entry point    Bail-out stenting    True lumen confirmation |
| **穿孔與填塞** | Wire perforation    Cardiac tamponade | Echo surveillance    Pericardiocentesis    Correct wire route    Monitor recurrence |
| **導管併發症** | Catheter kinking    Catheter entrapment | Multiple retrieval techniques    Snaring    Sheathless guide    Creative access routes |
| **器械問題** | Stent dislodgement    Unable to retrieve | Adequate preparation    Guide extension    Snaring techniques    Accept deformation if necessary |
| **血栓併發症** | CIT (catheter-in-transit)    PE | Heart team approach    Balance bleeding/thrombosis risk    Individualized treatment |


### 設備準備檢查表


importantbox
****導管室應備器材****


1. **影像設備：**IVUS catheter, OCT catheter
2. **Bail-out設備：**        各種尺寸covered stents        Guide extension catheters        Microcatheters
3. **Retrieval設備：**        Snares (various sizes)        Retrieval balloons        Additional wires (hydrophilic)
4. **Drainage設備：**        Pericardiocentesis kit        Pigtail catheters
5. **其他：**        Larger sheaths        Temporary pacing system        IABP/ECMO standby

### 團隊訓練重點


**併發症管理訓練**


1. **定期模擬訓練**：Practice retrieval techniques, pericardiocentesis
2. **案例討論**：Regular morbidity and mortality conferences
3. **流程演練**：Emergency protocols, team roles
4. **跨團隊溝通**：With cardiac surgery, imaging, anesthesia
5. **設備熟悉度**：Know where rescue equipment is located
6. **決策訓練**：Practice heart team decision making

## 總結與建議


### 給新進同仁的建議


importantbox
****學習重點****


1. **了解原理**：理解為何併發症發生，而非只是記住處置
2. **觀察資深醫師**：注意他們如何**預防**併發症
3. **練習基本技能**：導管操作、wire handling要扎實
4. **學習辨識**：訓練自己early recognition的能力
5. **培養系統性**：發展自己的checklist與routine
6. **重視團隊**：知道何時call for help
7. **永遠準備**：Mental preparation for complications
8. **終身學習**：持續更新知識與技術


importantbox


### 持續改進


- 定期review自己的併發症案例
- 參加相關教育訓練與workshop
- 與同事分享經驗與技巧
- 保持對新技術的開放態度
- 建立個人的complication management protocols


importantbox
****記住這三個核心****


1. **Prevention is key**：最好的併發症處置是預防發生
2. **Early beats late**：早期辨識與處置改變預後
3. **Team over individual**：團隊合作勝過個人英雄主義


導管室併發症雖然罕見，但發生時往往危及生命。通過系統性學習前人經驗，我們可以提升自己的處理能力，為病人提供更安全的醫療照護。
