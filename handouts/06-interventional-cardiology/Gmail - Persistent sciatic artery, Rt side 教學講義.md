## Persistent Sciatic Artery 簡介

### 定義

Persistent sciatic artery (PSA) 是一種罕見的先天性血管變異，正常情況下 sciatic artery 於胚胎發育早期退化，由 superficial femoral artery 取代成為下肢主要供血動脈。當 sciatic artery 未退化而持續存在時，即為 PSA。

### 臨床意義

- 發生率極低（約 0.03-0.06%）
- 可能合併 superficial femoral artery 發育不全或缺如
- 主要血供來自 **internal iliac artery**，經坐骨孔 (sciatic foramen) 延伸至下肢
- 易合併動脈瘤形成及血栓栓塞事件
- 進行下肢血管介入時需特別注意血管通路規劃

### 分型 (Pillet-Gauffre Classification)

| 類型 | 特徵 |
|------|------|
| Type 1 | PSA 為完整型，SFA 正常 |
| Type 2 | PSA 為完整型，SFA 發育不全 |
| Type 3 | PSA 為不完整型，僅殘留上段 |
| Type 4 | PSA 為不完整型，僅殘留下段 |
| Type 5 | PSA 合併動脈瘤形成 |

> **Pearl**: 當 angiography 發現導絲或導管行經 internal iliac artery 路徑到達下肢時，應高度懷疑 persistent sciatic artery 之存在。
