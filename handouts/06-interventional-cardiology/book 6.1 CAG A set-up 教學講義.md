## 前言


冠狀動脈造影（Coronary Angiography）是經皮冠狀動脈介入治療（PCI）14個步驟中的**第6步**，必須在導管成功嵌入（engagement）後才能執行。

本講義將詳細說明執行冠狀動脈造影的完整8個步驟，這些步驟對於確保病人安全、獲得高品質影像以及進行準確診斷都至關重要。

請使用本講義一起閱聽: [6.1 Manual of PCI - How to perform coronary angiography](https://www.youtube.com/watch?v=Y3bEFuzTpB8). 


### 這一系列教學文件的由來

### 冠狀動脈造影的重要性


- 提供冠狀動脈解剖的詳細資訊
- 評估病變的嚴重程度與特徵
- 指導介入治療策略
- 評估治療成效


importantbox
**安全警示**

執行冠狀動脈造影時，**嚴格遵守每個步驟**是防止嚴重併發症（如冠狀動脈剝離、急性血管閉塞、甚至死亡）的關鍵。切勿為求快速而省略任何步驟！
importantbox


## 冠狀動脈造影的八個步驟


### 步驟總覽


| p{5cm}p{7cm}}   **步驟** | **內容** | **目的** |
| --- | --- | --- |
| **1** | 確保良好的壓力波形 | 避免冠狀動脈剝離與其他併發症 |
| **2** | 小劑量測試注射 | 確認導管位置與評估潛在問題 |

## 步驟一：確保良好的壓力波形


### 重要性


importantbox
****關鍵步驟****：在注射任何對比劑之前，**必須**確認壓力波形正常！
importantbox


### 壓力波形衰減（Pressure Dampening）的風險


如果在壓力波形衰減的情況下注射對比劑，可能導致：


- **冠狀動脈剝離（Coronary dissection）**
- **心律不整**
- **其他嚴重併發症**


### 評估標準


**理想的壓力波形特徵：**

- 清晰的收縮壓峰值
- 明顯的舒張期下降
- 波形形態正常，無衰減（dampening）
- 數值與主動脈壓力相符


### 處理異常壓力波形


若發現壓力波形衰減：


1. **立即停止**：不可注射對比劑
2. **調整導管位置**：      - 輕微退出導管 - 旋轉導管角度 - 確認導管未過度嵌入（over-engagement）
3. **重新評估**：調整後再次檢查壓力波形
4. **考慮更換導管**：若持續衰減，考慮使用不同型號


## 步驟二：小劑量測試注射


### 測試注射的目的


在進行完整造影前，先給予**1-2 cc**的小劑量對比劑測試注射，目的包括：


| p{10cm}}   **評估項目** | **說明** |  |
| --- | --- | --- |
| **1. 確認導管嵌入** | 確保導管確實進入冠狀動脈而非其他位置 |  |
| **2. 檢查開口病變** | 評估冠狀動脈開口是否有狹窄（ostial lesion） |  |
| **3. 評估導管同軸性** | 確認導管與血管同軸（coaxial），未指向血管壁 |  |
| **4. 檢查嵌入深度** | 確保導管嵌入深度適當（理想為 2-3 mm），避免過深 |  |


### 測試注射的執行


1. 將注射器保持**垂直**（minimizing bubbles）
2. 緩慢注射 1-2 cc 對比劑
3. 觀察對比劑流向與分布
4. 根據結果調整導管位置


### 臨床實例


**案例一**：測試注射顯示導管指向左主幹頂部（roof of left main）而非同軸，需調整導管角度後再進行完整造影。

**案例二**：繞道血管（bypass graft）測試注射顯示血管完全閉塞，避免了不必要的大量對比劑注射。


### 注射前的關鍵檢查


importantbox
****防止氣泡栓塞****


- 注射前將 manifold 保持**垂直**
- 確保所有氣泡上升至頂部
- 避免氣泡進入導管線路
- 氣泡栓塞可能導致嚴重後果


importantbox


## 步驟三：給予冠狀動脈 Nitroglycerin


### 為何必須給予 Nitroglycerin


importantbox
****關鍵概念****

冠狀動脈是**動態結構**，可能發生痙攣（spasm）。若不給予 nitroglycerin，可能將血管痙攣誤判為固定性狹窄，導致錯誤診斷與不必要的介入治療。
importantbox


### 給藥劑量指引


| p{9cm}}   **血壓範圍（收縮壓）** | **Nitroglycerin 劑量** |  |
| --- | --- | --- |
| 正常血壓患者 | 200 mcg |  |
| 收縮壓 100-120 mmHg | 100 mcg |  |
| 收縮壓 $<$ 100 mmHg | 0-50 mcg（視情況給予） |  |
| 低血壓患者 | ****禁忌**** |  |


### 給藥方式


**方法一：經由 Manifold 側孔**

1. 從 manifold 側孔抽吸
2. 注射 nitroglycerin
3. 等待藥效（約 30-60 秒）


**方法二：Nitroglycerin-on-a-Stick 技術**

1. 經由導引鞘（introducer）直接注射
2. 藥物經由 Y-connector 後端進入
3. 適用於需要快速給藥的情況


### 臨床實例：血管痙攣


**重要案例**

初始造影顯示右冠狀動脈開口有顯著狹窄（significant ostial lesion）。然而，給予 nitroglycerin 後，該「病變」完全消失！

**結論**：這是由於導管操作引起的血管痙攣，而非真正的狹窄。若未給予 nitroglycerin，可能導致錯誤診斷與不必要的介入治療。


### Nitroglycerin 引起低血壓的處理


若給藥後出現低血壓：


1. **等待**：通常數分鐘內會自行恢復
2. **給予輸液**：Normal saline 靜脈輸注
3. **升壓劑**：罕見情況下（通常是迷走神經反應），可考慮給予 phenylephrine
4. **延遲造影**：等待血壓穩定後再進行


## 步驟四：最佳化位置設定


### 位置設定的雙重目標


1. **最佳化影像品質**：獲得清晰、完整的冠狀動脈影像
2. **最小化輻射暴露**：保護操作者免於散射輻射


### 正確的設置原則


| p{10cm}}   **設置項目** | **最佳化方法** |  |
| --- | --- | --- |
| **檢查台高度** | 盡可能升高（提高影像品質） |  |
| **影像接收器** | 盡可能靠近病人（降低輻射劑量，提高影像品質） |  |
| **鉛屏蔽** | 放置於病人與操作者之間 |  |
| **操作者位置** | 盡可能遠離散射源 |  |


### 輻射防護的物理原理


importantbox
****平方反比定律（Inverse Square Law）****

散射輻射強度與距離的**平方**成反比：

$$輻射強度 \propto 1{距離^2}$$

**實務意義**：操作者距離散射源越遠，輻射暴露呈指數下降。即使增加少許距離，也能顯著降低輻射暴露！
importantbox


### 鉛屏蔽的正確使用


**常見錯誤**


- **錯誤**：鉛屏蔽放置距離病人過遠
- **錯誤**：鉛屏蔽位置過高，無法阻擋來自病人背部的散射


**正確做法**


- **正確**：鉛屏蔽緊鄰病人
- **正確**：鉛屏蔽位置較低，阻擋病人背部散射輻射


### 延長管路的使用


使用延長管路可讓 manifold 距離散射源更遠，進一步降低操作者的輻射暴露。


### 肥胖患者的特殊考量


對於極度肥胖的患者：


- 影像品質會受影響
- **減少角度**：使用較少的成角（angulation）可獲得較清晰影像
- 可能需要增加輻射劑量以穿透組織
- 更需要注意輻射防護


### 常用投影角度


#### 左冠狀動脈（5個投影）


| p{4.5cm}p{7cm}}   **順序** | **投影角度** | **主要顯示血管** |
| --- | --- | --- |
| 1 | AP（Anterior-Posterior） | 左主幹（Left Main） |
| 2 | RAO Caudal | 迴旋支（Circumflex）與鈍緣支（Obtuse Marginals） |
| 3 | RAO Cranial | 左前降支（LAD） |
| 4 | LAO Cranial | LAD 與對角支（Diagonals） |
| 5 | LAO Caudal（Spider） | 左主幹分叉、Circumflex、近端 LAD |


#### 右冠狀動脈（3個投影）


| p{4.5cm}p{7cm}}   **順序** | **投影角度** | **主要顯示血管** |
| --- | --- | --- |
| 1 | LAO | 近端、中段、遠端 RCA |
| 2 | LAO Cranial | 遠端 RCA、PDA、後外側分叉 |
| 3 | RAO | 中段 RCA、PDA、後外側血管 |


**個人化調整**

每位操作者可能有自己偏好的投影順序與角度。重點是確保：

- 所有冠狀動脈節段都有評估
- 每個節段至少有兩個正交投影
- 避免遺漏任何病變


## 步驟五：檢查壓力與心電圖


### 再次確認的重要性


在實際注射對比劑前，**必須再次**檢查：


1. ****壓力波形****：確保仍然正常，無衰減
2. ****心電圖****：確認無 ST 段變化或心律不整


importantbox
****關鍵安全檢查****

**為何要再次檢查？**

從步驟一到步驟五之間，可能發生：

- 導管位置改變
- 導管過度嵌入
- 冠狀動脈痙攣
- 壓力波形衰減


如果此時壓力波形衰減，**不可注射對比劑**，必須先處理問題！
importantbox


## 步驟六：執行造影


### 造影前準備


1. 用對比劑充滿注射器
2. **保持注射器垂直**，確保氣泡在頂端
3. 轉動三通開關至正確位置
4. 準備開始造影


### 造影執行順序


| p{12cm}}   **順序** | **動作** |  |
| --- | --- | --- |
| **第一步** | Collimation（縮小視野）與調整 X 光影像接收器位置 |  |
| **第二步** | 啟動 cine（影像記錄） |  |
| **第三步** | 等待 1-2 秒（若有鈣化，此時可顯現） |  |
| **第四步** | 注射對比劑 |  |
| **第五步** | ****確認對比劑回流至主動脈****（評估開口病變） |  |
| **第六步** | 必要時進行 panning（移動影像）以顯示所有分支 |  |
| **第七步** | 在正交投影重複造影 |  |


### 對比劑注射技巧


**注射強度的掌握**


- **開始**：緩慢、溫和的注射
- **過程**：逐漸增加注射壓力
- **目的**：避免導管因過強注射而脫離（kick back）


**臨床案例**

如果一開始就用力注射，可能導致導管被推出血管。但若從緩慢注射開始，逐漸加壓，同一支導管可以穩定地完成造影。


### Guide Extension 的使用


若使用 guide extension：


- 造影時應**固定**住 guide extension
- 否則注射壓力可能將其推向冠狀動脈深處
- 可能造成血管損傷


### 確保開口病變可被評估


importantbox
****重要檢查點****

**必須看到對比劑回流至主動脈！**

如果對比劑沒有回流至主動脈，可能遺漏開口病變（ostial lesion）。開口病變是重要且常被遺漏的病變，必須確保每次造影都有評估。
importantbox


## 常見造影問題與解決方案


### 問題一：冠狀動脈顯影不完全


#### 臨床表現


- 冠狀動脈或繞道血管顯影不清楚
- 對比劑未充滿整個血管
- 影像品質不足以進行診斷


#### 可能原因


| p{9cm}}   **原因** | **說明** |  |
| --- | --- | --- |
| 導管過小 | 無法提供足夠的對比劑流量 |  |
| 導管有側孔 | 對比劑從側孔流失 |  |
| 冠狀動脈過大或擴張 | 需要更多對比劑 |  |
| 導管嵌入不良 | 對比劑未進入冠狀動脈 |  |
| 高冠狀動脈血流 | 常見於嚴重主動脈瓣狹窄或肥厚性心肌病變 |  |
| 對比劑黏度高 | 流動性差 |  |
| 注射力度不足 | 未提供足夠壓力 |  |
| 左心室舒張末期壓力高 | 增加冠狀動脈阻力 |  |


#### 解決方案


1. 改善導管嵌入
2. 使用更大的導管（如 6F 改為 7F）
3. 增加注射力度
4. 考慮使用低黏度對比劑
5. 評估並處理潛在的血流動力學問題


**實例對照**

**左側影像**：導管嵌入不良，右冠狀動脈顯影極差，無法進行診斷。

**右側影像**：改善導管嵌入後，整條 RCA 及開口都清晰可見，可進行準確診斷。


### 問題二：造影過程中導管脫離


#### 臨床表現


- 注射對比劑時導管退出血管
- 造影中斷
- 可能造成血管損傷


#### 可能原因


| p{9cm}}   **原因** | **說明** |  |
| --- | --- | --- |
| 導管嵌入不良 | 最常見原因 |  |
| 注射過於用力 | 特別是一開始就用力注射 |  |
| 導管支撐力不足 | 導管太軟或型號不適合 |  |
| 病人深呼吸 | 造成導管位置改變，橈動脈路徑尤其常見 |  |


#### 解決方案


1. **改善嵌入技術**：確保導管穩定嵌入
2. **使用不同導管**：選擇支撐力更好的型號
3. **放置導引鋼絲**：可穩定導管位置
4. **使用 guide extension**：提供額外支撐
5. **調整注射技巧**：**從緩慢開始，逐漸加壓**


**正確的注射技巧**

**錯誤做法**：一開始就用力注射 → 導管被推出

**正確做法**：緩慢開始注射，逐漸增加壓力 → 導管穩定，造影成功


### 問題三：部分冠狀動脈節段無法顯示


#### 可能原因


- 導管嵌入不良
- 注射力度不足
- 冠狀動脈節段重疊
- 病人體型過大
- 投影角度不足
- 病人位置不當（如：手臂擋住遠端 LAD）


#### 解決方案


1. 改善導管嵌入
2. 使用更大的導管
3. 增加並使用不同投影角度
4. 調整病人位置
5. 確保影像接收器可移動到所需位置


## 步驟七：再次檢查壓力與心電圖


### 造影後評估的重要性


造影後**立即**檢查：


1. ****心電圖****：確認無新的 ST 段變化或心律不整
2. ****壓力波形****：確認仍然正常


### 可能的併發症


importantbox
****立即識別併發症****

造影後可能立即發生：


- **心室顫動**：需立即去顫電擊
- **ST 段變化**：可能表示冠狀動脈剝離或急性閉塞
- **心搏過緩**：可能需要 atropine
- **壓力波形衰減**：可能表示導管引起的問題


早期識別與立即處理是關鍵！
importantbox


### 臨床實例


**案例一：心室顫動**

造影後立即發生心室顫動。因為操作者持續監測心電圖，立即識別並給予去顫電擊，成功恢復正常心律。

**案例二：一過性 ST 段變化**

注射對比劑後出現一過性 ST 段變化。確認為對比劑引起的暫時性變化，而非冠狀動脈剝離或閉塞。持續監測直到恢復正常。


## 步驟八：解讀造影


### 解讀的前提條件


解讀造影**完全依賴**於前七個步驟的優秀執行：


- 影像品質必須良好
- 所有冠狀動脈節段都已充分顯影
- 每個節段都有正交投影
- 有對比劑回流至主動脈


importantbox
****品質決定一切****

如果造影品質不佳，解讀可能**不可能**或**不準確**。

常見的品質問題：

- 冠狀動脈顯影不完全
- 缺少正交投影
- 未評估開口

## 避免併發症的關鍵原則


### 冠狀動脈剝離的預防


importantbox
****最嚴重的併發症****

冠狀動脈剝離（特別是左主幹剝離）可能導致：

- 急性血管閉塞
- 嚴重血流動力學惡化
- 死亡


**預防措施：**

1. **永遠不要**在壓力波形衰減時注射對比劑
2. 確保導管同軸性
3. 避免導管過度嵌入
4. 注射時從緩慢開始
5. 測試注射確認安全性


importantbox


### 氣泡栓塞的預防


1. 注射前將 manifold 與注射器保持垂直
2. 確保所有氣泡上升至頂端
3. 仔細檢查管路中有無氣泡
4. 抽吸時避免產生氣泡


### 心律不整的處理


- 持續監測心電圖
- 準備好去顫器
- 立即識別心室顫動並去顫
- 準備 atropine 處理心搏過緩


## 總結與最佳實務建議


### 成功的關鍵


1. **嚴格遵守所有步驟**：不可省略任何步驟
2. **重視技術細節**：每個細節都可能影響安全與影像品質
3. **持續監測**：造影前、中、後都要監測
4. **預期問題**：準備好處理常見問題
5. **保護操作者**：重視輻射防護


### 給新進同仁的建議


**學習重點**


1. **理解原理**：了解每個步驟為何重要
2. **觀察資深醫師**：注意他們如何處理問題
3. **練習基本技能**：導管操作、注射技巧
4. **學習識別問題**：壓力波形衰減、造影不良
5. **培養系統性**：使用檢查表，養成習慣
6. **重視安全**：永遠把病人安全放第一位


### 持續改進


- 定期回顧自己的造影品質
- 分析失敗案例，學習改進
- 參加教育訓練與討論會
- 保持對新技術的學習
- 與同事分享經驗


importantbox
****記住這三個核心原則****


1. **安全第一**：永遠不要為了速度而犧牲安全
2. **品質至上**：優秀的造影技術是準確診斷的基礎
3. **系統化**：遵循標準流程，使用檢查表


優秀的冠狀動脈造影技術需要時間和經驗累積，但嚴格遵守這八個步驟，可以確保每次造影都是安全且高品質的！
