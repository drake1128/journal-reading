1.3 前言 Introduction

經皮冠狀動脈介入術 (Percutaneous Coronary Intervention, PCI) 的成功關鍵之一在於導絲 (guidewire) 能否順利通過病灶。不同的病灶型態需要選擇適當的導絲，並運用正確的技術才能安全有效地完成手術。

本講義將詳細介紹導絲選擇的原則、polymer jacket 導絲的特性，以及 3D wiring method 等進階技術，幫助臨床醫師提升 PCI 的成功率與安全性。

1.4 1. 不同病灶類型的導絲選擇

1.4.1 1.1 近端彎曲合併鈣化病灶 — Proximal Tortuosity with Calcified Lesions

1.4.1.1 理想導絲特性 Ideal Guidewire Characteristics

特性 (Characteristic) 中文說明 English Description
操控性 (Handling) 能精確控制導絲尖端位置 Precise control of wire tip position
滑順性 (Smoothness) 減少通過病灶時的阻力 Minimize resistance when crossing lesions
柔軟度 (Flexibility) 能順應血管彎曲 Conform to vessel tortuosity
支撐力 (Support) 提供足夠的推送力 Provide adequate pushing force

1.4.1.2 實際選用導絲 Practical Guidewire Selection

製造商 (Manufacturer) 建議導絲型號 (Recommended Wire Models)
Terumo Runthrough NS Hypercoat, Runthrough NS Fullcoat

1.4.1.3 臨床考量 Clinical Considerations

Pearl: 在鈣化病灶中，導絲通過彎曲或鈣化區域時會遇到困難。除非鈣化病灶的狹窄非常嚴重，否則操作並不太困難。然而，近端彎曲會對導絲產生極大阻力，使得在鈣化病灶內的操作變得困難。

重要警告：過度施力可能導致進入 false lumen 的風險！

1.4.1.4 安全策略 Safety Strategy

Pearl — 最佳實踐建議: 1. 最小 coating 原則：使用最少量 hydrophilic coating 或 polymer jacket 的導絲可防止操作壓力和不必要的併發症 2. 導絲更換策略：通過病灶後，建議將 polymer-jacketed wire 更換為 coiled wire 以確保最大安全性

1.4.2 1.2 開口病灶 Ostial Lesions

1.4.2.1 理想導絲特性

  • 操控性 (Maneuverability)：能在開口處精確定位
  • 柔軟度 (Flexibility)：避免對血管壁造成損傷
  • 支撐力 (Support)：提供穩定的平台以利支架置放

1.4.2.2 實際選用導絲

製造商 建議導絲型號
Terumo Runthrough NS Extra Floppy, Runthrough NS Ultra Floppy
Asahi Intecc Rinato, Route, SION blue
Abbott HI-TORQUE TRAVERSE, HI-TORQUE ELITE II
Japan Lifeline ATHLETE Passista

1.4.2.3 PCI 成功關鍵 Keys to Successful PCI

  1. 支架位置 (Stent Positioning)：支架能否放置在適當位置是成功的關鍵
  2. Guide Catheter Support：需要適當尺寸的導引導管和足夠的 backup
  3. 導絲穩定性 (Wire Stability)：導絲的穩定性至關重要

1.4.2.4 導絲選擇原則

Pearl: 雖然 hydrophilic coating 導絲容易通過病灶，但放置支架時常需要將導引導管從冠狀動脈開口移出。因此，無 hydrophilic coating 的導絲較為適合。若因遠端血管型態不得不使用強 coating 導絲，基於安全理由應在適當時機更換導絲。

1.5 2. Polymer Jacket 導絲

1.5.1 2.1 定義與結構 Definition and Structure

PCI 使用的導絲依結構分為金屬基底型和 polymer jacket 型。Polymer 是由多個單體聚合而成的化合物，通常為高分子量有機化合物。以 polymer sleeve 包覆金屬內芯並施加 hydrophilic coating 的產品稱為 polymer jacket 導絲。

1.5.2 2.2 特性 Characteristics

1.5.2.1 優異的滑動特性 Excellent Lubricity

Polymer jacket 導絲最重要的特點是其滑順性 (lubricity)。

  • 優點：能夠滑入金屬導絲無法到達的病灶
  • 缺點：會減弱觸感，較難傳達血管和病灶的特性

1.5.2.2 導絲尖端負荷 Tip Load

Pearl: 此處的 tip load 不同於製造商標示的數值，而是指導絲尖端數毫米到 1 公分範圍內的硬度。因 polymer 施加在柔軟金屬線圈表面，導絲尖端意外地較硬。最簡單的評估方式：一手握住導絲尖端附近，另一手手指墊感受尖端硬度。

1.5.3 2.3 使用時機 Indications for Use

臨床情境 理由
鈣化狹窄病灶 (Calcified stenotic lesions) 因滑順性使導絲更容易通過，金屬導絲因摩擦力難以通過
支架網眼上的側枝選擇 (Side branch selection through stent struts) 支架網眼對導絲產生摩擦阻力，polymer jacket 導絲更易插入側枝
彎曲血管 (Tortuous vessels) 連續之字形彎曲和高度曲率使得血管壁對導絲摩擦阻力高
Reverse wire method 用於將導絲插入從主幹以鈍角分支的側枝，或狹窄病灶後立即分支的側枝（尤其是偏心病灶）
STAR 技術 (STAR technique) Subintimal tracking and re-entry (STAR) 技術涉及有意的 subintimal tracking

1.5.4 2.4 使用注意事項 Precautions for Use

Pearl 1 — 滲入斑塊 (Plaque Penetration)：因滑順性和局部較硬的尖端，容易滑入脆弱斑塊。若未注意而推進至周邊，可能造成 dissection。在急性冠狀動脈症候群等不穩定、脆弱斑塊病灶需特別小心。

Pearl 2 — 氣球擴張後的病灶通過：病灶經氣球擴張後會產生人工 dissection。導絲尖端進入 dissection 腔會導致 dissection 擴大。

Pearl 3 — 穿孔風險 (Perforation Risk)： - 可無阻力穿透小的周邊血管 - 因尖端較硬，容易發生穿孔 - 必須始終注意確保尖端在正確位置 - 與 microcatheter 併用進行 CTO 穿透時，導絲尖端也會受到強大作用力，需謹慎操作

1.6 3. 3D Wiring Method

1.6.1 3.1 正交方向影像 Orthogonal Directions

3D wiring method 的核心概念是使用兩個約 90 度角的影像角度來精確導引導絲。

1.6.1.1 各冠狀動脈分支的建議角度

分支 (Branch) 第一角度 (First Angle) 第二角度 (Second Angle)
LAD Seg 5-2 LAO 45° → AP CRA 30°
LAD Seg 6 LAO 45° RAO 45°
LAD Seg 7 (proximal) RAO 30° CRA 45° RAO 30° CAU 45°
LAD Seg 7 (distal) LAO 45° CRA 45° LAO 45° CAU 45°
LAD Seg 8 CAU 30° LAO 45°
LAD Seg 9 LAO 45° CRA 30° AP CAU CRA 30° RAO 45°
LCX Seg 11 CAU 30° RAO 45°

Pearl — 縮寫說明：LAO = Left Anterior Oblique 左前斜位、RAO = Right Anterior Oblique 右前斜位、CRA = Cranial 頭位、CAU = Caudal 尾位、AP = Anteroposterior 正位

1.6.2 3.2 實驗模型驗證 Experimental Model Validation

實驗設計：研究團隊使用脈動模型 (Pulsatile model, Terumo Corporation) 進行 3D wiring 驗證：

  • CTO 段製作：20mm 長，使用 2-3% agarose 製作
  • 血管模型直徑：約 3mm
  • 遠端目標：插入目標管（內徑 0.4mm 或 0.6mm）
  • 測試部位：RCA 或 LAD

實驗結果

導絲類型 (Wire Type) 結果 (Results)
Gaia wire 能順利通過 Progreat tube (內徑 0.6mm)
FineCross tube (內徑 0.4mm) 有時難以通過 10mm tip curve 的導絲 tip-body
Conquest Pro wires (9g 和 12g) Tip curve < 0.8mm 較有效
手動製作 (Manual shaping) 需要 0.8mm 40° tip curve，有學習曲線

1.6.3 3.3 臨床應用關鍵 Key Clinical Applications

1.6.3.1 2D vs 3D Wiring 的適用時機

位置 (Location) 優先方法 (Priority) 說明 (Description)
CTO 入口 (CTO entry) 2D > 3D 導絲位於血管腔內，移動不穩定且難以操控。需旋轉 tip curve 尋找可插入點
CTO 內部 (Within CTO) 2D = 3D 若有 landmarks 且可維持操作性，應嘗試 3D wiring。若無進展，可用 3D 影像確定約 180° 方向並擺動尖端
CTO 出口 (CTO exit) 3D > 2D 可視需要進行選擇性造影，盡可能使用 3D wiring 達成 pinpoint puncture

1.6.3.2 3D Wiring 穿透技術要點

導絲推進的必要條件 Prerequisites for Wire Advancement

  1. 充足的 Backup Support
    • 6 Fr 或更大的 guide catheter
    • Corsair catheter
    • 必要時使用 anchor
  2. 充足的 Tip Load
    • 常需要 Conquest wire
    • 對出口處的纖維囊尤其重要
    • 仔細考慮尖端推進的縱向向量並進行第一次穿刺

1.6.3.3 穿透過程中的處理

情境一：導管無法通過硬組織 1. 嘗試增加 tip load 2. 在 3D 影像下稍微改變穿刺點 3. 若仍無效，考慮替代路徑或視情況在出口處刻意旋轉尖端

情境二：尖端在出口處容易推進但偏離預定位置 懷疑導絲移入 false lumen： - 從 false lumen 穿刺至 true lumen 幾乎不可能 - 可從前方重新推進導絲 - 或使用 IVUS guided 或 retrograde approach

1.6.4 3.4 臨床實踐秘訣 Clinical Practice Tips

Pearl: 對於 CTO 病灶，應嘗試使用 Gaia 和 Conquest wires 的 3D wiring method。然而，實際上會處理各種病灶，3D wiring method 不應被視為唯一推薦技術且不應固守。

關鍵原則：重要的是根據情況適當地結合使用 2D 和 3D wiring method。

1.7 4. 總結 Summary

1.7.1 4.1 核心要點回顧

  1. 導絲選擇應基於病灶特性
    • 近端彎曲：選擇操控性佳的導絲
    • 鈣化病灶：需要滑順性和支撐力的平衡
    • 開口病灶：優先使用無 coating 導絲
  2. Polymer Jacket 導絲的雙面性
    • 優點：優異的滑順性，能通過困難病灶
    • 缺點：穿孔和 dissection 風險增加
    • 使用後應適時更換為安全性較高的導絲
  3. 3D Wiring Method 的靈活應用
    • 在 CTO 出口處特別有價值
    • 需與 2D method 靈活搭配
    • 充足的 backup 和適當的 tip load 是成功關鍵
  4. 安全性始終是首要考量
    • 避免過度施力
    • 密切監測導絲位置
    • 必要時毫不猶豫地更換導絲

1.7.2 4.2 給新進同仁的建議

Pearl — 學習重點: 1. 理解原理：了解每種導絲設計的物理特性和臨床意義 2. 觀察資深醫師：注意他們如何根據病灶特性選擇導絲 3. 練習基本技能：從簡單病灶開始，逐步累積經驗 4. 學習識別問題：及早發現導絲位置異常或阻力異常 5. 培養系統性：建立標準化的導絲選擇和操作流程 6. 重視安全：永遠把病人安全放第一位

1.7.3 4.3 三個核心原則

  1. 安全第一 (Safety First)：永遠不要為了速度而犧牲安全
  2. 因材施教 (Individualized Approach)：根據病灶特性選擇最適合的導絲和技術
  3. 靈活應變 (Flexible Adaptation)：沒有單一技術適用所有情況，需要靈活調整
