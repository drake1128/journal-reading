> **核心原則**  
> [10pt]
>     導管室心跳停止雖不常見，但因需要**同時進行 CPR 與處理根本病因**，  
>     是極具挑戰性的緊急情況。

## 前言


臨床背景{

- 導管室心跳停止發生率雖低，但處理極具挑戰性
- 需要**同時**進行 CPR 與處理導致心跳停止的根本原因
- 大多數發生於 **STEMI** 與 **cardiogenic shock** 的緊急/急診程序
- 高風險 PCI 與結構性介入病人也有心跳停止風險


}


## 高風險病人辨識


### 導管室心跳停止高風險族群


| p{5cm}p{8cm}}  **** | **風險因子** | **說明** |
| --- | --- | --- |
| **** | **風險因子** | **說明** |
| a. | ST elevation MI (STEMI) | 急性心肌梗塞，尤其是大範圍梗塞 |
| b. | Severe cardiogenic shock | SCAI shock stages C, D, E   （Classic、Deteriorating、Extremis） |
| c. | High-risk PCI | 複雜病灶、左主幹、最後一條血管 |
| d. | TAVR | 經導管主動脈瓣置換術 |
| e. | PCI 併發症 | Class III perforation（心包填塞）    Severe no-reflow    Left main dissection 導致 abrupt closure    Severe air embolism |
| f. | 嚴重心室功能不全 | 嚴重 LV / RV / 雙心室收縮功能不全 |
| g. | 心跳停止病史 | 曾於導管室外發生心跳停止 |


## 預防性規劃


### 高風險病人的事前準備


對於心跳停止高風險病人，**預防性規劃**是決定預後的關鍵！


| p{13cm}}  **** | **預防措施** |  |
| --- | --- | --- |
| **** | **預防措施** |  |
| a. | **提早通知相關團隊** |  |
|  | 在病人抵達導管室前或程序開始前，若預期有高機率需要 ECMO，應提早通知：       • 麻醉科（Anesthesia）       • 體外循環師（Perfusion）       • 心衰竭團隊（Advanced Heart Failure）       • 心臟外科（Cardiac Surgery）       **→ 這些團隊的電話號碼應清楚張貼於導管室內** |  |
| b. | **所有病人術前貼上去顫貼片** |  |
|  | 確保所有病人在程序開始前已貼上 defibrillator pads |  |
| c. | **早期置入機械循環支持（MCS）/ ECMO** |  |
|  | 對於 pre-arrest 或快速惡化的病人，應提早置入 MCS/ECMO       MCS 類型應針對主要造成血流動力學不穩的心室：       • **右心衰竭**（如近端 dominant RCA 阻塞）→ RV support 可能比 LV support 更有價值 |  |
| d. | **延遲插管時機** |  |
|  | 對於嚴重低血壓合併心搏過速的病人：       • 若可能，延遲插管直到血管通路/MCS 建立完成       • 因為這類病人可能在麻醉誘導時發生心跳停止 |  |


## Refractory In-Lab Cardiac Arrest 處理要點


### 即時處置步驟


> enumerate[label=\arabic*., leftmargin=1.5cm]
>     \item **指示人員立即開始高品質 CPR**
>     \item **執行電擊去顫**
>     \item **依 ACLS guidelines 給予藥物**
>     \item **將動脈 sheath 接上壓力監測**  
>     確保胸部按壓時收縮壓達 90-100 mmHg
> enumerate


### 機械式胸部按壓裝置（Mechanical Compression Devices, MCDs）


**MCDs 的優點：**

- 案例系列顯示：使用 MCDs 在 CPR 期間執行 PCI，**存活出院率達 25
- 提供無疲勞的 CPR，符合指引建議的頻率、深度與回彈參數
- 釋放人力執行其他任務
- 減少工作人員暴露於輻射、受傷風險與傳染性感染


### 團隊角色分配


| p{10cm}}  **角色** | **職責** |  |
| --- | --- | --- |
| **角色** | **職責** |  |
| **介入心臟科醫師** | 專注於介入處置（PCI、處理併發症）    **將 Code Blue 指揮權委派給其他團隊成員** |  |
| **Code Leader** | 負責指揮急救流程、藥物給予、時間記錄 |  |
| **CPR 執行者** | 高品質胸部按壓或操作 MCD |  |
| **呼吸道管理** | Bag-mask ventilation 或協助插管 |  |
| **藥物/記錄** | 準備並給予 ACLS 藥物、記錄時間與事件 |  |


重要提醒{

- **呼叫支援**：若有另一位介入心臟科醫師可協助，應立即通知
- **通知心臟外科**：若發生廣泛冠狀動脈或主動脈剝離、難以處理的穿孔


}


## 機械循環支持（MCS）與 ECMO


### MCS 裝置在心跳停止的角色


| p{10cm}}  **裝置** | **說明** |  |
| --- | --- | --- |
| **裝置** | **說明** |  |
| Impella | 對於心跳停止的療效證據有限 |  |
| IABP   （Intra-aortic Balloon Pump） | 對於心跳停止的療效證據有限 |  |
| **ECMO**   （VA-ECMO） | **Refractory cardiac arrest 的首選裝置**（若可取得）    • 產生高平均動脈壓    • 提供氧合    • 高 ROSC（Return of Spontaneous Circulation）率    • 但存活出院率取決於根本原因是否可逆轉 |  |


### ECMO 啟動時機


**ECMO 應考慮較早啟動（CPR 開始後 10-20 分鐘內）**  

而非較晚（CPR 超過 30 分鐘後）


### 2019 AHA ACLS 更新對 ECPR 的建議


> **Class IIb（Level of Evidence: C-LD）**  
> [5pt]
> 「目前沒有足夠證據建議常規使用 ECMO 進行心跳停止期間的 CPR（ECPR）。  
> [5pt]
> 但在以下情況下，ECPR 可考慮作為**搶救治療（rescue therapy）**：
> itemize
>     \item 傳統 CPR 失敗時
>     \item 在能夠快速啟動並有熟練團隊支持的環境中
> itemize
> 」


### ECMO 絕對禁忌症


根據 Extracorporeal Life Support Organization（ELSO）指引：


| }  **ECMO 絕對禁忌症** |  |  |
| --- | --- | --- |
| **ECMO 絕對禁忌症** |  |  |
| 心臟無法恢復（unrecoverable heart）且不是移植或 VAD 候選人 |  |  |
| 高齡（advanced age） |  |  |
| 慢性器官功能不全（emphysema、cirrhosis、renal failure） |  |  |
| 順從性限制（compliance limitations） |  |  |
| 長時間 CPR 且組織灌流不足 |  |  |


## 流程/系統措施


### 建立導管室心跳停止處理系統


1. **多專科團隊合作**        對於執行複雜與高風險程序的導管室，應建立包含以下專科的多專科處理模式：      - 麻醉科 - 體外循環師（Perfusion） - 心衰竭團隊 - 心臟外科
2. **定期模擬演練（Mock Drills）**         - 保持人員熟練度 - 辨識知識與流程缺口 - 所有人員與醫師的 ACLS 認證應保持有效
3. **制定導管室心跳停止處理演算法**         - 根據導管室與醫院的特定環境與架構客製化 - 應包含 ECMO 病人篩選標準（因各機構與計畫而異）

## 結論


核心要點{

1. **團隊合作**：快速應用團隊合作模式
2. **維持灌流**：維持重要器官灌流
3. **矯正原因**：努力矯正根本原因
4. **ECMO/MCS**：角色持續演進，存活出院率顯著取決於根本原因是否可矯正


}

## 附錄：導管室心跳停止快速參考卡


> **【立即行動】**
> itemize[leftmargin=0.5cm]
>     \item[$\square$] 開始高品質 CPR（或啟動 MCD）
>     \item[$\square$] 電擊去顫（若為 shockable rhythm）
>     \item[$\square$] 動脈 sheath 接壓力監測，確認 SBP 90-100 mmHg
>     \item[$\square$] 依 ACLS 給予 Epinephrine 1mg IV q3-5min
> itemize
> **【角色分配】**
> itemize[leftmargin=0.5cm]
>     \item[$\square$] 介入醫師 → 專注 PCI / 處理併發症
>     \item[$\square$] 指定 Code Leader → 指揮急救流程
>     \item[$\square$] 呼叫支援：麻醉、perfusion、心臟外科
> itemize
> **【辨識並處理原因】**
> itemize[leftmargin=0.5cm]
>     \item[$\square$] Perforation → Balloon inflation + pericardiocentesis
>     \item[$\square$] No-reflow → IC Epinephrine / Adenosine / Nitroprusside
>     \item[$\square$] LM dissection → Emergent stenting
>     \item[$\square$] Air embolism → Aspiration + 100
> itemize
> **【Refractory Arrest → 考慮 ECMO】**
> itemize[leftmargin=0.5cm]
>     \item[$\square$] CPR 10-20 分鐘無效 → 考慮 VA-ECMO
>     \item[$\square$] 確認病人無 ECMO 禁忌症
>     \item[$\square$] 通知 ECMO team / perfusion
> itemize
