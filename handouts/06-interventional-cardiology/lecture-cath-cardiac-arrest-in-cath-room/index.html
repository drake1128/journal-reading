核心原則：導管室心跳停止雖不常見，但因需要同時進行 CPR 與處理根本病因，是極具挑戰性的緊急情況。快速的團隊合作 + 維持重要器官灌流 + 矯正根本原因 = 病人存活的關鍵。

原文作者：Christine Gasperetti, MD, FSCAI; Jayant Bagai, MD, FSCAI 資料來源：SCAI QI Tips, July 2020

2 1. 前言

2.1 臨床背景

  • 導管室心跳停止發生率雖低，但處理極具挑戰性
  • 需要同時進行 CPR 與處理導致心跳停止的根本原因
  • 大多數發生於 STEMI 與 cardiogenic shock 的緊急/急診程序
  • 高風險 PCI 與結構性介入病人也有心跳停止風險

3 2. 高風險病人辨識

3.1 2.1 導管室心跳停止高風險族群

風險因子 說明
ST elevation MI (STEMI) 急性心肌梗塞，尤其是大範圍梗塞
Severe cardiogenic shock SCAI shock stages C, D, E（Classic、Deteriorating、Extremis）
High-risk PCI 複雜病灶、左主幹、最後一條血管
TAVR 經導管主動脈瓣置換術
PCI 併發症 Class III perforation（心包填塞）、severe no-reflow、left main dissection 導致 abrupt closure、severe air embolism
嚴重心室功能不全 嚴重 LV / RV / 雙心室收縮功能不全
心跳停止病史 曾於導管室外發生心跳停止

4 3. 預防性規劃

4.1 3.1 高風險病人的事前準備

Pearl：對於心跳停止高風險病人，預防性規劃是決定預後的關鍵！

4.1.1 a. 提早通知相關團隊

在病人抵達導管室前或程序開始前，若預期有高機率需要 ECMO，應提早通知：

  • 麻醉科（Anesthesia）
  • 體外循環師（Perfusion）
  • 心衰竭團隊（Advanced Heart Failure）
  • 心臟外科（Cardiac Surgery）

這些團隊的電話號碼應清楚張貼於導管室內。

4.1.2 b. 所有病人術前貼上去顫貼片

確保所有病人在程序開始前已貼上 defibrillator pads。

4.1.3 c. 早期置入機械循環支持（MCS）/ ECMO

  • 對於 pre-arrest 或快速惡化的病人，應提早置入 MCS/ECMO
  • MCS 類型應針對主要造成血流動力學不穩的心室：
    • 右心衰竭（如近端 dominant RCA 阻塞）→ RV support 可能比 LV support 更有價值

4.1.4 d. 延遲插管時機

對於嚴重低血壓合併心搏過速的病人：

  • 若可能，延遲插管直到血管通路/MCS 建立完成
  • 因為這類病人可能在麻醉誘導時發生心跳停止

5 4. Refractory In-Lab Cardiac Arrest 處理要點

5.1 4.1 即時處置步驟

心跳停止發生時的即時行動：

  1. 指示人員立即開始高品質 CPR
  2. 執行電擊去顫
  3. 依 ACLS guidelines 給予藥物
  4. 將動脈 sheath 接上壓力監測，確保胸部按壓時收縮壓達 90–100 mmHg

5.2 4.2 機械式胸部按壓裝置（Mechanical Compression Devices, MCDs）

Pearl：MCDs 的優點：

  • 案例系列顯示：使用 MCDs 在 CPR 期間執行 PCI，存活出院率達 25%
  • 提供無疲勞的 CPR，符合指引建議的頻率、深度與回彈參數
  • 釋放人力執行其他任務
  • 減少工作人員暴露於輻射、受傷風險與傳染性感染

5.3 4.3 團隊角色分配

角色 職責
介入心臟科醫師 專注於介入處置（PCI、處理併發症），將 Code Blue 指揮權委派給其他團隊成員
Code Leader 負責指揮急救流程、藥物給予、時間記錄
CPR 執行者 高品質胸部按壓或操作 MCD
呼吸道管理 Bag-mask ventilation 或協助插管
藥物/記錄 準備並給予 ACLS 藥物、記錄時間與事件

Pearl：呼叫支援——若有另一位介入心臟科醫師可協助，應立即通知。通知心臟外科——若發生廣泛冠狀動脈或主動脈剝離、難以處理的穿孔。

6 5. 機械循環支持（MCS）與 ECMO

6.1 5.1 MCS 裝置在心跳停止的角色

裝置 說明
Impella 對於心跳停止的療效證據有限
IABP (Intra-aortic Balloon Pump) 對於心跳停止的療效證據有限
ECMO (VA-ECMO) Refractory cardiac arrest 的首選裝置（若可取得）：產生高平均動脈壓、提供氧合、高 ROSC 率，但存活出院率取決於根本原因是否可逆轉

6.2 5.2 ECMO 啟動時機

Pearl：ECMO 應考慮較早啟動（CPR 開始後 10–20 分鐘內），而非較晚（CPR 超過 30 分鐘後）。

6.3 5.3 2019 AHA ACLS 更新對 ECPR 的建議

AHA 2019 ACLS Focused Update — Class IIb（Level of Evidence: C-LD）

「目前沒有足夠證據建議常規使用 ECMO 進行心跳停止期間的 CPR（ECPR）。但在以下情況下，ECPR 可考慮作為搶救治療（rescue therapy）：傳統 CPR 失敗時，在能夠快速啟動並有熟練團隊支持的環境中。」

6.4 5.4 ECMO 絕對禁忌症

根據 Extracorporeal Life Support Organization（ELSO）指引：

  • 心臟無法恢復（unrecoverable heart）且不是移植或 VAD 候選人
  • 高齡（advanced age）
  • 慢性器官功能不全（emphysema、cirrhosis、renal failure）
  • 順從性限制（compliance limitations）
  • 長時間 CPR 且組織灌流不足

7 6. 流程/系統措施

7.1 6.1 建立導管室心跳停止處理系統

7.1.1 a. 多專科團隊合作

對於執行複雜與高風險程序的導管室，應建立包含以下專科的多專科處理模式：

  • 麻醉科
  • 體外循環師（Perfusion）
  • 心衰竭團隊
  • 心臟外科

7.1.2 b. 定期模擬演練（Mock Drills）

  • 保持人員熟練度
  • 辨識知識與流程缺口
  • 所有人員與醫師的 ACLS 認證應保持有效

7.1.3 c. 制定導管室心跳停止處理演算法

  • 根據導管室與醫院的特定環境與架構客製化
  • 應包含 ECMO 病人篩選標準（因各機構與計畫而異）

9 8. 結論

核心要點： 1. 團隊合作：快速應用團隊合作模式 2. 維持灌流：維持重要器官灌流 3. 矯正原因：努力矯正根本原因 4. ECMO/MCS：角色持續演進，存活出院率顯著取決於根本原因是否可矯正

11 10. 附錄：導管室心跳停止快速參考卡

=== 導管室心跳停止 — 快速處置流程 ===

【立即行動】
  ✦ 開始高品質 CPR（或啟動 MCD）
  ✦ 電擊去顫（若為 shockable rhythm）
  ✦ 動脈 sheath 接壓力監測，確認 SBP 90-100 mmHg
  ✦ 依 ACLS 給予 Epinephrine 1mg IV q3-5min

【角色分配】
  ✦ 介入醫師 → 專注 PCI / 處理併發症
  ✦ 指定 Code Leader → 指揮急救流程
  ✦ 呼叫支援：麻醉、perfusion、心臟外科

【辨識並處理原因】
  ✦ Perforation → Balloon inflation + pericardiocentesis
  ✦ No-reflow → IC Epinephrine / Adenosine / Nitroprusside
  ✦ LM dissection → Emergent stenting
  ✦ Air embolism → Aspiration + 100% O₂

【Refractory Arrest → 考慮 ECMO】
  ✦ CPR 10-20 分鐘無效 → 考慮 VA-ECMO
  ✦ 確認病人無 ECMO 禁忌症
  ✦ 通知 ECMO team / perfusion
