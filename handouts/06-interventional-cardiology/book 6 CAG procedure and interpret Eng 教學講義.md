## Introduction


Coronary angiography is **Step 6** of the 14-step Percutaneous Coronary Intervention (PCI) procedure and must be performed only after successful catheter engagement.

This manual provides detailed instruction on both the execution of coronary angiography and the systematic interpretation of angiographic images. These steps are critical for ensuring patient safety, obtaining high-quality images, and making accurate diagnoses.

## PART A: PERFORMING CORONARY ANGIOGRAPHY


### The Eight Steps of Coronary Angiography - Overview


| p{5.5cm}p{7cm}}   **Step** | **Action** | **Purpose** |
| --- | --- | --- |
| **1** | Ensure good pressure waveform | Prevent coronary dissection and other complications |
| **2** | Perform test injection | Confirm catheter position and assess potential problems |
| **3** | Administer intracoronary nitroglycerin | Prevent vasospasm and ensure accurate diagnosis |
| **4** | Optimize table positioning | Improve image quality and reduce radiation exposure |
| **5** | Check pressure and ECG | Confirm safe conditions before angiography |

## Step 1: Ensure Good Pressure Waveform


### Critical Importance


importantbox
****CRITICAL STEP****: Before injecting any contrast, you **MUST** confirm that the pressure waveform is normal!
importantbox


### Risks of Pressure Damping


Injecting contrast with a dampened pressure waveform may lead to:


| p{9cm}}   **Complication** | **Description** |  |
| --- | --- | --- |
| **Coronary dissection** | Tear in the vessel wall, potentially catastrophic |  |
| **Arrhythmias** | Including ventricular tachycardia or fibrillation |  |
| **Other serious complications** | Acute vessel closure, myocardial infarction |  |


### Assessment Criteria


notebox
**Characteristics of ideal pressure waveform:**

- Clear systolic peak
- Obvious diastolic decline
- Normal waveform morphology without damping
- Values consistent with aortic pressure


notebox


### Managing Abnormal Pressure Waveform


| p{12cm}}   **Step** | **Action** |  |
| --- | --- | --- |
| 1 | **STOP immediately** - Do NOT inject contrast |  |
| 2 | **Adjust catheter position:**  - Gently withdraw the catheter - Rotate catheter angle - Confirm catheter is not over-engaged |  |
| 3 | **Reassess** - Check pressure waveform again after adjustment |  |
| 4 | **Consider catheter change** - If damping persists, consider using different catheter |  |


## Step 2: Test Injection


### Purpose of Test Injection


Before performing complete angiography, give a **1-2 cc** test injection to:


| p{9.5cm}}   **Assessment Item** | **Description** |  |
| --- | --- | --- |
| 1. Confirm catheter engagement | Ensure catheter is truly in coronary artery, not elsewhere |  |
| 2. Check for ostial lesions | Assess whether coronary ostium has stenosis |  |
| 3. Evaluate catheter coaxiality | Confirm catheter is coaxial with vessel, not pointing at wall |  |
| 4. Check engagement depth | Ensure appropriate catheter engagement depth (ideal: 2-3 mm), avoid excessive depth |  |


### Performing Test Injection


| p{12cm}}   **Step** | **Procedure** |  |
| --- | --- | --- |
| 1 | Keep syringe **vertical** (minimizing bubbles) |  |
| 2 | Slowly inject 1-2 cc of contrast |  |
| 3 | Observe contrast flow and distribution |  |
| 4 | Adjust catheter position based on results |  |


### Clinical Examples


notebox
**Case 1**: Test injection shows catheter pointing at roof of left main rather than being coaxial. Need to adjust catheter angle before performing complete angiography.

**Case 2**: Bypass graft test injection shows complete vessel occlusion, avoiding unnecessary large volume contrast injection.
notebox


### Critical Check Before Injection


importantbox
****PREVENT AIR EMBOLISM****


- Keep manifold **vertical** before injection
- Ensure all air bubbles rise to the top
- Avoid air bubbles entering catheter system
- Air embolism can lead to serious consequences


importantbox


## Step 3: Administer Intracoronary Nitroglycerin


### Why Nitroglycerin is Essential


importantbox
****KEY CONCEPT****

Coronary arteries are **dynamic structures** that may develop spasm. Without nitroglycerin, vasospasm may be mistaken for fixed stenosis, leading to incorrect diagnosis and unnecessary intervention.
importantbox


### Dosing Guidelines


| p{8.5cm}}   **Blood Pressure (Systolic)** | **Nitroglycerin Dose** |  |
| --- | --- | --- |
| Normal BP patients | 200 mcg |  |
| SBP 100-120 mmHg | 100 mcg |  |
| SBP $<$ 100 mmHg | 0-50 mcg (use clinical judgment) |  |
| Hypotensive patients | ****CONTRAINDICATED**** |  |


### Administration Methods


| p{11cm}}   **Method** | **Procedure** |  |
| --- | --- | --- |
| **Method 1:** Via manifold side port | 1. Aspirate from manifold side port 2. Inject nitroglycerin 3. Wait for effect (approximately 30-60 seconds) |  |
| **Method 2:** Nitroglycerin-on-a-Stick technique | 1. Inject directly via introducer 2. Drug enters through Y-connector back end 3. Suitable for situations requiring rapid administration |  |


### Clinical Case: Vasospasm


notebox
**IMPORTANT CASE**

Initial angiography showed significant ostial lesion in right coronary artery. However, after administering nitroglycerin, this "lesion" completely disappeared!

**Conclusion**: This was catheter-induced vasospasm, not true stenosis. Without nitroglycerin, this could have led to incorrect diagnosis and unnecessary intervention.
notebox


### Managing Nitroglycerin-Induced Hypotension


| p{12cm}}   **Step** | **Management** |  |
| --- | --- | --- |
| 1 | **Wait** - Usually self-resolves within minutes |  |
| 2 | **Give fluids** - Normal saline IV bolus |  |
| 3 | **Vasopressors** - In rare cases (usually vagal reaction), consider phenylephrine |  |
| 4 | **Delay angiography** - Wait for stable blood pressure before proceeding |  |


## Step 4: Optimize Table Positioning


### Dual Goals of Positioning


| p{12cm}}   **Goal** | **Description** |  |
| --- | --- | --- |
| 1 | **Optimize image quality** - Obtain clear, complete coronary artery images |  |
| 2 | **Minimize radiation exposure** - Protect operator from scatter radiation |  |


### Correct Setup Principles


| p{9.5cm}}   **Setup Item** | **Optimization Method** |  |
| --- | --- | --- |
| Table height | Raise as high as possible (improves image quality) |  |
| Image receptor | As close to patient as possible (reduces radiation dose, improves image quality) |  |
| Lead shield | Place between patient and operator |  |
| Operator position | As far from radiation source as possible |  |


### Physics of Radiation Protection


importantbox
****INVERSE SQUARE LAW****

Scatter radiation intensity is inversely proportional to the **square** of the distance:

$$Radiation Intensity \propto 1{Distance^2}$$

**Practical Significance**: The farther the operator is from the radiation source, the exponentially lower the radiation exposure. Even a small increase in distance can significantly reduce radiation exposure!
importantbox


### Correct Use of Lead Shield


notebox
**COMMON ERRORS**


- **ERROR**: Lead shield placed too far from patient
- **ERROR**: Lead shield positioned too high, unable to block scatter from patient's back


**CORRECT METHOD**


- **CORRECT**: Lead shield close to patient
- **CORRECT**: Lead shield positioned lower to block scatter radiation from patient's back


notebox


### Use of Extension Tubing


Using extension tubing allows the manifold to be farther from the radiation source, further reducing operator radiation exposure.


### Special Considerations for Obese Patients


| p{10cm}}   **Consideration** | **Strategy** |  |
| --- | --- | --- |
| Image quality affected | Image quality will be compromised |  |
| **Reduce angulation** | Using less angulation can obtain clearer images |  |
| Increased radiation may be needed | May require increased radiation dose to penetrate tissue |  |
| Greater need for radiation protection | Even more attention to radiation protection required |  |


### Common Angiographic Projections


#### Left Coronary Artery (5 Projections)


| p{5cm}p{7.5cm}}   **Order** | **Projection** | **Main Vessels Visualized** |
| --- | --- | --- |
| 1 | AP (Anterior-Posterior) | Left Main |
| 2 | RAO Caudal | Circumflex and Obtuse Marginals |
| 3 | RAO Cranial | Left Anterior Descending (LAD) |
| 4 | LAO Cranial | LAD and Diagonal branches |
| 5 | LAO Caudal (Spider) | Left Main bifurcation, Circumflex, proximal LAD |


#### Right Coronary Artery (3 Projections)


| p{5cm}p{7.5cm}}   **Order** | **Projection** | **Main Vessels Visualized** |
| --- | --- | --- |
| 1 | LAO | Proximal, mid, and distal RCA |
| 2 | LAO Cranial | Distal RCA, PDA, posterolateral bifurcation |
| 3 | RAO | Mid RCA, PDA, posterolateral vessels |


notebox
**INDIVIDUALIZATION**

Each operator may have preferred projection sequences and angles. The key is to ensure:

- All coronary artery segments are evaluated
- Each segment has at least two orthogonal projections
- No lesions are missed


notebox


## Step 5: Check Pressure and ECG


### Importance of Reconfirmation


Before actually injecting contrast, you **MUST recheck**:


| p{12cm}}   **Check** | **Assessment** |  |
| --- | --- | --- |
| 1 | ****Pressure waveform**** - Ensure it remains normal, without damping |  |
| 2 | ****ECG**** - Confirm no new ST segment changes or arrhythmias |  |


importantbox
****CRITICAL SAFETY CHECK****

**Why recheck?**

Between Steps 1 and 5, the following may occur:

- Catheter position change
- Excessive catheter engagement
- Coronary vasospasm
- Pressure waveform damping


If pressure waveform is dampened at this point, **DO NOT inject contrast**. Must address the problem first!
importantbox


## Step 6: Perform Angiography


### Pre-Angiography Preparation


| p{12cm}}   **Step** | **Action** |  |
| --- | --- | --- |
| 1 | Fill syringe with contrast |  |
| 2 | **Keep syringe vertical**, ensure bubbles are at the top |  |
| 3 | Turn three-way stopcock to correct position |  |
| 4 | Prepare to begin angiography |  |


### Angiography Execution Sequence


| p{12cm}}   **Step** | **Action** |  |
| --- | --- | --- |
| **First** | Collimation (narrow field of view) and adjust X-ray image receptor position |  |
| **Second** | Start cine (image recording) |  |
| **Third** | Wait 1-2 seconds (if calcification present, can be visible at this time) |  |
| **Fourth** | Inject contrast |  |
| **Fifth** | ****Confirm contrast reflux to aorta**** (assess ostial lesions) |  |
| **Sixth** | Pan as needed to show all branches |  |
| **Seventh** | Repeat angiography in orthogonal projections |  |


### Contrast Injection Technique


notebox
**Mastering Injection Force**


- **Start**: Slow, gentle injection
- **Process**: Gradually increase injection pressure
- **Purpose**: Avoid catheter dislodgement from excessive injection force (kick back)


**CLINICAL CASE**

If you inject forcefully from the start, the catheter may be pushed out of the vessel. But if you start with slow injection and gradually increase pressure, the same catheter can stably complete angiography.
notebox


### Using Guide Extension


| }   **Guide Extension Considerations** |  |  |
| --- | --- | --- |
| If using guide extension: |  |  |
| • Should **hold** the guide extension during angiography |  |  |
| • Otherwise injection pressure may push it deeper into coronary artery |  |  |
| • May cause vessel injury |  |  |


### Ensure Ostial Lesion Assessment


importantbox
****CRITICAL CHECKPOINT****

**Must see contrast reflux to aorta!**

If contrast does not reflux to the aorta, ostial lesions may be missed. Ostial lesions are important and commonly missed lesions that must be assessed in every angiography.
importantbox


## Common Angiographic Problems and Solutions


### Problem 1: Incomplete Coronary Opacification


#### Clinical Presentation


| }   **Signs of Incomplete Opacification** |  |  |
| --- | --- | --- |
| • Unclear coronary or bypass graft visualization |  |  |
| • Contrast not filling entire vessel |  |  |
| • Image quality insufficient for diagnosis |  |  |


#### Possible Causes


| p{8.5cm}}   **Cause** | **Description** |  |
| --- | --- | --- |
| Catheter too small | Cannot provide sufficient contrast flow |  |
| Catheter with side holes | Contrast lost through side holes |  |
| Coronary artery too large or dilated | Requires more contrast |  |
| Poor catheter engagement | Contrast not entering coronary artery |  |
| High coronary flow | Common in severe aortic stenosis or hypertrophic cardiomyopathy |  |
| High contrast viscosity | Poor flowability |  |
| Insufficient injection force | Not providing adequate pressure |  |
| Elevated LVEDP | Increases coronary resistance |  |


#### Solutions


| p{12cm}}   **Step** | **Solution** |  |
| --- | --- | --- |
| 1 | Improve catheter engagement |  |
| 2 | Use larger catheter (e.g., change from 6F to 7F) |  |
| 3 | Increase injection force |  |
| 4 | Consider using low-viscosity contrast |  |
| 5 | Evaluate and address potential hemodynamic issues |  |


### Problem 2: Catheter Dislodgement During Angiography


#### Clinical Presentation and Causes


| p{8.5cm}}   **Cause** | **Description** |  |
| --- | --- | --- |
| Poor catheter engagement | Most common cause |  |
| Excessive injection force | Especially forceful injection from the start |  |
| Insufficient catheter support | Catheter too soft or inappropriate model |  |
| Patient deep breathing | Causes catheter position change, especially common with radial approach |  |


#### Solutions


| p{12cm}}   **Step** | **Solution** |  |
| --- | --- | --- |
| 1 | **Improve engagement technique** - Ensure catheter is stably engaged |  |
| 2 | **Use different catheter** - Select model with better support |  |
| 3 | **Place guidewire** - Can stabilize catheter position |  |
| 4 | **Use guide extension** - Provides additional support |  |
| 5 | **Adjust injection technique** - **Start slowly, gradually increase pressure** |  |


### Problem 3: Unable to Visualize Certain Coronary Segments


#### Possible Causes and Solutions


| p{8.5cm}}   **Cause** | **Solution** |  |
| --- | --- | --- |
| Poor catheter engagement | Improve engagement |  |
| Insufficient injection force | Use larger catheter |  |
| Severe coronary segment | Increase injection force |  |
| Patient too large | Adjust patient position |  |
| Insufficient projection angles | Use more and different projections |  |
| Inappropriate patient positioning | Ensure image receptor mobile |  |
| (e.g., arm blocking distal LAD) | to needed position |  |

### Importance of Post-Angiography Assessment


**Immediately** after angiography, check:


| p{12cm}}   **Check** | **Assessment** |  |

| 1 | ****ECG**** - Confirm no new ST segment changes or arrhythmias |  |

### Potential Complications


importantbox
****IMMEDIATE COMPLICATION RECOGNITION****

Complications that may occur immediately after angiography:


| p{8cm}}   **Complication** | **Management** |  |
| --- | --- | --- |
| **Ventricular fibrillation** | Immediate defibrillation |  |
| **ST segment changes** | May indicate dissection or acute closure |  |
| **Bradycardia** | May require atropine |  |
| **Pressure damping** | May indicate catheter-induced problem |  |


Early recognition and immediate treatment are critical!
importantbox


### Clinical Cases


notebox
**Case 1: Ventricular Fibrillation**

Ventricular fibrillation occurred immediately after angiography. Because operator continuously monitored ECG, immediately recognized and gave defibrillation, successfully restored normal rhythm.

**Case 2: Transient ST Segment Changes**

Transient ST segment changes appeared after contrast injection. Confirmed as temporary change induced by contrast, not coronary dissection or occlusion. Continued monitoring until return to normal.

## PART B: INTERPRETING CORONARY ANGIOGRAPHY


## Step 8: Angiographic Interpretation


### Prerequisites for Interpretation


Interpretation **completely depends** on excellent execution of the previous seven steps:


| }   **Required Conditions for Accurate Interpretation** |  |  |
| --- | --- | --- |
| • Image quality must be good |  |  |
| • All coronary artery segments adequately opacified |  |  |
| • Each segment has orthogonal projections |  |  |
| • Contrast reflux to aorta visible |  |  |


importantbox
****QUALITY DETERMINES EVERYTHING****

If angiography quality is poor, interpretation may be **impossible** or **inaccurate**.

Common quality issues:

- Incomplete coronary opacification
- Lack of orthogonal projections
- Ostium not assessed
- Some segments not visualized


importantbox


### Image Quality Assessment


Before interpretation, first confirm image quality is sufficient for accurate reading.


#### Common Causes of Poor Image Quality


| }   **Poor Quality Indicators** |  |  |
| --- | --- | --- |
| • Poor catheter engagement causing incomplete coronary opacification |  |  |
| • Coronary lumen not fully opacified (unclear visualization) |  |  |
| • No contrast reflux to aorta |  |  |
| • Not all coronary segments evaluated with orthogonal projections |  |  |


> itemize
>     \item Each coronary artery must be evaluated in **at least two orthogonal projections**
>     \item Must have contrast reflux to aorta (watch for ostial stenosis)
>     \item All coronary segments should be clearly opacified
> itemize


## The 13 Assessment Elements of Coronary Angiography


### 1. Coronary Blood Flow (TIMI Grading System)


TIMI (Thrombolysis In Myocardial Infarction) grading evaluates coronary blood flow:


| p{11.5cm}}   **Grade** | **Definition** |  |
| --- | --- | --- |
| TIMI 0 | Complete occlusion, no antegrade flow beyond stenosis |  |
| TIMI 1 | Contrast passes stenosis but does not reach coronary terminus |  |
| TIMI 2 | Contrast reaches coronary terminus but flow slower than contralateral normal coronary |  |
| TIMI 3 | Normal antegrade flow, same velocity as unaffected artery |  |


### 2. Stenosis Severity


#### Stenosis Assessment


| }   **Key Concepts for Stenosis Evaluation** |  |  |
| --- | --- | --- |
| • Reported as **percent diameter stenosis ( |  |  |
| • Calculation formula:  \[Stenosis |  |  |
| • Reference Diameter = $\text{Proximal Reference + Distal Reference{2}$ |  |  |


#### Stenosis Severity Classification


| p{10cm}}   **Stenosis** | **Classification** |  |
| --- | --- | --- |
| **$\geq$ 70 | Usually considered significant stenosis |  |
| **50-69 | Moderate stenosis |  |
| **$<$ 50 | Mild stenosis |  |


> Percent diameter stenosis does NOT perfectly correlate with hemodynamic significance!
> itemize
>     \item Among 71-90
>     \item Among $<$ 70
>     \item Recommendation: When uncertain, use FFR or iFR for evaluation
> itemize


#### Importance of Orthogonal Projections


| }   **Critical Points About Orthogonal Views** |  |  |
| --- | --- | --- |
| • Eccentric lesions may be completely invisible in some projection angles |  |  |
| • Must report stenosis severity in most severe projection angle |  |  |
| • At least two orthogonal projections needed to avoid underestimation or overestimation |  |  |


### 3. Nitroglycerin Administration


> itemize
>     \item Should give nitroglycerin before each angiography
>     \item Purpose: Avoid mistaking multiple vasospasms for multiple stenoses
>     \item Not giving nitroglycerin may lead to false-positive stenosis diagnosis
> itemize

## Clinical Recommendations and Best Practices


### Golden Rules for Angiographic Interpretation


| p{12cm}}   **Rule** | **Description** |  |
| --- | --- | --- |
| 1 | **Systematic evaluation** - Use checklist to ensure no important findings are missed |  |
| 2 | **Orthogonal projections** - At least two angles for each vessel |  |
| 3 | **Dynamic assessment** - Observe vascular changes throughout cardiac cycle |  |
| 4 | **Comparative evaluation** - Compare with contralateral or other vessel flow |  |
| 5 | **Integrate information** - Combine clinical presentation, ECG, echocardiography |  |


### Common Errors and Prevention


| p{7.5cm}}   **Common Error** | **Prevention Method** |  |
| --- | --- | --- |
| Not giving nitroglycerin | Routinely give before each angiography |  |
| Single projection evaluation only | At least two orthogonal projections |  |
| Underestimating calcification | Consider using IVUS/OCT |  |
| Missing ostial lesions | Ensure contrast reflux to aorta |  |
| Ignoring pressure waveform | Must check waveform before injection |  |
| Over-relying on visual estimation | Use FFR/iFR when uncertain |  |


### When Additional Testing is Needed


| p{10.5cm}}   **Test** | **Indication** |  |
| --- | --- | --- |
| **FFR/iFR** | 50-90 |  |
| **IVUS** | - Left main lesion assessment - Severe calcification assessment - Stent optimization post-implantation - Unclear cause of in-stent restenosis - Dissection extent assessment |  |
| **OCT** | - Thrombus assessment - Severe calcification assessment - Stent apposition assessment |  |
