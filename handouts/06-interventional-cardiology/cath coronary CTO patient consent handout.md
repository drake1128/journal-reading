本文件旨在幫助您了解冠狀動脈慢性完全閉塞(Coronary Chronic Total Occlusion, CTO)的相關資訊,包括疾病成因、症狀、治療選項、手術過程及術後照護等內容。

### 什麼是冠狀動脈慢性完全閉塞(CTO)?


importantbox
**CTO定義**

冠狀動脈慢性完全閉塞(Coronary Chronic Total Occlusion, CTO)是指:

- 冠狀動脈血管**100
- 阻塞時間**超過3個月**
- 無法讓血液流通至心臟肌肉


importantbox


## 常見問題解答


### 問題一:什麼是冠狀動脈慢性完全閉塞(CTO)?


#### 冠狀動脈的功能


冠狀動脈是供應心臟肌肉血液和氧氣的血管。當這些血管發生阻塞時,心臟肌肉無法獲得足夠的養分。


#### CTO的特徵


| p{10cm}}   **特徵** | **說明** |  |
| --- | --- | --- |
| 阻塞程度 | 100 |  |
| 阻塞時間 | 超過3個月以上的慢性阻塞 |  |
| 血流情形 | 阻塞處完全沒有血流通過 |  |
| 側枝循環 | 身體可能會發展出其他血管(側枝循環)來部分代償血流 |  |


#### 如何診斷CTO?


**冠狀動脈攝影(Coronary Angiography):**

- 這是診斷CTO的標準檢查
- 透過導管注射顯影劑
- 在X光下觀察冠狀動脈
- 可清楚看到血管阻塞的位置和程度


**側枝循環的意義**

當冠狀動脈發生慢性完全閉塞時,身體會試圖建立新的血管路徑(側枝循環)來繞過阻塞處,為心臟肌肉提供血流。然而,這些側枝循環通常:

- 無法完全代償原本血管的功能
- 可能來自其他冠狀動脈分支
- 在運動時仍無法提供足夠血流


### 問題二:為什麼會發生CTO?


#### 冠狀動脈疾病(Coronary Artery Disease)


CTO是冠狀動脈疾病的一種表現形式。

**定義:**冠狀動脈疾病是指心臟動脈內斑塊(plaque)的形成和積聚。


#### 危險因子


| p{10cm}}   **危險因子** | **說明** |  |
| --- | --- | --- |
| 吸菸 | 大幅增加冠狀動脈疾病風險 |  |
| 高血壓 | 長期高血壓損傷血管內壁 |  |
| 高膽固醇 | 膽固醇沉積形成斑塊 |  |
| 糖尿病 | 加速動脈硬化過程 |  |
| 家族史 | 遺傳因素增加發病風險 |  |
| 年齡 | 隨年齡增長風險上升 |  |


#### CTO的形成過程


1. **初期:**冠狀動脈內開始形成斑塊
2. **進展:**斑塊逐漸增大,造成部分阻塞(例如70
3. **急性事件:**斑塊可能破裂,形成血栓
4. **慢性閉塞:**血栓組織化,形成100


importantbox
****重要概念****

並非所有的冠狀動脈阻塞都會發展成CTO。某些部分阻塞(如70
importantbox


### 問題三:CTO會引起什麼症狀?


#### 典型症狀


| p{10cm}}   **症狀類型** | **具體表現** |  |
| --- | --- | --- |
| 胸痛或胸悶 | 運動或活動時出現胸部不適    休息後症狀緩解    可能放射至左臂、下巴或背部 |  |
| 呼吸困難 | 活動時感到喘不過氣    可能在爬樓梯或快走時出現    休息後改善 |  |
| 疲勞感 | 容易疲倦    體力明顯下降    無法進行以前能做的活動 |  |
| 其他症狀 | 噁心    盜汗    心悸 |  |


#### 症狀的漸進性


**重要觀察**

許多CTO患者的症狀是**逐漸發展**的:

- 患者可能**潛意識地減少活動量**以避免症狀出現
- 本人可能未察覺活動能力下降
- 配偶或家人往往更容易注意到患者活動量的減少
- 與幾個月或幾年前相比,活動能力明顯下降


**醫師建議:**與家人一起就診,有助於提供更完整的病史資訊。


#### 無症狀的CTO


**部分患者可能沒有明顯症狀:**

- 側枝循環發展良好,能部分代償血流
- 在例行健康檢查時意外發現
- 可能在壓力測試或影像檢查中被診斷


### 問題四:CTO一定要治療嗎?


#### 治療指徵


importantbox
****介入治療的主要理由****

**第一優先考量:症狀改善**

- 這是進行CTO介入治療的**首要原因**
- 當患者有明顯症狀(胸痛、呼吸困難、活動受限)
- 症狀影響日常生活品質
- 藥物治療效果不佳


importantbox


#### 治療時機的考量


| p{9cm}}   **情況** | **建議** |  |
| --- | --- | --- |
| 有明顯症狀 | 強烈建議考慮介入治療    可顯著改善生活品質 |  |
| 無症狀但心肌缺血 | 可考慮介入治療    需評估風險效益 |  |
| 完全無症狀 | 通常以藥物治療為主    定期追蹤觀察 |  |
| 曾接受繞道手術 | 若繞道血管阻塞    優先考慮打通原生血管(CTO) |  |


#### 不治療的選項


**以下情況可能選擇藥物治療而非介入:**

- 完全沒有症狀
- 高齡且有多重共病
- 手術風險過高
- 預期壽命有限
- 患者個人意願


### 問題五:CTO有哪些治療選項?


#### 三種主要治療方式


| p{10cm}}   **治療方式** | **說明** |  |
| --- | --- | --- |
| **1. 藥物治療** | **適用:**症狀輕微或無症狀患者    **藥物包括:**    - 抗血小板藥物(aspirin, clopidogrel)    - 降膽固醇藥物(statins)    - 控制血壓藥物    - 控制心絞痛藥物(nitrates, beta-blockers)    **目標:**控制症狀,預防疾病進展 |  |
| **2. 經皮冠狀動脈介入治療(PCI)** | **又稱:**心導管手術、支架置放術    **方式:**透過導管將阻塞血管打通並放置支架    **優點:**    - 微創手術,恢復快    - 住院時間短(通常1-2天)    - 可立即改善血流    **適用:**單一或少數血管阻塞 |  |
| **3. 冠狀動脈繞道手術(CABG)** | **方式:**開胸手術,使用血管繞過阻塞處    **優點:**    - 可同時處理多處阻塞    - 長期通暢率佳    **缺點:**    - 手術規模較大    - 恢復時間較長    **適用:**多條血管阻塞,特別是糖尿病患者 |  |


#### 如何選擇治療方式?


**治療決策考量因素**

選擇治療方式需考慮多個面向:

1. **症狀嚴重程度:**這是最重要的考量
2. **阻塞血管數量:**      - 單一血管:通常選擇PCI(心導管) - 多條血管:可能選擇CABG(繞道手術)
3. **是否有糖尿病:**糖尿病患者多血管病變時,繞道手術效果較佳
4. **是否曾接受繞道手術:**若已有繞道手術,通常選擇PCI
5. **患者整體健康狀況**
6. **患者及家屬的偏好**


### 問題六:為什麼CTO的導管手術較困難?


#### 技術挑戰


| p{10cm}}   **困難點** | **說明** |  |
| --- | --- | --- |
| 完全閉塞 | 100  沒有可見的血管腔 |  |
| 閉塞時間長 | 阻塞處組織纖維化、鈣化    質地堅硬,不易穿透 |  |
| 路徑複雜 | 閉塞段可能很長    血管可能扭曲或彎曲 |  |
| 需要特殊技術 | 可能需要使用多種專業技術:    - Antegrade approach(順向技術)    - Retrograde approach(逆向技術)    - Dissection re-entry技術 |  |
| 需要專業設備 | 特殊導線、導管、微導管    影像導引系統    可能需要心臟支持裝置 |  |
| 手術時間長 | 平均約2小時    複雜案例可能需要更長時間 |  |


#### 為什麼需要經驗豐富的醫師?


importantbox
****手術成功率與經驗相關****

**有經驗的CTO中心:**

- 成功率: 85-90
- 併發症風險: 2-3


**經驗較少的中心:**

- 成功率**顯著較低**
- 併發症風險可能較高


**建議:**選擇有CTO處理經驗的醫療中心和醫師進行手術。
importantbox


### 問題七:手術有哪些風險?


#### 主要風險


| p{10cm}}   **風險類型** | **說明** |  |
| --- | --- | --- |
| 心肌梗塞 | 手術過程中可能發生    機率:約1-2 |  |
| 緊急手術需求 | 罕見情況下可能需要緊急冠狀動脈繞道手術    機率:<1 |  |
| 中風 | 極罕見但可能發生    機率:<0.5 |  |
| 死亡 | 機率很低但不是零    機率:<0.5 |  |
| 支架再狹窄 | 支架內可能再次狹窄    需要後續追蹤和可能的再次治療 |  |
| 穿刺部位併發症 | 出血、血腫、假性動脈瘤    通常可以處理 |  |
| 顯影劑相關 | 過敏反應    腎功能影響 |  |
| 輻射暴露 | 接受X光輻射    醫療團隊會盡量減少輻射劑量 |  |


importantbox
****整體風險評估****

**在有經驗的中心:**

- 總併發症風險: 約2-3
- 大多數患者手術順利完成
- 嚴重併發症罕見


**風險因子:**

- 高齡
- 腎功能不全
- 左心室功能不良
- 多重共病
- CTO病變複雜度高


importantbox


### 問題八:PCI(心導管)和繞道手術如何選擇?


#### 決策考量


| p{9cm}}   **考量因素** | **說明** |  |
| --- | --- | --- |
| **首要:患者症狀** | 症狀是否影響生活品質    藥物治療效果如何 |  |
| 阻塞血管數量 | **單一血管:**通常選擇PCI    **多條血管:**考慮CABG,特別是糖尿病患者 |  |
| 是否曾接受CABG | 若已有繞道手術,通常選擇PCI治療原生血管(CTO)    原生血管長期通暢率較繞道血管佳 |  |
| 患者整體狀況 | 年齡、共病症、心肺功能    手術耐受度評估 |  |
| 患者及家屬偏好 | 充分討論後的共同決策    考慮恢復期、風險等因素 |  |


#### 一般建議原則


**治療選擇指引**


1. **曾接受繞道手術者:**      - 優先選擇PCI - 原生血管(即使是CTO)的長期通暢率較繞道血管佳
2. **僅有CTO一條血管阻塞:**      - 大多數情況選擇PCI
3. **多條血管阻塞:**      - 特別是糖尿病患者 - 冠狀動脈繞道手術(CABG)為較佳選擇


**最終決定:**需經由心臟科醫師、心臟外科醫師、患者及家屬共同討論決定。

### 問題十:手術前中後應該注意什麼?


#### 手術前準備


| p{10cm}}   **準備事項** | **說明** |  |
| --- | --- | --- |
| **術前檢查** | 可能需要的檢查:    - 血液檢查    - 心電圖    - 心臟超音波(Echocardiogram)    - 電腦斷層掃描(CT)    - 心臟磁振造影(Cardiac MRI) |  |
| **禁食** | 手術前一晚午夜後**禁食禁水**    包括食物和飲料 |  |
| **藥物調整** | 某些藥物需要暫停:    - 抗凝血藥物(warfarin, NOAC)    - 部分糖尿病藥物(metformin)    **請務必告知醫師所有用藥** |  |
| **過敏史** | 告知是否對以下過敏:    - 顯影劑    - 藥物    - 碘 |  |
| **攜帶物品** | - 身份證明文件    - 健保卡    - 之前的病歷資料    - 目前服用的藥物清單 |  |


#### 手術過程


| p{10cm}}   **手術階段** | **說明** |  |
| --- | --- | --- |
| **進入導管室** | - 躺在X光檢查台上    - 連接心電圖監視器    - 連接血壓、血氧監測 |  |
| **鎮靜藥物** | - 給予輕度鎮靜藥物放鬆    - 通常不會完全睡著    - 大多數患者不記得手術細節 |  |
| **穿刺部位** | 通常在以下部位之一:    - 手腕(橈動脈)    - 鼠蹊部(股動脈)    通常會放置**兩根導管** |  |
| **心臟支持裝置** | 複雜案例可能需要:    - 主動脈內氣球幫浦(IABP)    - Impella等裝置    通常手術結束時移除 |  |
| **打通血管** | 使用多種器材:    - 導線(guidewires)    - 氣球導管(balloons)    - 支架(stents)    - 其他特殊器械 |  |
| **手術時間** | **平均約2小時**    - 可能更短(<1小時)    - 可能更長(>3小時)    視病變複雜度而定 |  |


**導管室環境說明**

**導管室設備:**

- X光機:用於即時觀察血管內情況
- 多個監視螢幕:顯示心電圖、血壓、血管影像
- 無菌環境:醫療團隊穿著手術衣、戴口罩


**醫療團隊:**

- 主刀醫師
- 助手醫師
- 護理師
- 放射技師
- 麻醉護理師或麻醉科醫師


**手術中:**

- 持續監測心率、血壓、心電圖
- 確保過程安全順利
- 家屬可在等候區等待,醫療團隊會定期更新狀況


#### 手術後照護


| p{10cm}}   **照護項目** | **說明** |  |
| --- | --- | --- |
| **立即術後** | - 穿刺部位可能有輕微不適    - 需要平躺數小時(若從鼠蹊部穿刺)    - 持續監測生命徵象 |  |
| **住院時間** | 大多數患者:    - 住院**1晚**    - 隔天出院    除非需要額外監測 |  |
| **症狀改善** | - 有些患者立即感到改善    - 有些患者需要數週才感到明顯差異    - 逐漸增加活動量 |  |
| **活動限制** | 術後初期(**約1週**):    - 避免提重物(>10磅/4.5公斤)    - 避免劇烈運動    - 預防穿刺部位出血 |  |
| **心臟復健** | **強烈建議**參加心臟復健:    - 監督下的運動訓練    - 增加體能和活動能力    - 學習健康生活方式    - 特別適合長期缺乏活動的患者 |  |


#### 重要用藥


importantbox
****術後藥物治療至關重要****

**雙重抗血小板治療(DAPT):**

1. **Aspirin(阿斯匹靈)**      - 通常需**終身**服用 - 預防血栓形成
2. **P2Y12抑制劑**      - 最常用: Clopidogrel(保栓通) - 其他: Ticagrelor, Prasugrel - 通常需服用**至少12個月** - 某些情況可能需要更長時間


**為什麼這些藥物如此重要?**

- 預防支架內血栓形成
- 支架內血栓可能導致心肌梗塞或死亡
- ****絕對不可自行停藥****
- 若需其他手術,請先諮詢心臟科醫師


importantbox


#### 其他重要藥物


| p{10cm}}   **藥物類別** | **作用與注意事項** |  |
| --- | --- | --- |
| Statins(降膽固醇藥) | - 降低LDL膽固醇    - 穩定斑塊    - 減少心血管事件    **通常需長期使用** |  |
| Beta-blockers(乙型阻斷劑) | - 降低心率和血壓    - 減少心臟負擔    - 改善預後 |  |

| 糖尿病藥物 | - 嚴格控制血糖    - 減少併發症 |  |


#### 生活方式調整


importantbox
****健康生活方式預防再發****

**重要提醒:**手術不是終點,冠狀動脈疾病可能繼續進展,新的阻塞仍可能形成。

**關鍵生活方式改變:**

**1. 飲食調整**

- 低飽和脂肪、低膽固醇飲食
- 多吃蔬菜、水果、全穀類
- 選擇魚類、家禽、豆類作為蛋白質來源
- 減少加工食品和含糖飲料
- 控制鹽分攝取


**2. 規律運動**

- 每週至少150分鐘中等強度運動
- 從輕度活動開始,逐漸增加
- 考慮參加心臟復健計畫
- 運動前諮詢醫師


**3. 戒菸**

- 吸菸是最重要的可改變危險因子
- 戒菸可立即開始改善心血管健康
- 必要時尋求戒菸門診協助


**4. 控制危險因子**

- 血壓: 目標<130/80 mmHg
- 膽固醇: LDL<70 mg/dL(甚至更低)
- 血糖: HbA1c<7
- 體重: 維持健康BMI


**5. 充足睡眠與壓力管理**

- 每晚7-9小時睡眠
- 學習壓力管理技巧
- 保持良好心理健康


importantbox


#### 手術失敗的處理


**若手術未成功怎麼辦?**

約10-15

**選項一:再次嘗試**

- 通常在幾週或幾個月後
- 使用不同策略或技術
- 成功率仍有可能


**選項二:考慮繞道手術**

- 若患者症狀明顯
- 若適合手術
- 繞道手術成為替代方案


**選項三:藥物治療**

- 優化藥物治療
- 生活方式調整
- 症狀管理


**重要:**手術失敗不代表沒有其他選擇,與醫師討論最適合您的方案。

## 手術同意書


### 手術執行方式詳細說明


#### 手術過程


冠狀動脈慢性全阻塞(CTO)的經皮冠狀動脈介入(PCI)手術執行方式:


1. **血管穿刺**      - 大多數情況需要**兩個以上**的動脈穿刺部位 - 穿刺位置可能在:腹股溝(股動脈)、手腕(橈骨動脈)或手臂(肱動脈) - 由心臟專家根據病情判斷最適合的穿刺部位
2. **導線穿越技術**      - **順行穿越(Antegrade):**從閉塞上游的血管穿越 - **逆行穿越(Retrograde):**通過其他冠狀動脈的側支循環,從閉塞下游穿越 - 導線可能通過閉塞本身或血管內膜下空間 - 可能需要特殊導管協助重新進入血管真腔
3. **處理鈣化病灶**      - **血管內震波碎石術:**使用產生聲波的特殊球囊導管 - **旋磨術(Rotablation):**使用螺旋鑽磨碎裂鈣質 - 促進後續血管擴張
4. **支架置放**      - 擴張後幾乎都需要植入一個或多個金屬支架 - 支架材質:惰性金屬(不鏽鋼基礎合金),不會產生排斥或過敏 - **塗藥支架:**漸進式釋放藥物,減少組織增生和再狹窄風險 - 支架在高壓球囊擴張下嵌入動脈壁,裝置好後不會移動


#### 手術時間與複雜性


importantbox
**手術時間考量**

- CTO手術通常需要**較長時間**
- 穿越完全閉塞平均需要**90-120分鐘**
- 嚴重鈣化病灶可能需要更長時間
- 根據閉塞的位置、長度、側支循環、血管曲折度和鈣化程度而定


importantbox


#### 術後抗血小板治療


importantbox
****極為重要的術後用藥****

**雙重抗血小板治療(DAPT):**

- 成功開通CTO並植入支架後,**必須每天服用**抗血小板藥物
- 通常為阿司匹林(Aspirin)與其他抗血小板藥物的組合
- 持續時間取決於:     itemize
- 冠狀動脈疾病的臨床複雜性
- 患者的其他合併症
- 植入的支架數量和長度


    \item 之後必須**終身**繼續單一抗血小板藥物
itemize

**請務必告知醫師:**

- 是否對阿司匹林過敏
- 過去是否有重大出血病史

## 常見問題補充


### 關於藥物


**Q: 我需要終身服用抗血小板藥物嗎?**

A: Aspirin通常需要終身服用。第二種抗血小板藥物(如Clopidogrel)通常需要服用至少12個月,某些情況可能需要更長時間。請勿自行停藥,若有疑問請諮詢醫師。

**Q: 服用抗血小板藥物後容易瘀青出血,怎麼辦?**

A: 這是正常現象。避免受傷,使用軟毛牙刷,小心使用刀具。若出血不止或有異常出血,請立即就醫。

**Q: 需要其他手術時,是否要停用抗血小板藥物?**

A: 請先諮詢心臟科醫師和即將手術的醫師,共同決定最安全的方案。多數小手術可以繼續服用。


### 關於活動


**Q: 何時可以開車?**

A: 通常術後1-2週,若傷口復原良好且無不適症狀。請先諮詢醫師。

**Q: 何時可以恢復工作?**

A: 視工作性質而定,辦公室工作通常1-2週內可恢復,粗重工作可能需要4-6週。

**Q: 可以搭飛機嗎?**

A: 通常術後2-4週可以搭飛機,建議先諮詢醫師。記得攜帶藥物和病歷摘要。


### 關於飲食


**Q: 有特殊飲食限制嗎?**

A: 建議低脂、低鹽、高纖維飲食。避免高膽固醇食物。若有糖尿病,需控制血糖。營養師可提供個人化建議。

**Q: 可以喝酒嗎?**

A: 建議限制或避免飲酒。若要飲用,男性每日不超過2個單位,女性不超過1個單位。


### 關於性生活


**Q: 何時可以恢復性生活?**

A: 通常術後1-2週,若傷口復原良好且無症狀。性生活時若有胸痛或不適,應立即停止並就醫。


## 結語


冠狀動脈慢性完全閉塞(CTO)的治療是一個專業且複雜的過程,但隨著技術進步,在有經驗的醫療團隊照護下,多數患者能獲得良好的治療結果和生活品質改善。

**成功治療的關鍵:**

1. 選擇有經驗的醫療團隊
2. 術後確實服藥
3. 改變生活方式
4. 定期追蹤
5. 與醫療團隊保持良好溝通

- 您有充分的時間考慮是否接受手術
- 您可以隨時撤回同意
