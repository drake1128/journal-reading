## 開口病變的血管攝影角度


### 最佳開口可視化原則


對於主動脈開口病變患者進行冠狀動脈攝影是最具挑戰性的血管攝影任務之一，需要正確描繪開口狹窄並處理導管引起的缺血和damping。

關鍵原則是獲得一個角度，使開口與瓣尖和主動脈的交界在二維角度下呈現為**一條線**。不要讓瓣尖覆蓋在開口上方，這會遮蔽正確的描繪。


### 各血管標準視角

| \tablename  -- 續 |  |  |

- **前向起源RCA**：需要更陡峭的LAO（或陡峭RAO）以展開開口
- **CT分析**：研究顯示最佳正交視角為：     itemize
- LCA：LAO cranial
- RCA：陡峭LAO合併cranial角度（雖然LAO caudal可能更好顯示最近端部分）


    \item **實際限制**：非常陡峭的LAO增加軟組織穿透和影像模糊
    \item **標記環**：如果移植血管開口周圍有標記環，獲取使圓環變成一條線的視角

## Damping與心室化


### 定義與識別


| p{10cm}} 導管壓力波形變化 |  |  |
| --- | --- | --- |
| **狀態** | **描述** |  |
| \tablename  -- 續 |  |  |
| **狀態** | **描述** |  |
| **正常接合** | 導管尖端傳導主動脈壓力波形 |  |
| **Damping** | 導管楔入動脈；壓力波形變鈍，脈壓降低 |  |
| **輕度心室化** | 峰值間的波段變水平，有時有小a波 |  |
| **重度心室化** | 尖端壓力反映下游心室壓力；幾乎像真正的LV壓力 |  |
| **完全阻塞** | 平線——導管抵住開口壁或斑塊 |  |


**絕對不可在damped或心室化時注射顯影劑！**

- 顯影劑會滯留在冠狀動脈內無法清除
- 進一步加重導管已造成的缺血
- **心肌染色** = 顯影劑染色實際心肌 = **距離VF僅一秒之遙**
- 高壓注射有冠狀動脈夾層風險


**處置**：一旦發現damping/心室化，立即脫離導管！


## Damped導管的影像技術


當導管接合時發生damping或心室化，有四種技術可獲得診斷影像：


### 技術1：微脫離位置（首選）


- 脫離導管至「剛好非選擇性」狀態
- 導管稍微退出時注射
- 可獲得良好影像，無阻塞性或心室化波形
- **RCA**：逆時針旋轉（與接合相反）使導管保持同一水平但稍微退出
- **LCA**：輕微順時針或逆時針旋轉，或輕柔回拉
- 考慮使用較短尖端的導管（如JL而非Tiger或EBU）


### 技術2：快進快出注射法


- 在damped/心室化時快速注射1-2 cc（RCA）或4-5 cc（LCA）
- 注射後立即撤回導管
- **限制**：     itemize
- 仍有夾層風險
- 需要多次重新接合以完成評估
- 有心肌染色風險


itemize


### 技術3：輕度心室化時攝影


- 僅適用於**輕度**心室化（非重度）
- 可以快速拍攝幾張影像
- 不建議用於重度心室化


### 技術4：快進快出導線法（最推薦）


**快進快出導線技術**——介入醫師最「專業」的技術：

1. 將導管推進至主動脈；接合前先將導線準備在導管內
2. 接合冠狀動脈（接受damping，但不要注射）
3. 在damping狀態下快速盲目導線冠狀動脈
4. 脫離導管——導線錨定導管防止飛出
5. 在穩定的微脫離位置獲取影像
6. 考慮使用**iFR/FFR導線**同時完成影像和生理評估


**重要**：進行開口iFR/FFR時，測量期間務必脫離導管。接合的導管會因阻塞限制血流下降而導致iFR值偽高。


## Damped導管下的PCI技術


### 球囊和支架的快進快出操作


快進快出導線的相同原則適用於支架置放：


1. 在damped時導線，然後脫離獲取影像
2. 決定球囊/支架
3. 在導管脫離時（無缺血）將支架推進並定位在導管內
4. 接合（接受damping）以獲得支撐
5. 在damping狀態下將支架推進至目標
6. 脫離導管以緩解缺血
7. 在微脫離狀態下釋放支架


**技巧**：對於嚴重左主幹病變，先球囊擴張LM，讓患者能更好耐受後續操作期間的缺血。


### 微脫離狀態下支架置放


如果導管可以保持穩定的微脫離位置：

- 在導管部分退出時導線和支架
- 不需要快進快出
- 主要適用於開口支架置放（不適用於需要深入接合以獲得支撐的遠端病變）


## 側孔導管：為何不建議使用


**大多數開口PCI病例應避免使用側孔導管。**

**反對側孔導管的五個理由**：

1. 提供的血流很少（正常的幾個百分點）——不能顯著減輕缺血
2. 通過側孔傳導主動脈壓力來改善波形，而非緩解缺血——提供**虛假的安全感**
3. 失去damping作為診斷工具——damping告訴你何時需要在支架定位期間脫離導管
4. **絕對不可**用於左主幹PCI——你不希望隱藏的LM缺血
5. 對開口RCA PCI無價值——本來就應該大部分時間脫離


**唯一可能的例外**：通過中度開口病變/小開口進行中遠端RCA PCI，且不需要支架該處，在RCA供血區域較小的患者。可能降低顯影劑染色和VF風險。

**經橈動脈RCA開口支架置放使用JR4——問題所在：**

- 呼吸時導管運動過大，尤其是肥胖患者
- 每次深呼吸時，導管可能飛出至主動脈
- 噩夢場景：部分釋放的支架隨深呼吸飛入主動脈


**解決方案**：使用延伸方式的Amplatz Left（非鴨形）、AR2或Ikari導管，這些導管錨定更好，能抵抗呼吸運動。


## 支架定位：避免地理位置偏差


### 主動脈開口交界的解剖

> **下緣比上緣更向前：**

>     \item **下緣**：在瓣尖水平
>     \item **上緣**：靠近管狀主動脈
> itemize
> **完美的支架定位：**
> itemize[leftmargin=*]
>     \item 將支架定位在**下緣**近端幾毫米處
>     \item 你**必須**覆蓋下緣進入瓣尖
>     \item 你**不需要**覆蓋上緣
>     \item 血液通過主動脈和瓣尖流入支架創造的「煙囪」
>     \item 開口病變使用支架長度$\geq$12 mm
> itemize


**這與分支開口病變不同：**

- 分支開口病變必須覆蓋上方的隆突
- 主動脈開口病變覆蓋上緣可能有害（支架延伸太遠進入主動脈）


### 開口擴張技術


釋放開口支架後：

1. 回拉支架球囊
2. 球囊部分懸在主動脈內時以**更高壓力**充氣
3. 這樣「擴張」近端支架以：      - 使未來接合更容易 - 解決開口處高彈性回縮 - 創造進入主動脈的錐形外觀


**Ostial Flush球囊**：一種專門的雙球囊導管，近端有球形球囊用於對稱擴張。標準球囊也可以。


## 精確支架定位的手部操作


### 三元素問題


任何介入中，你必須觀察三個元素：**導管**、**導線**和**支架**。對其中一個的動作會在其他兩個上產生相反反應。

**開口支架置放挑戰**：

- 將支架拉回到開口 $\rightarrow$ 導管和導線潛入
- 將導管拉回以暴露開口 $\rightarrow$ 支架進一步潛入
- 這個循環重複，阻止正確定位


### 左手掌技術


**掌握此技術以成功進行開口支架置放：**

**步驟1 - 定位支架：**

- 右手：拉支架使其定位在開口
- 左手掌：同時對導管施加回拉張力
- 左手指：回拉導線（防止穿孔）
- 觀看螢幕——調整張力使導管保持穩定而支架移動


**步驟2 - 脫離導管：**

- 左手掌：拉導管以脫離
- 右手：同時拉支架/導線以防止潛入
- 結果：導管脫離而支架保持位置


### 部分充氣穩定技術


為增加穩定性，特別是有呼吸運動時：

1. 當支架接近最佳位置時，部分充氣至2-3大氣壓
2. 這錨定支架並防止：      - 導管操作時向前潛入 - 深呼吸時跳動
3. 花時間微調位置
4. 可以在支架仍部分在導管內時開始部分充氣


**絕對噩夢：**

- 患者在釋放支架時深呼吸
- 支架飛出並最終部分釋放在主動脈內
- **這無法糾正**——你必須接受支架懸在主動脈內


**如果支架潛入太遠端** = 可以加另一個支架（可糾正）  

**如果支架飛入主動脈** = 災難（無法糾正）


## 不推薦的技術


### 浮動導線技術


- 概念：在導管脫離時將導線推進至瓣尖以描繪開口並防止導管潛入
- **問題**：     itemize
- 導線可能漂離真正的開口
- 給出開口位置的錯誤印象
- 可能導致過度近端支架置放（進入主動脈）
- 引入第四個要觀察的元素——增加複雜性


itemize


### Szabo技術


- 概念：浮動導線加上近端支架支柱鉤在上面
- 在體外預擴張支架，將導線後端裝入遠端支柱
- **問題**：     itemize
- 與浮動導線相同的限制
- 造成顯著支架變形
- 體外和IVUS研究顯示過度近端支架突出
- 防止遠端偏差但**不能防止終極噩夢**（支架進入主動脈）


itemize


### Ostial Pro裝置


- 一種短導管延伸件，有腳貼在主動脈上
- 防止導管深潛
- **限制**：     itemize
- 仍可能造成過度近端支架置放
- 引入第四個要管理的元素


itemize


**關鍵原則**：在介入心臟學中，堅持基礎並保持盡可能簡單。不要添加不必要的元素和技術。掌握基本的手部操作。


## IVUS驗證開口覆蓋


**支架置放後務必使用IVUS驗證開口覆蓋。**


> 在IVUS回拉中，你必須看到至少一幀畫面顯示：
> itemize[leftmargin=*]
>     \item **支架的一部分**貼在冠狀動脈壁上（白色外觀）
>     \item **支架的另一部分**在主動脈內（同一幀）
> itemize
> 這確認支架正確覆蓋開口。
> **另一個可接受的發現**：一幀顯示支架完全被主動脈包圍（如果完全覆蓋上下緣）。


如果IVUS顯示錯過開口，在結束手術前加另一個支架。

## 總結與重點摘要


1. **視角**：LM用LAO cranial，RCA用陡峭LAO（40-50度）——使瓣尖與開口呈一條線
2. **絕對不可在damped/心室化時注射**——有VF和夾層風險
3. **快進快出導線法**是damped開口病變的首選技術；考慮使用iFR導線
4. **快進快出操作**球囊和支架用於damped導管下的PCI
5. **避免側孔導管**——虛假安全感，掩蓋缺血，對LM尤其危險
6. **經橈動脈避免JR4**處理開口RCA——呼吸運動過大
7. **專注於下緣**——必須覆蓋它，不必執著於上緣
8. 使用**左手掌技術**進行精確支架定位
9. **部分充氣**（2-3 atm）有助於微調期間穩定支架
10. **避免**浮動導線、Szabo和Ostial Pro——增加複雜性而無益處
11. **務必使用IVUS驗證**——尋找同一幀中支架同時在冠狀動脈和主動脈內
12. **Rotablation**：從導管內開始，向前旋磨進入開口
13. **Orbital atherectomy**：深入接合，回程時切割（絕對不能向前通過開口）
