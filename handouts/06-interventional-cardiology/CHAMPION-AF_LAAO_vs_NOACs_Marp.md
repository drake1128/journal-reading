# 第一部分：研究背景

---

# 研究背景

- 經皮 left atrial appendage closure (LAAC) 原為無法使用 OAC 患者的替代方案
- PROTECT-AF / PREVAIL 證實 WATCHMAN vs **warfarin** 的安全性與有效性
- **NOACs 已取代 warfarin** 成為 AF stroke prevention 首選
- 缺乏 WATCHMAN FLX vs NOACs 的直接隨機比較數據

> **知識缺口**：LAAC 能否作為 NVAF 患者的**第一線**中風預防策略？

---

# CHAMPION-AF 研究目標

## 前瞻性評估 WATCHMAN FLX 能否作為 NOACs 的第一線替代方案

- **首個也是最大規模** LAAC vs NOACs 的 RCT
- 納入**廣泛族群**：包括低至中度出血風險患者
- 3,000 patients, 141 centers globally
- 36 個月 primary analysis, 5 年延伸追蹤

---

<!-- _class: divider -->

# 第二部分：試驗設計

---

# 試驗設計

- **設計**：Prospective, randomized, multi-center, global
- **分配**：1:1 → WATCHMAN FLX (N=1,499) vs NOAC (N=1,501)
- **納入**：NVAF, CHA₂DS₂-VASc ≥2 (男) / ≥3 (女), 適合 OAC
- **主要分析**：36 個月 (ITT)

```text
3,000 patients → 隨機 1:1
  ├── WATCHMAN FLX (n=1,499)
  │     └─ 術後: DOAC±ASA (≥3mo) → ASA/P2Y12 (至1yr)
  └── NOACs (n=1,501)
        └─ 持續服用 approved DOAC
```

---

# Primary Endpoints 定義

| 終點 | 定義 | 分析方法 | 時點 |
|------|------|---------|------|
| **Efficacy** | CV death + Stroke + SE | Non-inferiority | 3 年 |
| **Safety** | ISTH non-procedural major + CRNM bleeding | Superiority | 3 年 |
| **Efficacy** | Ischemic stroke + SE | Non-inferiority | 5 年 |

- **Net clinical benefit** = CV death + Stroke + SE + Non-procedural bleeding
- ISTH = International Society on Thrombosis and Haemostasis

---

# 第三部分：基線特徵

---

# 基線特徵 — 兩組均衡

| 特徵 | NOACs (n=1,501) | WATCHMAN FLX (n=1,499) |
|------|-----------------|----------------------|
| 年齡 (歲) | 71.8 ± 7.5 | 71.6 ± 7.5 |
| 女性 | 31.5% | 32.4% |
| **CHA₂DS₂-VASc** | **3.5 ± 1.3** | **3.5 ± 1.2** |
| **HAS-BLED** | **1.3 ± 0.8** | **1.3 ± 0.8** |
| 先前缺血性中風 | 7.6% | 7.9% |
| 重大出血病史 | 3.3% | 3.5% |
| Paroxysmal AF | 68.5% | 69.3% |
| 先前 AF ablation | 46.9% | 48.7% |

> **關鍵**：HAS-BLED 平均 1.3 → 包含大量低至中度出血風險患者

---

<!-- _class: divider -->

# 第四部分：主要結果

---

# Primary Efficacy Endpoint ✓

## CV Death + Stroke + Systemic Embolism — Non-inferiority Met

| | WATCHMAN FLX | NOACs |
|------|-------------|-------|
| **事件率 (36 mo)** | **5.7%** | **4.8%** |
| **HR (95% CI)** | **1.20 (0.87, 1.66)** | |
| **P non-inferiority** | **< 0.0001** | |

- CV death: 2.7% vs 2.7% (HR 1.01) — 完全相同
- All stroke: 3.6% vs 2.5% (HR 1.46, NS)
- Systemic embolism: 0.0% vs 0.1%

---

# Primary Safety Endpoint ✓

## ISTH Non-procedural Bleeding — Superiority Achieved

| | WATCHMAN FLX | NOACs |
|------|-------------|-------|
| **事件率 (36 mo)** | **10.9%** | **19.0%** |
| **HR (95% CI)** | **0.55 (0.45, 0.67)** | |
| **P superiority** | **< 0.0001** | |

- ISTH major bleeding: 5.1% vs 6.4% (HR 0.78)
- **Modified CRNM bleeding: 7.0% vs 14.2% (HR 0.48)**

> **45% 相對風險降低**，NNT ≈ 12（每治療 12 位患者可預防 1 次出血事件）

---

# 手術相關數據

## Procedural Success: **98.8%**

| 指標 | 數值 |
|------|------|
| Procedural success | **98.8%** |
| 年化缺血性中風率 | **1.1%/年** |
| NOAC 組年化缺血性中風率 | 0.7%/年 |

> **即使計入手術出血**，總 ISTH bleeding 仍顯著低於 NOAC 組
> (12.8% vs 19.0%, **HR 0.66**, P<0.0001) → **34% 降低**

---

# 第五部分：次要終點

---

# 次要終點與淨臨床效益

| 終點 | NOACs | WATCHMAN FLX | HR (95% CI) | P |
|------|-------|-------------|-------------|---|
| ISTH major bleeding (含 procedural) | 6.4% | 5.9% | 0.92 (0.68, 1.24) | <0.0001 |
| **Net clinical benefit** | **21.8%** | **15.1%** | **0.66 (0.56, 0.79)** | **<0.0001** |

## Net Clinical Benefit = CV Death + Stroke + SE + Non-procedural Bleeding

> **Superiority achieved**: WATCHMAN FLX 降低 **34%** 淨臨床不良事件風險

# 第七部分：臨床意義

---

# CHAMPION-AF vs 先前 LAAC 試驗

| | PROTECT-AF / PREVAIL | CHAMPION-AF |
|------|---------------------|-------------|
| 對照組 | Warfarin | **NOACs** |
| 族群 | 高中風/出血風險 | **含低至中度風險** |
| Device | WATCHMAN 1st gen | **WATCHMAN FLX** |
| 植入成功率 | ~95% | **98.8%** |
| 定位 | Warfarin 替代 | **NOAC 第一線替代** |

> **典範轉移**：LAAC 從「無法抗凝的替代」→「第一線治療選擇」

---

# 臨床實務要點

> **Pearl 1**: LAAC 可作為廣泛 NVAF 族群（含低至中度出血風險）的**第一線**選擇

> **Pearl 2**: 出血獲益顯著（**45% 降低**），一次性手術免除終身抗凝

> **Pearl 3**: 年化缺血性中風率 1.1%，與當代最佳 NOAC 相當

> **Pearl 4**: Net clinical benefit **superior**（HR 0.66）— 整體而言 LAAC 患者獲益更多

---

# 適合 LAAC 的患者特徵

```text
適合考慮 WATCHMAN FLX 的患者：
  ├── NVAF + CHA₂DS₂-VASc ≥2/3
  ├── 擔憂長期抗凝出血風險
  ├── 抗凝藥物遵從性不佳
  ├── 活動性生活方式（運動、跌倒風險）
  ├── 曾有出血事件或高 HAS-BLED
  └── 偏好一次性治療免除長期用藥

術前評估重點：
  ├── TEE 評估 LAA 解剖
  ├── 排除 LAA thrombus
  └── 確認可接受短期抗血栓治療
```

---

# 研究限制

1. **Open-label** 設計（但由 independent CEC 裁定終點）
2. **36 個月**結果，5 年 ischemic stroke endpoint 待報告
3. Trial 中 NOAC 遵從率可能高於真實世界
4. 未詳述 device-related thrombus (DRT) 與 peridevice leak (PDL) 數據
5. 平均 HAS-BLED 1.3 — 真實世界高出血風險患者獲益可能更大

---

# 結論

## WATCHMAN FLX 達成所有三年終點，確立 LAAC 的第一線地位

| 終點 | 結果 | 意義 |
|------|------|------|
| Efficacy (CV death/Stroke/SE) | **Non-inferior** ✓ | 中風預防不劣於 NOACs |
| Safety (Non-procedural bleeding) | **Superior** ✓ | 出血風險降低 45% |
| Net clinical benefit | **Superior** ✓ | 整體臨床獲益更佳 |
| Procedural success | **98.8%** | 技術成熟度高 |
| Annualized ischemic stroke | **1.1%/年** | 與最佳 NOAC 相當 |
