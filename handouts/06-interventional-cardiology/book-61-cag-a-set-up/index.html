心血管中心教學文件 冠狀動脈造影 步驟一：確保良好的壓力波形 3.1 重要性 關鍵步驟：在注射任何對比劑之前，必須確認壓力波形正常！ 3.2 壓力波形衰減（Pressure Dampening）的風險 如果在壓力波形衰減的情況下注射對比劑，可能導致： • 冠狀動脈剝離（Coronary dissection） • 心律不整 • 其他嚴重併發症 3.3 評估標準 重點提醒 理想的壓力波形特徵： • 清晰的收縮壓峰值 • 明顯的舒張期下降 • 波形形態正常，無衰減（dampening） • 數值與主動脈壓力相符 3.4 處理異常壓力波形 若發現壓力波形衰減： 1. 立即停止：不可注射對比劑 2. 調整導管位置： • 輕微退出導管 • 旋轉導管角度 • 確認導管未過度嵌入（over-engagement） 3. 重新評估：調整後再次檢查壓力波形 4. 考慮更換導管：若持續衰減，考慮使用不同型號

心血管中心教學文件 冠狀動脈造影 步驟二：小劑量測試注射 4.1 測試注射的目的 在進行完整造影前，先給予1-2 cc 的小劑量對比劑測試注射，目的包括： 評估項目 說明 1. 確認導管嵌入 確保導管確實進入冠狀動脈而非其他位置 2. 檢查開口病變 評估冠狀動脈開口是否有狹窄（ostial lesion） 3. 評估導管同軸性 確認導管與血管同軸（coaxial），未指向血管壁 4. 檢查嵌入深度 確保導管嵌入深度適當（理想為2-3 mm），避免過深 4.2 測試注射的執行 1. 將注射器保持垂直（minimizing bubbles） 2. 緩慢注射1-2 cc 對比劑 3. 觀察對比劑流向與分布 4. 根據結果調整導管位置 4.3 臨床實例 重點提醒 案例一：測試注射顯示導管指向左主幹頂部（roof of left main）而非同軸，需調整導管角 度後再進行完整造影。 案例二：繞道血管（bypass graft）測試注射顯示血管完全閉塞，避免了不必要的大量對 比劑注射。 4.4 注射前的關鍵檢查 防止氣泡栓塞 • 注射前將manifold 保持垂直 • 確保所有氣泡上升至頂部 • 避免氣泡進入導管線路 • 氣泡栓塞可能導致嚴重後果 步驟三：給予冠狀動脈Nitroglycerin 5.1 為何必須給予Nitroglycerin
