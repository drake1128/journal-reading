## 前言


冠狀動脈remodeling（血管重塑）是指動脈壁對粥狀硬化斑塊形成的適應性反應。這個概念由病理學家Seymour Glagov在1987年首次系統性描述，他發現冠狀動脈在斑塊形成早期會代償性擴張，以維持管腔面積。

理解冠狀動脈remodeling對於臨床診斷、風險分層和治療策略選擇都具有重要意義。不同類型的remodeling模式與斑塊穩定性、臨床表現和預後密切相關。


## Remodeling的定義與分類


### 基本定義


冠狀動脈remodeling是指動脈外彈性膜（external elastic membrane, EEM）面積的改變，作為對動脈粥狀硬化斑塊積累的反應。這個過程涉及血管壁結構的重新組織，包括細胞外基質的重塑、平滑肌細胞的遷移增殖，以及炎症反應。


### Glagov現象


Glagov在屍體解剖研究中觀察到，當冠狀動脈斑塊負荷（plaque burden）小於40


### 分類標準


根據血管橫截面的變化，remodeling可分為以下類型：


| p{11.5cm}}  **類型** | **定義與特徵** |  |
| --- | --- | --- |
| Positive Remodeling   （正向重塑） | 病灶處的血管外彈性膜面積\textgreater 參考節段的外彈性膜面積    血管代償性擴張，管腔面積相對維持    Remodeling index \textgreater 1.05 |  |
| Negative Remodeling   （負向重塑） | 病灶處的血管外彈性膜面積\textless 參考節段的外彈性膜面積    血管收縮，管腔狹窄加劇    Remodeling index \textless 0.95 |  |
| Intermediate/No Remodeling   （中間型/無重塑） | Remodeling index介於0.95-1.05之間    血管面積變化不明顯 |  |


### Remodeling Index計算


Remodeling index的計算公式為：

\[
Remodeling Index = \text{病灶處EEM面積}{參考節段平均EEM面積}
\]

參考節段通常選擇病灶近端和遠端相對正常的血管段。


## Positive Remodeling（正向重塑）


### 病理生理機轉


Positive remodeling代表血管的代償性擴張反應。在斑塊形成早期，血管壁會向外擴張以容納增長的斑塊，從而維持管腔面積。這個過程涉及多種機制：


| p{11cm}}  **機制** | **說明** |  |
| --- | --- | --- |
| 基質金屬蛋白酶   （MMPs）活化 | MMP-2、MMP-9等酶降解細胞外基質，特別是膠原蛋白和彈性纖維，使血管壁更易擴張 |  |
| 炎症細胞浸潤 | 巨噬細胞、T淋巴細胞等炎症細胞釋放細胞激素，促進血管擴張 |  |
| 平滑肌細胞凋亡 | 血管中層平滑肌細胞凋亡減少對血管擴張的阻力 |  |
| 內皮功能障礙 | 內皮細胞產生較少的血管收縮物質，較多的血管舒張物質 |  |
| 新生血管形成 | 斑塊內和外膜的新生血管增加，可能促進血管擴張 |  |


### 斑塊特徵


Positive remodeling通常與以下斑塊特徵相關：


- **大的脂質核心**：斑塊中心含有豐富的脂質，占斑塊體積的很大比例
- **薄纖維帽**：覆蓋脂質核心的纖維帽較薄（\textless 65 \textmu m），易於破裂
- **高炎症活性**：大量巨噬細胞浸潤，炎症標記物升高
- **斑塊內出血**：脆弱的新生血管容易破裂出血
- **鈣化較少**：相對於negative remodeling，鈣化程度較低


Positive remodeling的斑塊通常是「易損斑塊」（vulnerable plaque），具有較高的破裂風險。這些斑塊在血管攝影可能看起來狹窄程度不嚴重，但實際上具有高度危險性。


### 臨床意義


| p{11cm}}  **臨床特徵** | **說明** |  |
| --- | --- | --- |
| 臨床表現 | 常見於急性冠心症（ACS），特別是急性心肌梗塞（AMI）    患者可能在斑塊破裂前沒有明顯症狀或僅有輕微症狀 |  |
| 血管攝影表現 | 管腔狹窄程度可能不嚴重（\textless 50  可能被低估病灶的危險性    需要IVUS或OCT等腔內影像確認 |  |
| 斑塊破裂風險 | 高風險：薄纖維帽、大脂質核心、高炎症活性    可導致急性血栓形成和心肌梗塞 |  |
| 預後 | 與較高的心血管事件風險相關    需要積極的藥物治療和風險因子控制 |  |


### 影像學表現


**血管內超音波（IVUS）**：

- 病灶處外彈性膜面積增大
- 可見低回音的脂質核心
- 纖維帽薄或不清晰
- 可能見到斑塊內出血或鈣化


**光學同調斷層掃描（OCT）**：

- 高解析度觀察纖維帽厚度（\textless 65 \textmu m定義為薄纖維帽）
- 清楚顯示脂質核心範圍
- 可見巨噬細胞聚集
- 評估斑塊內微血管


**冠狀動脈電腦斷層（CCTA）**：

- Napkin-ring sign：環形強化，中心低密度
- Spotty calcification：點狀鈣化
- Low attenuation plaque：低密度斑塊（\textless 30 HU）
- Positive remodeling：remodeling index \textgreater 1.1


## Negative Remodeling（負向重塑）


### 病理生理機轉


Negative remodeling代表血管的收縮性反應。血管壁向內收縮，不僅無法代償斑塊的生長，反而使管腔狹窄程度加劇。這個過程的機制包括：


| p{11cm}}  **機制** | **說明** |  |
| --- | --- | --- |
| 纖維化過程 | 膠原蛋白大量沉積，細胞外基質增加    纖維母細胞和肌纖維母細胞活化    組織金屬蛋白酶抑制劑（TIMPs）增加 |  |
| 平滑肌細胞增殖 | 血管中層平滑肌細胞增殖和遷移    收縮型表現型增加 |  |
| 炎症程度較低 | 相對於positive remodeling，炎症反應較輕    慢性低度炎症狀態 |  |
| 鈣化沉積 | 斑塊鈣化程度較高    血管壁僵硬度增加 |  |


### 斑塊特徵


Negative remodeling通常與以下斑塊特徵相關：


- **纖維斑塊為主**：斑塊主要由纖維組織構成，脂質核心相對較小
- **厚纖維帽**：纖維帽較厚（\textgreater 65 \textmu m），結構較穩定
- **高度鈣化**：斑塊內鈣化程度高，可能有密集或片狀鈣化
- **低炎症活性**：巨噬細胞浸潤較少，炎症標記物較低
- **慢性穩定過程**：通常是慢性動脈粥狀硬化的表現


Negative remodeling的斑塊通常是「穩定斑塊」（stable plaque），破裂風險相對較低，但由於管腔狹窄程度較重，更容易引起慢性心絞痛症狀。


### 臨床意義


| p{11cm}}  **臨床特徵** | **說明** |  |
| --- | --- | --- |
| 臨床表現 | 常見於穩定型心絞痛    症狀與活動相關，可預測性    運動耐受力逐漸下降    較少出現急性冠心症 |  |
| 血管攝影表現 | 管腔狹窄程度通常較嚴重（\textgreater 50-70  狹窄長度可能較長    血管攝影較能準確評估狹窄程度 |  |
| 斑塊破裂風險 | 相對低風險：厚纖維帽、纖維化斑塊    斑塊較穩定，較少急性血栓事件 |  |
| 預後 | 與慢性症狀相關，但急性事件風險較低    可能需要血管重建治療改善症狀    PCI成功率通常較高 |  |
| 治療考量 | 功能性缺血評估重要（FFR, iFR）    適合進行選擇性PCI    支架擴張較佳，併發症較少 |  |


### 影像學表現


**血管內超音波（IVUS）**：

- 病灶處外彈性膜面積減小
- 可見高回音的纖維組織或鈣化
- 纖維帽厚而清晰
- 脂質核心較小或不明顯


**光學同調斷層掃描（OCT）**：

- 厚纖維帽（\textgreater 65 \textmu m）
- 纖維組織為主，呈現均質高訊號
- 鈣化斑塊邊界清晰
- 較少見巨噬細胞聚集


**冠狀動脈電腦斷層（CCTA）**：

- 密集鈣化斑塊
- 管腔狹窄明顯
- Remodeling index \textless 1.0
- 纖維斑塊呈現中等密度

## 臨床評估與影像學方法


### 血管內超音波（IVUS）


IVUS是評估remodeling的標準工具之一：

**測量參數**：

- 外彈性膜面積（EEM area）
- 管腔面積（Lumen area）
- 斑塊加血管中層面積（Plaque + Media area）
- 斑塊負荷（Plaque burden = [EEM - Lumen]/EEM × 100
- Remodeling index


**優點**：

- 準確測量血管壁結構
- 評估斑塊組成
- 即時測量
- 指導PCI策略


**限制**：

- 侵入性檢查
- 解析度有限（100-150 \textmu m）
- 無法穿透鈣化陰影
- 需要額外時間和成本


### 光學同調斷層掃描（OCT）


OCT提供更高解析度的影像：

**優勢**：

- 超高解析度（10-20 \textmu m）
- 精確測量纖維帽厚度
- 清楚顯示斑塊微結構
- 評估支架貼壁情況


**限制**：

- 組織穿透深度有限（1-2 mm）
- 需要血液清除（對比劑注射）
- 較難評估血管外彈性膜
- 成本較高


### 冠狀動脈電腦斷層（CCTA）


CCTA是非侵入性評估remodeling的重要工具：

**高危斑塊特徵**（與positive remodeling相關）：

- Positive remodeling（RI \textgreater 1.1）
- Low attenuation plaque（\textless 30 HU）
- Napkin-ring sign
- Spotty calcification


**優點**：

- 非侵入性
- 全面評估冠狀動脈樹
- 評估血管外結構
- 長期追蹤理想工具


### 整合評估策略


| p{11cm}}  **臨床情境** | **建議評估策略** |  |
| --- | --- | --- |
| 急性冠心症 | 冠狀動脈攝影為主    考慮IVUS/OCT評估culprit lesion    評估remodeling模式指導治療 |  |
| 穩定型心絞痛 | CCTA初步評估    功能性評估（FFR-CT, 運動測試）    必要時侵入性評估+IVUS |  |
| 無症狀但高風險 | CCTA篩檢    評估高危斑塊特徵    積極控制危險因子 |  |
| PCI術前評估 | IVUS/OCT評估病灶    測量血管大小    評估鈣化程度    指導支架選擇 |  |


## 治療策略


### Positive Remodeling的治療


**藥物治療**（斑塊穩定為主）：

- **高強度statin**：Atorvastatin 40-80 mg或Rosuvastatin 20-40 mg
- **抗血小板藥物**：Aspirin + P2Y12抑制劑（如Ticagrelor或Prasugrel）
- **PCSK9抑制劑**：考慮用於極高危患者（LDL-C目標\textless 55 mg/dL）
- **血壓控制**：ACEI或ARB
- **血糖控制**：糖尿病患者積極控制（GLP-1 RA或SGLT2i優先）
- **生活方式**：戒菸、飲食控制、規律運動


**介入治療考量**：

- 急性冠心症時需要緊急PCI
- 考慮藥物塗層支架（DES）
- 評估是否需要影像導引（IVUS/OCT-guided PCI）
- 注意防止no-reflow現象
- 考慮使用血栓抽吸或遠端保護裝置


對於positive remodeling的易損斑塊，即使管腔狹窄不嚴重，如果有高危特徵（大脂質核心、薄纖維帽、高炎症活性），仍應積極藥物治療以穩定斑塊，預防急性事件。


### Negative Remodeling的治療


**藥物治療**（症狀控制為主）：

- **抗心絞痛藥物**：Beta-blocker、鈣離子阻斷劑、長效硝酸鹽
- **Statin治療**：中至高強度
- **抗血小板藥物**：通常Aspirin即可
- **危險因子控制**：血壓、血糖、血脂


**介入治療考量**：

- 症狀導向的選擇性PCI
- 功能性評估（FFR \textless 0.80或iFR \textless 0.89）
- 鈣化病灶可能需要特殊處理：     itemize
- Rotational atherectomy
- Cutting balloon
- Intravascular lithotripsy（IVL）


    \item 支架擴張通常較佳
    \item 併發症風險相對較低
itemize


### 長期追蹤策略


| p{11cm}}  **Remodeling類型** | **追蹤建議** |  |
| --- | --- | --- |
| Positive Remodeling | 密集追蹤（3-6個月）    嚴格控制LDL-C（目標\textless 55 mg/dL）    考慮CCTA評估斑塊進展    評估其他血管的高危斑塊    持續雙重抗血小板治療（視情況延長） |  |
| Negative Remodeling | 常規追蹤（6-12個月）    症狀評估為主    必要時功能性評估    優化藥物治療    運動復健計畫 |  |


## 臨床案例分析


### 案例一：Positive Remodeling導致的急性心肌梗塞


**病史**：52歲男性，吸菸史30年，輕度高血脂，無糖尿病。平時無明顯症狀，某日晨起時突發胸痛、冒冷汗。

**檢查發現**：

- ECG：ST elevation in leads V2-V4
- Troponin I：明顯升高（25 ng/mL）
- 緊急冠狀動脈攝影：LAD近端50
- IVUS發現：明顯positive remodeling（RI = 1.3），大脂質核心，薄纖維帽已破裂，血栓形成


**治療**：

- 緊急PCI with DES
- 雙重抗血小板治療（Aspirin + Ticagrelor）
- 高強度statin（Atorvastatin 80 mg）
- 後續加上PCSK9抑制劑
- 戒菸輔導


**學習重點**：

- Positive remodeling的病灶可能在血管攝影上看起來狹窄不嚴重
- 易損斑塊可在無明顯前驅症狀時突然破裂
- IVUS有助於正確診斷和治療決策
- 需要積極的藥物治療預防再發


### 案例二：Negative Remodeling的穩定型心絞痛


**病史**：68歲女性，糖尿病、高血壓病史10年。近半年來運動時胸悶，休息後緩解。症狀逐漸加重。

**檢查發現**：

- 運動心電圖：陽性（ST depression in leads II, III, aVF）
- CCTA：RCA近端70
- 冠狀動脈攝影確認：RCA 75
- FFR測量：0.75（有功能性意義）


**治療**：

- Rotational atherectomy輔助PCI
- DES植入
- Aspirin長期使用
- Statin、ACEI、Beta-blocker
- 糖尿病控制優化


**學習重點**：

- Negative remodeling通常表現為穩定型心絞痛
- 功能性評估有助於決定是否需要PCI
- 鈣化病灶可能需要特殊技術處理
- 預後相對良好，症狀可有效控制


## 最新研究進展


### 分子機制研究


近年來對remodeling的分子機制有更深入的了解：

**Positive Remodeling相關因子**：

- **MMPs/TIMPs失衡**：MMP-2、MMP-9表達增加，TIMP-1、TIMP-2減少
- **炎症因子**：IL-1β、IL-6、TNF-α升高
- **氧化壓力**：ROS生成增加，抗氧化能力下降
- **血管新生**：VEGF表達增加，脆弱的新生血管形成


**Negative Remodeling相關因子**：

- **TGF-β路徑**：促進纖維化和膠原沉積
- **PDGF**：促進平滑肌細胞增殖和遷移
- **骨形成蛋白**：促進血管鈣化


### 生物標記


研究發現某些生物標記與remodeling類型相關：


| p{5cm}p{5.5cm}}  **標記** | **Positive Remodeling** | **Negative Remodeling** |
| --- | --- | --- |
| 高敏感C反應蛋白 | 顯著升高 | 輕微升高或正常 |
| MMP-9 | 升高 | 正常或降低 |
| TIMP-1 | 降低 | 升高 |
| Osteopontin | 升高 | 升高（但機制不同） |
| Troponin | 可能升高（微小損傷） | 通常正常 |


### 影像學新技術


**近紅外光譜（NIRS）**：

- 檢測脂質核心
- 可與IVUS結合（NIRS-IVUS）
- 預測高危斑塊


**血管內光聲成像（IVPA）**：

- 同時提供結構和組成資訊
- 偵測脂質和血紅素
- 仍在研發階段


**人工智慧應用**：

- 自動分析CCTA影像
- 預測斑塊進展
- 風險分層


## 總結與臨床建議


importantbox
**關鍵要點**


1. **Positive remodeling**代表血管代償性擴張，常伴隨易損斑塊，急性事件風險高
2. **Negative remodeling**代表血管收縮，通常是穩定纖維斑塊，症狀較明顯但急性風險較低
3. 血管攝影可能低估positive remodeling的危險性，腔內影像（IVUS/OCT）有助於準確評估
4. 治療策略應根據remodeling類型個別化：positive remodeling重視斑塊穩定，negative remodeling重視症狀控制
5. 長期追蹤和嚴格控制危險因子對兩種類型都很重要

### 臨床實務建議


| p{10.5cm}}  **情境** | **建議** |  |
| --- | --- | --- |
| 急性冠心症 | 考慮IVUS評估culprit lesion的remodeling模式    評估其他非culprit lesion的高危特徵    積極藥物治療穩定所有易損斑塊 |  |
| 中度狹窄病灶 | 考慮腔內影像評估是否有positive remodeling    不要僅憑血管攝影低估危險性    Positive remodeling伴高危特徵應積極治療 |  |
| 嚴重鈣化病灶 | 通常提示negative remodeling    評估功能性意義（FFR/iFR）    準備適當的鈣化處理策略 |  |
| 無症狀高危患者 | CCTA篩檢高危斑塊特徵    評估positive remodeling和其他高危特徵    積極一級預防 |  |


### 未來展望


- **精準醫療**：基於基因、生物標記和影像學的個別化風險評估

- **影像技術**：更高解析度、非侵入性的斑塊評估工具
