## 前言


超音波導引穿刺是一項團隊合作的醫療技術。護理同仁在此過程中扮演關鍵角色，協助備物、維持無菌環境、監測病患狀況，能大幅提升穿刺成功率與病患安全。本手冊旨在幫助護理及其他醫療同仁了解超音波導引穿刺的基本概念及配合準備事項。

importantbox
**目標：**護理師兩人一組，就可以幫忙備好探頭。探頭準備時間目標：$<$ 60 秒。
importantbox


## 什麼是超音波導引穿刺？


超音波導引穿刺是利用超音波即時影像，引導醫師將針準確地插入目標血管（如靜脈或動脈）的技術。相較於傳統盲穿法，此技術能：


1. 視覺化血管位置與走向
2. 即時追蹤針尖位置
3. 避免誤穿動脈或其他組織
4. 降低併發症發生率
5. 提高穿刺成功率

## 超音波探頭準備步驟


探頭準備是護理同仁最重要的配合項目之一。建議由兩人一組合作完成，可大幅縮短準備時間。


### 方法一：無菌套包覆法


步驟 1：準備凝膠{
在探頭發射面上塗抹適量超音波凝膠。
}

步驟 2：套入無菌套{
由另一位同仁將無菌套打開，操作者將探頭伸入無菌套中。
}

步驟 3：排除空氣{
將探頭推至無菌套底部，確保探頭發射面與無菌套緊密貼合，排除中間空氣。
}

步驟 4：固定無菌套{
將無菌套多餘部分反摺，使用橡皮筋固定於探頭手把處。
}


### 方法二：OP site 輔助法


步驟 1：準備凝膠{
在探頭發射面上塗抹適量超音波凝膠。
}

步驟 2：覆蓋 OP site{
使用 OP site（透明敷料）將探頭發射面包覆，確保與探頭緊密貼合。
}

步驟 3：套入無菌套{
再將整個探頭套入無菌套中。
}

步驟 4：固定{
確認包覆完整，無菌套固定妥當。
}


### 不同環境的準備方式


| p{0.25\textwidth}>{\arraybackslash}p{0.65\textwidth}}  **環境** | **建議準備方式** |  |  |  |  |
| --- | --- | --- | --- | --- | --- |

## 介質選擇注意事項


importantbox
**重要提醒：**

用抽好 local anesthesia 的 syringe 往皮膚噴灑，產生的液面就足以當作介質（jelly）。不一定要用凝膠，但若超音波視窗過於模糊，可考慮使用凝膠。

**注意：**當針穿刺時，凝膠可能造成針具阻塞，降低針具回血的能力。
importantbox


## 皮膚準備注意事項


1. 穿刺部位皮膚若消毒完全，可考慮不覆蓋 OP site
2. 若需要覆蓋 OP site，**一定要將空氣趕出**並且盡可能密合皮膚
3. 若 OP site 與皮膚間有空氣，會讓超音波穿透更差，影像品質下降


沒有好的超音波影像，即時超音波導引穿刺的危險性會大幅上升。


## 術中護理配合事項


### 術前


1. 確認病人身份及預定穿刺部位
2. 確認知情同意已完成
3. 協助醫師確認術前實驗室檢查（尤其是凝血功能）
4. 確認術前相關用藥（如抗凝血劑）
5. 備齊所有所需器材
6. 協助病人擺位
7. 準備好超音波探頭


### 術中


1. 維持無菌區域
2. 協助遞送器材
3. 若需要，協助維持負壓（使用 T-型延長管接 syringe）
4. 持續監測病人生命徵象
5. 觀察病人反應及主訴


### 術後


1. 協助固定導管
2. 確認導管功能（回抽血液、沖洗通暢）
3. 記錄穿刺過程及導管資訊
4. 清潔超音波探頭
5. 整理用物

## 需要立即通知醫師的情況


importantbox
遇到以下情況時，請**立即通知醫師**：

1. 穿刺時間過久
2. 回抽血時看到空氣
3. 皮下血腫快速擴展
4. 病患生命徵象轉趨不穩
5. 病患主訴劇烈疼痛或呼吸困難

## 常見問題 Q&A


**Q1: 為什麼一定要用 linear probe？**

A: Linear probe 能提供高解析度的淺層組織影像，對於直徑較小的血管（如頸靜脈、橈動脈等）能清楚顯示，是血管穿刺的最佳選擇。其他探頭如 curvilinear probe 適用於深部組織，不適合血管穿刺。

**Q2: 探頭準備需要多久？**

A: 目標是 60 秒內完成。經過練習，熟練的團隊可以在 18 秒內完成探頭準備。

**Q3: 可以不用凝膠嗎？**

A: 可以。用 local anesthesia 噴灑在皮膚上產生的液面即可作為介質。但若影像品質不佳，仍可使用凝膠。需注意凝膠可能造成針具阻塞。

**Q4: 護理師需要會操作超音波嗎？**

A: 護理師主要負責協助準備探頭、備物、維持無菌環境、監測病患。不需要會操作超音波進行穿刺，但了解基本概念有助於更好地配合醫師。


## 總結


護理同仁在超音波導引穿刺過程中的角色至關重要。透過妥善的術前準備、術中配合與術後照護，能有效提升穿刺成功率並確保病患安全。


1. **熟悉器材**：了解各項器材的用途與放置位置
2. **練習探頭準備**：兩人一組練習，目標 60 秒內完成
3. **維持無菌**：這是預防感染的關鍵
4. **持續監測**：術中密切觀察病患狀況
5. **及時溝通**：發現異常立即通知醫師


建議帶團隊試作一次探頭準備，可以找到最適合自己團隊的方法。
