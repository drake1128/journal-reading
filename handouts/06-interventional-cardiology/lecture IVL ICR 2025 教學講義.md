\maketitle


> **重大突破：**血管內震波碎石術（IVL）自2018年引入英國以來，已經革新了鈣化冠狀動脈病變的治療方式。本專家意見文章提出IVL技術的進階應用策略，強調其不僅是球囊血管成形術工具，而是使用球囊進行治療的傳輸系統。
> **研究意義：**通過借鏡周邊血管介入的經驗，特別是球囊過大化（oversizing）策略，本文提出了針對不同鈣化型態（同心性、結節性、偏心性及大型分叉病變）的優化IVL使用方法，旨在提升支架擴張效果，改善長期預後。

## 研究背景與問題


### 鈣化冠狀動脈疾病的治療挑戰


儘管各種技術不斷進步，治療鈣化冠狀動脈疾病仍然是一項重大挑戰。鈣化病變的主要困難在於達到最佳支架擴張效果，當無法達到充分擴張時，往往會出現次佳的治療結果。


- **鈣化盛行率被低估：**藥物洗脫支架研究的核心實驗室分析顯示，血管攝影/視覺評估經常低估鈣化的存在程度
- **實際鈣化比例：**匯總數據顯示近26
- **血管內影像發現：**使用血管內影像檢查發現，24.5
- **偏心性鈣化普遍：**這突顯了偏心性鈣化存在於半數病例中


### IVL技術的引入與發展


2018年引入英國的血管內震波碎石術（IVL）為管理鈣化冠狀動脈疾病的治療選擇增添了新的利器。與去除鈣化技術（如旋磨術、軌道旋磨術和雷射斑塊切除術）相比，IVL的簡便性促進了其廣泛採用。


> **技術原理：**
> enumerate
>     \item IVL發射器產生電火花，在整合球囊內的周圍液體介質中產生蒸汽泡
>     \item 低水平電能導致蒸汽泡的形成和快速膨脹
>     \item 產生聲壓波以非聚焦方式環繞並穿透性地輻射
>     \item 震波以最小效應穿透軟組織，但前緣對鈣化施加壓縮應力，導致鈣化斷裂
>     \item 反射聲波進一步加深並交叉這些斷裂，改變鈣化並改善血管順應性
> enumerate
> **雙重作用：**這種雙重作用確保全面的鈣化改變，使血管更具順應性，以實現最佳支架置入。

### 研究背景與目標


- IVL最初被定位為針對同心性鈣化的有效工具
- 標準建議為球囊充氣至4大氣壓以傳遞治療，然後增加至6大氣壓以評估血管順應性
- 然而這導致某些介入醫師將IVL視為球囊血管成形術的延伸，關注靜水壓力而非將球囊僅視為傳遞治療的輔助工具
- **研究目標：**精煉IVL機制特性的知識，強調其作為使用球囊的治療傳遞系統，而非球囊血管成形術工具


## 當前實務做法及其限制


### 傳統IVL使用的誤解


| p{11cm}}  **誤解項目** | **說明與糾正** |  |
| --- | --- | --- |
| **誤解項目** | **說明與糾正** |  |
| **4大氣壓的意義** | minipage[t]{11cm} **誤解：**4大氣壓是標稱壓力    **正確：**4大氣壓是防止「乾式震波」的最低壓力，確保發射器有液體覆蓋    **重要性：**更高壓力不會增加能量傳遞，依據反平方定律可能減少總體能量傳遞，並增加球囊損壞風險 |  |
| **球囊尺寸選擇** | minipage[t]{11cm} **當前做法：**基於遠端血管直徑選擇球囊尺寸    **問題：**在遠端細小的漸細性長病變或分叉病變中，球囊過小限制了技術效能    **原因：**基於傳統球囊血管成形術的安全性和可送達性考量 |  |
| **空氣排除** | minipage[t]{11cm} **關鍵重要性：**徹底排除空氣至關重要    **證據：**研究顯示在IVL球囊內引入空氣會減少能量傳遞約20  **機制：**空氣在超音波界面的存在會降低震波振幅 |  |
| **導管定位** | minipage[t]{11cm} **重要性：**正確定位IVL導管對確保有效鈣化改變至關重要    **發射器位置：**    　• 近端發射器：位於近端放射線不透明標記遠端約2 mm    　• 遠端發射器：位於遠端放射線不透明標記近端4 mm    **重疊策略：**球囊位置間需保持2 mm重疊以避免治療地理性遺漏 |  |


### 球囊尺寸過小的影響


球囊尺寸過小會限制技術效能的原因包括：


1. **能量傳遞範圍受限：**最大聲壓直接垂直於發射器位置
2. **接觸不良：**在結節性鈣化中，球囊可能無法與鈣化基底充分接觸
3. **穿透深度不足：**震波穿透鈣化的深度可能不足
4. **邊緣效應：**球囊邊緣可能無法與病變區域的血管壁接觸

### IVL與其他技術的安全性比較


> **傳統scoring balloon和cutting球囊的風險：**
> itemize
>     \item 在結節性和偏心性鈣化中，特別是在非鈣化區域，存在夾層和穿孔的風險
>     \item 可能造成血管壁創傷
> itemize
> **IVL的安全性：**
> itemize
>     \item 聲波安全地穿越軟組織而不造成損傷
>     \item IVL治療引起的斷裂通常發生在組織界面處
>     \item 增強血管順應性並實現最佳支架擴張
>     \item 與其他基於球囊的技術相比，具有獨特的安全性特徵
> itemize
> **注意事項：**目前尚無IVL與評分/cutting球囊的比較研究

### 大型分叉病變


> **建議方法（基於作者經驗）：**
> enumerate
>     \item **球囊尺寸：**根據主血管直徑選擇球囊
>     \item **定位：**跨越分叉放置球囊
>     \item **充氣壓力：**在2大氣壓下傳遞治療
>     \item **優勢：**同時改變主血管和分支血管以及分支開口處的鈣化
>     \item **後續效果：**促進分支通路和分支開口處的病變準備
> enumerate
> **操作技巧：**
> itemize
>     \item 確保分叉處的充分重疊和治療傳遞
>     \item 改善支架置入並減少併發症
>     \item 如果兩個分支都需要鈣化改變，建議首先在角度較大的分支使用IVL導管
>     \item 通過更全面的病變準備，避免支架擴張不足的風險
> itemize
> **研究需求：**需要前瞻性IVL分叉研究提供安全性和臨床有效性的證據

「枕狀效應」（pillowing effect）是作者提出的術語，用於描述以下假說：


| p{10cm}}  **策略元素** | **技術細節與理論基礎** |  |
| --- | --- | --- |
| **策略元素** | **技術細節與理論基礎** |  |
| **球囊過大化** | minipage[t]{10cm} 　• 過大化0.5 mm（10  　• 在2大氣壓下充氣    　• 創造「枕狀效應」 |  |

| **結節性鈣化應用** | minipage[t]{10cm} **確保改變所有鈣化結節面向：**    　• 上升和下降邊緣    　• 結節頂點    　• 邊緣至相鄰鈣化板的過渡區域    **效果：**實現最佳病變準備 |  |


#### 操作技術要點


1. **避免傳統方法：**不要將IVL球囊中間放在結節性鈣化上，以1:1球囊尺寸充氣至4大氣壓
2. **傳統方法問題：**      - 可能減少發射器與結節基底的對準 - 可能導致球囊邊緣與感興趣區域的血管壁接觸減少
3. **治療週期數：**儘管證據顯示治療週期數與更多鈣化斷裂相關，但偏心性和結節性鈣化是否需要比同心性鈣化更多的治療週期仍不清楚

### 球囊壓力的物理原理


> **關鍵物理概念：**
> enumerate
>     \item **反平方定律：**能量隨距離的平方而減少
>     \item **壓力與能量：**更高壓力不等於更高能量傳遞
>     \item **最佳壓力：**2-4大氣壓可能更有效於能量傳遞
>     \item **球囊功能：**球囊的主要作用是確保發射器有液體界面覆蓋
>     \item **乾式震波風險：**防止「乾式震波」可能導致球囊破裂等併發症
> enumerate
> **臨床應用：**
> itemize
>     \item 最小充氣壓力足以確保發射器的液體覆蓋即可
>     \item 4大氣壓不是標稱壓力，而是防止乾式震波的方法
>     \item 過高壓力可能增加球囊損壞風險
> itemize

## 結論與臨床建議


### 主要結論


1. ****IVL理解演進：****自2018年引入以來，IVL技術的理解和使用已經演進
2. ****去除空氣的重要性：****強調徹底去除空氣和認識到4大氣壓不是標稱壓力，而是防止乾式震波的機制，可以優化治療傳遞
3. ****過大化策略：****球囊過大化0.5 mm並使用較低壓力（2大氣壓）可以增強療效，特別是在結節性、偏心性和分叉病變中
4. ****安全性優勢：****鑑於IVL在軟組織中的安全性，稍微過大球囊尺寸可以確保全面的鈣化改變
5. ****術語修正：****將技術重新命名為「IVL導管」而非「IVL球囊」可能糾正誤解

### 對介入心臟病學的影響


這種對IVL的細緻理解，加上周邊血管研究的經驗教訓，將技術定位為處理複雜鈣化冠狀動脈疾病全譜的多功能工具，包括同心性環狀、結節性、偏心性病變和板狀鈣化。未來的研究和真實世界數據可能會進一步精煉這些方法，鞏固IVL在介入心臟病學中的角色。


> **記住關鍵概念：**
> enumerate
>     \item IVL是治療傳遞系統，不是球囊血管成形術工具
>     \item 球囊的作用是確保發射器有液體覆蓋，而非施加靜水壓力
>     \item 稍微過大化可能更有效，因為聲波安全穿越軟組織
>     \item 低壓充氣（2-4大氣壓）可能比高壓更有效
>     \item 影像導引對優化結果至關重要
>     \item 正確的發射器定位和重疊策略防止治療遺漏
> enumerate
