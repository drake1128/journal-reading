## 1. 重點摘要

> **Pearl 1**: 僅依賴透析單位血壓會造成近 **1/3 病人被錯誤分類**，標準化居家血壓監測可提升診斷精準度

> **Pearl 2**: **Sustained hypertension (SHT)** 和 **Masked hypertension (MHT)** 與最大的心臟及血管負擔相關

> **Pearl 3**: **White-coat phenotype (WCH)** 並非完全良性 — 雖無 LVH 增加，但與較高的動脈硬化相關

> **Pearl 4**: 未治療的 WCH 病人 LVH 盛行率高於未治療的正常血壓病人，提示 WCH 在透析族群需要持續追蹤

## 2. 背景與臨床意義

### 透析血壓測量的挑戰

| 問題 | 說明 |
|------|------|
| 超濾影響 | 透析中急性體液轉移影響血壓讀數 |
| 交感神經活化 | 透析前環境壓力導致暫時性血壓升高 |
| 容積波動 | 透析間期體重增加造成血壓變異 |
| 診斷錯誤 | 單純依賴透析單位血壓可能錯過或高估高血壓 |

### 為何需要居家血壓？

- **Ambulatory BP monitoring (ABPM)** 雖為黃金標準，但在 HD 病人執行困難
- **Home BP monitoring** 為國際指引推薦的可行替代方案
- 可捕捉透析日與非透析日的血壓變化
- 更能反映 **interdialytic BP exposure**

## 3. 研究設計

### 研究族群：Hsinchu Vascular Study

| 項目 | 內容 |
|------|------|
| 設計 | 橫斷面分析（前瞻性世代研究） |
| 時間 | 2022 年起 |
| 地點 | 新竹地區 5 個透析中心 |
| 樣本數 | **217 位**成人，每週 3 次 HD |
| 遵從率 | 居家血壓監測 **93%** |

### 排除條件

- 透析頻率非每週 3 次
- 症狀性心衰竭 (NYHA III-IV)
- 透析後心胸比 > 60%
- 慢性心房顫動
- 近 2 週內乾體重或降壓藥物改變
- 近 1 個月內住院

### 血壓測量方法

**透析單位血壓**
- 標準化訓練的護理師測量
- 每次透析前後各測量 2 次（間隔 1 分鐘取平均）
- 連續 1 週共 3 次透析

**居家血壓 (7-day protocol)**
- 使用經驗證的 Omron HEM-705 CP
- 每日早晚各測量 2 次（間隔 1 分鐘取平均）
- 連續 7 天完整資料

## 4. 血壓表型定義

### 主要分類標準

| 表型 | Predialysis BP | Home BP |
|------|----------------|---------|
| **SNT** (Sustained Normotension) | < 140/90 mmHg | < 135/85 mmHg |
| **WCH** (White-coat Phenotype) | ≥ 140/90 mmHg | < 135/85 mmHg |
| **MHT** (Masked Hypertension) | < 140/90 mmHg | ≥ 135/85 mmHg |
| **SHT** (Sustained Hypertension) | ≥ 140/90 mmHg | ≥ 135/85 mmHg |

                    未治療 (n=91)      已治療 (n=126)
    ────────────────────────────────────────────────
    SNT             27.3%              35.5%
    WCH             47.8%              66.7%

## 6. 臨床意涵

### WCH 在透析族群的特殊意義

| 面向 | 發現 |
|------|------|
| **心臟** | LVH 盛行率與 SNT 相當（整體族群） |
| **血管** | 動脈硬化顯著增加 (OR 2.84) |
| **解讀** | WCH 可能反映血管脆弱性而非純粹的情境反應 |
| **建議** | 不應積極加藥，但需持續監測 |

### 為何透析病人 WCH 較多、MHT 較少？

1. **容積敏感性**：透析前為最大體液累積時間點
2. **環境因素**：透析單位的壓力與交感神經活化
3. **居家 BP 限制**：僅捕捉日間測量，可能遺漏夜間或透析間期血壓波動

### 實務建議

```text
┌────────────────────────────────────────────────────────┐
│  結構化居家血壓監測應納入常規透析照護                  │
├────────────────────────────────────────────────────────┤
│  ✓ 提升診斷精準度                                      │
│  ✓ 改善風險分層                                        │
│  ✓ 指導個人化血壓管理                                  │
│  ✓ 避免僅依據透析單位血壓而過度治療或治療不足          │
└────────────────────────────────────────────────────────┘
```

### 研究限制

- 橫斷面設計，無法推論因果
- MHT 個案數少 (n=15)，信賴區間寬
- 未執行 ABPM
- 單一區域研究，需驗證於其他族群
