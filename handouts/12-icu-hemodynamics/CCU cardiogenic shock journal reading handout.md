## 心因性休克概論


### 定義與病因


心因性休克 (Cardiogenic Shock, CS) 是一種急性心臟衰竭導致的臨床症候群,特徵為心輸出量嚴重下降,導致組織灌流不足與器官功能障礙。


#### 主要分類


description[leftmargin=2cm,style=nextline]
\item[AMI-CS (急性心肌梗塞相關心因性休克)] \hfill

- STEMI (ST段上升型心肌梗塞)
- NSTEMI (非ST段上升型心肌梗塞)


\item[HF-CS (心臟衰竭相關心因性休克)] \hfill

- 急性心肌炎 (Acute myocarditis)
- Takotsubo 心肌病變
- 周產期心肌病變 (Peripartum cardiomyopathy)
- 心搏過速相關心肌病變 (Tachycardia-related cardiomyopathy)
- 肥厚性心肌病變 (Hypertrophic cardiomyopathy)
- 浸潤性疾病 (Infiltrative diseases)


description


心因性休克的死亡率極高,即使在現代醫療下仍達 40-50


### SUSPECT CS 記憶口訣


用於快速辨識心因性休克的臨床表徵:

| --- |
| **S**ymptoms/Signs |
| **U**rine output |

importantbox
**注意「正常血壓休克」(Normotensive Shock)!**

並非所有心因性休克患者都會表現低血壓。患者可能因周邊血管阻力代償性增加,血壓維持在正常範圍,但仍有組織灌流不足的表現。臨床醫師必須警覺以下徵象:

- Lactate 升高 ($\geq$2.5 mmol/L)
- 尿量減少
- 意識改變
- 四肢冰冷
- 代謝性酸中毒

## 黃金一小時 (Golden One Hour)


### 概念與重要性


心因性休克的處理強調「黃金一小時」概念,在急診或診斷後的 60 分鐘內完成四大任務:


1. **辨認及評估** (Identification and Evaluation)
2. **病因及分級** (Etiology and Staging)
3. **支持及穩定** (Support and Stabilization)
4. **計劃與啟動** (Planning and Activation)


### 時間軸與處理流程


| p{2.5cm}>{\arraybackslash}p{5cm}>{\arraybackslash}p{7cm}}  **時間點** |
| --- |
| Time zero |
| 30 min |
| 30-60 min |
| 60 min |


在黃金一小時內,最關鍵的是:  

1. **早期辨識**心因性休克,不要被「正常」血壓誤導  

2. **SCAI 分期**,決定處置強度  

3. **Shock team 啟動**,多專科團隊合作  

4. **及早考慮轉院**至有能力處理的中心

### 三大核心策略


在 24 小時內,治療策略可歸納為三大重點:


1. **最佳化 (Optimization):** 達成去鬱血、代謝與血行動力學目標
2. **個別化 (Personalization):** 根據病因進行特定介入 (如 PCI, 瓣膜手術, 電燒等)
3. **規劃下一步 (Exit Strategy):** 心臟恢復 vs. 心臟移植/長期 VAD vs. 緩和醫療


## SCAI 分期系統


### SCAI Shock Stage 定義


SCAI (Society for Cardiovascular Angiography and Interventions) Shock Stage 是目前最廣為使用的心因性休克分期系統,將患者分為 A 至 E 五個階段:


| p{1.5cm}p{4cm}p{9cm}}  **Stage** | **描述** | **臨床特徵** |  |  |  |  |
| --- | --- | --- | --- | --- | --- | --- |
| **A** | At risk    (有風險但目前無休克症狀) | **Physical exam:**    JVP 正常,Perfusion 溫暖且良好,Distal pulses 強,Mentation 正常,Lung sounds 清澈    **Biochemical:**    Lactate 正常,Renal function 正常 (或基礎值)    **Hemodynamics:**    SBP $>$100 mmHg (或基礎值),CI $\geq$2.5 (若急性),CVP $\leq$10,PCWP $<$15,PA Sat $\geq$65 |  |  |  |  |
| **B** | Beginning cardiogenic shock    (血行動力學不穩定或有低灌流的臨床證據) | **Physical exam:**    JVP 升高,Perfusion 溫暖且良好,Distal pulses 強,Mentation 正常,Lung sounds 囉音    **Biochemical:**    Lactate 正常,Renal function 輕微急性腎損傷,BNP 升高    **Hemodynamics:**    SBP $<$90 mmHg 或 MAP $<$60 mmHg,或從基礎值下降 $>$30 mmHg,HR $\geq$100 bpm |  |  |  |  |
| **C** | Classic cardiogenic shock    (需要藥物或機械性介入以改善灌流,超過容量再灌注) | **Physical exam:**    JVP 升高,Mental status 改變,Distal pulses 冰冷且濕黏,Lung sounds 囉音廣泛,Urine output $<$30 mL/h    **Biochemical:**    Lactate $\geq$2,Cr 1.5倍基礎值或 GFR $>$50  **Hemodynamics:**    SBP $<$90 mmHg 或 MAP $<$60 mmHg,或從基礎值下降 $>$30 mmHg,CI $<$2.2 (若執行侵入性血行動力學監測,[強烈建議]),PCWP $>$15 |  |  |  |  |
| **D** | Deteriorating    (初始支持策略無法恢復灌流) | **Physical exam:**    Stage C 的任何表現且惡化 (或未改善)    儘管初始治療,仍有低灌流徵象    **Biochemical:**    Stage C 的任何表現    Lactate 升高且持續 $\geq$2    Renal function 惡化    LFT 惡化    BNP 升高    **Hemodynamics:**    Stage C 的任何表現且需要升階劑量的 pressors 或增加 tMCS devices 數量以維持灌流 |  |  |  |  |
| **E** | Extremis    (急性或即將發生的循環衰竭) | **Mental status:** 意識不清    **Pulse:** 接近無脈搏    **Cardiac collapse**    **Multiple defibrillations**    **Biochemical:**    Lactate $\geq$8,CPR,Acidosis 嚴重 (pH $<$7.2)    **Hemodynamics:**    Profound hypotension 儘管給予最大的血行動力學支持    需要 bolus doses of vasopressors |  |  |  |  |


**實務應用重點:**  

1. SCAI stage 可幫助決定治療強度:Stage B 可能只需藥物,Stage C-D 需考慮 tMCS,Stage E 需要最積極的支持  

2. SCAI stage 是動態的,應每 6 小時重新評估  

3. Stage 的**惡化或改善趨勢**比單次測量更重要  

4. 不要單純依賴血壓,要整合所有臨床、生化與血行動力學指標


### 簡化版 SCAI 分期判斷


為方便臨床使用,可用以下簡化標準快速判斷:


- **SCAI C:** Lactate 2.5-5.0, SBP $>$60, MAP $>$50
- **SCAI D:** Lactate 5.0-10, SBP $>$60, MAP $>$50
- **SCAI E:** Lactate $>$10 或 SBP $<$60 或 MAP $<$50


## 多專科 Shock Team 啟動


### 團隊組成


研究顯示,多專科 shock team 的啟動與以下結果相關:

- 更多使用侵入性血行動力學監測
- 更多使用進階型 tMCS
- **降低死亡風險**


**標準 Shock Team 成員應包含:**


1. Critical care cardiology (重症心臟專科)
2. Advanced HF and transplant cardiology (進階心衰與移植專科)
3. Interventional cardiology (介入性心臟專科)
4. Cardiac surgery (心臟外科)
5. ECMO intensivists 或 perfusionist (ECMO 專家或體外循環師)


### Shock Team 啟動時機


應在以下情況立即啟動 shock team:


- 診斷為心因性休克 (符合 SUSPECT CS 標準)
- SCAI Stage C 或以上
- Stage B 但對初始治療反應不佳
- 需要考慮 tMCS 或進階支持
- 需要轉院至 shock center


## 侵入性血行動力學監測


### Pulmonary Artery Catheter 的角色


儘管**沒有隨機對照試驗證據**,PA catheter 在心因性休克中仍被**強烈建議**使用。


#### PA Catheter 的三大功能


1. **Diagnostic tool (診斷工具):** 測量 CI, CPO, PAPI
2. **Therapeutic guidance (治療指引):** 區分 LV, RV, BiV failure,調整治療策略
3. **Prognosticator (預後評估):** RAP, RAP/PCWP, PAPI 等指標可預測預後


### 重要血行動力學參數


| p{3.5cm}p{4cm}p{7cm}}  **參數** |
| --- |
| Cardiac Power Output (CPO) |
| Pulmonary Artery Pulsatility Index (PAPI) |


**臨床應用建議:**  

1. 對 SCAI C 以上或血行動力學不明確的患者,強烈建議放置 PA catheter  

2. 初始測量後,應持續或每 2-8 小時重新測量,評估治療反應  

3. CPO 與 PAPI 可快速計算,應成為常規評估項目  

4. PA catheter 也可指引容量管理與 tMCS 的調整


## 藥物治療


### 血管活性藥物選擇


importantbox
**核心原則: The Lowest Dose and The Shortest Duration**  

Inotropes/vasopressors 應使用**最低有效劑量**與**最短時間**,以減少心律不整、心肌氧耗增加與死亡風險。

### Norepinephrine 作為首選


根據 SOAP II trial 與 DoReMi trial,**Norepinephrine 是合理的第一線選擇**:


- 比 dopamine 有更低的心律不整風險
- 提供適度的 inotropic support 與較強的 vasopressor effect
- 適用於大多數低血壓伴低心輸出量的情況


### 藥物選擇階梯式策略


1. **第一線:** Norepinephrine (0-0.3 $\mu$g/kg/min) 或 Dobutamine    若 dobutamine 劑量 $>$10 $\mu$g/kg/min,改為 milrinone (不給 bolus; 0-0.5 $\mu$g/kg/min)
2. **若需更多 vasopressor (low MAP):**    加上 epinephrine (0.01-0.5 $\mu$g/kg/min) 或 vasopressin (0-0.04 IU/min)
3. **若需增加 flow (low CI) 或 inotropic therapy $>$48 hours:**    Milrinone 可改為 levosimendan (0.1-0.2 $\mu$g/kg/min) without prior bolus,並視需要補充 vasopressor
4. **若以上方式仍未達治療目標:**    應評估升階治療 (escalation of therapy),考慮 tMCS


**Dopamine 使用注意:**  

Dopamine 在歷史上常用,但現代證據顯示其心律不整風險較高,除非特定情況(如嚴重心搏過慢),一般**不建議**作為第一線。

**2025 ACS Guideline 更新:**  

**Class 2a, Level B-R:** In **selected** patients with STEMI and severe or refractory cardiogenic shock, insertion of a **microaxial intravascular flow pump** (Impella CP) is reasonable to reduce death.  

  

**Class 3: Harm, Level B-R:** Routine PCI of a noninfarct-related artery at the time of PPCI should NOT be performed because of higher risk of death or renal failure.  

**Class 3: No benefit, Level B-R:** Routine use of IABP or VA-ECMO in AMI-CS is NOT recommended due to lack of survival benefit.


### DanGer Shock Trial 重點摘要


#### 試驗設計


- **假說:** 在 STEMI 相關心因性休克,Impella CP 加上標準治療比單獨標準治療有更低的死亡率
- **時間:** 2013-2021
- **地點:** 丹麥、德國、英國 (多中心隨機對照試驗)
- **收案數:** 360 位患者 (Impella CP + 標準治療組 179 人,標準治療組 176 人)


#### 納入條件


**A: STEMI and cardiogenic shock**

1. STEMI $<$36 hours
2. Cardiogenic shock $<$24 hours,confirmed by:  - Peripheral signs of tissue hypoperfusion (lactate $\geq$2.5 mmol/L and/or SvO$_2$ $<$55 - SBP $<$100 mmHg and/or need for vasopressor therapy
3. LVEF $<$45


**B: STEMI equivalent and cardiogenic shock**

1. STEMI equivalent $<$36 hours and emergency angiography demonstrating acute proximal occlusion of coronary
2. Cardiogenic shock $<$24 hours (同上述定義)
3. LVEF $<$45


**C: STEMI, where shock develops up to 12 hours after revascularization**

- STEMI $<$36 hours duration treated with primary PCI that develop cardiogenic shock within 12 hours of intervention


#### 排除條件 (重要)


1. Shock duration $>$24 hours
2. Other causes of shock
3. **Shock due to mechanical complication to myocardial infarction**
4. **OHCA with persistent GCS $<$8 after ROSC**
5. **Evidence of severe right ventricular failure by echocardiography**
6. Severe aortic valve regurgitation/stenosis
7. Severe peripheral arterial obstructive disease that would preclude Impella device placement
8. Abnormalities of the aorta that would preclude Impella device placement
9. Presence of a mechanical aortic valve prosthesis
10. Left ventricular thrombus
11. Infective endocarditis
12. Life expectancy of less than 1 year due to comorbidities
13. Mental disorder or language barrier that precludes informed consent


#### Impella CP 使用方式


- **Randomization 在心導管室進行**
- **Impella 組:** Impella 放置 $\rightarrow$ 再灌流
- **標準治療組:** 再灌流
- Impella CP full support (P8) for at least 48 hours (除非併發症需提早移除)
- 依 algorithm 進行 weaning
- Impella CP 支持最多 5 天
- 若仍無法脫離,考慮心臟移植、長期 VAD 或緩和醫療


#### 標準治療組 ICU 管理


**Cardiovascular Treatment Goals:**

- Aim for sinus rhythm, 70-110/min
- MAP $>$65 mmHg (possibly higher in patients with hypertension)
- CI $>$2.0 L/min/m$^2$
- SvO$_2$ $>$55
- Lactate $<$2.5 mmol/L


**Inotrope/Vasopressor 階梯:**

1. Norepinephrine (0-0.3 $\mu$g/kg/min) / Dobutamine    若 dobutamine $>$10 $\mu$g/kg/min,改為 milrinone (0-0.5 $\mu$g/kg/min)
2. 需要更多 vasopressor $\rightarrow$ 加 epinephrine (0.01-0.5) 或 vasopressin (0-0.04 IU/min)
3. 需要增加 flow 或 inotropic therapy $>$48 hours $\rightarrow$ Milrinone 可改為 levosimendan (0.1-0.2 $\mu$g/kg/min) without bolus
4. 若仍未達目標 $\rightarrow$ 評估升階治療


**Renal:**

- Furosemide infusion 5-40 mg/h iv if diuresis $<$1 mL/kg/h
- CVP 目標 10-12 mmHg
- CRRT 適應症: Hyperkalemia ($>$6.5 mmol/L), treatment-resistant metabolic acidosis (pH $<$7.30), volume overload (pulmonary edema) or U/O $<$50 mL for 6 consecutive hours
- Loop diuretics 可補充 metolazone 2.5-10 mg/day PO


**Nutrition:**

- Enteral nutrition (EN) 在到達 ICU 後 6 小時內開始
- Hemodynamically unstable: EN 10-20 mL/h
- Hemodynamically stable: Individual diet calculation
- 若第一週內無法達到足夠的 enteral nutrition,可考慮 TPN


**Infection:**

- No prophylactic antibiotic treatment is given

使用 Impella CP 顯著增加以下併發症:  

1. **Moderate to severe bleeding** (21.8

2. **Limb ischemia** (5.6

3. **需要 renal replacement therapy** (41.9

因此,使用 Impella 需要:  

- 仔細評估周邊血管條件  

- 密切監測出血與肢體缺血  

- 積極的感染監控  

- 腎功能追蹤與早期 RRT 考量


#### 長期追蹤結果 (10 年)


**Death from any cause:**

- **Impella 組:** 94/179 (52.5
- **標準治療組:** 121/176 (68.8
- **Hazard Ratio: 0.70 (95


**Median time to death:**

- **Impella 組:** 577 days
- **標準治療組:** 61 days


10 年追蹤顯示,Impella 組的存活優勢持續存在,且大部分死亡發生在早期。若患者能度過急性期,長期預後相對較好。


#### Subgroup Analysis

- 性別
- 年齡 ($\leq$67 yr vs. $>$67 yr)
- Lactate level ($\leq$4.5 mmol/L vs. $>$4.5 mmol/L)
- Mean arterial pressure ($\leq$63 mmHg vs. $>$63 mmHg)
- LVEF ($\leq$25
- STEMI location (Nonanterior vs. Anterior)
- Number of diseased vessels (1 vs. $\geq$2)
- Year of randomization (2013-2018 vs. 2019-2023)
- **SCAI-CSWG stage (C vs. D or E)**


**所有亞組的 HR 皆指向 Impella 有益,無顯著交互作用 (interaction)。**


## 機械通氣策略


### PEEP 對心因性休克的影響


PEEP (Positive End-Expiratory Pressure) 在心因性休克病患的使用需要謹慎,因為其對左右心室有不同影響:


#### 對右心室的影響


- **$\downarrow$ RV venous return (降低靜脈回流)**
- **$\uparrow$ Pulmonary vascular resistance due to vascular compression (增加肺血管阻力)**
- **$\uparrow$ RV dilation $\rightarrow$ left shift in septum (RV 擴張導致中隔左移)**
- **$\uparrow$ Compensatory increase in systemic vascular resistance (代償性增加全身血管阻力)**
- **$\downarrow$ Hypoxia-mediated pulmonary vasoconstriction (降低缺氧導致的肺血管收縮)**


#### 對左心室的影響


- **$\downarrow$ Preload due to lower RV output (因 RV 輸出降低而降低前負荷)**
- **$\downarrow$ Stroke volume due to interventricular dependence (因心室相互依賴而降低每搏輸出量)**
- **$\downarrow$ LV afterload (降低 LV 後負荷)**
- **$\downarrow$ LV preload and LV dilation (降低 LV 前負荷與擴張)**
- **$\downarrow$ Myocardial oxygen demand (降低心肌氧需求)**
- **$\uparrow$ Pressure gradient from thorax to periphery (增加胸腔至周邊的壓力梯度)**
- **$\uparrow$ Hydrostatic displacement of alveolar edema (增加肺泡水腫的靜水壓移位)**


#### 臨床決策


importantbox
**PEEP 對 CO 的淨效應取決於:**

- RV/LV function
- Preload
- Afterload
- Ventricular interdependence


**實務原則:**

- 在 **RV failure/preload dependence**,moderate to high PEEP (10-15 cmH$_2$O) 可能**降低 RV CO**
- 在 **afterload dependent states** (如 LV failure),moderate to high PEEP (10-15 cmH$_2$O) 可能**改善 CO**


importantbox


### 機械通氣策略


**Potential indications for invasive mechanical ventilation in CS:**

- Severe hypoxia
- High work of breathing
- Clinical instability due to refractory shock
- Airway protection for invasive procedures


**Early intubation to decrease myocardial oxygen consumption**

**Special consideration:** If RV-dominant shock present, use mechanical ventilation with caution only when necessary


#### Initial Ventilator Strategy


- **Mode:** AC (PC-CMV, VC-CMV)
- **FiO$_2$:** Start with 100
- **TV:** Start with 6-8 mL/kg of ideal body weight $\rightarrow$ adjust to pCO$_2$ while maintaining a plateau pressure $<$30 cmH$_2$O and [peak - plateau] $<$15 cmH$_2$O
- **RR:** Start with 10-20 breaths/min $\rightarrow$ adjust to pCO$_2$ while monitoring for auto-PEEP
- **Ensure adequate sedation to maintain synchrony**


**If RV-dominant shock present:** Start PEEP 3-5 cmH$_2$O and use the lowest possible to achieve O$_2$Sat $\geq$92


#### Adjust PEEP


- **Ensure adequate volume status**
- If PCWP 12-15 mmHg, start PEEP at 5-10 cmH$_2$O
- If PCWP elevated ($>$15 mmHg), PEEP can be increased to 8-10 cmH$_2$O with monitoring of cardiac output
- Adjust to achieve O$_2$Sat $\geq$92


## 重症照護管理


### 連續重新評估


importantbox
**Critical Care Management 五大原則:**

1. **反覆評估支持足夠度**
2. **治療可逆病因**
3. **降低併發症機率**
4. **降階治療 ASAP**
5. **跨團隊討論治療模式及目標**

## 重點整理


### Take Home Messages


1. **Golden Hour Concept is Critical**    在 60 分鐘內完成初始評估、SCAI 分期、shock team 啟動    早期辨識可防止惡化 - 注意「正常血壓休克」
2. **Multidisciplinary Shock Team Activation Improves Outcomes**    組成團隊包括: cardiology, CCU, interventional, cardiac surgery, perfusionist    使用侵入性血行動力學 (PA catheter) 進行診斷與治療指引 (CPO, PAPI)
3. **DanGer Shock Trial: Impella CP Improves Survival in AMI-CS**    180 天死亡率絕對風險降低 26  但增加出血併發症與 RRT 需求
4. **Medical Management Principles**    Norepinephrine 是首選 vasopressor    使用最低劑量/最短時間的 inotropes    小心優化 PEEP (可能惡化 RV 功能但幫助 LV)    每 6 小時連續重新評估,決定升階/降階
5. **Individualized Strategy**    根據病因 (AMI-CS vs. HF-CS) 與 hemodynamic profile (LV vs. RV vs. BiV) 選擇治療    及早規劃 exit strategy: 心臟恢復 vs. 移植/VAD vs. 緩和醫療
