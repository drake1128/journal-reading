## 人工瓣膜種類


### 概述

人工瓣膜用於置換受損的心臟瓣膜，主要分為機械瓣膜和生物瓣膜。選擇瓣膜類型需考慮患者年齡、抗凝治療需求及瓣膜耐久性。機械瓣膜適合年輕患者，需終身抗凝；生物瓣膜適合老年患者，無需抗凝但耐久性較低。


機械瓣膜故障風險極低，新型號如Medtronic Open Pivot和OnX機械瓣膜性能優異。生物瓣膜含支架和縫合環，可在超音波中辨識。


### 機械瓣膜


- 結構：金屬殼和閉合器
- 類型：球籠式、傾斜盤式、雙葉式
- 需終身抗凝治療
- 高耐久性
- 複合移植（如Bentall手術，含主動脈管移植）

- 結構：含支架或無支架環
- 無需抗凝治療
- 耐久性低於機械瓣膜
- 同種移植（屍體瓣膜）

## 人工瓣膜超音波評估


### 2D超音波評估


- 閉合器/瓣葉運動
- 瓣膜搖晃運動
- 生物瓣膜瓣葉增厚/鈣化
- 瓣環異常（如空洞、假性動脈瘤、血栓/贅生物）


### 都卜勒評估


- 連續波都卜勒測量跨瓣最大和平均壓力梯度
- 彩色都卜勒評估瓣內和瓣周逆流


### 機械瓣膜流型


- 前向血流：依瓣膜類型而異
- 生理逆流：機械瓣膜常見，呈V形噴射


機械瓣膜主動脈位置的瓣葉運動難以評估，需使用非典型視角。經胸超音波（TTE）顯示心室側，經食道超音波（TEE）顯示心房側，必要時結合兩者。

### 壓力恢復與瓣膜-患者不匹配


- **壓力恢復**：都卜勒高估梯度，常见于小型雙葉瓣膜，特別在高流量或小主動脈根 (<30 mm) 時
- **瓣膜-患者不匹配**：有效瓣口面積 (EOA) 不足，導致高跨瓣梯度，影響左心室肥厚消退和預後
- EOA計算：$EOA = Stroke volume{VTI}$，若索引化EOA <0.85 cm$^2$/m$^2$，考慮不匹配


importantbox
**重要提醒：** 主動脈瓣膜最大速度 >3 m/s 或平均梯度 >20 mmHg，提示功能異常；二尖瓣最大速度 $\geq$2 m/s 或平均梯度 $\geq$8 mmHg，需懷疑阻塞。
importantbox


## 人工瓣膜併發症


### 常見併發症


- 瓣周漏
- 瓣膜阻塞（血栓/瘢膜）
- 心內膜炎
- 機械瓣膜故障
- 生物瓣膜退行性變化
- 假性動脈瘤/瘺管


### 瓣膜阻塞


- **超音波徵象**：瓣葉運動受限、閥區回聲增強、異常血流模式、高梯度、延長壓力半時間（二尖瓣）
- **血栓 vs. 瘢膜**：

| --- | --- | --- |
| 抗凝治療範圍內 | 抗凝不足 |  |
| 症狀緩慢 | 症狀突發 |  |
| 瓣膜年限較長 | 腦中風/栓塞 |  |
| 梯度穩定 | 梯度變化 |  |

### 逆流


- **生理逆流**：機械瓣膜常見
- **病理性逆流**：瓣周逆流（6-32
- 瓣周逆流風險因素：鈣化瓣環、心內膜炎、縫合技術

### 其他併發症


- **瓣膜脫落**：搖晃運動
- **醫源性室中隔缺損**：罕見
- **三尖瓣逆流**：左心瓣膜手術後常見，與肺動脈高壓或三尖瓣環擴張相關
- **假性動脈瘤**：常由心內膜炎引起，呈搏動性空洞，收縮期流入、舒張期流出


二尖瓣高梯度時，測量壓力半時間；若延長，提示阻塞；若正常，考慮逆流或高流量狀態。假性動脈瘤需警惕心內膜炎病史。


## 二尖瓣修復


### 概述

二尖瓣修復通常涉及瓣環成形術（環植入），以防止瓣環擴張。修復技術包括四邊形/三角形切除，伴或不伴滑動成形術。


### 超音波特徵


- 瓣環成形環：類似瓣環鈣化
- 後葉可能較短
- 人工腱索可見


### 併發症


- 殞餘逆流
- 左心室流入道阻塞（環過小）
- 環脫落（逆流起源於環外）


importantbox
**重要提醒：** 二尖瓣修復後需測量平均梯度和壓力半時間，環過小可能導致二尖瓣狹窄。修復失敗患者預後較差。
importantbox

1. 索引化EOA <0.85 cm$^2$/m$^2$

4. 壓力半時間 >130 ms

1. 抗凝治療範圍
2. 症狀起病速度
3. 梯度變化
4. 瓣膜類型

1. 高流量狀態

4. 無殞餘逆流

1. 無搏動性空洞
2. 收縮期流入、舒張期流出

4. 正常瓣葉運動

### 考題詳解


1. **正確答案：c) 含金屬閉合器**      **詳解：** 機械瓣膜由金屬殼和閉合器組成，需抗凝治療，耐久性高，適用於年輕患者。
2. **正確答案：b) $14.4 \pm 5.7$ mmHg**      **詳解：** Carpentier-Edwards Perimount主動脈生物瓣膜的平均梯度為$14.4 \pm 5.7$ mmHg，見參考值表。
3. **正確答案：b) 平均梯度 >20 mmHg**      **詳解：** 主動脈瓣膜平均梯度 >20 mmHg或最大速度 >3 m/s提示功能異常。都卜勒速度指數 <0.3更敏感。
4. **正確答案：c) 正常壓力半時間**      **詳解：** 二尖瓣阻塞表現為瓣葉運動受限、高梯度、延長壓力半時間 (>130 ms) 和異常血流。正常壓力半時間提示無阻塞。
5. **正確答案：a) 索引化EOA <0.85 cm$^2$/m$^2$**      **詳解：** 瓣膜-患者不匹配定義為索引化EOA <0.85 cm$^2$/m$^2，導致高梯度和不良預後。
6. **正確答案：d) 瓣膜類型**      **詳解：** 血栓與瘢膜鑑別基於抗凝狀態、症狀起病、梯度變化和栓塞史，與瓣膜類型無關。
7. **正確答案：b) 鈣化瓣環**      **詳解：** 瓣周逆流風險因素包括鈣化瓣環、心內膜炎和縫合技術問題，非正常縫合或無心內膜炎。
8. **正確答案：b) 環脫落**      **詳解：** 二尖瓣修復併發症包括殞餘逆流、環脫落和左心室流入道阻塞（環過小）。正常壓力半時間非併發症。
9. **正確答案：b) 收縮期流入、舒張期流出**      **詳解：** 假性動脈瘤呈搏動性空洞，收縮期流入、舒張期流出，常與心內膜炎相關。
10. **正確答案：c) 生理逆流**      **詳解：** 機械瓣膜常有生理逆流，呈V形噴射。瓣內和瓣周逆流為病理性，瓣葉鈣化見於生物瓣膜。
