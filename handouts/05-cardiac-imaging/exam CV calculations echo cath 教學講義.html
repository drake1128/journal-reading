Fick 公式- 心輸出量（Cardiac Output, CO） 患者氧耗（VO2）為250 mL/min，動脈氧含量（CaO2）為20 mL/dL，混合靜脈氧含 量（CvO2）為15 mL/dL。請計算心輸出量（L/min）。 公式： CO = VO2 CaO2 −CvO2 提示：注意單位轉換，氧含量單位為mL/dL，需轉為mL/L。 2. Qp/Qs - Atrial Septal Defect (ASD) A patient presents with an atrial septal defect (ASD). The following Doppler echocardio- graphy measurements are obtained: • Pulmonary artery (PA): Velocity Time Integral (VTIPA) = 22cm, diameter = 2.4cm • Left ventricular outflow tract (LVOT): Velocity Time Integral (VTILVOT) = 18cm, diameter = 2.0cm Calculate the Qp/Qs ratio. Formula: Qp/Qs = CSAPA · VTIPA CSALVOT · VTILVOT where the cross-sectional area (CSA) is calculated as: CSA = π · (diameter )2 Use π ≈3.1416 for calculations. Solution To calculate the Qp/Qs ratio, we need to compute the pulmonary blood flow (Qp) and systemic blood flow (Qs) using the provided measurements.
