## 心內膜炎原理


### 概述

心內膜炎是心血管結構的血管內微生物感染，涉及瓣膜、大血管、心室/心房內膜、人工材料及導管相關結構。人工瓣膜和起搏器導線相關心內膜炎的發生率逐漸增加。


贅生物是附著於內膜結構（如瓣膜或人工材料）的感染性團塊，在2D超音波中呈振盪性結構，大小和形態多變。


### 病理生理學


- 內膜損傷引發血栓形成
- 菌血症時，血栓可能被細菌感染，形成「超感染血栓」
- 後續可能導致：完全修復、組織破壞、栓塞或缺陷癒合

## 原生瓣膜心內膜炎


### 診斷與症狀


- 發燒/盜汗
- 易感因素（如瓣膜病變）
- 結膜瘀點、Janeway斑、Roth斑、指甲下出血
- 贅生物、逆流、併發症（如膿腫、破壞）
- 心包積液


### 超音波檢查


- **經胸超音波（TTE）**：初始檢查，懷疑心內膜炎時必做
- **經食道超音波（TEE）**：若TTE陰性但仍懷疑，7-10天內重複TEE
- 右側心內膜炎若TTE影像清晰，無需TEE


### 鑑別診斷


- 纖維化/鈣化
- 黏液瘤樣變性（如二尖瓣脫垂）
- Lambl’s突起/細絲


心內膜炎症狀多變，感染伴心雜音或非典型症狀時，需考慮心內膜炎。血培養陰性不排除診斷。


### 其他檢查重點


- 其他瓣膜受累
- 逆流及容量超載
- 心肌功能（左/右心室）
- 心包/胸膜積液
- 瓣膜阻塞（罕見）
- 冠狀動脈栓塞（罕見）


## 原生瓣膜心內膜炎併發症


### 常見併發症


- 栓塞
- 瓣膜破壞
- 逆流/心衰
- 膿腫
- 假性動脈瘤
- 穿孔
- 瘺管
- 黴菌性動脈瘤


### 瓣膜破壞類型


- **穿孔**：瓣葉孔洞，彩色都卜勒顯示內膜連續性中斷
- **瘺管**：與鄰近腔室連通，不直接涉及瓣膜（如主動脈-左心房）
- **假性動脈瘤**：搏動性瓣周空洞，與心血管腔相通
- **瓣環膿腫**：瓣周感染性腔，呈非均質回聲
- **瓣葉破裂**：主動脈瓣撕裂或腱索斷裂，導致偏心逆流


importantbox
**重要提醒：** 癒合過程常導致瓣膜纖維化或鈣化，需長期隨訪。假性動脈瘤和膿腫需警惕嚴重後果。
importantbox


## 右心心內膜炎


### 病因


- 靜脈藥物濫用
- 免疫抑制
- 留置導管
- 起搏器


### 三尖瓣心內膜炎特徵


- 常見病原體：金黃色葡萄球菌（60-80
- 併發症：肺動脈高壓、肺栓塞、三尖瓣逆流、右心衰
- 預後：院內死亡率10


### 併發症


- 瓣膜破壞
- 鄰近心臟結構受累


三尖瓣心內膜炎在肺膿腫、藥物濫用及新發心雜音患者中高度可能。贅生物可能極大，需非典型視角成像。


## 人工瓣膜心內膜炎


### 風險因素


- 心衰
- 傷口併發症
- 手術中直接污染


### 鑑別診斷


- 偽影
- 瓣下殞餘結構
- 手術材料
- 細絲
- 血栓
- 血腫


### 併發症


- 瓣周膿腫
- 假性動脈瘤
- 瓣周漏
- 瓣膜脫落
- 瓣膜阻塞
- 瘺管


importantbox
**重要提醒：** 人工瓣膜心內膜炎難以檢測，推薦使用TEE。需與外科醫師確認手術細節以排除手術材料偽影。
importantbox


## 起搏器/導管相關心內膜炎


### 易感因素


- 囊袋感染
- 免疫功能低下
- 全身感染
- 植入前臨時起搏
- 糖尿病
- 外科醫師經驗
- 高齡


### 臨床表現


- 發燒/低熱（反覆）
- 肺栓塞
- 局部併發症
- 敗血性休克
- 全身狀況差


### 典型感染部位


- 上腔靜脈
- 右心房
- 三尖瓣
- 三尖瓣環


導管尖端增厚或附著贅生物提示感染，TTE和TEE均需檢查。起搏器感染預後差，需及早干預。


## 非感染性/無菌性心內膜炎


### 類型


- Libman-Sacks心內膜炎
- 惡病性心內膜炎
- 抗磷脂症候群
- 高凝狀態


### 超音波特徵


- 瓣膜增厚
- 小型贅生物
- 輕至中度逆流
- 心包積液


### Libman-Sacks心內膜炎特徵


- 瓣膜增厚及贅生物
- 左/右心室功能障礙
- 壁內血栓
- 心包積液
- 自發對比


importantbox
**重要提醒：** 非感染性心內膜炎常見於狼瘡或抗磷脂症候群患者，需結合臨床病史鑑別。
importantbox

## 考題


以下為針對「心內膜炎」章節的十道選擇題，旨在檢驗關鍵概念的理解。


1. 心內膜炎的定義是：     enumerate[label=a)]
2. 心肌的細菌感染
3. 心血管結構的血管內微生物感染
4. 心包的病毒感染
5. 冠狀動脈的真菌感染


    \item 最常見的心內膜炎病原體是：
    
1. 腸球菌
2. 金黃色葡萄球菌
3. 表皮葡萄球菌
4. 牛型鏈球菌


    \item 原生瓣膜心內膜炎的典型超音波徵象是：
    
1. 瓣膜增厚無贅生物
2. 振盪性贅生物
3. 正常瓣膜運動
4. 無逆流


    \item 右側心內膜炎的常見病因不包括：
    
1. 靜脈藥物濫用
2. 免疫抑制
3. 起搏器
4. 高血壓


    \item 人工瓣膜心內膜炎的鑑別診斷包括：
    
1. 瓣膜鈣化
2. 手術材料偽影
3. 瓣葉脫垂
4. 正常血流模式


    \item 假性動脈瘤的超音波特徵是：
    
1. 非搏動性空洞
2. 瓣周搏動性空洞
3. 均質回聲
4. 無心血管腔連通


    \item 三尖瓣心內膜炎最常見的病原體是：
    
1. 牛型鏈球菌
2. 金黃色葡萄球菌
3. 腸球菌
4. 黴菌


    \item 起搏器相關心內膜炎的典型感染部位不包括：
    
1. 上腔靜脈
2. 右心房
3. 左心室
4. 三尖瓣


    \item Libman-Sacks心內膜炎的超音波特徵包括：
    
1. 大型贅生物
2. 瓣膜增厚及小型贅生物
3. 嚴重逆流
4. 無心包積液


    \item 根據ESC 2009，緊急手術適應症不包括：
    
1. 嚴重急性逆流導致心源性休克
2. 局部不受控感染
3. 嚴重逆流無心衰
4. 大型贅生物伴栓塞

### 考題詳解


1. **正確答案：b) 心血管結構的血管內微生物感染**      **詳解：** 心內膜炎是心血管結構（如瓣膜、內膜）的微生物感染，非心肌、心包或冠狀動脈感染。
2. **正確答案：b) 金黃色葡萄球菌**      **詳解：** 金黃色葡萄球菌占心內膜炎病原體25
3. **正確答案：b) 振盪性贅生物**      **詳解：** 原生瓣膜心內膜炎的典型超音波徵象為振盪性贅生物，常伴逆流，非正常瓣膜運動。
4. **正確答案：d) 高血壓**      **詳解：** 右側心內膜炎病因包括藥物濫用、免疫抑制及起搏器。高血壓與右側心內膜炎無直接關聯。
5. **正確答案：b) 手術材料偽影**      **詳解：** 人工瓣膜心內膜炎需與手術材料偽影、血栓及細絲鑑別，瓣膜鈣化或脫垂非主要鑑別點。
6. **正確答案：b) 瓣周搏動性空洞**      **詳解：** 假性動脈瘤呈瓣周搏動性空洞，與心血管腔連通，伴非均質回聲，常由心內膜炎引起。
7. **正確答案：b) 金黃色葡萄球菌**      **詳解：** 三尖瓣心內膜炎最常由金黃色葡萄球菌（60-80
8. **正確答案：c) 左心室**      **詳解：** 起搏器相關心內膜炎感染部位包括上腔靜脈、右心房及三尖瓣，左心室非典型部位。
9. **正確答案：b) 瓣膜增厚及小型贅生物**      **詳解：** Libman-Sacks心內膜炎表現為瓣膜增厚、小型贅生物、輕至中度逆流及心包積液，非大型贅生物。
10. **正確答案：c) 嚴重逆流無心衰**      **詳解：** ESC 2009緊急手術適應症包括心源性休克、不受控感染及大型贅生物伴栓塞。嚴重逆流無心衰為選擇性手術。
