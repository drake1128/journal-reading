## 心包概述


### 結構與功能

心包由內臟層和壁層組成，正常心包液量 <50 ml。心包功能包括：

- 限制心臟過度擴張
- 促進心室/心房相互作用與耦合
- 促進心臟扭轉與旋轉


心包在超音波中高反射。開胸手術或心包切開患者常呈現異常收縮模式。


## 心包積液


### 分類


- **滲出性**：結核病、膿胸擴散，含纖維條帶
- **漏出性**：充血性心衰、甲狀腺功能減退、腎病症候群
- **出血性**：創傷、動脈瘤破裂、惡性積液、醫源性
- **惡性**：常為出血性，預後差


### 病因


- 特發性：全面檢查後無明確原因
- 感染性：病毒感染（直接或免疫反應）
- 醫源性：起搏器、導管操作、心臟手術
- 腫瘤性：惡性積液，預後差
- 心肌梗死：心肌破裂、早期心外膜反應或Dressler症候群
- 腎衰竭：尿毒症或透析相關
- 自身免疫病：系統性紅斑狼瘡、類風濕性關節炎
- 放射線：20
- 風濕性：通常小量積液
- 創傷：心臟挫傷或主動脈破裂
- 內分泌疾病：甲狀腺功能減退
- 肺動脈高壓：機制不明，預後差
- 心臟手術後：常為血腫，局部積液
- 主動脈破裂：45


### 超音波診斷


- 舒張末期無回聲區
- 使用多視角（尤其肋下視角）及非典型視角（右心室周圍）
- **特徵**：     itemize[nosep]
- 大量積液：腫瘤、尿毒症、結核病、甲減
- 局部積液：術後、創傷、化膿性


itemize


### 鑑別診斷


- 胸膜積液
- 心外膜脂肪
- 心包囊腫
- 腹水


### 心外膜脂肪特徵


- 隨心包正常運動
- 與腹部脂肪相關
- 非完全無回聲（低強度回聲）
- 右心房上方無脂肪，常見於房室溝及心耳周圍


### 積液量量化


- 方法：心外膜輪廓體積減去心臟輪廓體積
- 最佳視角：肋下視角


### 超音波重要性


- 確立診斷
- 尋找病因（如心肌梗死、肺動脈高壓）
- 評估血流動力學影響
- 引導心包穿刺


importantbox
**重要提醒：** 心包積液病因因地區及實驗室環境而異（如發展中國家多結核病）。胸膜積液位於降主動脈後方，心包積液位於前方。
importantbox


## 心包壓塞


### 定義


- **壓塞**：心包腔內積液壓迫心臟
- **縮窄**：心包僵硬
- **滲出性縮窄**：僵硬心包伴積液


### 病理生理學


- 吸氣時全身靜脈回流增加，右心容量增加
- 心包無法適應，室間隔向左移位（吸氣時左心室充盈受限）
- 心室相互依賴性增強


### 臨床表現


- **症狀**：胸痛、呼吸困難、休克
- **體徵**：心動過速、水腫、低血壓


### 慢性積液壓塞觸發因素


- 低血容量（低壓壓塞）
- 陣發性心動過速
- 急性心包炎


### 超音波徵象


- 右心房塌陷（早期徵象，單獨通常不顯著）
- 下腔靜脈及肝靜脈擴張
- 右心室塌陷（伴症狀，搖擺心臟時難評估）
- 左心室塌陷（嚴重壓塞，需緊急穿刺）
- 三尖瓣流入過度呼吸變異（PW都卜勒）
- 肝靜脈舒張期逆流（PW都卜勒）


使用呼吸曲線記錄吸氣/呼氣相，評估室間隔移位。壓塞為醫療急症，快速積液易引發。


## 心包縮窄


### 特徵


- 心包鈣化/纖維化/瘢痕
- 亞急性/慢性病程
- 正常收縮功能
- 充盈受限


### 病因


- 炎症（細菌/結核病）
- 放射線
- 心臟手術後
- 靜脈充血
- 水腫
- 肝腫大
- 腹水
- 結締組織病
- 特發性


### 類型


- 環形縮窄
- 左側縮窄
- 右側縮窄
- 全局縮窄伴心肌萎縮
- 全局縮窄伴心肌周圍纖維化
- 限制性心肌病


### 超音波徵象


- 下腔靜脈及肝靜脈擴張
- 早期舒張期前向血流突出（PW都卜勒E波增高）
- 吸氣時三尖瓣血流過度增加（PW都卜勒）
- 呼氣時肝靜脈逆流（PW都卜勒）
- 室間隔震顫（振盪室間隔）
- 吸氣時室間隔向左移位
- 心臟輪廓變形（局部縮窄）
- 影像質量差
- 增厚心包
- 心室/心房較小
- 胸膜積液


importantbox
**重要提醒：** 放射線相關縮窄預後差。吸氣時室間隔移位及震顫是縮窄診斷的最佳指標，必要時需血流動力學檢查。
importantbox


## 其他心包疾病


### 心包囊腫


- 良性腫瘤，占縱隔腫瘤6
- 心包囊形成時間葉腔未融合
- 通常無症狀
- 單房/多房
- 典型位於右心膈角


### 心包囊腫鑑別診斷


- 局部心包積液
- 肝/腎/縱隔囊腫
- 包蟲囊腫
- 膈疝
- 心房憩室
- 動脈瘤樣血管


### 心包惡性疾病


- 原發性惡性腫瘤
- 轉移性腫瘤
- 心包癌
- 常伴心包積液（標誌性）


### 先天性心包缺如


- 1/10,000屍檢發現
- 形式：完全/左側/右側缺如
- 常無症狀或胸痛
- 創傷性剝離風險高
- 併發症：心腔疝或嵌頓（如左心耳）


### 先天性心包缺如超音波徵象


- 心臟移位
- 過度心臟運動
- 異常室間隔運動
- 左心耳擴大


心包囊腫常於胸部X光發現，惡性心包積液預後差（中位存活4個月）。心包缺如患者心室形狀異常，需MRI/CT確診。

    \item 滲出性心包積液的特徵是：
    
1. 無纖維條帶
2. 含纖維條帶
3. 完全無回聲
4. 與心外膜脂肪相同

1. 胸膜積液
2. 心外膜脂肪
3. 主動脈剝離
4. 心包囊腫

1. 左心室塌陷
2. 右心房塌陷
3. 正常IVC直徑
4. 無室間隔移位


    \item 心包縮窄的超音波特徵包括：
    
1. 正常E波
2. 室間隔震顫
3. 正常IVC直徑
4. 無呼吸變異


    \item 心包囊腫的典型位置是：
    
1. 左心膈角
2. 右心膈角
3. 主動脈根部
4. 左心房上方


    \item 先天性心包缺如的超音波徵象不包括：
    
1. 心臟移位
2. 正常室間隔運動
3. 左心耳擴大
4. 過度心臟運動


    \item 惡性心包積液的特徵是：
    
1. 無出血性
2. 常為出血性
3. 無心包積液
4. 預後良好


    \item 心包壓塞的病理生理學涉及：
    
1. 左心室過度充盈
2. 室間隔向右移位
3. 心室相互依賴性增強
4. 吸氣時右心室塌陷


    \item 心包縮窄的常見病因不包括：
    
1. 結核病
2. 放射線
3. 心臟手術後
4. 高血壓

### 考題詳解


1. **正確答案：a) <50 ml**      **詳解：** 正常心包液量 <50 ml，過多提示心包積液。
2. **正確答案：b) 含纖維條帶**      **詳解：** 滲出性心包積液含纖維條帶，與漏出性或心外膜脂肪不同，非完全無回聲。
3. **正確答案：c) 主動脈剝離**      **詳解：** 心包積液需與胸膜積液、心外膜脂肪、心包囊腫及腹水鑑別，主動脈剝離非直接鑑別對象。
4. **正確答案：b) 右心房塌陷**      **詳解：** 右心房塌陷是壓塞早期徵象，左心室塌陷見於嚴重病例，IVC擴張及室間隔移位為後續徵象。
5. **正確答案：b) 室間隔震顫**      **詳解：** 縮窄特徵包括室間隔震顫、E波增高、呼吸變異及IVC擴張，正常IVC或E波不符合。
6. **正確答案：b) 右心膈角**      **詳解：** 心包囊腫典型位於右心膈角，常為無症狀良性腫瘤。
7. **正確答案：b) 正常室間隔運動**      **詳解：** 心包缺如表現為心臟移位、過度運動、異常室間隔運動及左心耳擴大，正常室間隔運動不符合。
8. **正確答案：b) 常為出血性**      **詳解：** 惡性心包積液常為出血性，伴心包積液，預後差（中位存活4個月）。
9. **正確答案：c) 心室相互依賴性增強**      **詳解：** 壓塞病理涉及吸氣時右心容量增加，室間隔向左移位，心室相互依賴性增強，非右心室塌陷或左心室過充盈。
10. **正確答案：d) 高血壓**      **詳解：** 縮窄病因包括結核病、放射線、心臟手術及結締組織病，高血壓非直接原因。
