## 左心室


### 左心室直徑測量

左心室直徑測量是評估心臟大小和功能的基本參數。有兩種主要測量方法：胸骨旁長軸M模式和心尖四腔心切面二維測量。


| p{4cm}p{4cm}}  **左心室舒張末期直徑參考值** | **男性** | **女性** |
| --- | --- | --- |
| 正常值(mm) | 42 - 59 | 39 - 53 |
| 輕度擴大(mm) | 60 - 63 | 54 - 57 |
| 中度擴大(mm) | 64 - 68 | 58 - 61 |
| 重度擴大(mm) | $\geq$ 69 | $\geq$ 62 |


| p{4cm}p{4cm}}  **左心室舒張末期直徑/體表面積** | **男性** | **女性** |
| --- | --- | --- |
| 正常值(cm/m$^2$) | 2.2 - 3.1 | 2.4 - 3.2 |
| 輕度擴大(cm/m$^2$) | 3.2 - 3.4 | 3.3 - 3.4 |
| 中度擴大(cm/m$^2$) | 3.5 - 3.6 | 3.5 - 3.7 |
| 重度擴大(cm/m$^2$) | $\geq$ 3.7 | $\geq$ 3.8 |


只有當測量線垂直於左心室腔和心壁時，才能使用M模式測量值。測量時應測量心內膜邊界之間的距離，而非心外膜(外側)。M模式測量與二維測量結果應有一致性。正常心腔大小與體表面積(以及體型)成正比。


#### 心尖四腔心左心室直徑測量技術

左心室舒張末期直徑(LVEDD)測量是在舒張末期，從左心室側壁心內膜到隔壁心內膜之間進行測量，測量位置在二尖瓣尖端與乳頭肌之間。如有隔壁凸起，則應在更靠近基底部位置測量。


### 左心室容積測量

容積測量比直徑和面積測量更能準確反映左心室大小。

| 中度擴大(mL) | 179 - 200 | 118 - 130 |
| 重度擴大(mL) | $\geq$ 201 | $\geq$ 131 |

#### Simpson法測量技術

Simpson法是經由描繪舒張末期心內膜邊界來測量舒張末期容積(LVEDV)。進行雙平面測量時，需確保四腔心和二腔心切面中的心室長度相匹配。


測量時不應描繪乳頭肌輪廓。乳頭肌容積應包含在計算中。


### 左心室功能生理病理學

左心室功能受多種因素影響：

- 收縮力
- 心室形狀
- 前負荷
- 後負荷


這些因素共同影響心肌力學和每搏輸出量。

importantbox
**左心衰竭代償機制：**

- 收縮力下降初期由交感神經系統激活代償(心率和收縮力增加)
- 左心室擴張以維持每搏輸出量
- 休息時每搏輸出量尚可維持，但運動時無法適當增加(功能儲備降低)
- 末期心衰竭時，休息狀態下每搏輸出量亦下降(代償失效)


importantbox


## 左心室功能


### 左心室功能評估參數

評估左心室功能的參數包括：

- 縮短分數(FS)
- 心輸出量/心指數
- 目視評估左心室功能
- 變形參數(應變、應變率)
- 射血分數(EF) - Simpson法
- 收縮力(dp/dt)
- 每搏輸出量
- Tei指數
- 組織都卜勒心肌速度
- 二尖瓣環收縮期位移(MAPSE)


| p{4cm}p{4cm}}  **縮短分數參考值** | **男性** | **女性** |
| --- | --- | --- |
| 正常值( | 25 - 43 | 27 - 45 |
| 輕度減低( | 20 - 24 | 22 - 26 |
| 中度減低( | 15 - 19 | 17 - 21 |
| 重度減低( | $\leq$ 14 | $\leq$ 16 |


縮短分數是評估整體左心室功能的粗略估計。不應使用Teichholz公式從這些值推導射血分數。小型心室的患者，即使射血分數「正常」，其心功能和(縱向)收縮力可能仍然減低。


### 縮短分數禁忌症

縮短分數在以下情況可能導致左心室功能的高估或低估：

- 左束支傳導阻滯/不同步/起搏器
- 異常隔壁運動
- 區域性壁運動異常
- 不適當(斜向)M模式定位
- 影像質量不佳
- 「假性縮短」(非常小的心室)


#### 左束支傳導阻滯的M模式表現

左束支傳導阻滯的M模式影像顯示不同步。早期收縮期向內運動與後外側壁運動分離。無法定義舒張末期和收縮末期以確定縮短分數。增加掃描速度可最佳顯示隔壁不同步。組織都卜勒成像有助於確定收縮時間。


### 射血分數 - Simpson法


| p{8cm}}  **程度** | **射血分數範圍** |  |
| --- | --- | --- |
| 正常 | > 55 |  |
| 輕度降低 | 45 - 54 |  |

importantbox
**射血分數相關注意事項：**

- 小型心室的射血分數傾向較高
- 運動員的射血分數常在正常範圍下限
- 射血分數無法預測運動能力或功能儲備
- 後負荷降低(如二尖瓣返流)患者的射血分數可能異常高


importantbox

射血分數計算公式：$EF = EDvol - ESvol{EDvol} \times 100$

dP/dt通過測量二尖瓣返流信號初始部分在1 m/s和3 m/s之間的斜率計算。通過目視評估二尖瓣返流曲線的斜率也可粗略估計收縮力。


## 右心室


### 右心室特徵

右心室的特點包括：

- 心壁較薄(< 5 mm)
- 有調節帶(moderator band)
- 肌小梁豐富
- 「環繞」左心室


**結構縮寫：**

- PV = 肺動脈瓣
- RAA = 右心房附耳
- RVIT = 右心室流入道
- RVOT = 右心室流出道


右心室的幾何形狀比左心室更為複雜：類似風笛形狀。

| 中部右心室直徑(mm) | 27-33 | 34-37 |

| p{2.5cm}p{2.5cm}p{2.5cm}}  **參數** | **正常範圍** | **輕度異常** |

當探頭位置過於頭側時，右心室直徑會顯得較大。


#### 右心室直徑測量技術

在舒張末期測量基底部和中部右心室直徑。為提高準確性，使用優化右心室的四腔心切面。當心室縮短時，右心室直徑會被高估。

| p{5cm}}  **參數** | **參考值** |  |
| --- | --- | --- |
| 三尖瓣環收縮期位移(TAPSE) | < 16 - 18 mm |  |
| 基底外側壁組織都卜勒最大速度(S') | > 10 cm/s |  |
| 脈衝波都卜勒心肌功能指數 | > 0.4 |  |
| 組織都卜勒心肌功能指數 | > 0.55 |  |


斑點追蹤衍生的右心室自由壁縱向應變可提供額外信息量化右心室功能。它也反映了右心室心尖段的功能。


#### TAPSE測量技術

TAPSE通過將M模式線放置在三尖瓣環並測量從舒張到收縮的位移來測量。


#### 右心室組織都卜勒成像技術

取樣容積放置在右心室基底外側壁。S'表示右心室縱向功能。


### 右心室舒張功能


| p{5cm}}  **參數** | **異常值** |  |
| --- | --- | --- |
| E/A比率 | < 0.8 或 > 2.1 |  |
| E/e' | > 6 |  |
| 減速時間(ms) | < 120ms |  |


右心室舒張功能評估在臨床實踐中很少使用。


### 右心室擴張原因


- 擴張型心肌病
- 右心梗死
- 心肌炎
- 肺栓塞/肺高壓
- 右心室發育不良
- 右心室容積負荷過重(如房間隔缺損、肺動脈/三尖瓣返流)
- 運動員(對訓練的正常反應)


始終尋找右心室擴張的原因。


### 右心室面積變化分數(FAC)


| p{8cm}}  **程度** | **數值範圍** |  |
| --- | --- | --- |
| 正常 | 32-60 |  |
| 輕度降低 | 25-31 |  |
| 中度降低 | 18-24 |  |
| 重度降低 | $\leq$ 17 |  |


**測量方法：**在優化的四腔心切面上描繪舒張和收縮期右心室輪廓以獲得面積。計算變化百分比：
(右心室舒張末期面積 - 右心室收縮末期面積)/右心室舒張末期面積 $\times$ 100


描繪右心室輪廓可能困難(肌小梁多，心壁薄)。

### 左心房容積


| p{3cm}p{3cm}p{3cm}}  **程度** | **男性(mL)** | **女性(mL)** |

左心房容積測量優於M模式和二維直徑測量。左心房容積>200ml表示非常嚴重的心房擴張(左心房容積甚至可能超過1升)。優化四腔心切面，特別關注左心房以獲得最佳結果。


### 計算左心房容積的誤差來源


- 包含肺靜脈
- 二尖瓣帳篷區
- 對齊/心房縮短
- 側向分辨率
- 測量不在收縮末期進行
- 左心房斜視
- 心房縮短


### 左心房功能參數

左心房功能參數包括：

- 都卜勒（二尖瓣流入）
- 收縮/舒張面積變化
- 肺靜脈血流
- 組織都卜勒/二維應變


A波下面積與左心房將血液射入左心室（心房收縮，增壓泵功能）相關。A波較小可能意味著收縮力差、充盈阻力高，或在被動充盈階段已有較多血液進入心室。


### 左心房擴張原因


- 舒張功能障礙
- 二尖瓣狹窄/返流
- 主動脈狹窄
- 限制性/肥厚性心肌病
- 心房顫動
- 左心室功能受損


成人中最常見的左心房擴張原因是高血壓。

#### 心室間隔測量技術

心室間隔是顯著結構。隔膜中心高度回聲，特別是在高血壓患者中常觀察到隔膜凸起。凸起厚度應單獨報告。


測量問題包括：測量不在舒張末期進行(2D)、右心室結構干擾測量、隔膜形狀(基底隔膜凸起)、影像方向不正確(非垂直)。


### 左心室質量計算

左心室質量計算公式：
\[
LV mass = 0.8 \times \left\{ 1.04 \left[ (LVIDd + PWTd + SWTd)^2 - (LVIDd)^2 \right] \right\} + 0.6   g
\]

縮寫：

### 考題詳解


1. **正確答案：b) 從心內膜到心內膜，位於二尖瓣尖端與乳頭肌之間**      **詳解：** 左心室舒張末期直徑(LVEDD)的測量應在舒張末期，從左心室側壁心內膜到隔壁心內膜之間進行，位置在二尖瓣尖端與乳頭肌之間。此方法確保測量反映心室腔的真實大小，而不包括心外膜或其他結構。選項a和c錯誤，因為它們包含心外膜；選項d錯誤，因為測量位置不正確。
2. **正確答案：d) 描繪乳頭肌輪廓**      **詳解：** Simpson法測量左心室容積時，應描繪舒張末期心內膜邊界，且乳頭肌容積應包含在計算中，而不應單獨描繪其輪廓。選項a是正確做法；選項b和c是必要步驟；選項d是錯誤做法，因為描繪乳頭肌輪廓會導致容積計算不準確。
3. **正確答案：b) 交感神經激活 → 左心室擴張 → 功能儲備降低 → 休息時每搏輸出量下降**      **詳解：** 左心衰竭的代償機制遵循特定順序：收縮力下降時，交感神經系統首先激活，增加心率和收縮力；隨後左心室擴張以維持每搏輸出量；休息時每搏輸出量可維持，但運動時功能儲備降低；最終，末期心衰竭導致休息時每搏輸出量下降。選項b正確反映此順序，其他選項顛倒或錯誤排列。
4. **正確答案：c) 非常小的心室(假性縮短)**      **詳解：** 縮短分數(FS)可能因「假性縮短」而高估左心室功能，特別是在非常小的心室中，因為相對較小的絕對縮短距離會導致較高的百分比。選項a和b（如左束支傳導阻滯或區域性壁運動異常）通常導致低估功能；選項d（影像質量不佳）可能導致測量不可靠，但不一定高估。
5. **正確答案：c) 32 - 60      **詳解：** 右心室面積變化分數(FAC)的正常範圍為32 - 60
6. **正確答案：b) 左心室衰竭**      **詳解：** 右心室擴張的原因包括肺高壓、房間隔缺損、運動員對訓練的正常反應等，但左心室衰竭主要影響左心系統，間接通過肺循環壓力增加影響右心室，而非直接引起右心室擴張。因此，選項b不是右心室擴張的常見直接原因。
7. **正確答案：a) 更能預測不良事件(如心房顫動、中風)**      **詳解：** 左心房容積測量優於M模式和二維直徑測量，因為它更能準確反映左心房大小，並與不良事件（如心房顫動、中風）的風險密切相關。選項b錯誤，因為容積測量較複雜；選項c錯誤，因為影像質量仍影響測量；選項d錯誤，因為需要優化四腔心切面。
8. **正確答案：c) 主動脈擴張**      **詳解：** 左心房擴張的常見原因包括高血壓、二尖瓣狹窄、心房顫動等，這些直接影響左心房壓力或容積負荷。主動脈擴張通常與主動脈疾病相關，間接影響心臟，但不是左心房擴張的直接或常見原因。
9. **正確答案：a) $LV mass = 0.8 \times \left\{ 1.04 \left[ (LVIDd + PWTd + SWTd)^2 - (LVIDd)^2 \right] \right\** + 0.6   g$}      **詳解：** 左心室質量(LV mass)的標準計算公式為選項a，根據ESC/ASE 2005指南，使用左心室內徑(LVIDd)、後壁厚度(PWTd)和隔壁厚度(SWTd)。其他選項的係數或結構錯誤，不符合標準公式。
10. **正確答案：b) 正常或超正常舒張功能**      **詳解：** 運動員心臟的特徵包括正常或超正常舒張功能，反映心臟適應長期訓練的能力。選項a錯誤，因為隔膜厚度很少超過13 mm；選項c錯誤，因為右心室收縮功能通常正常或增強；選項d錯誤，因為左心房增壓泵功能通常超正常。
