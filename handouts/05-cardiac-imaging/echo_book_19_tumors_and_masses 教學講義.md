## 偽腫瘤


### 概述

偽腫瘤是模擬腫瘤團塊的正常或異常心臟結構，需與真性腫瘤、血栓及心內膜炎贅生物鑑別。超音波檢查需熟悉這些結構以避免誤診。


建議參與屍檢以了解偽腫瘤真實形態。右心房偽腫瘤常見，需仔細鑑別。


### 右心房偽腫瘤


- 梳狀肌
- 尤斯塔基瓣
- 齊氏網
- 終嵴
- 房間隔脂肪肥厚（啞鈴徵）
- 顯著三尖瓣環（脂肪性）
- 導管/起搏器
- 卵圓孔未閉（PFO）/房中隔缺損（ASD）封堵器


### 右心室偽腫瘤


- 導管（重症監護室）
- 小梁
- 起搏器
- 調節帶
- 肌束


### 左心室偽腫瘤


- 異常/側腱索
- 小梁
- 乳頭肌


importantbox
**重要提醒：** 左心室異常腱索可能被誤認為贅生物或模擬肥厚性梗阻性心肌病的前向運動，需使用肋下視角鑑別。
importantbox

### 血栓


#### 風險因素


- **心房**：     itemize[nosep]
- 心房顫動
- 二尖瓣置換
- 二尖瓣狹窄
- 左心室功能下降


    \item **左心室**：
    
- 左心室功能下降
- 心尖動脈瘤
- 急性心肌梗死
- STEMI後首週


itemize


#### 超音波特徵


- 大小
- 活動性
- 回聲性（新鮮低回聲 vs.陳舊高回聲）
- 位置


#### 發生率與風險


- 總發生率：20
- 大梗死伴動脈瘤：高達60
- 全身栓塞風險：2


### 心臟腫瘤


#### 轉移性腫瘤來源


- 黑色素瘤
- 軟組織肉瘤
- 甲狀腺癌
- 肺癌
- 乳腺癌
- 食道癌
- 腎細胞癌
- 肝細胞癌
- 白血病/淋巴瘤繼發侵犯


#### 良性原發性心臟腫瘤


- **成人**：黏液瘤（50
- **兒童**：橫紋肌瘤、纖維瘤、畸胎瘤
- 約75


#### 黏液瘤


- 常見於左心房（卵窩處房間隔），右心房次之
- 其他心腔或瓣膜少見
- 特徵：有蒂（短柄）、圓形/橢圓、表面光滑或絨毛狀
- 大黏液瘤可引起瓣膜阻塞
- 可導致全身或微栓塞


#### 脂肪瘤


- 第二常見良性腫瘤
- 常見位置：左心室、右心房、房間隔
- 可位於心肌內
- CT/MRI對脂肪特異性高


#### 纖維彈性瘤


- 最常位於主動脈瓣，次為二尖瓣
- 很少引起瓣膜功能障礙
- 活動性預測栓塞風險
- 罕見引起冠狀動脈閉塞


#### 惡性原發性心臟腫瘤


- **成人**：血管肉瘤、橫紋肌肉瘤、間皮瘤、纖維肉瘤、淋巴瘤、骨肉瘤
- **兒童**：惡性畸胎瘤
- 95


### 團塊成像技巧


- 使用非典型視角聚焦團塊
- 進行全面檢查，不僅限於團塊
- 調整增益設置，心肌內腫瘤難以辨識
- 使用彩色都卜勒判斷腫瘤血管化及內部血流
- 使用超音波造影劑勾勒腫瘤輪廓及血管化
- 檢查肝臟、下腔靜脈及胸膜


### 惡性腫瘤併發症


- 局部壓迫
- 阻塞
- 心包積液伴壓塞
- 向周圍結構擴散
- 心律失常
- 瓣膜功能障礙


### 後果與治療選項


- 不確定是否為腫瘤時，進行TEE、MRI、CT及隨訪
- 良性腫瘤：左心或左心室腫瘤（高栓塞風險，如纖維彈性瘤）考慮手術切除
- 血栓：抗凝治療並複查，應變小
- 惡性腫瘤：確定原發腫瘤類型（心包穿刺、實驗室檢查），部分腫瘤（如淋巴瘤）對放療/化療反應良好


importantbox
**重要提醒：** 右心房惡性腫瘤常沿房間隔生長，需仔細檢查此結構。黏液瘤典型表現幾乎可確診，必要時行TEE或MRI。
importantbox


## 考題


以下為針對「心臟腫瘤與團塊」章節的十道選擇題，旨在檢驗關鍵概念的理解。


1. 右心房偽腫瘤不包括：     enumerate[label=a)]
2. 尤斯塔基瓣
3. 異常腱索
4. 齊氏網
5. 房間隔脂肪肥厚


    \item 血栓與心內膜炎團塊的共同特徵是：
    
1. 發燒/感染
2. 栓塞
3. 擴張性生長
4. 自發對比


    \item 左心室血栓的高風險因素是：
    
1. 心房顫動
2. 心尖動脈瘤
3. 二尖瓣狹窄
4. 正常左心室功能


    \item 黏液瘤最常見的位置是：
    
1. 右心房
2. 左心房
3. 右心室
4. 主動脈瓣


    \item 纖維彈性瘤的特徵是：
    
1. 常引起瓣膜功能障礙
2. 最常位於三尖瓣
3. 活動性預測栓塞風險
4. 無冠狀動脈閉塞風險


    \item 惡性原發性心臟腫瘤中最常見的是：
    
1. 黏液瘤
2. 血管肉瘤
3. 脂肪瘤
4. 纖維彈性瘤


    \item 團塊成像時不推薦的做法是：
    
1. 使用非典型視角
2. 僅聚焦團塊
3. 使用彩色都卜勒
4. 使用超音波造影劑


    \item 惡性腫瘤的併發症不包括：
    
1. 心包積液
2. 心律失常
3. 瓣膜阻塞
4. 正常心室功能


    \item 左心室偽腫瘤包括：
    
1. 尤斯塔基瓣
2. 乳頭肌
3. 齊氏網
4. PFO封堵器


    \item 良性腫瘤的治療選項包括：
    
1. 左心腫瘤手術切除
2. 右心腫瘤抗凝治療
3. 無栓塞風險時放療
4. 無症狀時化療


enumerate


### 考題詳解


1. **正確答案：b) 異常腱索**      **詳解：** 右心房偽腫瘤包括尤斯塔基瓣、齊氏網及房間隔脂肪肥厚，異常腱索為左心室偽腫瘤。
2. **正確答案：b) 栓塞**      **詳解：** 血栓、腫瘤及心內膜炎團塊均可引起栓塞。發燒/感染及自發對比為心內膜炎及血栓特徵，擴張性生長為腫瘤特徵。
3. **正確答案：b) 心尖動脈瘤**      **詳解：** 左心室血栓風險因素包括心尖動脈瘤、左心室功能下降及急性心梗。心房顫動及二尖瓣狹窄為心房血栓風險。
4. **正確答案：b) 左心房**      **詳解：** 黏液瘤最常位於左心房（卵窩處房間隔），右心房次之，其他位置罕見。
5. **正確答案：c) 活動性預測栓塞風險**      **詳解：** 纖維彈性瘤常位於主動脈瓣，少引起瓣膜功能障礙，活動性預測栓塞風險，罕見引起冠狀動脈閉塞。
6. **正確答案：b) 血管肉瘤**      **詳解：** 惡性原發性心臟腫瘤以肉瘤為主，血管肉瘤最常見。黏液瘤、脂肪瘤及纖維彈性瘤為良性腫瘤。
7. **正確答案：b) 僅聚焦團塊**      **詳解：** 團塊成像需全面檢查，使用非典型視角、彩色都卜勒及造影劑，僅聚焦團塊可能遺漏其他病變。
8. **正確答案：d) 正常心室功能**      **詳解：** 惡性腫瘤併發症包括心包積液、心律失常及瓣膜阻塞，正常心室功能非併發症。
9. **正確答案：b) 乳頭肌**      **詳解：** 左心室偽腫瘤包括乳頭肌、異常腱索及小梁。尤斯塔基瓣、齊氏網及PFO封堵器為右心房偽腫瘤。
10. **正確答案：a) 左心腫瘤手術切除**      **詳解：** 良性腫瘤如左心或左心室腫瘤（高栓塞風險）需手術切除。抗凝適用血栓，放療/化療針對惡性腫瘤。
