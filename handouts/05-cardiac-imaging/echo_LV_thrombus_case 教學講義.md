## 左心室血栓概述


### 前言

本講義依據《CASE: Cardiovascular Imaging Case Reports, 2025;9:90-94》中的案例報告整理，參考文獻詳見[《Thrombus Migration Resulting From Cessation of Cardiopulmonary Bypass Flow in a Patient With Left Ventricular Thrombus》](https://doi.org/10.1016/j.case.2024.12.007)。內容聚焦於左心室（LV）血栓的超音波診斷、術中管理與併發症預防。


### 定義與病理

左心室血栓（LV Thrombus）是急性心肌梗塞（AMI）或左心室功能受損的常見併發症，具有系統性栓塞的高風險。血栓通常形成於左心室心尖部，與心室壁運動異常（如前壁梗塞導致的心尖部運動減弱）密切相關。


**LV 血栓的臨床風險**

- 系統性栓塞：可能導致腦中風或外周動脈栓塞
- 術中脫落：體外循環（CPB）操作可能導致血栓脫落
- 診斷挑戰：需依賴超音波或增強造影技術進行精確診斷


### 臨床適應症

LV 血栓的診斷與管理適用於以下情況：


| p{12cm}}  **適應症** | **說明** |  |
| --- | --- | --- |
| 急性心肌梗塞 | 特別是前壁梗塞，伴隨心尖部運動減弱或左心室射血分數（LVEF）降低 |  |
| 高凝狀態 | D-dimer 升高或有血栓形成傾向的患者 |  |
| 術前評估 | 需進行心臟手術的患者，特別是涉及體外循環 |  |


LV 血栓的早期診斷對預防栓塞性併發症至關重要。心臟超音波技術員需熟練掌握經胸超音波（TTE）與經食道超音波（TEE）的操作，並在術中密切監測血栓位置與動態變化。


## 診斷與影像學技術


### 經胸超音波（TTE）

TTE 是診斷 LV 血栓的首選影像學工具，特別適用於心尖部血栓的檢測。


| p{12cm}}  **步驟** | **內容** |  |
| --- | --- | --- |
| 標準切面 | 心尖四腔心切面與旁胸骨短軸切面 |  |
| 血栓特徵 | 高回聲團塊，附著於心室壁，常位於心尖部 |  |
| 造影增強 | 使用超音波造影劑提高血栓與心肌邊界的辨識度 |  |


### 經食道超音波（TEE）

TEE 在術中或 TTE 影像質量不足時使用，提供更高解析度。


| p{12cm}}  **步驟** | **內容** |  |
| --- | --- | --- |
| 切面選擇 | 中段食道左心室聚焦切面（約 72°） |  |
| 血栓定位 | 確認血栓是否附著於心尖部或脫落至左心室流出道（LVOT） |  |
| 術中監測 | 實時監測血栓動態，特別在體外循環啟動後 |  |


### 其他影像學方法


| p{12cm}}  **方法** | **說明** |  |
| --- | --- | --- |
| 增強造影 CT | 確認血栓位置與大小，特別適用於術前評估 |  |
| 心臟磁共振（CMR） | 晚期釓增強（LGE）技術對血栓檢測具有高敏感性 |  |
| 術中心外膜超音波 | 在體外循環啟動後，檢測血栓是否脫落至 LVOT |  |


**影像學診斷注意事項**

- TTE 的敏感性可能低於 CMR，需結合臨床指標（如 D-dimer 升高）綜合判斷
- TEE 在術中動態監測血栓脫落至關重要
- 術中超音波需注意體外循環流量的影響，避免誤判血栓位置


## 術中管理與併發症預防


### 體外循環（CPB）中的血栓管理

LV 血栓在體外循環操作中可能脫落，需採取以下策略：


| p{12cm}}  **步驟** | **內容** |  |
| --- | --- | --- |
| 主動脈夾閉 | 在血栓脫落至 LVOT 時，緊急夾閉主動脈以防止系統性栓塞 |  |
| CPB 流量調整 | 維持主動脈壓力，確保主動脈瓣關閉，減少血栓排出風險 |  |
| 靜脈引流優化 | 確保心臟完全排空，降低左心室內殘餘血量 |  |


體外循環操作中的血栓脫落是嚴重併發症。超音波技術員需在 CPB 啟動後立即通過心外膜超音波確認血栓位置，協助麻醉與外科團隊制定緊急處理策略。


### 術後併發症監測


| p{12cm}}  **併發症** | **監測與處理** |  |
| --- | --- | --- |
| 外周栓塞 | 術後立即進行全身 CT 檢查，確認栓塞部位；必要時進行血管內血栓切除術 |  |
| 神經系統併發症 | 監測神經學症狀，必要時進行腦血管造影 |  |
| 抗凝治療 | 術後持續肝素或口服抗凝劑，預防血栓復發 |  |


## 臨床案例分析


### 案例概述

一名 53 歲男性患者因急性 ST 段抬高型心肌梗塞（STEMI）接受經皮冠狀動脈介入治療（PCI）。術後第 13 天，TTE 與增強造影 CT 發現左心室心尖部血栓。術中因體外循環流量停止，血栓脫落至 LVOT，導致下肢動脈栓塞。


| p{12cm}}  **階段** | **事件與處理** |  |
| --- | --- | --- |
| 診斷 | 第 13 天 TTE 與 CT 確認心尖部血栓，D-dimer 升高至 3.5 $\mu$g/mL |  |
| 術中 | CPB 啟動後，血栓脫落至 LVOT，緊急主動脈夾閉無效，血栓已進入系統循環 |  |
| 術後 | 緊急血管內血栓切除術恢復下肢血流，術後第 21 天無神經學後遺症出院 |  |

### 教訓與建議


| p{12cm}}  **問題** | **解決方案** |  |
| --- | --- | --- |
| CPB 流量停止 | 維持 CPB 流量，確保主動脈瓣關閉 |  |
| 心臟未完全排空 | 優化靜脈引流，減少左心室內血量 |  |
| 術前診斷不足 | 考慮術前 CMR 提高血栓檢測敏感性 |  |


**術中血栓管理的關鍵**

- 術中超音波監測需實時確認血栓位置
- CPB 流量調整與靜脈引流優化可顯著降低栓塞風險
- 術後立即進行全身影像學檢查，快速診斷栓塞部位

## 教學考題


### 考題設計

以下為 10 道考題，用以檢驗心臟超音波技術員對左心室血栓診斷與管理的理解。請於培訓後完成。


1. LV 血栓的主要臨床風險是什麼？
2. TTE 診斷 LV 血栓的標準切面有哪些？
3. TEE 在 LV 血栓管理中的主要作用是什麼？
4. 體外循環中如何防止 LV 血栓脫落至系統循環？
5. 術後應監測哪些併發症以發現外周栓塞？
6. 什麼情況下應考慮使用心臟磁共振（CMR）診斷 LV 血栓？
7. LV 血栓形成的高風險因素有哪些？
8. 術中血栓脫落至 LVOT 時的緊急處理措施是什麼？
9. 超音波技術員在 CPB 啟動後的關鍵任務是什麼？
10. 術後抗凝治療的目的是什麼？


---


### 考題詳解


1. **答案**：系統性栓塞，包括腦中風和外周動脈栓塞。              **詳解**：LV 血栓可能脫落進入系統循環，導致嚴重併發症，如腦中風或肢體缺血，需高度警惕。
2. **答案**：心尖四腔心切面與旁胸骨短軸切面。              **詳解**：這兩個切面可清晰顯示心尖部血栓的高回聲團塊，結合造影劑可提高診斷準確性。
3. **答案**：術中動態監測血栓位置，特別是確認是否脫落至 LVOT。              **詳解**：TEE 提供高解析度影像，適用於術中實時監測，尤其在 TTE 影像質量不足時。
4. **答案**：維持 CPB 流量確保主動脈瓣關閉，或優化靜脈引流使心臟完全排空。              **詳解**：這兩種策略可減少左心室內血量或防止血栓排出，降低系統性栓塞風險。
5. **答案**：外周動脈栓塞與神經系統併發症。              **詳解**：術後需監測肢體缺血症狀與神經學變化，必要時進行全身 CT 或腦血管造影。
6. **答案**：當 TTE 敏感性不足或懷疑血栓存在但未明確診斷時。              **詳解**：CMR 的晚期釓增強技術對 LV 血栓具有高敏感性，適用於複雜病例。
7. **答案**：急性前壁心肌梗塞、心尖部運動減弱、左心室射血分數降低、高凝狀態。              **詳解**：這些因素增加血栓形成風險，需密切監測並及早診斷。
8. **答案**：緊急主動脈夾閉，同時調整 CPB 流量或優化靜脈引流。              **詳解**：這些措施旨在防止血栓進入系統循環，需在超音波引導下快速實施。
9. **答案**：通過心外膜超音波確認血栓位置，協助團隊制定處理策略。              **詳解**：術中超音波監測是確保血栓管理成功的关键，需實時提供影像資訊。
10. **答案**：預防血栓復發與栓塞併發症。              **詳解**：術後抗凝治療（如肝素或口服抗凝劑）可降低血栓再形成風險，改善預後。
