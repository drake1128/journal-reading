## 先天性心臟病基礎


### 概述

先天性心臟病（CHD）是出生時存在的心臟結構異常，成人中複雜CHD患病率為418/百萬。ASD占所有先天性缺陷的20

## 房中隔缺損（ASD）


### 血流動力學


- 右心室容量超載
- 肺動脈高壓
- 矛盾性栓塞風險
- 左心室順應性下降


### ASD類型及相關病變


| |p{6cm}|}  **類型** | **相關病變** |  |
| --- | --- | --- |
| ASD I（初級缺損） | 裂隙二尖瓣（恒有）、入口型VSD、房間隔動脈瘤 |  |
| ASD II（次級缺損） | 二尖瓣脫垂、肺動脈瓣狹窄、部分異常靜脈回流 |  |
| 靜脈竇缺損 | 部分異常靜脈回流、上腔靜脈騎跨 |  |
| 冠狀竇缺損 | 無頂冠狀竇、左上腔靜脈存留、部分/完全異常靜脈回流 |  |
| ASD類型及相關病變 |  |  |


### 超音波檢測視角


- 傾斜四腔心視角
- 胸骨旁短軸（SAX）視角
- 肋下視角
- TEE優於TTE，特別是用於靜脈竇缺損診斷及ASD形態量化


### 分流檢測難點


- 影像質量差
- 分流血流角度不佳
- 低流速
- 下腔靜脈流入模擬ASD
- 三尖瓣逆流掩蓋ASD訊號
- 分流依左右心室順應性及右心壓變化


### 懷疑ASD的情況


- 右心室擴大
- 肺動脈擴張
- 陽性造影試驗
- 房間隔形態異常（動脈瘤、連續性中斷）
- 肺動脈血流增加（VTI >25 cm）
- 病史（心律失常、呼吸困難、心房顫動）及ECG右心室增大使


### ASD量化


| |p{3cm}|}  **大小** | **直徑** |  |
| --- | --- | --- |
| 大 | >10 mm |  |
| 小 | 5-10 mm |  |
| 無顯著分流 | <5 mm |  |
| ASD大小量化 |  |  |


### 分流量計算

\[
Qp/Qs = \text{Flow_{pulm} = (PA diameter/2)^2 \cdot \pi \cdot VTI_{PA/RVOT}}{Flow_{syst} = (LVOT diameter/2)^2 \cdot \pi \cdot VTI_{LVOT}}
\]
其中：PA = 肺動脈，RVOT = 右心室流出道，LVOT = 左心室流出道，VTI = 速度時間積分


### 介入封閉適應症


- **ESC 2010 Class I**：     itemize[nosep]
- 顯著分流（右心室容量超載），肺血管阻力 <5 Wood單位，無論有無症狀
- 次級ASD首選器械封閉


    \item **適合性**：拉伸直徑 <38 mm，朝主動脈邊緣 >5 mm
    \item **禁忌症**：Eisenmenger症候群（右至左分流，ESC Class III）

### 介入後超音波評估


- 殞餘分流（彩色都卜勒降低PRF，造影檢查）
- 封閉器位置及穩定性
- 封閉器上血栓
- 心包積液


importantbox
**重要提醒：** TEE為ASD診斷及介入監測的關鍵工具。右心室擴大必須排除ASD，即使無彩色分流或陰性造影試驗。
importantbox


## 卵圓孔未閉（PFO）


### 流行病學


- 一般人群：25
- 隱源性卒中患者：40


### 超音波評估


- 常伴活動性及動脈瘤樣房間隔
- 陽性造影試驗：右心房顯影後3個心動週期內左心房出現造影
- 彩色都卜勒：小噴射至右心房（靠近主動脈緣）
- 最佳視角：肋下或傾斜四腔心（都卜勒對齊），四腔心用於造影
- Valsalva動作增強PFO檢測


TTE陰性造影試驗不能排除PFO，需行TEE。降低PRF有利於彩色都卜勒檢測。


## 室中隔缺損（VSD）


### 概述

VSD占成人先天性心臟病10


### VSD類型


- 膜周型
- 流出道下嵴型
- 流出道上嵴型
- 入口型
- 小梁型


### 超音波視角與位置


- 膜周型：胸骨旁短軸，位於主動脈瓣下方，噴射向右心室流入道
- 其他類型：多視角評估（胸骨旁長軸、短軸、肋下）


importantbox
**重要提醒：** 懷疑VSD時使用聽診器輔助。造影對VSD無幫助，需依賴彩色及PW都卜勒。
importantbox


## 動脈導管未閉（PDA）


### 超音波特徵


- **視角**：胸骨旁短軸（肺動脈分叉）、鎖骨上視角
- 收縮期及舒張期血流（光譜及彩色都卜勒）
- 肺動脈擴張


### 治療


- 高速度PDA噴射患者適合封閉
- 例外：小型無症狀PDA


PDA噴射於肺動脈分叉處，呈連續性血流，需與冠狀動脈瘺管鑑別。


## 冠狀動脈瘺管


### 概述

冠狀動脈與心腔異常連通，90


### 超音波特徵


- 冠狀動脈擴張（>0.6 cm）
- 心腔擴大
- 紊流
- 連續性血流（右心分流）


## 法洛氏四聯症


### 特徵


- 肺動脈狹窄（右心室流出道阻塞）
- 室中隔缺損
- 主動脈右偏（騎跨主動脈）
- 右心室同心肥厚


### 超音波評估


- **VSD及騎跨主動脈**：     itemize[nosep]
- 多視角評估VSD特徵及數量
- 胸骨旁長軸及心尖視角評估主動脈騎跨程度
- 短軸視角評估VSD與三尖瓣及主動脈瓣關係


    \item **右心室流出道阻塞**：
    
- 胸骨旁短軸評估漏斗部及肺動脈瓣
- 漏斗部肌束常加重阻塞
- 肺動脈瓣環常發育不全


    \item **血流動力學**：
    
- 大型VSD導致左右心室壓力均衡
- 分流方向及程度依右心室流出道阻塞嚴重性


    \item **主動脈弓及冠狀動脈**：
    
- 鎖骨上視角檢查主動脈弓及動脈導管
- 短軸視角評估近端冠狀動脈
- 排除右側主動脈弓（25


itemize


### 術後評估


- 殞餘肺動脈逆流
- 右心室流出道阻塞程度


右心室流出道阻塞程度決定血流動力學，嚴重阻塞時VSD呈右至左分流。術後需關注肺動脈逆流。


## 大動脈轉位


### 概述


- **D-TGA**：主動脈起自右心室，肺動脈起自左心室（心室-動脈不協調），患病率4.7/10,000活產，導致青紫症
- **L-TGA**：先天矯正型，靜脈血經右心房至左心室至肺動脈，氧合血經左心房至右心室至主動脈（心房-心室及心室-動脈不協調），通常無青紫症


### D-TGA相關病變


- VSD（50
- 左心室流出道阻塞（25
- 二尖瓣/三尖瓣異常（如跨騎三尖瓣）
- 冠狀動脈異常


### D-TGA超音波評估


- 肋下視角：肺動脈起自後方左心室
- 胸骨旁短軸：主動脈起自前方右心室
- 檢查相關病變


### L-TGA相關病變


- VSD（70-80
- 肺流出道（左心室）阻塞（50-60
- 三尖瓣異常（90
- 二尖瓣異常（50


### L-TGA超音波評估


- 三尖瓣位於全身循環側，較二尖瓣更心尖（四腔心或短軸視角）
- 右心室重度小梁化，位於左心室位置
- 肋下視角清晰顯示肺動脈起自左心室
- 檢查相關病變


importantbox
**重要提醒：** D-TGA需分流（ASD/VSD/PDA）維持生命，L-TGA易被成人超音波實驗室漏診，因右心室承擔全身循環易心衰。
importantbox
