## 正常心音


### S1心音


#### 生理機制


S1心音代表心室收縮開始時,房室瓣膜關閉所產生的聲音。具體來說包括:


- **二尖瓣關閉**(Mitral valve closure)
- **三尖瓣關閉**(Tricuspid valve closure)


importantbox
****關鍵概念****

當心室壓力上升超過心房壓力時:

1. 二尖瓣與三尖瓣關閉
2. 主動脈瓣與肺動脈瓣開啟
3. 血液被射出至主動脈與肺動脈
4. 這個瓣膜關閉的聲音就是S1


importantbox


#### 臨床特徵


| p{10cm}}   **特徵** | **說明** |  |
| --- | --- | --- |
| 時間點 | 心室收縮期開始 |  |
| 音色 | 較S2低沉 |  |
| 持續時間 | 較長 |  |
| 最佳聽診位置 | 心尖部位 |  |
| 與頸動脈關係 | 與頸動脈搏動同時發生 |  |


### S2心音


#### 生理機制


S2心音代表心室舒張開始時,半月瓣關閉所產生的聲音:


- **主動脈瓣關閉**(Aortic valve closure) - A2成分
- **肺動脈瓣關閉**(Pulmonary valve closure) - P2成分


**A2與P2的時間差**

正常情況下,主動脈壓力較肺動脈壓力高,因此:

- A2(主動脈瓣關閉)先發生
- P2(肺動脈瓣關閉)稍後發生
- 兩者的時間差形成S2的組成


#### 臨床特徵


| p{10cm}}   **特徵** | **說明** |  |
| --- | --- | --- |
| 時間點 | 心室舒張期開始 |  |
| 音色 | 較S1高亢、清脆 |  |
| 持續時間 | 較短 |  |
| 最佳聽診位置 | 主動脈區與肺動脈區 |  |
| 組成 | A2 + P2 |  |


## 額外心音


### S3心音


#### 生理機制


importantbox
****核心概念: 左心室擴張****

S3心音的產生機制:

1. 左心室明顯擴張(Left ventricular dilation)
2. 舒張早期血液快速充填
3. 血液撞擊心室壁產生湍流
4. 形成低頻的S3心音


importantbox

S3心音發生在S2之後,代表**舒張早期快速充填**(Rapid ventricular filling)階段。


#### 臨床意義


| p{10cm}}   **情境** | **意義** |  |
| --- | --- | --- |
| 生理性S3 | - 年輕人(<40歲) - 運動員 - 孕婦 - **屬於正常變異** |  |
| 病理性S3 | - 心衰竭(Heart failure) - 容積負荷過重(Volume overload) - 二尖瓣逆流 - 擴張性心肌病變 - **需要進一步評估** |  |


#### 聽診特徵


- 音色: 低頻、鈍音
- 時間: S2之後
- 最佳聽診位置: 心尖部位,病人左側臥位
- 使用聽診器的鐘型端(Bell)
- 節奏描述: "Kentucky" (Ken-tuc-ky = S1-S2-S3)


### S4心音


#### 生理機制


importantbox
****核心概念: 左心室順應性降低****

S4心音的產生機制:

1. 左心室順應性降低(Decreased compliance)
2. 左心室肥厚或僵硬
3. 心房需用力收縮以充填心室
4. 血液撞擊僵硬心室壁
5. 形成S4心音


importantbox

S4心音發生在S1之前,代表**心房收縮**(Atrial contraction)階段。


#### 臨床意義


S4心音**幾乎總是病理性的**,常見於:


| p{9cm}}   **疾病** | **機制** |  |
| --- | --- | --- |
| 高血壓性心臟病 | 左心室肥厚導致順應性下降 |  |
| 主動脈狹窄 | 壓力負荷增加,心室肥厚 |  |
| 肥厚性心肌病變 | 心室壁異常肥厚 |  |
| 缺血性心臟病 | 心肌缺血導致僵硬度增加 |  |
| 限制型心肌病變 | 心室順應性嚴重下降 |  |


#### 聽診特徵


- 音色: 低頻、鈍音
- 時間: S1之前
- 最佳聽診位置: 心尖部位
- 使用聽診器的鐘型端(Bell)
- 節奏描述: "Tennessee" (Ten-nes-see = S4-S1-S2)


**S3與S4的鑑別**


- S3在S2**之後** → "Kentucky"
- S4在S1**之前** → "Tennessee"
- S3: 容積負荷 → 心室擴張
- S4: 壓力負荷 → 心室肥厚
- S3: 可能為生理性(年輕人)
- S4: 幾乎總是病理性


### Summation Gallop


當心搏過速時,S3與S4可能融合成單一聲音,稱為summation gallop,常見於嚴重心衰竭。


## S2分裂


### 生理性分裂(Physiologic Splitting)


#### 機制


importantbox
****正常生理現象****

吸氣時S2分裂的機制:

1. **吸氣**增加右心靜脈回流
2. 右心室射血時間延長
3. 肺動脈瓣關閉延後(P2延後)
4. A2與P2分開,可聽到分裂
5. **吐氣**時分裂消失


importantbox


#### 臨床特徵


| p{10cm}}   **特徵** | **說明** |  |
| --- | --- | --- |
| 吸氣時 | S2明顯分裂(A2-P2分開) |  |
| 吐氣時 | S2分裂消失(A2與P2合併) |  |
| 臨床意義 | **正常生理現象** |  |
| 最佳聽診位置 | 肺動脈區(左上胸骨緣第2肋間) |  |


### 固定性分裂(Fixed Splitting)


#### 機制


importantbox
****病理性: 心房中膈缺損(ASD)****

固定性分裂的機制:

1. 心房中膈缺損造成左向右分流
2. 右心持續容積負荷過重
3. 右心室射血時間持續延長
4. P2持續延後
5. S2分裂**不隨呼吸改變**


importantbox


#### 臨床意義


固定性S2分裂幾乎**專一診斷**心房中膈缺損(ASD):


- 吸氣與吐氣時都有分裂
- 分裂程度不隨呼吸改變
- 需要進一步心臟超音波確診
- 可能需要手術或導管治療


### 反常性分裂(Paradoxical Splitting)


#### 機制


importantbox
****病理性: 左心傳導或射血異常****

反常性分裂的機制:

1. 左心室傳導或射血延遲
2. A2(主動脈瓣關閉)延後
3. A2落在P2**之後**
4. 正常順序顛倒(P2-A2)
5. **吸氣時**分裂反而消失
6. **吐氣時**出現分裂


importantbox


#### 病因


| p{9cm}}   **病因** | **機制** |  |
| --- | --- | --- |
| 左束支傳導阻滯(LBBB) | 左心室去極化延遲 |  |
| 主動脈狹窄(嚴重) | 左心室射血時間延長 |  |
| 肥厚性心肌病變 | 左心室流出道阻塞 |  |
| 右心室節律器 | 右心室先收縮,左心室延遲 |  |


**S2分裂總整理**


| p{3cmp{3cm}p{5cm}}   **類型** | **吸氣** | **吐氣** |
| --- | --- | --- |
| 生理性 | 分裂↑ | 分裂消失 |
| 固定性 | 分裂 | 分裂 |
| 反常性 | 分裂消失 | 分裂↑ |

}


## 收縮期雜音


### 主動脈狹窄(Aortic Stenosis)


#### 病理生理


主動脈瓣狹窄導致左心室射血時血流通過狹窄瓣膜產生湍流,形成收縮期雜音。

importantbox
****嚴重主動脈狹窄的三大症狀****

記憶口訣: "SAD"

1. **S**yncope (昏厥)
2. **A**ngina (心絞痛)
3. **D**yspnea (呼吸困難)


出現症狀表示**預後不佳**,需要儘快介入治療。
importantbox


#### 聽診特徵


| p{10cm}}   **特徵** | **說明** |  |
| --- | --- | --- |
| 時期 | 收縮期中段達高峰 |  |
| 型態 | Crescendo-decrescendo (漸強-漸弱) |  |
| 音色 | 粗糙、低沉 |  |
| 最佳聽診位置 | 右上胸骨緣第2肋間(主動脈區) |  |
| 傳導 | 向頸動脈傳導 |  |
| S2變化 | 嚴重時S2減弱或消失(A2消失) |  |
| 伴隨徵象 | Pulsus parvus et tardus (脈搏小而慢) |  |


#### 嚴重度評估


- 雜音峰值越晚 → 狹窄越嚴重
- S2減弱或消失 → 嚴重狹窄
- S4心音出現 → 左心室肥厚
- 頸動脈搏動延遲且減弱


### 二尖瓣逆流(Mitral Regurgitation)


#### 病理生理


二尖瓣閉鎖不全,收縮期血液從左心室逆流回左心房,產生全收縮期雜音。


#### 聽診特徵


| p{10cm}}   **特徵** | **說明** |  |
| --- | --- | --- |
| 時期 | 全收縮期(Holosystolic/Pansystolic) |  |
| 型態 | 高音調、恆定強度(Blowing quality) |  |
| 音色 | 吹風樣(Blowing) |  |
| 最佳聽診位置 | 心尖部位 |  |
| 傳導 | 向左腋下傳導 |  |
| S1變化 | 可能減弱 |  |
| S3心音 | 慢性嚴重MR可能出現S3 |  |


#### 病因分類


| p{9cm}}   **病因類別** | **常見疾病** |  |
| --- | --- | --- |
| 瓣膜本身問題 | - 風濕性心臟病 - 感染性心內膜炎 - 二尖瓣脫垂 - 黏液性瓣膜退化 |  |
| 腱索問題 | - 腱索斷裂 - 乳頭肌功能異常 - 乳頭肌梗塞 |  |
| 心室問題 | - 左心室擴張 - 缺血性心肌病變 - 擴張性心肌病變 |  |


### 二尖瓣脫垂(Mitral Valve Prolapse, MVP)


#### 病理生理


二尖瓣葉在收縮期向左心房脫垂(bulging),可能伴隨不同程度的逆流。

importantbox
****特殊的Click-Murmur複合音****

MVP的特徵性聽診發現:

1. **Mid-systolic click**: 瓣膜脫垂的瞬間
2. **Late systolic murmur**: 脫垂後的逆流
3. Click與murmur的時間可隨姿勢改變而改變


importantbox


#### 聽診特徵


| p{10cm}}   **特徵** | **說明** |  |
| --- | --- | --- |
| Click | 收縮期中段高音調喀啦聲 |  |
| 雜音 | 收縮期晚段,高音調 |  |
| 最佳聽診位置 | 心尖部位 |  |
| 姿勢影響 | - 站立/Valsalva → Click提早,雜音變長 - 蹲下/握拳 → Click延後,雜音變短 |  |


#### 臨床意義


- 最常見的瓣膜異常(2-3
- 多數為良性,預後良好
- 少數病例可能進展為嚴重MR
- 極少數與心律不整或猝死相關
- 需要感染性心內膜炎預防(有MR者)


### 肥厚性心肌病變(Hypertrophic Cardiomyopathy, HCM)


#### 病理生理


importantbox
****左心室流出道動態性阻塞****

HCM的特殊機制:

1. 心室中膈異常肥厚
2. 收縮時中膈向二尖瓣前葉靠近
3. 造成左心室流出道阻塞(LVOTO)
4. 產生收縮期雜音
5. **阻塞程度隨心室容積改變而改變**


importantbox


#### 聽診特徵


| p{10cm}}   **特徵** | **說明** |  |
| --- | --- | --- |
| 時期 | 收縮期中晚段 |  |
| 型態 | 粗糙、漸強型 |  |
| 最佳聽診位置 | 左下胸骨緣 |  |
| S4心音 | 常見(左心室肥厚) |  |
| 特殊表現 | 雙重心尖搏動(Double apical impulse) |  |


#### 關鍵鑑別點


HCM與其他雜音最大的不同在於其對生理性maneuvers的**反常反應**:


| p{9cm}}   **Maneuver** | **HCM雜音的反應** |  |
| --- | --- | --- |
| **增加靜脈回流** | - 蹲下、抬腿 - 心室容積↑ - 中膈離二尖瓣較遠 - **雜音**減弱**** |  |
| **減少靜脈回流** | - 站立、Valsalva - 心室容積↓ - 中膈更靠近二尖瓣 - **雜音**增強**** |  |
| **增加後負荷** | - 握拳(Handgrip) - 心室容積↑ - 阻塞程度↓ - **雜音**減弱**** |  |
| **減少後負荷** | - Nitrate - 心室容積↓ - 阻塞程度↑ - **雜音**增強**** |  |


**記憶要點**

HCM與MVP是唯二會因**減少靜脈回流**而雜音**增強**的疾病。這是考試最愛考的鑑別點!

其他所有雜音都是: 靜脈回流↑ → 雜音↑


#### 臨床意義


- 年輕運動員猝死的重要原因
- 可能需要β-blocker治療
- 嚴重者需要手術或酒精中膈消融術
- 需要避免劇烈運動
- 有家族遺傳傾向,需家族篩檢


### 心室中膈缺損(Ventricular Septal Defect, VSD)


#### 病理生理


心室中膈缺損造成左心室高壓血液向右心室低壓處分流,產生全收縮期雜音。


#### 聽診特徵


| p{10cm}}   **特徵** | **說明** |  |
| --- | --- | --- |
| 時期 | 全收縮期(Holosystolic) |  |
| 型態 | 高音調、粗糙 |  |
| 音色 | 機械樣(Harsh) |  |
| 最佳聽診位置 | 左下胸骨緣第3-4肋間 |  |
| 傳導 | 局限,不廣泛傳導 |  |
| 胸壁震顫 | 小的VSD可能有震顫(Thrill) |  |


**Roger雜音**

VSD的雜音特別粗糙和響亮,以前稱為"Roger's murmur"。

**矛盾現象**: 小的VSD雜音反而比大的VSD更響亮,因為:

- 小缺損 → 壓力差大 → 血流速度快 → 雜音響亮
- 大缺損 → 壓力差小 → 血流速度慢 → 雜音較輕
- 非常大的缺損甚至可能沒有雜音(Eisenmenger症候群)


#### 臨床意義


- 最常見的先天性心臟病
- 小的VSD可能自行關閉
- 大的VSD需要手術修補
- 可能導致肺動脈高壓
- Eisenmenger症候群: 分流反轉(右→左)


## 舒張期雜音


### 主動脈瓣逆流(Aortic Regurgitation, AR)


#### 病理生理


主動脈瓣閉鎖不全,舒張期血液從主動脈逆流回左心室,產生舒張早期遞減型雜音。

importantbox
****慢性AR的代償機制****


1. 左心室容積負荷增加
2. 左心室逐漸擴張
3. 維持正常每搏輸出量
4. 最終導致左心室衰竭


importantbox


#### 聽診特徵


| p{10cm}}   **特徵** | **說明** |  |
| --- | --- | --- |
| 時期 | 舒張早期 |  |
| 型態 | 遞減型(Decrescendo) |  |
| 音色 | 高音調、吹風樣(Blowing) |  |
| 最佳聽診位置 | 左下胸骨緣,病人前傾坐姿 |  |
| S3心音 | 慢性嚴重AR可能出現 |  |
| Austin Flint雜音 | 嚴重AR可能出現低音調舒張期雜音 |  |


#### 周邊徵象


慢性嚴重AR有許多特徵性的周邊徵象:


| p{10cm}}   **徵象** | **機制** |  |
| --- | --- | --- |
| Corrigan pulse | 水槌脈,快速上升快速下降 |  |
| De Musset sign | 頭部隨心跳點頭 |  |
| Quincke sign | 指甲床毛細血管搏動 |  |
| Traube sign | 股動脈槍擊音 |  |
| Duroziez sign | 股動脈雙期雜音 |  |
| Hill sign | 下肢血壓>上肢血壓>20mmHg |  |
| 脈壓差 | 明顯增大(收縮壓↑,舒張壓↓) |  |


#### Austin Flint雜音


**Austin Flint雜音的機制**

嚴重AR時,逆流的血流衝擊二尖瓣前葉,造成功能性二尖瓣狹窄:

- 舒張期中晚段
- 低音調隆隆聲
- 可能被誤認為真正的二尖瓣狹窄
- 鑑別: AR雜音+Austin Flint雜音 vs 單純MS雜音


### 二尖瓣狹窄(Mitral Stenosis, MS)


#### 病理生理


二尖瓣開口狹窄,舒張期血流從左心房進入左心室時受阻,產生舒張期隆隆聲。

importantbox
****風濕性心臟病的主要表現****

二尖瓣狹窄最常見的病因是風濕性心臟病:

1. A型鏈球菌咽喉炎後遺症
2. 瓣膜纖維化、鈣化
3. 開口面積逐漸縮小
4. 左心房壓力升高
5. 最終導致肺高壓與右心衰竭


importantbox


#### 聽診特徵


| p{10cm}}   **特徵** | **說明** |  |
| --- | --- | --- |
| Opening snap | 二尖瓣開啟時的高音調聲音(S2後) |  |
| 舒張期雜音 | 低音調隆隆聲,舒張期中晚段 |  |
| S1增強 | 瓣膜仍有彈性時S1會增強 |  |
| Presystolic    accentuation | 心房收縮時雜音加劇(心房顫動時消失) |  |
| 最佳聽診位置 | 心尖部位,左側臥位,使用Bell |  |
| S2-OS間隔 | 越短表示狹窄越嚴重 |  |


**Opening Snap的臨床意義**

Opening snap出現的時間有診斷價值:

- S2到OS的間隔越短 → 左心房壓力越高 → 狹窄越嚴重
- 輕度MS: S2-OS間隔 > 100ms
- 重度MS: S2-OS間隔 < 60ms
- 非常嚴重的MS: OS可能消失(瓣膜完全鈣化)


#### 與MR的Graham Steell雜音鑑別


嚴重MS合併肺高壓時,可能出現Graham Steell雜音:

- 舒張早期遞減型雜音
- 來自肺動脈瓣逆流
- 需與AR鑑別
- 位置: 左上胸骨緣(肺動脈區)


#### 臨床進程


1. 早期: 無症狀或運動耐力稍差
2. 進展期: 呼吸困難、心房顫動
3. 晚期: 肺高壓、右心衰竭
4. 需要經皮球囊瓣膜成形術(PBMV)或瓣膜置換手術


## 連續性雜音


### 開放性動脈導管(Patent Ductus Arteriosus, PDA)


#### 病理生理


importantbox
****胎兒循環遺跡****

開放性動脈導管的形成:

1. 胎兒期: 動脈導管連接肺動脈與主動脈
2. 出生後應於數天內自然關閉
3. 若持續開放 → PDA
4. 主動脈高壓血液持續流入肺動脈
5. 收縮期與舒張期都有分流
6. 形成特徵性的連續性雜音


importantbox


#### 聽診特徵


| p{10cm}}   **特徵** | **說明** |  |
| --- | --- | --- |
| 時期 | 連續性(Continuous) |  |
| 型態 | "Machinery murmur" - 機械樣 |  |
| 特色 | 收縮期與舒張期都存在 |  |
| 最大音量時間 | S2時達到高峰 |  |
| 最佳聽診位置 | 左上胸骨緣第2肋間(肺動脈區下方) |  |
| 胸壁震顫 | 可能存在 |  |
| 周邊脈搏 | 邊界脈(Bounding pulse) |  |


**"Machinery Murmur"的由來**

PDA的雜音被形容為"machinery murmur"(機械雜音)或"washing machine murmur",因為:

- 持續不斷的雜音
- 收縮期漸強
- S2時達高峰
- 舒張期漸弱但持續存在
- 聽起來像機器運轉的聲音


#### 臨床意義與處理


- 早產兒較常見
- 可能導致肺充血與心衰竭
- 增加感染性心內膜炎風險
- 治療:      itemize
- Indomethacin促進關閉(新生兒期)
- 經導管關閉術
- 手術結紮(較少使用)


    \item 未治療可能進展為Eisenmenger症候群
itemize


## 系統化心臟聽診評估


### 聽診前準備


1. 環境安靜
2. 適當暴露胸部
3. 病人舒適姿勢
4. 選擇適當聽診器(膜片vs鐘型)
5. 溫暖聽診器


### 標準聽診位置


| p{5cm}p{5cm}}   **位置** | **解剖位置** | **主要評估** |
| --- | --- | --- |
| 主動脈區 | 右上胸骨緣第2肋間 | AS, AR |
| 肺動脈區 | 左上胸骨緣第2肋間 | PS, PR, S2分裂 |
| Erb's point | 左胸骨緣第3肋間 | AR |
| 三尖瓣區 | 左下胸骨緣第4-5肋間 | TR, VSD |
| 二尖瓣區(心尖) | 左第5肋間鎖骨中線 | MR, MS, MVP |


### 系統化評估流程


importantbox
****IPPA評估法****


1. **I**nspection (視診): 觀察胸壁搏動
2. **P**alpation (觸診): 感覺心尖搏動與震顫
3. **P**ercussion (叩診): 確定心臟邊界(較少使用)
4. **A**uscultation (聽診): 本節重點


importantbox


#### 聽診的系統化步驟


1. **聽取心率與節律**      - 規律或不規律 - 計算心率
2. **辨識S1與S2**      - 同時觸摸頸動脈脈搏 - S1與頸動脈搏動同時
3. **評估心音強度**      - S1是否正常 - S2是否分裂
4. **聽取額外心音**      - S3存在嗎? - S4存在嗎? - Opening snap? - Ejection click?
5. **辨識雜音**      - 收縮期或舒張期? - 在哪個位置最響? - 向哪裡傳導? - 音色如何?
6. **特殊姿勢評估**      - 左側臥位(二尖瓣雜音) - 前傾坐姿(主動脈雜音) - 深呼吸(S2分裂)


### 雜音的描述要素


描述任何雜音都應包含以下要素:


| p{11cm}}   **要素** | **內容** |  |
| --- | --- | --- |
| Timing    (時期) | 收縮期、舒張期或連續性 |  |
| Location    (位置) | 在哪個聽診區最響亮 |  |
| Radiation    (傳導) | 雜音傳到哪裡(頸部、腋下等) |  |
| Quality    (音色) | 粗糙、吹風樣、機械樣等 |  |
| Intensity    (強度) | Levine分級1-6級 |  |
| Configuration    (型態) | Crescendo, decrescendo, plateau等 |  |
| Pitch    (音調) | 高音調或低音調 |  |


### Levine分級系統


| p{12cm}}   **級數** | **定義** |  |
| --- | --- | --- |
| 1/6 | 非常輕微,需要在安靜環境仔細聽才能聽到 |  |
| 2/6 | 輕微但容易聽到 |  |
| 3/6 | 中等強度,無震顫 |  |
| 4/6 | 響亮,有震顫(thrill) |  |
| 5/6 | 非常響亮,聽診器只需輕觸胸壁即可聽到 |  |
| 6/6 | 極度響亮,聽診器離開胸壁仍可聽到 |  |


## 臨床手法(Maneuvers)


### 生理性手法總覽


臨床手法可以改變心臟的前負荷(preload)或後負荷(afterload),從而改變雜音的強度,幫助鑑別診斷。

importantbox
****關鍵概念****


- 大多數雜音: 血流↑ → 雜音↑
- **例外**: HCM與MVP會相反反應
- 理解血液動力學變化是關鍵


importantbox


### 改變靜脈回流的手法


#### 增加靜脈回流


| p{10cm}}   **手法** | **對雜音的影響** |  |
| --- | --- | --- |
| 蹲下(Squatting) | - 增加靜脈回流與後負荷 - **增強**大多數雜音 - **減弱**HCM與MVP - MVP: Click延後,雜音縮短 |  |
| 抬腿    (Leg raising) | - 被動增加靜脈回流 - 效果類似蹲下 - 適用於無法蹲下的病人 |  |


#### 減少靜脈回流


| p{10cm}}   **手法** | **對雜音的影響** |  |
| --- | --- | --- |
| 站立(Standing) | - 減少靜脈回流 - **減弱**大多數雜音 - **增強**HCM與MVP - MVP: Click提早,雜音延長 |  |
| Valsalva動作 | - 用力呼氣對抗關閉的聲門 - 增加胸內壓 - 減少靜脈回流 - 效果類似站立 - 最常用於HCM診斷 |  |


**Valsalva動作的執行**


1. 請病人深吸一口氣
2. 閉住氣門,用力呼氣(像用力解便)
3. 維持10-15秒
4. 在此期間聽診
5. 觀察雜音變化


**注意**: 此動作可能引起暈眩,需小心監測。


### 改變後負荷的手法


#### 增加後負荷


| p{10cm}}   **手法** | **對雜音的影響** |  |
| --- | --- | --- |
| 握拳(Handgrip) | - 雙手用力握拳30秒 - 增加全身血管阻力 - 增加後負荷 - **增強**: AR, MR - **減弱**: AS, MVP, HCM |  |


#### 減少後負荷


| p{10cm}}   **手法** | **對雜音的影響** |  |
| --- | --- | --- |
| Amyl nitrite    吸入 | - 強效短效血管擴張劑 - 減少後負荷 - **減弱**: AR, MR - **增強**: AS, MVP, HCM - **臨床較少使用** |  |


### 呼吸的影響


| p{10cm}}   **手法** | **對雜音的影響** |  |
| --- | --- | --- |
| 吸氣    (Inspiration) | - 增加右心靜脈回流 - **增強**右側心雜音 - TR, PR, PS - **減弱**左側心雜音 |  |
| 吐氣    (Expiration) | - 增加左心靜脈回流 - **增強**左側心雜音 - AS, AR, MR, MS, MVP - **減弱**右側心雜音 |  |


**Carvallo徵象**

三尖瓣逆流(TR)在吸氣時雜音增強的現象稱為Carvallo徵象,是TR的特徵性表現。


### 姿勢改變


| p{10cm}}   **姿勢** | **臨床用途** |  |
| --- | --- | --- |
| 左側臥位    (Left lateral    decubitus) | - 使二尖瓣靠近胸壁 - **增強**二尖瓣雜音 - MS, MR最佳聽診姿勢 - 使用Bell於心尖 |  |
| 前傾坐姿    (Leaning    forward) | - 使主動脈瓣靠近胸壁 - **增強**主動脈雜音 - AR最佳聽診姿勢 - 在左下胸骨緣聽診 - 吐氣末屏住呼吸 |  |

| p{1.5cm}p{1.5cm}p{1.5cm}p{1.5cm}p{1.5cm}p{1.5cm}}   **Maneuver** | **AS** | **AR** |
| --- | --- | --- |
| **Squat** | ↑ | ↑ |
| **Stand** | ↓ | ↓ |
| **Valsalva** | ↓ | ↓ |
| **Handgrip** | ↓ | ↑ |
| **Amyl    nitrite** | ↑ | ↓ |

importantbox
****記憶要點****

**最重要的鑑別**: 

- 站立/Valsalva時,只有**MVP**與**HCM**的雜音會**增強**
- 所有其他雜音都會減弱
- 這是考試最常考的點!


**生理機制**:

- MVP: 心室變小→瓣膜更容易脫垂
- HCM: 心室變小→中膈更靠近前葉→阻塞更嚴重

## 鑑別診斷流程圖


### 收縮期雜音的鑑別


1. **確定是收縮期雜音**      - 在S1與S2之間 - 與頸動脈搏動同時
2. **判斷雜音型態**      - Crescendo-decrescendo → AS或HCM - Holosystolic → MR或VSD或TR - Late systolic → MVP
3. **確定最響位置**      - 右上胸骨緣 → AS - 左下胸骨緣第3-4肋間 → VSD - 左下胸骨緣第4-5肋間 → HCM - 心尖部位 → MR或MVP
4. **評估傳導**      - 向頸動脈 → AS - 向腋下 → MR - 局限性 → VSD或HCM
5. **進行臨床手法**      - Valsalva增強 → HCM或MVP - Handgrip增強 → MR - Handgrip減弱 → AS, HCM, MVP


### 舒張期雜音的鑑別


1. **確定是舒張期雜音**      - 在S2與下一個S1之間
2. **判斷雜音型態**      - 舒張早期遞減型 → AR - 舒張中晚段隆隆聲 → MS
3. **確定最響位置**      - 左下胸骨緣 → AR - 心尖部位 → MS
4. **尋找特殊徵象**      - Opening snap → MS - 寬脈壓 → AR - S1增強 → MS
5. **特殊姿勢**      - 前傾坐姿 → AR更響 - 左側臥位 → MS更響


## 常見錯誤與注意事項


### 聽診常見錯誤


| p{9cm}}   **常見錯誤** | **正確做法** |  |
| --- | --- | --- |
| 只聽一個位置 | 系統性聽診所有標準位置 |  |
| 未使用特殊姿勢 | AR用前傾坐姿,MS用左側臥位 |  |
| 聽診器選擇錯誤 | 高頻用膜片,低頻用鐘型 |  |
| 未同時觸摸脈搏 | 觸摸頸動脈幫助分辨S1/S2 |  |
| 忽略呼吸影響 | 評估S2分裂需要深呼吸 |  |
| 未進行臨床手法 | 困難病例需要maneuvers幫助鑑別 |  |


### 診斷陷阱


**需要特別小心的情況**


1. **Innocent murmur**: 功能性雜音在兒童很常見      - 收縮期雜音 - 強度小(1-2/6) - 姿勢改變時消失或明顯減弱 - 無其他異常發現
2. **合併多個瓣膜病變**      - 風濕性心臟病常合併MS+MR - 需要仔細分辨各個雜音
3. **極度肥胖病人**      - 雜音可能聽不清楚 - 需要心臟超音波輔助診斷
4. **嚴重病變反而雜音輕微**      - 嚴重AS晚期心輸出量下降,雜音減弱 - 大型VSD分流量大,雜音反而較輕

## 臨床案例練習


### 案例一: 70歲男性,呼吸困難


**病史**: 70歲男性,過去3個月漸進性活動性呼吸困難,偶有胸痛。

**理學檢查**:

- 右上胸骨緣可聽到3/6收縮期雜音
- Crescendo-decrescendo型態
- 向頸動脈傳導
- S2減弱
- 頸動脈搏動延遲且減弱


**問題**: 最可能的診斷是什麼?

**答案**: **嚴重主動脈狹窄**

**理由**:

- 典型位置與傳導
- Crescendo-decrescendo型態
- S2減弱(A2消失)
- Pulsus parvus et tardus
- 出現症狀(呼吸困難、胸痛)


**下一步**: 心臟超音波評估狹窄嚴重度,考慮主動脈瓣置換術。


### 案例二: 25歲女性,體檢發現雜音


**病史**: 25歲女性,例行體檢發現心雜音,無症狀。

**理學檢查**:

- 心尖部位可聽到收縮期click
- Click後接短暫收縮晚期雜音
- 站立時click提早,雜音變長
- 蹲下時click延後,雜音縮短


**問題**: 最可能的診斷是什麼?

**答案**: **二尖瓣脫垂**

**理由**:

- 特徵性click-murmur複合音
- 站立時click提早
- 蹲下時click延後
- 年輕女性好發


**處置**: 大多數MVP為良性,定期追蹤即可。如有明顯MR才需要考慮介入。


### 案例三: 8歲男童,運動後暈厥


**病史**: 8歲男童,運動後曾兩次暈厥。家族史有年輕猝死。

**理學檢查**:

- 左下胸骨緣3/6粗糙收縮期雜音
- S4心音
- Valsalva動作時雜音明顯增強
- 蹲下時雜音減弱
- 雙重心尖搏動


**問題**: 最可能的診斷是什麼?需要注意什麼?

**答案**: **肥厚性心肌病變(HCM)**

**警訊徵象**:

- 運動後暈厥
- 家族史有猝死
- 這是年輕運動員猝死的重要原因!


**緊急處置**:

- **禁止劇烈運動**
- 心臟超音波確診
- 基因檢測
- 家族篩檢
- 考慮植入式去顫器(ICD)

### 1. S1與S2基本概念 (00:43-03:36)


**核心要點：**

- S1代表二尖瓣和三尖瓣關閉（房室瓣關閉）
- S2代表主動脈瓣和肺動脈瓣關閉（半月瓣關閉）
- S2由A2（主動脈瓣）和P2（肺動脈瓣）兩個成分組成
- 主動脈壓力較高，所以A2通常先於P2發生


### 2. S3心音 (03:36-04:52)


**機制：**左心室擴張→舒張早期快速充填→血液撞擊心室壁→湍流→S3

**臨床意義：**

- 生理性：年輕人(<40歲)、運動員、孕婦
- 病理性：心衰竭、容積負荷過重


### 3. S4心音 (04:52-05:52)


**機制：**左心室順應性降低（僵硬、肥厚）→心房用力收縮→血液撞擊僵硬心室壁→S4

**臨床意義：**幾乎總是病理性——高血壓性心臟病、AS、HCM


### 4. 生理性S2分裂 (11:28-15:33)


**機制：**

- **吸氣**：右側靜脈回流↑→右心室射血時間↑→P2延遲→分裂增大
- **呼氣**：右側靜脈回流↓→P2提早→分裂消失


**記憶：**正常人的吸氣分裂


### 5. 固定性S2分裂 (15:33-17:19)


**機制：**ASD→左→右分流→右心室持續過載→P2持續延遲→分裂固定

**特徵：**無論吸氣或呼氣，分裂都持續存在且不改變


### 6. 反常性S2分裂 (17:19-19:29)


**機制：**LBBB或AS→左心室活化/射血延遲→A2延遲→順序變成P2-A2

**特徵：**

- 呼氣時分裂（與生理性相反！）
- 吸氣時消失


### 7. 主動脈狹窄 (19:29-24:13)


**雜音特徵：**

- Crescendo-decrescendo（漸強-漸弱）收縮期雜音
- 右上胸骨緣，向頸動脈傳導
- 重度時S2減弱或消失


**嚴重度評估：**

- 雜音峰值時間：輕度=早期，重度=晚期
- Pulsus parvus et tardus（重度）


**SAD症狀：**Syncope（暈厥）、Angina（心絞痛）、Dyspnea（呼吸困難）

**臨床手法：**

- 增加靜脈回流→雜音增強
- Handgrip（增加後負荷）→雜音減弱


### 8. 肥厚性心肌病變 (24:13-27:00)


**機制：**心室中隔不對稱肥厚→LVOT阻塞→SAM現象

**反常反應：**

- **減少**靜脈回流（站立/Valsalva）→LV變小→阻塞增加→雜音**增強**
- 增加靜脈回流（蹲下）→LV變大→阻塞減少→雜音減弱
- Handgrip→雜音減弱


**臨床重要性：**年輕運動員猝死的主要原因


### 9. 二尖瓣脫垂 (27:00-28:40)


**特徵：**收縮中期Click + 收縮晚期雜音

**姿勢影響：**

- 站立→LV變小→Click提早，雜音變長
- 蹲下→LV變大→Click延後，雜音變短


### 10. 主動脈瓣逆流 (31:10-33:14)


**雜音：**舒張早期遞減型（decrescendo）

**聽診技巧：**前傾坐姿，呼氣後閉氣，用膜片在左下胸骨緣

**周邊徵象：**

- Corrigan's pulse（水錘脈）
- De Musset's sign（頭部點頭）
- Quincke's sign（指甲床搏動）
- Hill's sign（下肢血壓>上肢>20mmHg）


**Austin Flint雜音：**嚴重AR時，舒張中晚期低頻雜音，類似MS但無opening snap


### 11. 二尖瓣狹窄 (33:14-34:08)


**雜音：**舒張中晚期低頻隆隆聲

**Opening snap：**

- S2後的清脆聲音
- OS-S2間隔短(<0.08秒)→重度MS
- OS-S2間隔長(>0.12秒)→輕度MS


**其他特徵：**S1增強、心尖部位、左側臥位、用鐘型


### 12. 開放性動脈導管 (34:08-35:52)


**特徵：**連續性"machinery murmur"（機器雜音）

**機制：**主動脈壓力始終>肺動脈壓力→整個心動周期都有分流

**雜音：**

- 貫穿S1和S2
- 在S2處最響（壓力梯度最大）
- 左上胸骨緣第2肋間


### 13. 臨床手法總整理 (36:26-40:24)


**1. 呼吸變化：**

- 吸氣→右側雜音↑，左側雜音↓
- 呼氣→左側雜音↑，右側雜音↓


**2. 姿勢改變：**

- 前傾坐姿→主動脈瓣雜音↑
- 左側臥位→二尖瓣雜音↑，S3/S4更明顯


**3. 靜脈回流改變：**

- 蹲下/抬腿（↑回流）→大部分雜音↑，**但HCM和MVP↓**
- 站立/Valsalva（↓回流）→大部分雜音↓，**但HCM和MVP↑**


**4. 後負荷改變：**

- Handgrip（↑後負荷）→AR和MR↑，AS/HCM/MVP↓
- Amyl nitrate（↓後負荷）→AR和MR↓，AS/HCM/MVP↑


**記憶要點：**

- HCM和MVP對靜脈回流有**反常反應**
- 減少LV大小→HCM和MVP雜音增強
- 增加後負荷→逆流增強，阻塞減少
