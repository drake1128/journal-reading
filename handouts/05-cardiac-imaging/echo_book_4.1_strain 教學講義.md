## 心肌應變基礎


### 概述

心肌應變（Strain）是量化心肌變形的超音波指標，通過斑點追蹤超音波（STE）測量。相較於傳統參數（如射血分數，EF），應變對心肌功能異常更敏感，廣泛應用於收縮與舒張功能評估、腫瘤治療心臟毒性監測及心肌病診斷。[](https://www.escardio.org/Guidelines/Recommended-Reading/Cardiovascular-Imaging/Echocardiography)


應變可檢測EF正常時的亞臨床心肌損傷，是現代心臟超音波的核心工具。


### 應變的定義與測量


- **定義**：心肌變形百分比，負值表示縮短（如縱向應變），正值表示增厚（如徑向應變）。全球縱向應變（GLS）是最常用指標，正常值-18
- **測量方法**：     itemize[nosep]
- 使用STE，分析心尖四腔、三腔、二腔視角，追蹤心內膜斑點運動。
- 高幀率（>50 fps）確保準確性。
- 確保清晰心內膜邊緣，避免心率不整干擾。


    \item **類型**：
    
- 縱向應變：反映長軸縮短，敏感於心肌纖維化。
- 周向應變：反映環形收縮。
- 徑向應變：反映壁增厚。


itemize


### 與傳統參數的比較


- **射血分數（EF）**：     itemize[nosep]
- 反映整體心室收縮，易受負荷影響。
- 對早期心肌異常不敏感。


    \item **E/A比值**：
    
- 反映舒張充盈模式，受前負荷、心率影響。
- 無法直接評估心肌特性。


    \item **應變優勢**：負荷獨立，敏感於區域性和亞臨床異常。(相關連結[about Strain](https://www.escardio.org/Guidelines/Recommended-Reading/Cardiovascular-Imaging/Echocardiography))
itemize

importantbox
**重要提醒：** GLS是診斷早期心肌功能障礙的首選指標，特別適用於EF正常的患者。
importantbox


## 臨床應用


### 收縮功能障礙


- **早期檢測**：GLS下降（>-18     itemize[nosep]
- 心肌病（如限制型心肌病，RCM）：GLS降低反映纖維化或浸潤。
- 缺血性心臟病：區域性應變異常提示冠狀動脈病變。


    \item **預後評估**：GLS <-16
itemize


### 舒張功能障礙


- **機制**：舒張應變率（SR-E）反映心肌鬆弛，正常值>1.0 s⁻¹。SR-E降低提示舒張功能障礙。
- **應用**：     itemize[nosep]
- 心衰竭保留射血分數（HFpEF）：GLS和SR-E降低，反映潛在心肌異常。
- RCM：GLS降低，伴舒張應變異常，提示限制型生理。


    \item **鑑別診斷**：RCM中GLS顯著降低，縮窄性心包炎則接近正常。
itemize


### 腫瘤治療心臟毒性


- **監測**：GLS下降（>15
- **指南支持**：美國心臟協會推薦GLS用於癌症患者心臟功能監測。
- **實例**：乳癌患者接受蒽環類治療，GLS監測可指導早期干預。
- Additional reading: [Circulation, Volume 148, Number 16.](https://www.ahajournals.org/doi/10.1161/CIR.0000000000001174)


### 特定心肌病


- **澱粉樣變性**：GLS呈“心尖保留模式”（基底和中段應變降低，心尖正常）。
- **類肉瘤病**：節段性應變異常，反映局部心肌受累。
- **肥厚型心肌病**：GLS降低，提示心肌纖維化程度。
- Additional reading: [JACC: Cardiovascular Imaging Volume 18, Issue 3, March 2025, Pages 340-381](https://www.sciencedirect.com/science/article/pii/S1936878X24003012)


應變的區域性分析有助於鑑別心肌病病因，如澱粉樣變性的心尖保留模式。


## 心臟超音波評估


### 應變測量技術


- **視角**：心尖四腔、三腔、二腔視角，獲取全心室GLS。
- **注意事項**：     itemize[nosep]
- 確保高幀率（>50 fps）和清晰心內膜追蹤。
- 標準化ROI（感興趣區域）設置，避免過寬或過窄。
- 校正心率變異影響。


    \item **異常標準**：GLS >-18
itemize


### 與其他參數的整合


- **組織都卜勒（E'）**：E'降低常伴GLS下降，共同提示舒張功能障礙。
- **E/A比值**：E/A > 2與GLS降低結合，確認RCM的限制型生理。
- **E/E'比值**：E/E'升高（>14）與GLS降低相關，反映充盈壓力增加。


importantbox
**重要提醒：** 應變與都卜勒參數結合可提高診斷特異性，尤其在複雜心肌病中。
importantbox

## 考題


以下為針對「心肌應變」章節的五道選擇題，旨在檢驗對關鍵概念的理解。


1. 全球縱向應變（GLS）的正常值範圍是：     enumerate[label=\alph*)]
2. -5
3. -18
4. -30
5. 0


    \item 以下哪項是應變相較於射血分數（EF）的優勢？
    
1. 受前負荷影響大
2. 對亞臨床心肌異常更敏感
3. 僅適用於晚期心衰
4. 無法進行區域性分析


    \item 澱粉樣變性心臟病的應變特徵是：
    
1. 均勻性GLS降低
2. 心尖保留模式
3. GLS正常
4. 徑向應變增強


    \item 應變在腫瘤治療心臟毒性監測中的作用是：
    
1. 僅用於晚期心衰患者
2. 預測早期心肌損傷
3. 替代心電圖檢查
4. 無臨床意義


    \item 以下哪項有助於區分限制型心肌病（RCM）與縮窄性心包炎？
    
1. RCM中GLS正常
2. 縮窄性心包炎中GLS顯著降低
3. RCM中GLS降低
4. 縮窄性心包炎中E/A < 0.8


enumerate


---


### 考題詳解


1. **正確答案：b) -18      **詳解：** GLS正常值為-18
2. **正確答案：b) 對亞臨床心肌異常更敏感**      **詳解：** 應變（GLS）對早期心肌異常更敏感，且負荷獨立。其他選項不正確。
3. **正確答案：b) 心尖保留模式**      **詳解：** 澱粉樣變性中應變呈心尖保留模式（apical sparing, 基底和中段應變降低）。其他選項不典型。
4. **正確答案：b) 預測早期心肌損傷**      **詳解：** GLS下降（>15
5. **正確答案：c) RCM中GLS降低**      **詳解：** RCM中GLS顯著降低，而縮窄性心包炎中GLS正常或輕度降低，是關鍵鑑別特徵。其他選項不正確。
