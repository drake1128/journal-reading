## 三尖瓣疾病基礎


### 概述

三尖瓣疾病主要包括三尖瓣逆流（TR）和三尖瓣狹窄（TS）。超音波檢查是診斷、量化和確定病因的主要工具，可指導治療決策。三尖瓣較二尖瓣更難成像，需使用更高肋間隙的四腔心視角。


三尖瓣後葉通常較小，乳頭肌位置和大小變異大，成像需調整視角。


### 形態學


- 三片瓣葉（前葉、後葉、隔葉）
- 面積較二尖瓣大（3.2-6.4 cm$^2$）
- 瓣葉較二尖瓣薄且更靠近心尖


### 成像方法

| --- | --- | --- |
| 右心室長軸（RV PLAX） | 前葉、隔葉、後葉 |  |
| 右心室流入-流出視角 | 前葉、後葉 |  |
| 右心室優化四腔心視角 | 隔葉、前葉 |  |
| 三尖瓣超音波成像視角 |  |  |


### 血流動力學


- 右心室流入E/A波低於二尖瓣
- 血流速度隨呼吸變化

**重要提醒：** 三尖瓣成像需使用較高肋間隙（比四腔心高1肋間），以確保清晰可視。

## 三尖瓣逆流原因


### 功能性三尖瓣逆流


- 左心疾病
- 二尖瓣疾病
- 肺動脈高壓
- 右心室擴張（如房中隔缺損/左-右分流）


### 原發性三尖瓣逆流


- 風濕性（常合併三尖瓣狹窄）
- 外傷（鈍傷、斷裂）
- 心臟起搏器導線相關
- 心內膜炎
- 先天性（如瓣膜發育不良、Ebstein異常）


### 預後


- 存活率取決於：     itemize[nosep]
- 三尖瓣逆流嚴重程度
- 肺動脈高壓程度
- 左/右心室功能減退


itemize


### 特殊病因


#### 類癌心臟病


- 心內膜纖維化
- 三尖瓣瓣葉受限
- 寬大接合缺陷
- 若存在房中隔缺損或卵圓孔未閉，可能影響左心


#### Ebstein異常


- 形態變異大
- 前葉顯著增大
- 常見併發症：     itemize[nosep]
- 房中隔缺損（>1/3患者）
- 室中隔缺損
- 動脈導管未閉
- 心內膜炎


itemize


Ebstein異常患者常伴房中隔缺損，需仔細檢查右心結構。


## 三尖瓣逆流量化


### 整合方法


- 使用多個視角（右心室優化四腔心、彩色都卜勒）
- 觀察數個心動週期，因吸氣可能增加逆流程度
- 聚焦流匯聚區（PISA）和狹窄段（vena contracta）


### 嚴重三尖瓣逆流徵象


- 右心室/心房顯著擴大（容量超載）
- 下腔靜脈擴大，無呼吸變化
- 肝靜脈收縮期逆流
- 舒張期室間隔扁平
- 可見接合缺陷


### 參考值

最新研究（Hahn et al., 2022）提出三尖瓣逆流（TR）分級已擴展為五級：輕度（mild）、中度（moderate）、重度（severe）、極重度（massive）和超重度（torrential）。此分級反映了TR嚴重程度的連續譜，並對經導管治療的預後評估具有重要意義。下表列出相關定量參數的參考值。


| |p{2.2cm}|p{2.2cm}|p{2.2cm}|p{2.2cm}|p{2.2cm}|}  **指標** | **輕度** | **中度** |
| --- | --- | --- |
| 狹窄段寬度（mm） | <3 | 3-6.9 |
| 有效逆流孔面積（mm$^2$） | <20 | 20-29 |
| 逆流體積（mL） | <15 | 15-29 |
| 逆流分數（3D超音波） | <25 | 25-44 |
| 3D狹窄段面積（mm$^2$） | - | - |
| 三尖瓣逆流分級參考值（Hahn et al., 2022） |  |  |


新型TR分級（mild, moderate, severe, massive, torrential）有助於精準評估經導管治療效果，特別是極重度和超重度TR常見於介入治療後殘餘逆流。


### 超音波觀察


- 嚴重逆流時，右心室功能初期亢進，後期惡化
- 三尖瓣逆流降低右心室後負荷，可能高估右心室功能
- 極重度（massive）和超重度（torrential）TR通常伴隨顯著右心室和右心房擴大，需3D超音波評估狹窄段面積以提高診斷準確性


importantbox
**重要提醒：** 嚴重三尖瓣逆流可能導致D形心室（室間隔扁平），需注意收縮期和舒張期變化。新型分級系統強調極重度和超重度TR的臨床重要性。
importantbox


## 三尖瓣狹窄


### 概述

三尖瓣狹窄（TS）較少見，主要與風濕性心臟病相關。超音波檢查需尋找瓣葉隆起和彩色都卜勒上的紊流。


### 病因


- 風濕性心臟病（9
- 心內膜炎（極罕見）
- 先天性三尖瓣狹窄（極罕見）
- 功能性狹窄（心內/心外腫塊阻塞或壓迫）
- 三尖瓣修復後（瓣環成形過小）


### 血流動力學


- 舒張期右心房-右心室壓力梯度
- 右心房擴大及壓力升高
- 下腔靜脈擴大


### 量化和診斷


- **Pressure half time (壓力半時間)**：三尖瓣面積（TVA）= 190/PHT，TVA<1 cm$^2$表示嚴重TS（未驗證）
- **平均梯度**：>5 mmHg表示顯著狹窄
- 吸氣時流入速度增加，需平均多個心動週期


### 超音波特徵


- 瓣葉隆起（2D）
- 彩色都卜勒顯示紊流
- 連續波都卜勒顯示高流速（平均梯度>5 mmHg）


三尖瓣狹窄症狀可能類似右心衰竭，需檢查所有風濕性二尖瓣狹窄患者的紊流。


## 三尖瓣手術適應症


### ESC 2012指南


- 嚴重原發性或繼發性TR，接受左側瓣膜手術的患者
- 嚴重孤立性原發性TR，有症狀且無嚴重右心室功能障礙


### 手術時機


- 右心衰竭徵象（如胸腔積液、周邊水腫、腹水）出現時，手術可能已晚（不可逆右心室功能障礙）
- 左側手術時添加三尖瓣修復不增加手術風險


importantbox
**重要提醒：** 及早進行三尖瓣修復可避免不可逆右心室功能衰竭。
importantbox


## 考題


以下為針對「三尖瓣疾病」章節的十道選擇題，旨在檢驗關鍵概念的理解。


1. 三尖瓣逆流預後的主要決定因素不包括：     enumerate[label=a)]
2. 三尖瓣逆流嚴重程度
3. 肺動脈高壓程度
4. 左心房壓力
5. 右心室功能減退


    \item 嚴重三尖瓣逆流的超音波徵象包括：
    
1. 左心室擴大
2. 肝靜脈收縮期逆流
3. 肺靜脈收縮期逆流
4. 左心房擴大


    \item 根據Hahn et al. (2022)，超重度三尖瓣逆流的狹窄段寬度為：
    
1. 3-6.9 mm
2. 7-13 mm
3. 14-20 mm
4. $\geq$21 mm


    \item Ebstein異常的常見特徵是：
    
1. 後葉顯著增大
2. 前葉顯著增大
3. 隔葉缺失
4. 右心室縮小


    \item 三尖瓣狹窄的平均壓力梯度閾值為：
    
1. >2 mmHg
2. >5 mmHg
3. >10 mmHg
4. >15 mmHg


    \item 功能性三尖瓣逆流的主要原因不包括：
    enumerate][label=a)]
        \item 肺動脈高壓
        \item 右心室擴張
        \item 心內膜炎
        \item 二尖瓣疾病
    \end{enumerate

    \item 三尖瓣成像的最佳視角是：
    enumerate][label=a)]
        \item 左心室長軸
        \item 右心室優化四腔心
        \item 左心室短軸
        \item 心尖二腔心
    \end{enumerate

    \item 類癌心臟病的三尖瓣逆流特徵包括：
    enumerate][label=a)]
        \item 瓣葉增厚
        \item 寬大接合缺陷
        \item 腱索斷裂
        \item 瓣環縮小
    \end{enumerate

    \item 三尖瓣手術的ESC Class I適應症包括：
    enumerate][label=a)]
        \item 輕度TR，無症狀
        \item 嚴重TR，接受左側瓣膜手術
        \item 嚴重TS，無右心室功能障礙
        \item 中度TR，有右心衰竭
    \end{enumerate

    \item 三尖瓣狹窄的超音波特徵不包括：
    enumerate][label=a)]
        \item 瓣葉隆起
        \item 彩色都卜勒紊流
        \item 左心房擴大
        \item 平均梯度>5 mmHg
    \end{enumerate
enumerate


### 考題詳解


1. **正確答案：c) 左心房壓力**      **詳解：** 三尖瓣逆流預後取決於逆流嚴重程度、肺動脈高壓及左右心室功能。左心房壓力主要影響二尖瓣疾病。
2. **正確答案：b) 肝靜脈收縮期逆流**      **詳解：** 嚴重三尖瓣逆流導致右心房/室擴大、下腔靜脈擴大及肝靜脈收縮期逆流。左心室或左心房擴大與三尖瓣無直接關聯。
3. **正確答案：d) $\geq$21 mm**      **詳解：** Hahn et al. (2022)定義超重度（torrential）三尖瓣逆流狹窄段寬度$\geq$21 mm。3-6.9 mm為中度，7-13 mm為重度，14-20 mm為極重度。
4. **正確答案：b) 前葉顯著增大**      **詳解：** Ebstein異常特徵為前葉顯著增大及形態變異。右心室通常擴大，非縮小。
5. **正確答案：b) >5 mmHg**      **詳解：** 三尖瓣狹窄的平均壓力梯度>5 mmHg表示顯著狹窄。其他選項不符合診斷標準。
6. **正確答案：c) 心內膜炎**      **詳解：** 功能性三尖瓣逆流由左心疾病、肺動脈高壓或右心室擴張引起。心內膜炎為原發性逆流原因。
7. **正確答案：b) 右心室優化四腔心**      **詳解：** 右心室優化四腔心視角（較高肋間隙）是三尖瓣成像最佳視角，可視隔葉和前葉。
8. **正確答案：b) 寬大接合缺陷**      **詳解：** 類癌心臟病導致心內膜纖維化、三尖瓣瓣葉受限及寬大接合缺陷。腱索斷裂或瓣環縮小非典型特徵。
9. **正確答案：b) 嚴重TR，接受左側瓣膜手術**      **詳解：** ESC 2012 Class I適應症包括嚴重TR患者接受左側瓣膜手術或有症狀的嚴重孤立性原發性TR。輕度或中度TR不適用。
10. **正確答案：c) 左心房擴大**      **詳解：** 三尖瓣狹窄超音波特徵包括瓣葉隆起、紊流及平均梯度>5 mmHg。左心房擴大與三尖瓣無關。
