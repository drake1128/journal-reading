心血管中心教育訓練 主動脈瓣狹窄(AS) 考卷 主動脈瓣狹窄(AS) 考卷 答案與詳解 主動脈瓣狹窄(AS) 考題答案 1. B (心尖五腔窗口(Apical 5-chamber view)) 詳解: 心尖五腔窗口是評估主動脈瓣狹窄最常用的超音波窗口，因為此窗口可同時顯示 左心室、左心室流出道及主動脈瓣，且超音波探頭與主動脈瓣血流方向較為平行，有 利於獲得最大的跨瓣壓力梯度。雖然胸骨右緣窗口有時也能獲得較好的角度，但在多 數病人中較難取得。 2. C (藍色(遠離探頭方向)) 詳解: 在連續多普勒測量主動脈瓣流速時，從心尖窗口測量的主動脈瓣收縮期血流是遠 離探頭的方向，因此在多普勒譜圖上顯示為藍色。正確識別血流方向對於區分主動脈 瓣順行血流和主動脈瓣逆流很重要，尤其是當兩者同時存在時。 3. C (>4.0 m/s) 詳解: 根據美國超音波心臟醫學會(ASE) 與歐洲心臟病學會(ESC) 指南，重度主動脈 瓣狹窄的診斷標準為：最大流速>4.0 m/s、平均壓力梯度>40 mmHg、瓣膜面積<1.0 cm² 或瓣膜面積指數<0.6 cm²/m²。最大流速是最容易測量且重複性良好的參數。 4. B (連續多普勒(Continuous-wave Doppler)) 詳解: 測量主動脈瓣狹窄的最大流速(Vmax) 時，應使用連續多普勒(CW Doppler)。 因為主動脈瓣狹窄的血流速度通常超過2 m/s，甚至可達5-6 m/s，而脈衝多普勒(PW Doppler) 因採樣深度限制，無法準確測量超過奈奎斯特極限(通常為2 m/s 左右) 的高 流速。連續多普勒沒有這種限制，可測量任何速度範圍的血流。 5. A (與血流方向平行(0 度)) 詳解: 使用CW 多普勒測量主動脈瓣狹窄流速時，最理想的取樣角度是與血流方向平行 (0 度)。根據多普勒效應原理，測量的流速與血流實際速度和夾角余弦值的乘積成正比 (V 測量=V 實際×cosθ)。當夾角為0 度時，cosθ=1，測得的流速最接近實際流速。隨 著夾角增大，測得的流速會被低估。 6. B (高密度、高速度信號) 詳解: 在重度主動脈瓣狹窄病患，使用心尖窗口的CW 多普勒檢查時，多普勒信號通常 表現為高密度、高速度信號。高密度反映了大量紅血球同時經過狹窄的瓣膜口徑，高 速度則是由於狹窄通道造成的血流加速。完整的多普勒包絡線清晰可見，頂峰呈尖狀， 表現出典型的” 刀刃” 形態。 7. C (左心室流出道直徑、左心室流出道流速時間積分、主動脈瓣流速時間積分) 詳解: 計算主動脈瓣有效口面積(AVA) 使用連續性方程式(continuity equation) 時，需 要測量以下三項參數：左心室流出道直徑(LVOT diameter)、左心室流出道流速時間積 分(LVOT VTI) 和主動脈瓣流速時間積分(AV VTI)。公式為：AVA = (LVOT 面積× LVOT VTI) ÷ AV VTI，其中LVOT 面積= π × (LVOT 直徑/2)²。這基於流量守恆原理，即通過 瓣膜前後的血流量相等。 第5 頁 心血管中心2025 年
