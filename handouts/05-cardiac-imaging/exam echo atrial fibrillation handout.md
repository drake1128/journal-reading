## 摘要


### 背景

心房顫動(AF)在一般族群中日益增加的負擔，也轉化為重症照護環境中發生率的增加，並因疾病嚴重度而進一步惡化。新發心房顫動已被確立為獨立的死亡率預測因子。心臟科管理指引基於納入門診病人的大型試驗，這些病人具有不同程度的心室收縮和舒張功能障礙，以及左心室充盈對心房收縮的不同依賴性。重點在於速率控制結合抗凝治療，並仔細考慮在已經有某種結構性心臟病的病人中，抗心律不整藥物可能造成的心肌抑制。

### 目標

重症超音波檢查(CCE)已被確立為血流動力學不穩定病人的廣泛應用診斷和監測工具。它有助於識別與心律不整相關的危險因子，揭示與心律不整持續性相關的參數，並引導治療以促進恢復竇性節律。CCE幫助指導節律控制或速率控制的關鍵管理決策，並在節律控制時，監測心臟電擊整流後竇性節律的機械性恢復和左心房功能恢復。超音波檢查也可協助處理衝突的管理目標，例如在有顯著出血風險的重症病人中，指引驅動的治療性抗凝。


### 結果

本回顧旨在協助重症醫療人員管理AF病人，重點關注CCE提供的諸多益處，將特定的重症醫學數據與當前心臟科心律不整管理指引相結合，應用於這些重症病人。


關鍵要點：重症超音波檢查是重症照護中心房顫動和其他上心室心律不整管理的寶貴工具。左心房二維靜態和功能參數，結合二尖瓣脈衝波都卜勒，可協助決定心律不整管理的節律控制與速率控制策略。


## 引言


心律不整的盛行率上升，對全球重症照護系統造成日益增加的負擔。上心室心律不整，特別是心房顫動(AF)，是最常遇到的心律不整，影響超過25

AF是重症病人的獨立死亡危險因子。新發心房顫動(NOAF)引發一個合理的問題：其有害效應是否可以通過心臟電擊整流和恢復竇性節律(SR)而逆轉。關於節律控制與速率控制策略的急性益處，目前數據有限，儘管長期追蹤記錄到類似的結果。非持續性心律不整可能自發終止，僅需觀察，而NOAF的電擊(DC)或藥物心臟電擊整流的決策仍有爭議。

心臟科的主要試驗未能證明節律控制優於速率控制策略。然而，大多數心臟科隨機對照試驗未考慮既有左心室(LV)收縮和舒張功能障礙的程度，這在急性失代償病人中可能至關重要。實際上，按既有心臟功能分層的亞組，對抗心律不整治療的反應可能存在相關差異。在關於最佳策略存在臨床均勢的情況下，有生理學理由使用超音波檢查來協助判斷節律控制在某些族群中是否更有利。

重症超音波檢查(CCE)已成為ICU中調查血流動力學不穩定病人的首選影像方式。此外，CCE在心律不整的心臟結構、功能、風險分層和管理指引評估中具有獨特而重要的作用。


## 相關心臟疾病


### 左心室收縮功能


左心室收縮功能降低會增加心房和心室心律不整的發生率。雖然AF的盛行率在中度降低LV射血分數(LVEF)的病人中為10-15


### 右心室功能、肺動脈高壓與肺心症


右心室(RV)擴張與新發心律不整相關。在所有形式的肺動脈高壓(PH)中，AF的盛行率為26-31

心臟科指引建議通過三尖瓣壓力梯度評估PH的超音波檢查，而不考慮從下腔靜脈直徑及其呼吸變異估計的RAP。然而，大多數ICU病人有中心靜脈導管，可直接測量RAP並更精確估計SPAP。

急性肺心症(RV擴張伴隨矛盾性室間隔運動)可能觸發AF，導致血流動力學不穩定。在慢性PH中，RV通常發展適應性壁肥厚，使其能產生增加的收縮壓以抵抗升高的後負荷。在RV收縮力急性降低的情況下，NOAF或高速心搏過速可迅速導致RV衰竭和心源性休克。


### 舒張功能與左心房


LV舒張功能障礙在ICU病人中很常見。這與中位年齡55-65歲伴隨LV鬆弛的固有生理改變、高血壓盛行率、各種形式的心衰竭、缺血性心臟病和敗血症相關。在AF病人中，二尖瓣A波缺失；然而，間歇性心臟電擊整流時，頻譜二尖瓣都卜勒圖形可提供關於預後和LV充盈對LA收縮依賴性的寶貴見解。例如，異常(延長)鬆弛和「正常化」(第II型)二尖瓣都卜勒圖形的病人，在暴露於重症疾病時，維持SR比晚期LV舒張功能障礙反映的限制性(第III型)二尖瓣都卜勒模式的病人獲益更大，後者速率控制可能更適當。

有趣的是，尚無心臟科試驗嘗試根據LV舒張功能障礙的嚴重程度(因此是心房收縮對LV充盈貢獻的程度)來分層病人，當評估非重症病人的管理策略(節律versus速率控制)時。令人驚訝的是，儘管重症病人AF發生率高且與存活率有顯著關係，但關於重症病人AF成功心臟電擊整流預測因子的知識缺口明顯。了解潛在的病理生理學可能是更有效治療的指引。

LA在LV舒張功能障礙病人中逐漸擴張，反映LA收縮對充盈鬆弛不良(第I期)和逐漸降低順應性(較高期別)的LV的增加貢獻。限制性LV充盈伴隨升高的心房壓促進AF；相反，LA擴張可能是長期AF的後果。LA大小增大(面積或容積)和LV舒張功能障礙被反覆確認為ICU病人NOAF的獨立預測因子。

## AF時的超音波評估


### 心室收縮功能參數


如果在AF存在時平均3-5個心動週期，或測量前兩個RR間期相等的指標節拍，心室收縮功能參數顯示可接受的觀察者內和觀察者間變異性。測量的有效性和可重複性取決於心率和心室反應的規律性。不幸的是，除了排除有顯著瓣膜疾病的病人外，大多數已發表的超音波和侵入性測量比較都基於心率低於100次/分的病人。LVEF(例如雙平面Simpson法)、目測收縮力估計、二尖瓣環平面收縮期位移(MAPSE)或二尖瓣組織速度S'可協助估計心律不整復發的可能性，並決定抗心律不整藥物的使用。


### 舒張功能與前負荷評估


左心房容積指數(LAVI)是LV舒張功能障礙的診斷標準之一，並整合在LV前負荷評估中。此外，最近的指引建議在AF下使用多個都卜勒參數診斷舒張功能改變。E/e'>11的升高比值(二尖瓣流入和室間隔端環位移)伴隨LAVI>34 ml/m²可預測AF的發生。LAVI每增加1 ml/m²，心律不整復發風險增加21

importantbox
**LAVI的臨床意義：**

- 正常LAVI在竇性節律時應<35 ml/m²
- AF病人應<40 ml/m²
- 35 ml/m²的切點反映升高的LV舒張末期壓力
- 大的LAVI可預測長期或復發性AF
- 可能引導臨床醫師傾向速率控制而非節律控制

### 心臟電擊整流後的監測


心臟電擊整流恢復SR後，LA功能可能無法立即轉化為可測量的超音波變化，特別是在明顯升高的LV充盈壓和LA衰竭的情況下。雖然LA收縮有時在心臟電擊整流後立即恢復，但機械性恢復通常更漸進，直到心臟電擊整流後4-6小時才能見到。心電圖上為SR但無實質性機械收縮(二尖瓣流入模式無A波)的病人，心律不整復發風險高。這類病人長期應優先考慮速率控制策略，而非徒勞地反覆嘗試恢復SR。

- 即刻：確認心電圖SR恢復

## 超音波引導的抗凝策略


### TOE的角色


持續時間不明或已知存在超過24小時的AF病人，在DC心臟電擊整流前應接受經食道超音波(TOE)檢查，以排除潛在的心內血栓並降低手術期間全身性栓塞的風險。這個心臟科建議可能適用於經常表現出多種促凝血因子高血漿濃度但也容易出血併發症的重症病人。因此，在決定抗凝之前，應仔細評估每個個體的益處/風險平衡。

TOE在插管和鎮靜的重症病人中容易執行。該檢查對檢測心內沉澱或血栓的敏感性和特異性超過90


### 血栓風險分層


在ICU病人中，心內血栓形成的風險可根據LA大小、瓣膜疾病存在和LV收縮功能分層：


| p{3cm}p{7cm}} 心內血栓風險與超音波發現 |  |  |
| --- | --- | --- |
| **超音波發現** | **血栓風險** | **臨床處置** |
| AF合併正常超音波 | 1.5 | 考慮出血風險     可能不需抗凝 |
| AF + LA擴張 +     中重度LV功能障礙 +     無二尖瓣逆流 | 20 | 強烈考慮抗凝     權衡出血風險 |
| 瓣膜狹窄 | 增加 | 提高血栓風險     需要抗凝 |
| 機械瓣膜 | 顯著增加 | 必須抗凝 |
| 中重度二尖瓣逆流 | 降低 | 沖洗效應     降低血栓風險 |


### TOE檢查重點


- **自發性超音波對比：**由低血流速度促進紅血球聚集引起，旋渦運動將其與過度增益設定區分
- **LAA血流速度：**<0.4 m/s表示血栓形成風險；≤0.3 m/s加重凝血風險
- **血栓外觀：**LAA中的移動/固定腫塊

### 治療決策


#### 節律控制指徵


- LAVI <40 ml/m²
- 無嚴重瓣膜疾病
- 血流動力學不穩定
- 低心輸出量


**藥物選擇(依優先順序)：**

- LVEF $\geq$35
- LVEF <35
- 合併DC心臟電擊整流


#### 速率控制指徵


- LAVI $\geq$40 ml/m²(特別是>48 ml/m²)
- 嚴重瓣膜疾病(主動脈狹窄、二尖瓣狹窄)
- 機械瓣膜


**藥物選擇：**

- 首選：$\beta$阻斷劑(短效型如landiolol或esmolol)
- 替代：Propafenone (Rytmonorm)、Digoxin、Verapamil

| Esmolol     (短效$\beta$阻斷劑) | **靜脈：**     負荷劑量：500 μg/kg，1分鐘內         維持劑量：50-300 μg/kg/min    （可每4分鐘調整） | • ICU速率控制     • 需要快速滴定     • 術後心房顫動     • 敗血性休克 |
| **三、節律/速率雙重控制藥物** |  |  |
| Amiodarone     (Class III) | **靜脈：**     負荷：150-300 mg，10-20分鐘內     或5 mg/kg，1小時內         維持：900-1200 mg/24小時         **口服：**     400-600 mg/天，1-3週後降至200 mg/天 | • LVEF <35  • 結構性心臟病     • 其他藥物失敗     • 肺動脈高壓     • RV功能不全 |
| **四、輔助性速率控制藥物** |  |  |
| Digoxin | **靜脈：**     負荷：0.25-0.5 mg，緩慢靜注     可於2-6小時後再給0.25 mg         維持：0.125-0.25 mg/天         **口服：**     0.125-0.25 mg/天 | • LVEF <35  • 合併心衰竭     • 其他藥物禁忌     • 輔助β阻斷劑     • 休息狀態速率控制 |


### 重要臨床注意事項


importantbox
**用藥安全提醒：**

1. **電解質校正：**所有抗心律不整藥物使用前必須確保：  - 血鉀 $\geq$4.0 mEq/L（理想4.5-5.0 mEq/L） - 血鎂 $\geq$2.0 mg/dL（理想>1.0 mmol/L）
2. **心電圖監測：**  - Propafenone (Rytmonorm)：QRS變寬>25 - Amiodarone/Vernakalant：QTc>500 ms需停藥 - 所有藥物：持續監測至少4小時
3. **血流動力學監測：**  - 短效β阻斷劑：每5-10分鐘測量血壓、心率 - 若收縮壓<90 mmHg或心率<50/min需立即停藥 - 準備升壓藥物（norepinephrine）
4. **轉律目標心率：**  - 初始目標：80-110次/分 - 避免過度心搏過緩（<60次/分）
5. **DC心臟電擊整流準備：**  - 所有藥物治療均應準備DC cardioversion - 同步電擊：起始能量100-200 J（雙相波） - 鎮靜/麻醉：propofol或midazolam

## 結論與限制


### 主要結論


1. 超音波檢查已成為ICU中調查血流動力學不穩定病人的首選方式
2. 對於日益增加的AF負擔，建議將收縮末期LAVI測量作為2D超音波評估的一部分
3. 治療前的LAVI和心臟電擊整流後的LAEF分別是成功或AF復發的預測因子
4. 提出了超音波驅動的方法來指引重症病人AF管理的節律或速率控制策略


### 主要限制


1. 重症醫學數據稀少
2. 單中心研究樣本量小
3. 需要進一步的前瞻性研究，評估超音波引導心律不整管理與標準治療的病人中心結果


### 未來方向


- 需要大型多中心前瞻性研究
- 建立ICU特定的參考值
- 評估新的影像技術(如LA應變)在重症環境中的應用
- 整合多模式監測工具
