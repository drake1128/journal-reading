## 心臟超音波LVEF與左心室收縮功能考題答案


1. A (心尖四腔與心尖二腔)        **詳解:** Simpson's方法(又稱雙平面法)需要獲取心尖四腔切面與心尖二腔切面，兩個相互垂直的切面，以便計算左心室容積。這種方法通過將左心室分為一系列圓盤，然後加總這些圓盤的體積來計算總容積。
2. B (55-70      **詳解:** 正常成人的左心室射出分數(LVEF)範圍為50-70
3. C (左心室為一系列圓盤堆疊)        **詳解:** 改良Simpson法基於將左心室視為一系列堆疊的圓盤，每個圓盤的高度相同，但直徑可能不同。通過測量每個圓盤的面積並加總，再乘以高度，可以計算出左心室容積。這種方法較其他幾何假設更準確，尤其當左心室形狀不規則時。
4. D (肺靜脈血流速度測量)        **詳解:** 肺靜脈血流速度測量主要用於評估左心房壓力和舒張功能，而非左心室收縮功能。Simpson's法、Teichholz法和M-mode測量都可用於評估左心室整體收縮功能，儘管它們的準確度和適用情況不同。
5. C (3D超音波容積測量法)        **詳解:** 當左心室形狀嚴重變形時(如心肌梗塞後)，3D超音波容積測量法最為準確，因為它不需要幾何假設，可以直接測量實際的左心室容積。M-mode和Teichholz法都基於特定的幾何假設，這些假設在心室變形時不再適用。
6. A (各節段評分總和除以評估節段數)        **詳解:** 左心室壁運動異常指數(WMSI)是通過將各節段評分總和除以評估的節段數計算得出。正常WMSI為1.0，數值越高表示壁運動異常越嚴重。在評分系統中，1分表示正常運動，2分表示輕度減弱，3分表示重度減弱，4分表示無收縮。
7. C (17節段)        **詳解:** 根據美國超音波醫學會(ASE)標準，左心室分為17個節段進行壁運動評估。這種分節方法考慮了左心室的三個主要冠狀動脈供應區域，有助於識別特定冠狀動脈病變相關的壁運動異常。
8. C (心室間隔厚度)        **詳解:** 心室間隔厚度主要用於評估左心室肥厚程度，而非舒張功能。E/A比值(二尖瓣早期與晚期舒張血流速度比)、E/e'比值(二尖瓣早期舒張血流速度與組織多普勒早期舒張速度比)以及左心房容積指數都是評估左心室舒張功能的重要參數。
9. A (正常運動)        **詳解:** 在壁運動評分系統中，評分1代表正常運動。評分2代表輕度運動減弱(舒張期向心臟中心運動減少)，評分3代表重度運動減弱(幾乎沒有向心臟中心的運動)，評分4代表無收縮運動或反常運動(舒張期遠離心臟中心運動)。
10. C (左心室心尖無法清晰顯影)        **詳解:** Simpson's法測量LVEF需要清晰顯影整個左心室輪廓，特別是心尖部。如果左心室心尖無法清晰顯影，將導致容積測量不準確，進而影響LVEF計算。此時可能需要使用對比劑增強顯影或考慮其他測量方法。
11. B (心肌形變程度)        **詳解:** Strain是測量心肌形變程度的指標，反映了心肌的伸縮性能。它測量心肌纖維在收縮和舒張期間長度變化的百分比，是評估局部和整體心肌功能的靈敏指標，可以早期發現傳統LVEF無法檢測到的心肌功能異常。
12. C (-18      **詳解:** 全局縱向應變(GLS)正常值約為-18
13. A (35-75 ml/m²)        **詳解:** 健康成人的左心室舒張末容積指數(LVEDVI)正常值範圍為35-75 ml/m²。LVEDVI值增加提示左心室擴大，可見於容量負荷增加(如二尖瓣或主動脈瓣逆流)或左心室收縮功能障礙(如擴張型心肌病)。
14. B (115 g/m²)        **詳解:** 左心室質量指數(LVMI)正常值上限為男性115 g/m²，女性95 g/m²。超過此值被定義為左心室肥厚，常見於高血壓、主動脈瓣狹窄等壓力負荷增加的疾病。
15. C (40      **詳解:** 根據最新心衰指南，LVEF低於40
16. D (肺動脈楔壓(PAWP)測量)        **詳解:** 肺動脈楔壓(PAWP)測量是一種有創的血流動力學監測方法，需要右心導管，而非超音波評估技術。心肌性能指數(MPI)、二維斑點追蹤(Speckle tracking)和組織多普勒(Tissue Doppler)都是心臟超音波評估左心室收縮功能的方法。
17. B (不需要幾何假設)        **詳解:** 使用三維超音波(3D Echo)測量LVEF的最大優勢是不需要幾何假設，可以直接測量心室的實際形狀和容積。這在左心室形狀變形的情況下尤為重要。然而，3D超音波的準確度尚未超過心臟MRI，MRI仍是LVEF測量的金標準。
18. C (心肌缺血與活性)        **詳解:** 應力性超音波(Stress Echocardiography)主要目的是評估心肌缺血與活性。通過觀察在運動或藥物負荷情況下心肌壁運動的變化，可以診斷冠狀動脈疾病，評估心肌存活性，以及預測心臟事件風險。
19. B (心肌病變早期)        **詳解:** 左心室收縮功能減退但LVEF正常的情況常見於心肌病變早期。這種現象通常可以通過應變成像等更敏感的技術檢測到，表現為縱向應變降低但整體LVEF仍在正常範圍。這種亞臨床心肌功能異常對於早期診斷和治療心肌病變至關重要。
20. C (右心室)        **詳解:** TAPSE(三尖瓣環收縮期位移)是用來評估右心室功能的指標，而非左心室功能。TAPSE測量三尖瓣環在收縮期向心尖方向移動的距離，正常值大於17mm。TAPSE減小表示右心室收縮功能減退。
21. B ((EDV-ESV)/EDV)        **詳解:** 左心室射出分數(LVEF)是通過以下公式計算得出：(EDV-ESV)/EDV，其中EDV為舒張末容積，ESV為收縮末容積。這個公式計算的是每次心臟收縮時射出的血液量佔舒張末期血液總量的百分比。
22. B ("槍彈"徵(Spade sign))        **詳解:** 心尖部肥厚型心肌病(Apical HCM)的特徵性超音波表現是"槍彈"徵(Spade sign)，指的是心尖部肌肉明顯肥厚，使左心室腔在舒張末期呈"槍彈"或"摺扇"形狀。這是診斷心尖部肥厚型心肌病的重要超音波特徵。
23. B (慢性心衰)        **詳解:** 收縮末期容積指數(ESVI)升高常見於慢性心衰。在慢性心衰中，由於左心室收縮功能減退，收縮末期容積增加，導致ESVI升高。這反映了左心室無法有效地排空血液，是心臟功能障礙的重要指標。
24. B (依賴操作者經驗)        **詳解:** 超音波測量LVEF與MRI相比的主要限制是依賴操作者經驗。超音波檢查質量和結果解釋在很大程度上受到操作者技術水平和經驗的影響，不同操作者之間可能存在較大差異。相比之下，MRI提供更客觀、更具重複性的測量結果。
25. B (25-40      **詳解:** 左心室縮短分數(Fractional Shortening)正常值範圍為25-40
26. B (心肌形變與速率)        **詳解:** 二維斑點追蹤(2D Speckle Tracking)技術測量的是心肌形變與速率。該技術通過追蹤超音波圖像中自然存在的聲學標記(斑點)在心動週期中的運動，可以評估心肌在縱向、環向和徑向的形變程度及速率，提供更全面的心肌功能評估。
27. C (多巴酚丁胺)        **詳解:** 負荷超音波中最常使用的負荷藥物是多巴酚丁胺(Dobutamine)。它是一種正性肌力藥物，能增加心肌收縮力和心率，模擬運動時的心臟負荷狀態。多巴酚丁胺負荷超音波用於評估冠狀動脈疾病、心肌存活性及瓣膜疾病。
28. C (心肌運動減弱)        **詳解:** 心內膜下心肌梗塞常表現為心肌運動減弱。心內膜下心肌梗塞通常只影響心肌壁的內層，導致局部運動減弱而非完全消失或反常運動。這與貫壁性心肌梗塞相比，後者可能表現為節段性反常運動(悖論性運動)。
29. B (左心房擴大)        **詳解:** 左心房擴大是左心室舒張功能障礙的重要超音波發現。長期的左心室舒張功能障礙導致左心房壓力增高，進而引起左心房擴大。左心房容積指數(LAVi)≥34 mL/m²被視為左心房擴大，是舒張功能障礙的結構性標誌。LVEF主要反映收縮功能，而非舒張功能。
30. B (心肌缺血)        **詳解:** 在應力超音波中發現某節段壁運動惡化表明存在心肌缺血。正常心肌在應力下會增強收縮，而缺血心肌則會出現新發壁運動異常或原有壁運動異常加重。這是診斷冠狀動脈疾病的重要指標，提示相應冠狀動脈的血流不足以滿足負荷增加時的心肌代謝需求。
