## 主動脈成像


### 概述

主動脈成像是心臟超音波檢查的重要部分，每份報告應描述升主動脈（正常/擴張）及測量值。經胸超音波（TTE）和經食道超音波（TEE）用於評估主動脈，TEE因食道靠近主動脈而提供更高解析度。


使用改良胸骨旁長軸視角（高一肋間）以觀察更多升主動脈。主動脈收縮期直徑略大於舒張期。


### TTE主動脈成像


- **視角**：胸骨旁長軸/短軸、鎖骨上視角、右胸骨旁視角
- **測量方法**：領先邊緣法（傳統）或內緣至內緣法（與CT/MRI一致）
- **注意事項**：影像解析度提升使兩種方法差異微小


### TEE優勢


- 高解析度（高頻探頭）
- 可視更多主動脈段
- 限制：升主動脈顱段可能難以成像


importantbox
**重要提醒：** 每份超音波報告需記錄升主動脈狀態。必要時結合TTE和TEE以全面評估。
importantbox


## 主動脈疾病基礎與動脈瘤


### 基礎


- 主動脈直徑隨血壓、尺寸及壁厚增加（拉普拉斯定律）
- 升主動脈動脈瘤需用胸骨旁長軸/短軸及鎖骨上視角評估，確認是否涉及主動脈根
- 動脈瘤患者主動脈可能右偏或延長


### 動脈瘤流行病學


- **死亡率**：動脈瘤0.7/100,000/年；主動脈剝離1.5/100,000/年
- **性別**：男女無差異
- **風險**：胸主動脈動脈瘤 >6 cm，年破裂/剝離風險6.9


### 動脈瘤形態


- 純升主動脈型
- 香腸型
- 球根型（常見於馬凡氏症候群）


### 遺傳性疾病


- 馬凡氏症候群
- 埃勒斯-丹洛斯症候群（IV型）
- 家族性結締組織病


### 馬凡氏症候群心臟表現


- 主動脈擴張
- 主動脈剝離
- 主動脈逆流（瓣環擴張）
- 二尖瓣脫垂
- 肺動脈擴張
- 大主動脈瓣瓣葉


### 炎症性疾病


- 梅毒
- 金黃色葡萄球菌感染
- 川崎病
- 巨細胞動脈炎
- 高安氏動脈炎


### 破裂風險分層

### 手術適應症（ACC Class I）


- 無症狀患者，升主動脈或主動脈竇直徑 ≥55 mm
- 馬凡氏症候群患者，主動脈直徑 40-50 mm
- 主動脈 <5.5 cm，年增長率 >0.5 cm
- 主動脈瓣修復患者，動脈瘤 ≥4.5 cm


馬凡氏症候群以主動脈剝離為主要死因。炎症可能導致主動脈擴張及分支入口狹窄，需結合CT/MRI精確測量。


## 主動脈剝離


### 概述

主動脈剝離特徵為內膜破裂形成真假腔，假腔通常較大、血流較慢，常含血栓。發病率2.6-3.5/100,000/年，男性占2/3。


### 分類


- **Stanford分類**：     itemize[nosep]
- Type A：涉及升主動脈
- Type B：僅涉及降主動脈


    \item **DeBakey分類**：
    
- Type I：升主動脈及降主動脈
- Type II：僅升主動脈
- Type III：僅降主動脈


itemize


### 主動脈症候群


- **壁內血腫**：主動脈壁內出血（如斑塊破裂）
- **局部剝離**：動脈粥樣硬化引起的局限性剝離
- **癒合剝離**：假腔血栓化並癒合
- **穿透性潰瘍**：動脈粥樣硬化斑塊破裂
- **腔內血栓**：主動脈增厚 >7 mm（需鑑別假腔血栓或壁內血腫）


### 超音波特徵


- 內膜瓣振盪
- 真假腔分離
- 假腔血流緩慢或血栓
- 內膜瓣可能經主動脈瓣脫垂


importantbox
**重要提醒：** 主動脈症候群具高破裂風險，需CT/MRI進一步評估。鎖骨上視角有助於觀察主動脈弓內膜瓣。
importantbox


## 主動脈縮窄（CoA）


### 概述

主動脈縮窄占先天性心臟病5-10


### 超音波特徵


- 左心室肥厚
- 主動脈狹窄
- 彩色都卜勒顯示紊流
- 連續波都卜勒（CW）顯示高梯度
- 收縮期及舒張期梯度提示血流動力學顯著阻塞


### 相關異常


- 二葉主動脈瓣
- 動脈導管未閉/室中隔缺損
- 主動脈弓發育不全
- 左心室流出道阻塞


鎖骨上視角為CoA最佳成像窗口。都卜勒測量常高估梯度，需結合血流動力學評估。偽縮窄（扭結）可能模擬CoA。


## 主動脈斑塊


### 概述

主動脈動脈粥樣硬化斑塊增加冠心病、心梗、卒中及主動脈剝離風險。斑塊 >4 mm時，卒中風險顯著升高（OR=9.1）。


### 典型部位


- 主動脈弓
- 降主動脈顱段


### 超音波評估


- TTE可檢測升主動脈斑塊
- 測量斑塊尺寸以評估風險


importantbox
**重要提醒：** 主動脈斑塊尺寸是卒中風險的重要指標，需精確測量並記錄。
importantbox


## 考題


以下為針對「主動脈疾病」章節的十道選擇題，旨在檢驗關鍵概念的理解。


1. 主動脈成像的最佳TTE視角包括：     enumerate[label=a)]
2. 心尖四腔心
3. 胸骨旁長軸
4. 肋下視角
5. 左胸骨旁短軸


    \item 馬凡氏症候群的心臟表現不包括：
    
1. 主動脈擴張
2. 二尖瓣脫垂
3. 三尖瓣逆流
4. 主動脈逆流


    \item 高風險主動脈破裂的尺寸是：
    
1. ≤2.75 cm/m²
2. 2.75-4.25 cm/m²
3. ≥4.25 cm/m²
4. <2.0 cm/m²


    \item Stanford Type A主動脈剝離涉及：
    
1. 僅降主動脈
2. 升主動脈
3. 主動脈弓
4. 腹主動脈


    \item 主動脈縮窄的超音波特徵包括：
    
1. 右心室肥厚
2. 主動脈狹窄及紊流
3. 正常CW梯度
4. 無左心室肥厚


    \item 主動脈斑塊 >4 mm的風險是：
    
1. 低卒中風險
2. 顯著卒中風險（OR=9.1）
3. 無冠心病風險
4. 低剝離風險


    \item ACC Class I手術適應症不包括：
    
1. 升主動脈直徑 ≥55 mm
2. 馬凡氏症候群，主動脈 40-50 mm
3. 年增長率 <0.5 cm
4. 主動脈瓣修復，動脈瘤 ≥4.5 cm


    \item 主動脈壁內血腫的特徵是：
    
1. 內膜瓣振盪
2. 主動脈壁內出血
3. 真假腔血流相同
4. 無破裂風險


    \item 主動脈縮窄常伴的異常是：
    
1. 三葉主動脈瓣
2. 二葉主動脈瓣
3. 無主動脈弓異常
4. 右心室流出道阻塞


    \item 主動脈剝離假腔的特徵是：
    
1. 血流較快
2. 較真腔小
3. 常含血栓
4. 無內膜瓣


enumerate


### 考題詳解


1. **正確答案：b) 胸骨旁長軸**      **詳解：** TTE主動脈成像最佳視角包括胸骨旁長軸/短軸、鎖骨上及右胸骨旁視角，心尖或肋下視角不適用。
2. **正確答案：c) 三尖瓣逆流**      **詳解：** 馬凡氏症候群心臟表現包括主動脈擴張、逆流、二尖瓣脫垂、肺動脈擴張等，三尖瓣逆流非典型。
3. **正確答案：c) ≥4.25 cm/m²**      **詳解：** 主動脈尺寸 ≥4.25 cm/m²為高風險，年破裂率20
4. **正確答案：b) 升主動脈**      **詳解：** Stanford Type A剝離涉及升主動脈，Type B僅涉及降主動脈。
5. **正確答案：b) 主動脈狹窄及紊流**      **詳解：** CoA超音波特徵包括主動脈狹窄、彩色都卜勒紊流、高CW梯度及左心室肥厚，右心室肥厚非典型。
6. **正確答案：b) 顯著卒中風險（OR=9.1）**      **詳解：** 主動脈斑塊 >4 mm顯著增加卒中風險（OR=9.1），同時伴冠心病及剝離風險。
7. **正確答案：c) 年增長率 <0.5 cm**      **詳解：** ACC Class I適應症包括升主動脈 ≥55 mm、馬凡氏症候群40-50 mm、動脈瘤 ≥4.5 cm或年增長 >0.5 cm。
8. **正確答案：b) 主動脈壁內出血**      **詳解：** 壁內血腫為主動脈壁內出血，無內膜瓣振盪，具破裂風險，需與剝離鑑別。
9. **正確答案：b) 二葉主動脈瓣**      **詳解：** CoA常伴二葉主動脈瓣、動脈導管未閉、室中隔缺損或主動脈弓發育不全。
10. **正確答案：c) 常含血栓**      **詳解：** 主動脈剝離假腔較真腔大，血流緩慢，常含血栓，伴內膜瓣分離。
