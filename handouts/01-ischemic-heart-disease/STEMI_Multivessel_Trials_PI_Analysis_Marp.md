# PHRI 學派特色

| 特色 | 說明 |
|------|------|
| **規模** | 動輒 4,000-20,000 人的 mega-trial |
| **設計** | Superiority design，追求 hard endpoints (CV death + MI) |
| **終點** | 刻意**排除** unplanned revascularization (主觀終點) |
| **風格** | Pragmatic、多國多中心、長期追蹤 |
| **導師影響** | Salim Yusuf — 強調 simple hypothesis, large sample size, hard outcomes |

> **PHRI 的哲學**：如果差異是真實的，在足夠大的樣本和足夠硬的終點下一定看得到。不追求花俏的次要終點。

> 這解釋了為什麼 COMPLETE 是唯一用 **CV death + MI** 作為主要終點的試驗，也是唯一達到 **positive superiority** 的試驗。

# OPTION-STEMI 的獨特視角

| 特色 | 說明 |
|------|------|
| **核心哲學** | FFR-first — 只治療真正有 ischemia 的病灶 |
| **與眾不同** | 兩組**都用 FFR** — 唯一差異是時機 (即時 vs. 住院期間) |
| **Deferred 定義** | 僅延遲 **1-5 天** (同次住院) — 是所有試驗中最短的 |
| **結果** | 即時策略**未達 noninferiority** — 打破「越早越好」的假設 |

> **為什麼「只延遲幾天」還是有差？**
> 可能原因：STEMI 急性期 FFR 值不穩定 → 即時測量可能導致 false positive → 不必要的 PCI → 增加 contrast load 和 procedural risk

> OPTION-STEMI 是從**生理學角度**質疑即時策略的第一個試驗

# 為什麼結果看似矛盾？

| 因素 | BIOVASC / MULTISTARS (支持即時) | OPTION-STEMI / iMODERN (不支持) |
|------|-------------------------------|-------------------------------|
| **終點** | 含 unplanned revasc (主觀) | 純 hard endpoints |
| **介入指引** | Angiography (全部做) | Physiology (選擇性做) |
| **學派信念** | "Complete revasc = 越早越好" | "只治療真正缺血的病灶" |
| **NI margin** | 較寬鬆 (4-6.3%) | — (superiority 或嚴格 NI) |
| **NSTEMI** | BIOVASC 60% | 0% |

> **矛盾的根源不在數據，而在 PI 的研究哲學和設計選擇**

# 從 PI 背景看文獻閱讀策略

> **Pearl 1**: 閱讀任何大型 RCT 之前，先了解 **PI 是誰、來自哪個機構、師承何人** — 這會幫助你理解試驗設計的邏輯和潛在偏好

> **Pearl 2**: 注意**終點定義的選擇** — 含 soft endpoints (unplanned revasc) 的試驗與用 hard endpoints 的試驗，結論可能截然不同

> **Pearl 3**: **Noninferiority margin** 的寬鬆程度是主觀決策 — 不同 PI 對「多少差距可接受」有不同的判斷

> **Pearl 4**: 同一個臨床問題可以有 5 種不同的試驗設計 — 因為每位 PI 帶著不同的**學術 DNA** 來提出問題

# Take-Home Messages

1. **PI 的學術背景決定試驗設計**：PHRI 做 superiority + hard endpoints；Lausanne 用 FFR；Radboud 用 MRI — 每個設計都是 PI 學術 DNA 的延伸

2. **「矛盾」的結果其實可以共存**：Angiography-guided + soft endpoints 的試驗支持即時策略；physiology-guided + hard endpoints 的試驗不支持 — 這不是矛盾，而是在回答不同的問題

3. **學術傳承 (lineage) 值得追蹤**：Yusuf → Mehta, Serruys → Diletti, Pijls → Muller, Davies → Al-Lamee → Nijveldt — 每一條線都代表一種思想體系

4. **臨床應用**：Complete revascularization 是共識 (COMPLETE)；但**時機與方法**取決於你信奉哪種證據等級 — 而你信奉哪種，往往取決於你受過哪種訓練
