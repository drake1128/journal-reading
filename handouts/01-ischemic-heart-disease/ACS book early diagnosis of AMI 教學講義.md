## 急性心肌梗塞早期診斷與心電圖判讀概述


### 前言

急性心肌梗塞（Acute Myocardial Infarction, AMI）是心血管急症中最致命的疾病之一，其早期診斷與正確的心電圖（ECG）判讀對於改善患者預後至關重要。心電圖作為AMI診斷的基石，提供快速、非侵入性且成本低廉的病理線索。本教學文件旨在幫助心臟內科、急診科醫師、住院醫師、實習醫師及心導管室醫師掌握AMI的早期診斷技巧與心電圖判讀能力，結合臨床症狀與鑑別診斷，提升診斷準確性與治療效率。本文件參考Cope的《Early Diagnosis of Acute Abdomen》一書的精神，強調臨床直覺與實證醫學相結合，特別針對心電圖的關鍵特徵進行深入講解。

importantbox
**教學目標：**

- 理解AMI心電圖的主要特徵：ST段抬高、增高T波及交互變化
- 掌握胸痛病史採集與AMI鑑別診斷的要點
- 認識典型與非典型AMI表現，提升診斷敏感性
- 熟悉冠狀動脈解剖與心電圖變化之間的關聯
- 熟悉STEMI與NSTEMI的診斷與治療策略
- 掌握特殊情境下的心電圖判讀（如LBBB、起搏器患者）
- 通過臨床案例與考題鞏固知識

### 機器判讀


完成心電圖檢查後，**務必立即審閱結果**。即使不熟悉心電圖判讀，也**必須檢查機器自動判讀的警訊**：


- **ACUTE MI / STEMI**：急性心肌梗塞/ST段上升型心肌梗塞
- **MEETS ST ELEVATION MI CRITERIA**：符合ST段上升型心肌梗塞標準
- **CONSIDER ACUTE INFARCT**：考慮急性心肌梗塞

### 背景與重要性

AMI的早期診斷依賴於對心電圖異常的快速識別，尤其是ST段抬高型心肌梗塞（STEMI），因其需要緊急血管重建。研究顯示，STEMI患者若能在症狀發作後90分鐘內接受經皮冠狀動脈介入治療（PCI），可顯著降低死亡率。然而，許多患者呈現非典型症狀或心電圖變化，特別是老年、女性或合併多重共病的患者，這增加了診斷難度。心電圖教學不僅需聚焦典型變化（如ST段抬高），還需涵蓋早期徵象（如增高T波）及交互變化（reciprocal change），以避免漏診。同時，連續心電圖監測比單次檢查更有診斷價值，特別在急診環境下，快速且準確的判讀能力直接影響患者預後。


心電圖是AMI診斷的「第一道防線」，但並非唯一依據。臨床醫師需結合病史、理學檢查及實驗室檢查（如肌鈣蛋白）進行綜合判斷，尤其在非典型病例中。急性胸痛患者應在10分鐘內完成心電圖檢查，且正常心電圖不能排除AMI。


## 心電圖基本原理與導程系統


### 心電圖基本概念

心電圖記錄心臟電氣活動，反映心肌細胞去極化和再極化的過程。在AMI時，缺血或壞死的心肌組織產生特徵性電氣變化，表現為特定的心電圖模式，包括ST段抬高、增高T波、病理性Q波及交互變化等。


### 標準12導程系統與心臟解剖

標準12導程心電圖包括肢體導程（I, II, III）、加強肢體導程（aVR, aVL, aVF）及胸前導程（V1-V6），每個導程提供特定心臟部位的電氣活動資訊。理解導程與心臟解剖的對應關係是正確判讀AMI的基礎：


| p{4cm}p{9cm}}  **導程分類** | **導程名稱** | **觀察部位與臨床意義** |
| --- | --- | --- |
| 肢體導程 | I, II, III | - I：左側壁 - II, III：下壁 - 評估整體心電軸 |
| 加強肢體導程 | aVR, aVL, aVF | - aVR：右上方觀察，常呈負向 - aVL：左側壁高位 - aVF：下壁 |
| 胸前導程 | V1-V6 | - V1-V2：間隔壁（右室面） - V3-V4：前壁 - V5-V6：左側壁 - 提供水平面心臟電氣活動資訊 |


- **前壁**：V1-V6（主要V3-V4），左前降支（LAD）供血
- **下壁**：II, III, aVF，右冠狀動脈（RCA）或迴旋支（LCX）供血
- **側壁**：I, aVL, V5-V6，LCX或LAD對角枝供血
- **後壁**：V7-V9或V1-V2對等變化，LCX或RCA供血
- **右室**：V1, V3R-V6R，RCA供血


單一導程變化不具特異性，需綜合多導程模式判斷。後壁與右室梗塞在標準12導程中易被遺漏，必要時檢查V7-V9及V3R-V6R導程。


## AMI的心電圖演變與診斷要點


### AMI心電圖變化的時序

AMI的心電圖呈現動態演變，分為以下階段：


| p{3cm}p{10cm}}  **時期** | **時間** | **心電圖特徵** |
| --- | --- | --- |
| 超急性期 | 數分鐘-數小時 | - 增高T波（高聳、對稱） - ST段輕微抬高 - 早期缺血徵象，易被忽略 |
| 急性期 | 數小時-數天 | - ST段明顯抬高（STEMI） - T波倒置開始出現 - 病理性Q波形成（透壁性梗塞） |
| 亞急性期 | 數天-數週 | - ST段逐漸回到基線 - T波持續倒置 - Q波更加明顯 |
| 慢性期 | 數週-終生 | - 病理性Q波持續存在 - T波可能正常化或持續倒置 - ST段正常 |


### STEMI與NSTEMI的鑑別


| p{5cm}p{6cm}}  **特徵** | **STEMI** | **NSTEMI** |
| --- | --- | --- |
| ST段變化 | 抬高$\geq$1mm（肢體導程）或$\geq$2mm（胸前導程） | 下降或T波倒置 |
| 梗塞範圍 | 透壁性梗塞 | 非透壁性梗塞 |
| Q波形成 | 常出現病理性Q波 | 通常無病理性Q波 |
| 治療急迫性 | 立即再灌流治療 | 依危險分層決定策略 |
| 預後 | 初期死亡率較高 | 長期預後可能相似 |


### ST段抬高（ST Elevation）

ST段抬高是STEMI的經典表現，反映急性冠狀動脈完全閉塞：

- **形態**：凸型抬高，$\geq$1 mm（V1-V4）或$\geq$0.5 mm（其他導聯），至少兩個連續導聯。
- **分佈**：     itemize
- V1-V4：LAD閉塞，前壁
- II, III, aVF：RCA或LCX，下壁
- I, aVL, V5-V6：LCX或LAD對角枝，側壁


  \item **交互變化**：與抬高導聯相對的導聯出現ST段壓低，如V1-V4抬高伴II, III, aVF壓低。
itemize

importantbox
**診斷提醒：**

- ST段抬高需至少兩個連續導聯
- 勿混淆ST段抬高與後續高T波，後者為缺血後期表現。

### 增高T波（Hyperacute T Wave）

增高T波是AMI早期徵象，通常出現在症狀後數分鐘至數小時：

- **形態**：高聳、對稱，幅度顯著增加，常伴ST段輕度抬高。
- **分佈**：對應閉塞血管供血區域。
- **進展**：可快速演變為ST段抬高，需密切監測。


增高T波常被忽視，但對早期診斷至關重要。發現高聳T波應立即重複心電圖，確認是否進展為STEMI。


### 交互變化（Reciprocal Change）


- **定義**：與ST段抬高導聯相對的導聯出現ST段壓低或T波倒置。
- **意義**：增強STEMI診斷特異性，排除心包炎等其他原因。
- **模式**：     itemize
- 前壁梗塞（V1-V4抬高）：II, III, aVF壓低
- 下壁梗塞（II, III, aVF抬高）：I, aVL或V1-V3壓低


itemize


## 冠狀動脈解剖與心電圖關聯


### 冠狀動脈分佈


- **左冠狀動脈（LCA）**：分為左前降支（LAD，供應前壁和心尖，V1-V4）和迴旋支（LCX，供應側壁，I, aVL, V5-V6）。
- **右冠狀動脈（RCA）**：供應下壁（II, III, aVF）及右室（V3R-V6R）。
- **後壁**：LCX或RCA，V7-V9或V1-V2對等變化。


### 心電圖與血管閉塞對應


| p{6cm}p{6cm}}  **閉塞血管** | **心電圖變化** | **受影響心肌區域** |
| --- | --- | --- |
| LAD | V1-V4 ST抬高，II, III, aVF交互變化 | 前壁、心尖 |
| LCX | I, aVL, V5-V6 ST抬高，V1-V3交互變化 | 側壁、後壁 |
| RCA | II, III, aVF ST抬高，I, aVL交互變化 | 下壁、右室 |


冠狀動脈造影可確認閉塞血管，常見視角（如RAO）有助於識別LAD、LCX或RCA病變。住院醫師需熟悉心電圖與造影的對應關係。


## 各部位AMI的心電圖特徵


### 前壁心肌梗塞


- **導程**：V1-V6（主要V3-V4）
- **特徵**：ST段抬高，Q波形成，II, III, aVF交互變化
- **分類**：     itemize
- 廣泛前壁：V1-V6, I, aVL（LAD近端）
- 前壁：V3-V4（LAD中段）
- 前間隔：V1-V3（LAD間隔枝）
- 前側壁：V4-V6, I, aVL（LAD遠端或對角枝）


itemize


### 下壁心肌梗塞


- **導程**：II, III, aVF（III最明顯）
- **特徵**：ST段抬高，I, aVL交互變化
- **責任血管**：     itemize
- RCA：III > II ST抬高，aVL明顯壓低
- LCX：II $\geq$ III ST抬高，常伴V5-V6變化


itemize


### 側壁心肌梗塞


- **導程**：I, aVL（高側壁），V5-V6（低側壁）
- **特徵**：ST段抬高，常與前壁或下壁梗塞合併


### 後壁心肌梗塞 (posterior wall MI


- **導程**：V7-V9 ST抬高 (Figure.  )，V1-V3 ST下降、R波增高
- **責任血管**：LCX或RCA

importantbox
**後壁梗塞診斷要點：**

- V1-V3 ST下降常被誤認為前壁缺血
- 常規檢查V7-V9導程
- 常與下壁或側壁梗塞合併


importantbox


### 右室心肌梗塞


- **導程**：V3R-V6R ST抬高（V4R最具診斷價值）
- **特徵**：常伴下壁梗塞，RCA近端閉塞
- **臨床**：血流動力學不穩定，禁用硝酸鹽，需容積管理


## 臨床病史與鑑別診斷


### 胸痛病史採集


- **典型胸痛**：壓迫性、胸骨後，放射至左臂、頸部或下頜，伴冷汗、噁心、呼吸困難。
- **非典型症狀**：上腹痛、消化不良、疲倦、暈厥（老年、女性、糖尿病患者常見）。
- **伴隨症狀**：冷汗、虛弱、心悸、焦慮。


非典型症狀易延誤就醫，醫師應主動詢問全身性症狀，如「您是否覺得特別累或頭暈？」以提高診斷敏感性。


### 鑑別診斷


| p{12cm}}  **疾病** | **特徵與鑑別要點** |  |
| --- | --- | --- |
| 主動脈剝離 | 突發撕裂樣胸痛，放射至背部，不對稱脈搏，胸片縱膈增寬 |  |
| 肺栓塞 | 急性呼吸困難、缺氧，ECG S1Q3T3或右心負荷，D-dimer升高 |  |
| 心包炎 | 胸痛隨體位變化，ECG瀰漫性ST抬高，無交互變化，PR段下降 |  |
| 嗜鉻細胞瘤 | 陣發性高血壓、心悸，尿兒茶酚胺升高 |  |
| 敗血症 | 全身炎症，ECG無特異性變化，需血培養與炎症標誌物 |  |
| 冠狀動脈栓塞 | 感染性心內膜炎患者，ECG類似AMI，伴發熱或瓣膜病 |  |


importantbox
**鑑別診斷提醒：**

- 優先排除主動脈剝離與肺栓塞
- 敗血症誘導心肌抑制為可逆性，需抗生素與支持療法
- 非典型症狀需結合肌鈣蛋白與影像學


importantbox


## 特殊情況下的AMI心電圖判讀


### 左束枝傳導阻斷（LBBB）


- **Sgarbossa準則**：     itemize
- QRS同向ST抬高$\geq$1mm（5分）
- V1-V3 ST下降$\geq$1mm（3分）
- QRS反向ST抬高$\geq$5mm（2分）
- 總分$\geq$3分提示AMI


  \item **修正Sgarbossa**：ST/QRS比值$\geq$-0.25，提高敏感性
itemize


### 右束傳導支阻斷 (RBBB)

    Enter Caption
    


#### 重要觀念


**右束支傳導阻斷（RBBB）可以合併STEMI，絕對不可遺漏！**


- RBBB本身會改變ST-T波形，使STEMI診斷更加困難
- 但RBBB並**不會**完全掩蓋STEMI的心電圖變化
- 若未及時診斷，將延誤治療黃金時間


#### RBBB STEMI的臨床意義


1. **病變位置：**      - 通常involve **LAD proximal**（左前降支近端） - 代表大範圍心肌受損 - 右束支血流供應也來自LAD，同時受影響
2. **預後較差：**      - 梗塞範圍通常較大 - 容易併發心因性休克 - 住院死亡率明顯上升 - 更容易發生完全房室傳導阻斷（Complete AV block）


#### 診斷要點


**RBBB存在時，STEMI診斷標準：**


| p{0.65\textwidth}}  **導程** |
| --- |
| V1-V3 |
| （右胸導程） |
| V4-V6, I, aVL |
| （左側導程） |
|  |
| II, III, aVF |
| （下壁導程） |


#### 關鍵提醒


1. **RBBB + 胸痛 = 高度警覺！**      - 不要因為看到RBBB就放鬆警戒 - 仔細檢查**所有導程**的ST變化 - 特別注意V4-V6, I, aVL導程
2. **新發生的RBBB本身就是警訊**      - 比對舊心電圖（如果有） - 新發RBBB + 胸痛 → 極可能是STEMI
3. **不確定時的處理原則**      - 有疑慮立即**諮詢心臟科醫師** - 寧可over-call，不可miss-call - RBBB STEMI預後差，時間更關鍵
4. **準備TCP（Transcutaneous Pacing）**      - RBBB STEMI容易進展成Complete AV block - 貼好TCP patches，隨時準備使用 - 備好temporary pacing wire


#### 記憶要點


0.9\textwidth{
**RBBB STEMI = LAD proximal病變 = 預後差**


「看到RBBB別放鬆，可能正在梗塞中！」


**檢查重點：V4-V6, I, aVL的ST變化**
}


### 起搏器患者


- 觀察自身心律、起搏器感知異常
- 比較先前ECG，依賴臨床與肌鈣蛋白

### 女性與年輕患者


- **女性**：ST抬高幅度較小，NSTEMI更常見，症狀不典型
- **年輕患者**：病因包括藥物、冠狀動脈剝離，ST抬高幅度大，少Q波


### 非典型表現與特殊情境


- **非典型症狀**：消化不良、呼吸困難、暈厥、疲倦
- **敗血症併發心肌抑制**：可逆性ECG變化，需抗生素
- **冠狀動脈栓塞**：感染性心內膜炎，需超音波鑑別

## 抗血小板 vs 抗凝血劑：一次搞懂！


### 核心概念：兩個完全不同的機轉


**重點：雖然都被稱為「抗血栓」，但作用機轉完全不同！**

tabular{cc}

\hline

- **抗血小板**：像是在士兵（血小板）身上噴「防黏劑」，讓他們無法手牽手聚集成團
- **抗凝血劑**：像是阻斷「骨牌效應」，讓凝血因子無法一個接一個啟動

### 臨床關鍵提醒


1. **ACS病人同時需要兩種！**      - 抗血小板：Aspirin + Plavix/Brilinta/Efient（雙重抗血小板） - 抗凝血劑：Heparin（急性期使用）
2. **不可互相取代！**      - 心肌梗塞急性期與導管時不能只給Heparin，**一定要給抗血小板** - 深部靜脈栓塞不能只給Aspirin，**一定要給抗凝血劑，同時不再多開aspirin**
3. **簡單判斷法：**      - 看到「支架」「心肌梗塞」「腦中風」→ 想到**抗血小板** - 看到「DVT」「PE」「AF」「人工瓣膜」→ 想到**抗凝血劑**

- Q: Heparin是抗血小板還是抗凝血劑？        A: **抗凝血劑**
- Q: 心房顫動病人主要用什麼預防中風？        A: **抗凝血劑**（Warfarin或NOAC）

## 臨床教育考題


### 考題設計


1. 65歲男性，胸痛2小時，ECG顯示V3-V4 ST上升3mm，V1-V2輕微上升，II, III, aVF ST下降1mm。最可能診斷與責任血管？
2. 55歲女性，胸痛，ECG顯示II, III, aVF ST上升（III 4mm, II 2mm），V5-V6 ST上升2mm。責任血管？如何進一步確認？
3. 70歲男性，LBBB病史，胸痛4小時。LBBB下哪些ECG變化提示AMI？說明Sgarbossa準則。
4. 45歲男性，胸痛1小時，ECG顯示V1-V3 ST下降2mm，R波增高，T波直立。應如何檢查？提示什麼？
5. 80歲女性，胸悶6小時，ECG顯示II, III, aVF及V4R ST上升。提示什麼？治療考量？
6. 35歲男性，運動後胸痛，ECG顯示V2-V5 ST上升1-2mm，凹面向上，無交互變化。鑑別診斷？
7. 60歲糖尿病患者，胸痛8小時，ECG顯示V3-V4 QS波，ST回到基線，T波倒置。提示什麼？可否再灌流？
8. 50歲女性，胸痛2小時，ECG顯示I, aVL, V5-V6 ST下降1-2mm，T波倒置，Troponin 15 ng/mL。診斷與處置？
9. 下壁AMI，ECG顯示II ST上升2mm，III 5mm，aVF 3mm，aVL下降3mm。責任血管是RCA還是LCX？
10. 65歲男性，ECG顯示廣泛ST上升，無交互變化，PR段下降。最可能診斷？與AMI鑑別？


### 考題詳解


1. **答案**：前壁AMI，LAD。              **詳解**：V3-V4 ST上升，II, III, aVF交互變化，提示LAD閉塞，需緊急PCI。
2. **答案**：LCX可能性高，檢查V7-V9及V3R-V6R。              **詳解**：III > II但伴V5-V6變化，提示LCX。V7-V9確認後壁，V3R-V6R評估右室。
3. **答案**：Sgarbossa準則：QRS同向ST抬高$\geq$1mm（5分），V1-V3 ST下降$\geq$1mm（3分），QRS反向ST抬高$\geq$5mm（2分），$\geq$3分提示AMI。              **詳解**：LBBB掩蓋典型變化，修正Sgarbossa提高敏感性。
4. **答案**：檢查V7-V9，提示後壁AMI。              **詳解**：V1-V3 ST下降、R波增高為後壁梗塞鏡像，V7-V9確認ST抬高。
5. **答案**：下壁與右室AMI，禁用硝酸鹽，需容積管理。              **詳解**：V4R ST抬高提示右室梗塞，需避免血管擴張劑。
6. **答案**：早期再極化症候群。              **詳解**：凹面ST抬高，無交互變化，需監測Troponin與ECG。
7. **答案**：亞急性前壁AMI，無緊急再灌流指徵。              **詳解**：QS波、ST回復提示亞急性期，著重藥物治療。
8. **答案**：NSTEMI，GRACE評分後24-72小時內PCI。              **詳解**：Troponin升高，無ST抬高，需抗血栓與危險分層。
9. **答案**：RCA。              **詳解**：III > II，aVL明顯下降，提示RCA閉塞。
10. **答案**：心包炎，鑑別點為瀰漫性ST抬高、PR下降、無交互變化。              **詳解**：心包炎無局部性變化，與AMI的局灶性ST抬高不同。

### 核心要點

- 連續ECG監測比單次檢查更有價值
